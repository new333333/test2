Blog Toolbar

The Blog toolbar provides the following options that enable you to manage your Blog folder:

  • Add Blog entry: Add a new entry to your Blog folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry and send an e-mail to specific users notifying them that you have submitted an entry.

  • New Blog Page: Add a new blog folder to your existing blog folder. This new folder is subordinate to your main blog folder.

  • Sort by: Sort your blogs by Number, Activity Date, Author, State, or Title.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Blog folder. This action adds the file as a new blog entry.
