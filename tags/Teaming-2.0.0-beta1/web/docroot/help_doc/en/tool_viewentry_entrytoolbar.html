Entry Toolbar

The entry toolbar provides only those tools you have permission to use. The toolbar appears for both entries and comments, and it can include the following buttons and menus:

  • Comment: Adds a comment to the entry. You can make a comment about a comment (this is called making nested comments.

  • Modify: Enables you to edit the entry or to add another attachment. (You can also use the Edit link in the lower-right section of the entry, just above the attachments.)

  • Reserved: Prevents others from editing the entry while you are editing it.

  • Move: Enables you to move the entry to another folder.

  • Copy: Enables you to copy the entry and save it to another location.

  • Delete: Removes the entry from the folder.

  • Reports: Displays a comma separated values file in Microsoft Excel*, which provides information about users and their activities with this entry.

    This menus also provides a tool that compares versions of the entry. Select one radio button in each column next to the edited versions you want to compare, and then click Compare.

  • Share This Entry: Enables you to share an entry with other Teaming users.
