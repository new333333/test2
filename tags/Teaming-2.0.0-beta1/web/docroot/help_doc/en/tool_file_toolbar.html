File Folder Toolbar

The File Folder toolbar provides the following options that enable you to manage your File folder:

  • Add File entry: Add a new entry to your File folder. You can add an attachment and description to your entry. You can also subscribe to the entry and send an e-mail to specific users notifying them that you have submitted an entry.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your File folder. This action adds the file as a new file entry.
