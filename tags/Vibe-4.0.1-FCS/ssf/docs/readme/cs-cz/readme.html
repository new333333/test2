Novell Vibe 4

1.0 Úvod

Úplné požadavky na systém a podrobné pokyny k instalaci určené administrátorům portálu Novell Vibe jsou uvedeny v příručce Vibe 4 Installation Guide.Nastavení portálu Vibe stejně jako popis jeho správy a údržby je popsáno v příručce Vibe 4 Administration Guide.

Po nastavení portálu Vibe naleznou uživatelé pokyny v příručkách Uživatelská příručka produktu Vibe 4 a Vibe 4 Advanced User Guide. (Pro jazykové verze, pro které není k dispozici Uživatelská příručka produktu Vibe 4, je k dispozici Stručná úvodní příručka řešení Vibe 4.)

Po přihlášení do portálu Vibe je možné zobrazit online nápovědu.
