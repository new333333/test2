Novell Vibe 3.2

1.0  开始使用

     完整系统要求和详细软件安装指导可在《"Vibe 3.2 安装指南》中找到，供 Novell Vibe 管理员参考。" Vibe
     站点设置及正在进行的站点管理和维护指导可在《"Vibe 3.2 管理指南"》中找到。

     Vibe 站点设置完成后，Vibe 用户可在《"Vibe 3.2 用户指南"》和《"Vibe 3.2 高级用户指南"》中获得指导。（《"Vibe 3.2
     快速入门"》以《"Vibe 3.2 用户指南"》不具备的其他语言提供。）

     登录 Vibe 站点时可以显示 Vibe 联机帮助。
