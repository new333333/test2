Ассоциации рабочего потока

Этот раздел позволяет связывать рабочие потоки с типами записей или типами комментариев (ответов). Когда рабочий поток ассоциируется с типом записей или комментариев, он автоматически запускается при создании записи или комментария.

Ассоциирование рабочих потоков с записями

Этот раздел содержит все типы записей, выбранные в разделе Типы записей по умолчанию на этой странице. Рядом с каждым типом записей в разделе Ассоциации рабочего потока Teaming отображает раскрывающийся список, содержащий все рабочие потоки, которые можно ассоциировать с типом записей.

Чтобы ассоциировать рабочий поток с типом записей, выполните следующие действия.

  1. Убедитесь, что тип записей выбран в разделе Типы записей по умолчанию и запись разрешена для создания в этой папке.

    После выбора типа записей в разделе Типы записей по умолчанию и нажатия кнопки Применить он появляется в разделе Ассоциации рабочего потока.

  2. В разделе Ассоциации рабочего потока в разделе Типы записей выберите рабочий поток в раскрывающемся списке рядом с типом записей, для которых будет применяться рабочий поток.

  3. Нажмите кнопки Применить > Закрыть.

    Теперь, когда будет создаваться запись этого типа в данной папке, Teaming автоматически поместит эту запись в начальное состояние рабочего потока.

При создании рабочего потока в папке этот рабочий поток автоматически добавляется в раскрывающийся список, и его можно будет ассоциировать с записями в папке. О том, как создать рабочий поток, см. в разделе Creating and Managing Workflows документа Расширенное руководство пользователя Novell Teaming 2.1.

Ассоциирование рабочих потоков с комментариями (ответами)

Раздел Типы комментариев или ответов позволяет назначать рабочие потоки, которые запускаются, когда пользователи вводят ответы определенного типа.

Teaming отображает все типы ответов для каждого типа записей по умолчанию, которые перечисляются в разделе Ассоциации рабочего потока.

Когда тип записей по умолчанию также является типом ответов, он не перечисляется второй раз.

Дополнительную информацию см. в Юридическая информация.
