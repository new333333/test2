Папка таблицы

При работе с папкой таблицы в каждой строке содержится информация одной записи. При работе с папкой таблицы используются следующие приемы.

  • Изменение размера столбца Сверху в таблице щелкните и удерживайте линию с двумя стрелками и перетащите ее влево или вправо.

  • Сортировка содержимого таблицы Щелкните заголовок столбца, чтобы сортировать таблицу по содержимому этого столбца. Чтоб обратить порядок сортировки, щелкните заголовок столбца еще раз. Значок стрелки рядом с заголовком столбца показывает, что в данный момент таблица сортируется по этому столбцу (стрелка вверх показывает сортировку по убыванию, а стрелка вниз сортировку по возрастанию).

  • Указатель на ячейке Наведите указатель мыши на ячейку (например, если длина заголовка превышает ширину ячейки), чтобы Novell Teaming полностью отобразил содержимое ячейки.

Дополнительную информацию см. в Юридическая информация.
