Панель инструментов просмотра

С помощью панели инструментов просмотра можно выбрать, задачи каких типов будут отображаться в списке задач. Если пользователь Teaming не находится в персональной рабочей области, он может видеть только те задачи, которые расположены в текущей папке.

  • Закрыто Отображаются задачи с состоянием "Закрыто".

  • Сегодня Отображаются задачи со сроком выполнения сегодня.

  • Неделя Отображаются задачи со сроком выполнения в течение следующих семи дней.

  • Месяц Отображаются задачи со сроком выполнения в течение следующих 30 дней.

  • Все активные Отображаются все текущие активные задачи.

  • Все записи Отображаются все задачи.

  • Назначенные задачи Доступно только в рабочих областях пользователей. Отображаются все задачи, назначенные владельцу определенной рабочей области, вне зависимости от расположения этих задач на портале Teaming.

  • Из папки Доступно только в рабочих областях пользователей. Отображаются все задачи, расположенные в текущей папке.

Дополнительную информацию см. в Юридическая информация.
