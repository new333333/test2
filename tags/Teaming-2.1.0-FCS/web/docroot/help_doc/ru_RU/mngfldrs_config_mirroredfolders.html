Отраженные папки

Отраженная папка является библиотечной папкой, файлы которой размещаются во внешней системе (в каталоге файловой системы или в папке на сервере SharePoint). Когда внешние папки отражаются из Teaming, к внешним файлам можно применять инструменты Novell Teaming (например, поиск и рабочие потоки).

Для включения этой функции необходимо работать с файлом, размещенным на сервере.

Дополнительные сведения о создании отраженных папок см. в разделе Working with Mirrored Folders документа Руководство пользователя Novell Teaming 2.1.

Дополнительную информацию см. в Юридическая информация.
