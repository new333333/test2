Моя рабочая область

Щелкните значок Моя рабочая область для просмотра своей персональной рабочей области Novell Teaming. Помимо прочего, персональная рабочая область может содержать следующие элементы.

  • Ваше изображение и другие избранные фотографии.

  • Контактную информацию.

  • Ваш блог.

  • Персональный календарь.

  • Гостевую книгу, в которую могут вносить записи другие пользователи.

Новым пользователям Teaming рекомендуется начинать работу со знакомства с персональной рабочей областью. Например, можно написать вводный текст, представляющий рабочую область, загрузить в нее несколько изображений и добавить запись блога (запись журнала). Посетители персональной рабочей области смогут увидеть Вашу контактную информацию, читать записи блога и комментировать их.

Дополнительную информацию см. в Юридическая информация.
