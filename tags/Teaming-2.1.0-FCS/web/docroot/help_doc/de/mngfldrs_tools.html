Verwaltungswerkzeuge für Ordner

Wenn Sie in der Symbolleiste eines Ordners auf Verwalten klicken, zeigt Novell Teaming nur die Verwaltungs- und Konfigurationswerkzeuge an, die zu verwenden Sie berechtigt sind. Die Administratoren der Website und die Eigentümer des Arbeitsbereichs oder Ordners haben Zugriff auf diese Werkzeuge und können anderen Benutzern Zugriffsberechtigungen für diese Werkzeuge erteilen.

Bei uneingeschränktem Zugriff werden die folgenden Optionen angezeigt:

  • Zugriffssteuerung: Damit können Sie festlegen, wer diesen Ort anzeigen und sich daran beteiligen darf.

  • Aktivitätsberichte: Damit können Sie die Aktivitäten anzeigen, die im Arbeitsbereich stattgefunden haben.

  • Konfigurieren: Damit können Sie den Ordner konfigurieren. Unter anderem können Sie die Standardansicht, die Standardeintragsarten, die zulässigen Ansichten und die URL des Ordners festlegen.

  • Diesen Ordner kopieren: Damit können Sie den aktuellen Ordner, sofern Sie über die entsprechenden Berechtigungen verfügen, an jedes beliebige Ziel innerhalb des Arbeitsbereichsbaums kopieren. Weitere Informationen hierzu finden Sie unter Navigation im Arbeitsbereichsbaum.

  • Diesen Ordner löschen: Damit können Sie den aktuellen Ordner löschen.

  • Email-Einstellungen: Damit können Sie den Ordner so konfigurieren, dass an andere Teaming-Benutzer automatisch Email-Benachrichtigungen gesendet werden.

  • Exportieren/Importieren: Nur Teaming-Administratoren. Hiermit können Sie den aktuellen Ordner exportieren bzw. einen Ordner oder Arbeitsbereich in den aktuellen Ordner importieren.

    Weitere Informationen zum Exportieren und Importieren von Arbeitsbereichen und Ordnern in Teaming finden Sie im Kablink Teaming 2.1 Administration Guide im Abschnitt zum Exportieren/Importieren von Arbeitsbereichen, Ordnern und Einträgen.

  • Formular- und Ansichtsdesigner: Damit können Sie benutzerdefinierte Eintragsformulare und Workflows erstellen, die Sie auf den Ordner und seine Unterordner anwenden können.

  • Diesen Ordner ändern: Damit können Sie den Ordner ändern. Unter anderem können Sie den Namen, die Beschreibung, die Marke und das Symbol des Ordners ändern.

  • Diesen Ordner verschieben: Damit können Sie den aktuellen Ordner, sofern Sie über die entsprechenden Berechtigungen verfügen, an jedes beliebige Ziel innerhalb des Arbeitsbereichsbaums verschieben. Weitere Informationen hierzu finden Sie unter Navigation im Arbeitsbereichsbaum.

  • Neuen Ordner hinzufügen: Damit können Sie dem Arbeitsbereich einen neuen Ordner hinzufügen.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
