Rollenmitgliedschaft erben

Wenn in diesem Abschnitt Ja ausgewählt ist, übernimmt der aktuelle Arbeitsbereich oder Ordner seine Benutzer und Rollen aus dem übergeordneten Ordner. In der Zugriffssteuerungstabelle des aktuellen Arbeitsbereichs oder Ordners können Sie in diesem Fall keine Änderungen vornehmen.

Wenn Sie die Tabelle des aktuellen Arbeitsbereichs oder Ordners bearbeiten möchten, müssen Sie sicherstellen, dass in diesem Abschnitt Nein ausgewählt ist. Ist dies nicht der Fall, dann wählen Sie zuerst Nein aus und klicken Sie dann auf Anwenden.

Dadurch ändern sich die Beschriftungen in der ersten Spalte der Tabelle.

Informationen zur Anpassung der Zugriffssteuerungstabelle an Ihre Anforderungen finden Sie im Abschnitt Zugriffssteuerung konfigurieren.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
