Tipps zum Hochladen von Dateien

Wenn Sie Probleme mit dem Hochladen einer Datei haben, weil die Datei zu groß ist, lässt sich das Problem eventuell beheben, indem Sie den von Java-Applets verwendeten Standardarbeitsspeicher erhöhen. Informationen hierzu erhalten Sie in den folgenden Abschnitten.

Erhöhen des Standardarbeitsspeichers für Java-Applets unter Windows

  1. Rufen Sie die Systemsteuerung auf (Klicken Sie auf Start > Einstellungen > Systemsteuerung).

  2. Doppelklicken Sie auf Java.

    Das Java Control Panel wird gestartet.

  3. Wählen Sie die Registerkarte Java aus.

  4. Klicken Sie im Abschnitt Java-Applet-Laufzeiteinstellungen auf Anzeigen.

  5. Fügen Sie für jede Ihrer Java Runtime-Versionen -Xmx256m als Java Runtime-Parameter hinzu.

  6. Klicken Sie auf OK.

    Nachdem Sie den Standardarbeitsspeicher für Java-Applets erhöht haben, müssen Sie alle Browser beenden und neu starten, damit die Änderung wirksam wird.

Durch diese Einstellung stehen dem Java-Applet zum Hochladen von Dateien in einen Teaming-Ordner zu Beginn 256 MB Arbeitsspeicher zur Verfügung. Wenn das Applet weiterhin angibt, dass nicht genügend Arbeitsspeicher zur Ausführung zur Verfügung steht, müssen Sie diesen Wert möglicherweise reduzieren.

Erhöhen des Standardarbeitsspeichers für Java-Applets unter Linux

  1. Suchen Sie die für das Browser-Plugin verwendete JRE oder JDK.

    Der Speicherort der JRE oder JDK ist systemabhängig.

  2. Führen Sie im Verzeichnis bin des JRE bzw. JDK die Datei ControlPanel aus.

    Das Control Panel wird gestartet.

  3. Wählen Sie die Registerkarte Java aus.

  4. Klicken Sie im Abschnitt Java-Applet-Laufzeiteinstellungen auf Anzeigen.

  5. Fügen Sie für jede Ihrer Java Runtime-Versionen -Xmx256m als Java Runtime-Parameter hinzu.

  6. Klicken Sie auf OK.

    Nachdem Sie den Standardarbeitsspeicher für Java-Applets erhöht haben, müssen Sie alle Browser beenden und neu starten, damit die Änderung wirksam wird.

Durch diese Einstellung stehen dem Java-Applet zum Hochladen von Dateien in einen Teaming-Ordner zu Beginn 256 MB Arbeitsspeicher zur Verfügung. Falls das Applet nach wie vor meldet, dass ihm nicht genügend Arbeitsspeicher zur Verfügung steht, müssen Sie diesen Wert eventuell reduzieren.
