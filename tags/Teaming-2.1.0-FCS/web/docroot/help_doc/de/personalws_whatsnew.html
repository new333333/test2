Neuheiten

Wenn Sie in einem Arbeitsbereich oder Ordner auf Neuigkeiten klicken, zeigt Novell Teaming alle neuen Einträge im aktuellen Arbeitsbereich oder Ordner sowie in den untergeordneten Arbeitsbereichen und Ordnern an (sofern Sie über die entsprechenden Berechtigungen verfügen). Aufgelistet werden nur Einträge, die innerhalb der letzten 30 Tage erstellt oder geändert wurden.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
