Email-Benachrichtigungen senden

Als Ordnereigentümer können Sie Novell Teaming so konfigurieren, dass andere Benutzer Email-Benachrichtigungen erhalten, durch die sie über jegliche Aktivitäten in diesem Ordner oder seinen Unterordnern informiert werden. Allerdings werden die Benachrichtigungseinstellungen der Ordnereigentümer durch die Einstellungen des Benutzers überschrieben. Das heißt, wenn der Benutzer keine Benachrichtigungen wünscht, erhält er auch keine, selbst wenn der Ordnereigentümer diese für ihn eingerichtet hat.

  1. Wählen Sie im Abschnitt Art der Email-Benachrichtigung auswählen die Art der Email-Benachrichtigung aus, die Sie senden möchten.

    Informationen zu den verschiedenen Arten von Email-Benachrichtigungen finden Sie im Abschnitt Auswahl der zu empfangenden Email-Benachrichtigungstypen.

  2. Geben Sie die Benutzer an, die Email-Benachrichtigungen erhalten sollen.

    Sie können bestimmte Email-Adressen, Benutzer und Gruppen angeben oder die Benachrichtigungen an alle Teammitglieder senden.

  3. Klicken Sie auf Anwenden > Schließen.

Weitere Informationen zu Email-Benachrichtigungen finden Sie im Abschnitt Verwenden von Email-Benachrichtigungen.
