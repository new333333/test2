Ändern der Standardansicht eines Ordners

  1. Wählen Sie die Ansicht aus, die Ihre Ordnerauflistung oder Ihren Arbeitsbereich optimal darstellt.

  2. Klicken Sie auf Anwenden > Schließen.

    Die Ordneransicht wird geändert.

Informationen zum Hinzufügen der in diesem Abschnitt zur Auswahl stehenden Standardansichten finden Sie im Abschnitt Hinzufügen zulässiger Standardansichten für einen Ordner.

Bei der Auswahl einer Ordneransicht sollten Sie die im Ordner enthaltenen Eintragsarten berücksichtigen. So lässt sich zum Beispiel eine Ordnerliste mit Kalendereinträgen in der Regel am besten in einer Kalenderansicht darstellen.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
