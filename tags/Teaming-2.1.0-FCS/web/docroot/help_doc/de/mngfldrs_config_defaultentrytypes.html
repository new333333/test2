Standardeintragsarten

In diesem Abschnitt können Sie die Eintragsarten auswählen, die Benutzer in diesem Ordner erstellen dürfen. Sie können für den Ordner weitere Eintragsarten zulassen oder die aktuelle Eintragsart ändern. Wenn Sie mehrere Eintragsarten zulassen, können Benutzer, die dem Ordner Einträge hinzufügen, die Art des neuen Eintrags (z. B. Aufgaben- oder Kalendereintrag) auswählen.

  1. Wählen Sie die Eintragsarten aus, die Sie für den aktuellen Ordner zulassen möchten.

  2. Klicken Sie auf Anwenden > Schließen.

Wenn Sie für einen Ordner mehrere Eintragsarten zulassen, können Benutzer, die dem Ordner Einträge hinzufügen, in der Symbolleiste "Ordnereintrag" auf Hinzufügen klicken und in der Dropdown-Liste aus den von Ihnen zugelassenen Eintragsarten die Art des neuen Eintrags auswählen.

Website-Administratoren und Ordnereigentümer können im Formular- und Ansichtsdesigner neue Eintragsarten (benutzerdefinierte Formulare) erstellen, die dann ebenfalls in der Liste der möglichen Eintragsarten angezeigt werden. (Eintragsarten, die von Ordnereigentümern erstellt wurden, werden im Abschnitt Lokale Definitionen der Liste angezeigt.)

Weitere Informationen zum Erstellen benutzerdefinierter Eintragsarten finden Sie im Novell Teaming 2.1 Advanced User Guide im Abschnitt zum Entwerfen benutzerdefinierter Formulare für Ordnereinträge.
