Barra degli strumenti Visualizza

La barra degli strumenti Visualizza consente di scegliere i tipi di task da visualizzare nell'apposito elenco. A meno che non ci si trovi nello spazio di lavoro personale di un utente di Teaming, è possibile visualizzare solo i task situati nella cartella attuale.

  • Chiuso: visualizza i task il cui stato è Chiuso.

  • Oggi: visualizza i task con scadenza odierna.

  • Settimana: visualizza i task con scadenza nei prossimi sette giorni.

  • Mese: visualizza i task con scadenza nei prossimi 30 giorni.

  • Tutto attivo: visualizza i task attualmente attivi.

  • Tutte le voci: visualizza tutti i task.

  • Task assegnati: disponibile solo negli spazi di lavoro utente. Permette di visualizzare tutti i task assegnati al proprietario dello spazio di lavoro, indipendentemente da dove i task sono ubicati in Teaming.

  • Da cartella: disponibile solo negli spazi di lavoro utente. Permette di visualizzare tutti i task che si trovano nella cartella attuale.

Per ulteriori informazioni, vedere Note legali.
