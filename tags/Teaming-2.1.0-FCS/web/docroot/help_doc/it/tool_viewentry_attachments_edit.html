Modifica dei file allegati

In Novell Teaming è consentito modificare i file allegati alle voci.

Se risulta impossibile modificare un file, è possibile che Teaming non sia configurato per supportare l'applicazione di modifica installata sul computer. Ad esempio, se Teaming è configurato in modo che Microsoft Word sia l'editor di default per tutti i file di testo, ma sul computer è installato OpenOffice, viene visualizzato un errore e sarà impossibile modificare il documento.

Per risolvere il problema, in Teaming è consentito ignorare l'impostazione dell'editor di default.

Modifica delle impostazioni dell'editor di default per un tipo di file singolo

  1. Nella barra laterale principale, nella sezione Preferenze personali, fare clic su Definisci priorità dell'editor.

  2. Fare clic su Aggiungi.

  3. Nell'elenco a discesa, selezionare l'estensione di file di cui modificare l'applicazione di modifica di default.

  4. Nel campo Applicazione specificare il nome dell'applicazione da utilizzare per modificare il tipo di file selezionato in Passaggio 3.

  5. Selezionare la casella di controllo, quindi fare clic su OK.

Modifica delle impostazioni dell'editor di default per tutti i tipi di file

È possibile configurare Teaming affinché venga utilizzata la stessa applicazione di modifica per tutti i tipi di file.

  1. Nella sezione Preferenze personali della barra laterale principale, fare clic su Definisci priorità dell'editor.

  2. Fare clic su Utilizza, quindi selezionare l'applicazione utilizzata da Teaming per modificare tutti i tipi di file.

  3. Selezionare la casella di controllo a sinistra di Estensione, quindi fare clic su OK.
