Controllo del blog

Tramite la barra degli strumenti Controllo del blog è possibile accedere rapidamente alle seguenti funzioni:

  • Aggiungi commento: consente di creare un commento su una voce del blog. Prima di creare un commento, è opportuno visualizzare i commenti creati precedentemente. Quando in un blog si aggiunge un commento a quelli esistenti, il nuovo commento viene aggiunto automaticamente per ultimo nell'elenco dei commenti nidificati.

  • Visualizza commenti: consente di visualizzare tutti i commenti di una determinata voce, senza doverla aprire.

  • Invia la voce ad amici: consente di inviare il contenuto della voce via e-mail a tutti coloro che dispongono dei diritti di accesso appropriati per visualizzare la voce.

  • Sottoscrivi: avvia un modulo che consente di effettuare la sottoscrizione della voce. Quando si effettua la sottoscrizione di una voce, è necessario configurare Teaming in modo da ricevere le notifiche e-mail sugli aggiornamenti della voce.

    Per ulteriori informazioni sulla sottoscrizione di una voce, consultare Subscribing to an Entry nella Novell Teaming 2.1 User Guide (in lingua inglese).
