Valutazione delle voci

Come numerose applicazioni per la gestione dei dati e per social network, Novell Teaming consente di determinare le voci più richieste. A tal fine, in Teaming è disponibile uno strumento di valutazione a cinque stelle.

Visualizzazione delle valutazioni delle voci

Quando si visualizza una voce, in Teaming vengono visualizzate delle stelle gialle a indicare la valutazione media della voce, dove una stella rappresenta la valutazione minima e cinque stelle quella massima. Se le stelle sono visualizzate in grigio, la voce non è stata valutata.

Invio delle valutazioni delle voci

Per inviare la propria valutazione, fare clic su una stella. (Ad esempio, per inviare una valutazione di tre stelle, fare clic sulla terza stella a partire da sinistra). È possibile modificare la valutazione tutte le volte che lo si desidera, ma nel calcolo della media viene utilizzata solo la valutazione più recente.
