Viste consentite

Poiché solo i proprietari delle cartelle dispongono dei diritti per impostare la vista di default di una cartella, la sezione Viste consentite consente al proprietario della cartella di selezionare le viste che gli utenti esterni della cartella sono autorizzati a utilizzare se non sono soddisfatti della vista di default o preferiscono utilizzare un'altra vista. Ad esempio, se si configura la cartella Calendario e si seleziona Task in questa sezione, gli utenti che visualizzano la cartella Calendario possono visualizzarla nella vista della cartella Task.

Abilitazione delle viste da utilizzare sulla propria cartella

  1. Nella sezione Viste consentite, selezionare le viste consentite desiderate per la cartella che si sta configurando.

  2. Fare clic su Applica > Chiudi.

  3. Nella barra laterale principale, nella sezione Opzioni cartella, fare clic su Vista cartella.

    Vengono visualizzate tutte le viste consentite per la cartella. (Si tratta delle viste selezionate nella sezione Viste consentite in Passaggio 1). Gli utenti della cartella possono scegliere le viste dall'elenco in base alla modalità di visualizzazione desiderata per la cartella.

Abilitazione delle viste da utilizzare come vista di default

Se si seleziona una vista nella sezione Viste consentite, tale vista viene aggiunta nella sezione Vista di default. In tal modo il proprietario della cartella è in grado di scegliere la vista come vista di default.

  1. Nella sezione Viste consentite, selezionare la vista da aggiungere nella sezione Vista di default.

  2. Fare clic su Applica.

    La vista selezionata viene aggiunta alla sezione Vista di default.

Per informazioni su come impostare la vista di default della cartella, vedere Modifica della vista cartella di default.

Per ulteriori informazioni, vedere Note legali.
