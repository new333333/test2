Tillåta e-post

I följande avsnitt beskrivs alternativen på den här sidan.

Aktivera utgående e-post

Du kan ändra följande konfigurationsalternativ för utgående e-post:

  • Aktivera utgående e-post: Markera den här kryssrutan om du vill aktivera all utgående e-post från Teaming. Om du inte vill tillåta användare att skicka e-postmeddelanden från Teaming avmarkerar du den här kryssrutan.

    Den här kryssrutan måste vara markerad för att användarna ska kunna skicka sammanfattningar.

  • Skicka meddelanden via e-post (utgående): I det här området kan du ange standardinställningar för sammanfattningsmeddelanden. Teaming-användare som vill skicka eller ta emot e-postmeddelanden med sammanfattningar måste använda de sammanfattningsinställningar som du väljer här. Du kan ange att sammanfattningar ska skickas varje dag, särskilda dagar i veckan, en viss tid på dagen eller varje timme.

Aktivera inkommande e-post med hjälp av den inbäddade e-postservern och enkla webbadresser

Markera kryssrutan Aktivera inkommande e-post med hjälp av den inbäddade e-postservern och enkla webbadresser om du vill att poster ska kunna skickas via e-post till mappar som har konfigurerats med en enkel webbadress.
