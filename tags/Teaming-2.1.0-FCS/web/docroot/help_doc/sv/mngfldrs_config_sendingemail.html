Skicka e-postmeddelanden

Mappägare kan konfigurera Novell Teaming så att e-postmeddelanden skickas till andra användare för att meddela dem om aktiviteter i mappen och mappens undermappar. Användarens egna inställningar åsidosätter emellertid meddelandeinställningar som definierats av mappägaren.

  1. Välj den typ av e-postmeddelande som du vill skicka under Markera typ av e-postmeddelande.

    Information om de olika typerna av e-postmeddelanden finns i avsnittet Bestämma vilken typ av e-postmeddelande du vill få.

  2. Ange vilka användare som e-postmeddelandena ska skickas till.

    Du kan ange särskilda e-postmeddelanden, adresser eller grupper, eller välja att skicka meddelandena till alla teammedlemmar.

  3. Klicka på Verkställ > Stäng.

Mer information om e-postmeddelanden finns i avsnittet Använda e-postmeddelanden.

Mer information finns i Juridisk information.
