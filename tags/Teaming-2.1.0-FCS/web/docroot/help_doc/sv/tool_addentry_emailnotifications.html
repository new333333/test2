Använda e-postmeddelanden

Du kan konfigurera Novell Teaming så att du får e-postmeddelanden för mappar och undermappar, eller för enskilda poster (kallas att prenumerera på en mapp eller post). Mappägare kan också konfigurera Teaming så att e-postmeddelanden skickas till dig. En användares prenumerationsinställningar åsidosätter emellertid mappägarens inställningar, och du kan använda det här formuläret för att åsidosätta konfigurationer på en högre nivå.

Välja meddelandetyper och leveransmål

I det här formuläret kan du välja vilken typ av meddelanden du vill få, och till vilken e-postadress eller till vilket telefonnummer meddelandena ska skickas.

Välj en e-postadress eller ett telefonnummer som meddelandena ska skickas till. Du kan välja följande leveransmål i fälten som visas, som representerar de olika typerna av e-postmeddelanden:

  • Sammanfattning

  • Enskilda meddelanden

  • Enskilda meddelanden utan bilagor

  • Meddelande som lämpar sig för textmeddelande

Mer information om de olika meddelandetyperna finns i avsnittet Bestämma vilken typ av e-postmeddelande du vill få.

Du kan välja flera e-postadresser eller telefonnummer som leveransmål för varje meddelandetyp genom att trycka på Ctrl och klicka på e-postadresserna eller telefonnumren.

Lägga till och ändra leveransmål

Du kan lägga till och ändra e-postadresserna och telefonnumren i din personliga profil. Så här öppnar du din personliga profil:

  1. Klicka på Min arbetsyta > Ändra profil.

  2. Ange information om leveransmålet i fälten E-postadress, E-postadress för mobil och E-postadress för textmeddelande.

  3. Klicka på OK.

När du väljer meddelandetyperna och leveransmålen visas den nya eller ändrade kontaktinformationen.

Bestämma vilken typ av e-postmeddelande du vill få

Du kan välja att få olika typer av e-postmeddelanden:

  • Sammanfattning: Ett meddelande skickas med en sammanfattning av alla aktiviteter i mappen och dess undermappar (endast tillgängligt för mappar). Sammanfattningarna skickas enligt ett schema som definieras av webbplatsadministratören.

  • Enskilda meddelanden: Ett meddelande skickas för varje ny eller ändrad post och meddelandena innehåller bifogade filer.

  • Enskilda meddelanden utan bilagor: Ett meddelande skickas för varje ny eller ändrad post. E-postmeddelandena innehåller inte de eventuella filer som är bifogade till de nya eller ändrade posterna. Alla typer av enskilda meddelanden skickas var femte minut.

  • Meddelande som lämpar sig för textmeddelande: Ett meddelande som kan visas som ett textmeddelande skickas. Meddelandet innehåller bara länkrubriken för den nya eller ändrade posten. Textmeddelanden skickas upp till var femte minut.

Åsidosätta förinställda meddelanden

När du väljer inställningar i det här formuläret och väljer Åsidosätt förinställda meddelanden med de här valen används dina alternativ i stället för de inställningar som gjorts av administratörer eller mappägare.

Mer information finns i Juridisk information.
