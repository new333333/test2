Tillämpa definitionsinställningarna rekursivt

Använd den här funktionen om du vill tillämpa definitionsinställningar på mappar eller poster som redan skapats.

Tillämpa definitionsinställningar på alla undermappar

Du kan tillämpa alla definitionsinställningar för en överordnad mapp på alla undermappar.

  1. Välj Ändra mappar som ingår så att de ärver den här mappens definitioner.

  2. Klicka på Verkställ.

Alla definitionsinställningar på sidan Konfigurera standardvärden tillämpas på alla undermappar.

Tillämpa definitionsinställningar på befintliga poster

Om du tidigare har lagt till poster i den här mappen med en annan definitionstyp än den för andra mapposter (du kanske exempelvis har en bloggmapp som huvudsakligen innehåller bloggdefinitionstyper, men även några poster som visas som diskussionsdefinitionstyper), kan du ändra definitionsinställningarna så att alla mapposter är samma.

  1. Välj Ändra mappar som ingår så att de ärver den här mappens definitioner och sedan Ändra poster som ingår så att de får följande posttyp.

  2. Välj den posttyp i listrutan som du vill tillämpa på alla befintliga poster i mappen.

    Om du vill lägga till posttyper i listrutan väljer du posttyperna i avsnittet Standardposttyper och klickar på Verkställ.

  3. Klicka på Verkställ.
