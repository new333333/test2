Visa händelserna i en kalender

I listrutan Visa kan du välja vilka typer av kalenderhändelser som ska visas i kalendern och hur du vill att de ska visas. Om du inte befinner dig i en Teaming-användares personliga arbetsyta, kan du endast visa de kalenderhändelser som finns i den aktuella mappen.

  • Tilldelade händelser: Endast tillgängligt i användararbetsytor. Visar alla kalenderhändelser som tilldelats arbetsytans ägare, oavsett var i Teaming händelserna finns.

  • Från mapp - Händelser: Kalenderhändelserna i den aktuella mappen visas på de dagar händelserna schemalagts till.

  • Från mapp - Alla efter skapelsedatum: Kalenderhändelserna visas på de dagar händelserna skapades, snarare än de dagar de schemalagts till.

  • Från mapp - Alla efter aktivitet: Kalenderhändelserna visas på de dagar då händelserna senast ändrades, snarare än de dagar de schemalagts till.
