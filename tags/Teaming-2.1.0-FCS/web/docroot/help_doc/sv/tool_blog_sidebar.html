Sidofältet Blogg

Sidofältet Blogg i Novell Teaming innehåller följande verktyg som du kan använda för att leta upp och visa bloggposter:

Bloggsidor

Om bloggmappen innehåller flera sidor visas sidorna överst i sidofältet Blogg. Om du vill gå till en annan sida i mappen klickar du på länknamnet för sidan.

Arkiv

Sidofältet Blogg innehåller området Arkiv, som visar alla månader då poster skapats i bloggmappen. När du klickar på en månad visas alla bloggposter som skapats under månaden i omvänd kronologisk ordning, med de nyaste posterna överst i listan.
