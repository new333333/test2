日期工具列

Novell Teaming 中的「日期」工具列可讓您以多種方式自定「行事曆」資料夾檢視窗。

  • 指定 Teaming 是顯示一天中的所有小時，還是僅顯示一般的工作小時。

  • 指定在單一資料夾頁面上顯示的時間長度 (1 天、3 天、5 天、1 週、2 週或 1 個月)。

  • 使用向左箭頭或向右箭頭圖示指定下一個或上一個資料夾頁面。例如，如果您目前正在檢視 3 天的資訊，按一下箭頭將顯示上一個或下一個 3 天的資訊。

  • 按一下「行事曆設定」圖示 可以指定下列選項：

    • 行事曆從星期幾開始。(星期六、星期日或星期一)。

    • 工作日開始的時間。

  • 按一下「行事曆」圖示 可以快速導覽至新的月份。在行事曆 Applet 中選取要檢視的新月份之後，務必要按一下「確定」以重新整理資料夾頁面。

有關更詳細的資訊，請參閱 法律聲明。
