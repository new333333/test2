檢視工具列

「檢視」工具列可讓您選擇要顯示在任務清單中的任務類型。除非您位於 Teaming 使用者的個人工作區，否則只可檢視位於目前資料夾中的那些任務。

  • 已停止： 顯示狀態為「已停止」的任務。

  • 今天︰ 顯示截止日期為今天的任務。

  • 週： 顯示截止日期為後 7 天中某個時間的任務。

  • 月： 顯示截止日期為後 30 天中某個時間的任務。

  • 所有進行中的項目： 顯示目前進行中的任務。

  • 所有項目： 顯示所有任務。

  • 指定的任務： 只適用於使用者工作區。顯示指定給工作區擁有者的所有任務，而不論這些任務是否位於 Teaming 中。

  • 來自資料夾： 只適用於使用者工作區。顯示目前資料夾中的所有任務。

有關更詳細的資訊，請參閱 法律聲明。
