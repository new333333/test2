以說明模式開始

頁面左上角中的面板稱為「說明模式控制台」。多數情況下，您都是使用它來存取目錄或線上文件。若要最小化「說明模式控制台」，請按一下控制台頂部的「最小化」圖示 。再次按一下此圖示會將控制台還原為其完整大小。

說明模式說明您目前正在檢視的頁面。按一下「資訊」圖示 後，Novell Teaming 將顯示「說明」面板，介紹頁面中的該區段。若要關閉「說明」面板，請按一下「說明」面板右上角的「關閉」圖示 (X)。或者，也可以按一下其他「資訊」圖示繼續閱讀。如果不使用「資訊」圖示，可以在「說明模式控制台」中按一下「說明主題」，然後按一下感興趣之主題的連結標題。

如需更完整的 Teaming 文件，可以在「說明模式控制台」中按一下「檢視手冊」。

使用完說明模式後，可以按一下控制台右上角的「關閉」圖示 (X) 結束，或者按一下頁面灰色部分的任何位置。

結束說明模式並開始使用工具之後，Teaming 會在需要其他說明的表單上提供「資訊」圖示 。按一下表單區段旁邊的「資訊」圖示可以閱讀有關表單的詳細資訊。

有關更詳細的資訊，請參閱 法律聲明。
