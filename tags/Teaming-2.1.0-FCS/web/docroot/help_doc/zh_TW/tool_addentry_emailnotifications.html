使用電子郵件通知

您可以將 Novell Teaming 設定為向您傳送有關資料夾及其子資料夾或有關個別項目的電子郵件通知 (稱為「訂閱」資料夾或項目)。此外，資料夾擁有者還可以將 Teaming 設定為向您傳送電子郵件通知。不過，因為使用者的個別訂閱設定會置換資料夾擁有者的訂閱設定，所以可以使用此表單置換任何更高層級的組態。

選取通知類型和傳送目的地

此表單可讓您選擇要接收哪些類型的通知，以及要將通知傳送至哪個電子郵件地址或哪個電話號碼。

選取要將通知送往的電子郵件地址或電話號碼。您可以在提供的表示下列電子郵件通知類型的任何區段中選取這些傳送目的地：

  • 摘要

  • 個人郵件

  • 不包含附件的個人郵件

  • 適用於文字訊息傳送的訊息

如需各種通知類型的詳細資訊，請參閱決定要接收哪些類型的電子郵件通知。

對於每種通知類型，只需按住 Ctrl 鍵並選取電子郵件地址或電話號碼，即可選取多個電子郵件或電話號碼傳送目的地。

新增和修改傳送目的地

您可以在「個人設定檔」中新增和修改電子郵件和電話號碼聯絡資訊。若要存取「個人設定檔」頁面：

  1. 按一下「我的工作區」>「修改設定檔」。

  2. 在「電子郵件地址」、「行動電子郵件地址」和「文字訊息電子郵件地址」欄位中，指定適當的傳送目的地資訊。

  3. 按一下「確定」。

現在，當您選取通知類型和傳送目的地時，將顯示新的或修改後的聯絡資訊。

決定要接收哪些類型的電子郵件通知

您可以接收不同類型的電子郵件通知：

  • 摘要： Teaming 傳送一則訊息，其中彙總了資料夾及其子資料夾中的所有活動 (僅可用於資料夾)。摘要通知將根據站台管理員建立的排程進行傳送。

  • 個人郵件： Teaming 將針對每個新的或變更的項目傳送一封郵件，且在郵件中附加檔案。

  • 不包含附件的個人郵件： Teaming 將針對每個新的或變更的項目傳送一封郵件。這些電子郵件不包含新的或已變更項目中可能附加的檔案。Teaming 將每五分鐘傳送一次所有類型的個人通知。

  • 適用於文字訊息傳送的訊息： Teaming 將傳送適合做為文字訊息檢視的訊息。訊息的內容僅包含新的或已變更項目的連結標題。Teaming 最多可以每五分鐘傳送一次文字訊息通知。

置換預設通知

在此表單中選擇設定並選取「以這些選擇置換預設通知」後，您的選擇將置換管理員或資料夾擁有者所做的選擇。
