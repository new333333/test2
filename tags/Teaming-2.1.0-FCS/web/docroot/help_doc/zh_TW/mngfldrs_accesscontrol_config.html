設定存取控制

此頁面用於變更此工作區或資料夾的存取控制設定。

依預設，工作區與資料夾會繼承其父代的存取控制設定。只有在您想要有不同的存取設定時才使用這個頁面。

在對存取控制表格進行任何變更之前，必須先確定資料夾或工作區未從父目錄繼承使用者和角色。如需此操作的相關資訊，請參閱繼承角色成員資格。

除這一節及後面幾小節包含的資訊之外，若要查看更詳細的存取控制資訊，請參閱《Novell Teaming 2.1 Advanced User Guide》(Novell Teaming 2.1 進階使用者指南) 中的「Controlling User Access」(控制使用者存取權限)。

以下幾部分說明如何根據您的存取需要修改存取控制表格。

新增使用者和群組

  1. 按一下表格的左側欄中的「新增群組」或「新增使用者」連結。

  2. 在該欄位中，輸入使用者名字或群組的前幾個字母，然後按一下下拉式清單中的連結名稱。

    Novell Teaming 將在表格中為該使用者或群組新增一列。

檢視角色任務

在存取控制表格中，角色名稱便是欄標題。

  1. 按一下欄標題 (參與者、團隊成員等等) 以查看與該角色關聯的權限清單。

變更使用者或群組的角色

角色是一組權限；將某人指定給角色時，他們會自動獲得與該角色關聯的所有權限。

  1. 透過選取和不選核取方塊可以調整人員或群組的角色。

  2. 按一下「確定」。

在其他位置調整存取控制

  1. 移至此頁面頂部的工作區樹狀結構，導覽至要調整存取控制的位置，然後按一下樹狀結構中該位置的連結。

    此時會顯示該位置的「存取控制」頁面。

有關更詳細的資訊，請參閱 法律聲明。
