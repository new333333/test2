我的最愛面板

按一下「我的最愛」後，Novell Teaming 將顯示「我的最愛」面板。可以將您慣用的工作區和資料夾新增至此面板，以便日後能更輕鬆地存取它們。

以下幾部分介紹如何修改「我的最愛」面板。

將位置新增至我的最愛面板

  1. 使用導覽列或工作區樹狀結構檢視工作區或資料夾。

  2. 到達某個最愛位置後，按一下「我的最愛」。

  3. 按一下「新增此位置」。

編輯我的最愛面板

「我的最愛」面板包含一些工具，可用於組織指向最愛位置的連結。

  1. 按一下「我的最愛」>「編輯我的最愛」。

  2. 選取要編輯的項目。

  3. 按一下「向下」、「向上」或「刪除」。

    「向下」和「向上」按鈕會將連結在「我的最愛」清單中上下移動。

  4. 按一下「確定」。

有關更詳細的資訊，請參閱 法律聲明。
