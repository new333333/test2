Tükrözött mappák

A tükrözött mappa egy könyvtár mappa, amelynek fájljai egy külső rendszerben helyezkednek el, (vagy egy fájlrendszer könyvtárában, vagy egy SharePoint szerver mappájában). Amikor külső mappákat tükröz a Teamingbe, akkor a külső fájlok esetében is használhatja a Novell Teaming eszközöket, (például keresés és munkafolyamat).

A szolgáltatás engedélyezéséhez a kiszolgálón található fájllal kell dolgoznia.

További tájékoztatás a tükrözött mappák létrehozásáról: Working with Mirrored Folders című fejezet, Novell Teaming 2.1 User Guide útmutató.
