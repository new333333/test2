Hivatkozás egy bejegyzésre különböző mappában

  1. Kattintson a Módosítás gombra.

  2. A megadott mezőben kezdje el beírni a bejegyzést tartalmazó mappa nevét, ahová hivatkozni szeretne.

  3. Amikor a kívánt mappa megjelenik a legördülő listában, válassza ki.

  4. A Hivatkozás erre az oldalnévre mezőben határozza meg a bejegyzés teljes nevét, amelyre hivatkozni szeretne vagy kattintson a Keresés beállításra a keresés gépelés közben funkció használatához.

  5. Miután befejezte a bejegyzés beírását vagy miután rákattintott a névre a gépelés közben megjelenő legördülő listából, kattintson a Beszúrás gombra.

    A bejegyzés hivatkozási címét beillesztette a Leírás mezőbe.

További információ a Szerzői jogi megjegyzések.
