Nézet eszköztár

A Nézet eszköztár lehetőséget ad arra, hogy kiválassza, milyen jellegű feladatokat szeretne megjeleníteni. Hacsak nincs egy Teaming-felhasználó személyes munkaterületén, megtekintheti csak az aktuális mappában lévő feladatokat.

  • Lezárt: Lezárt feladatok megjelenítése.

  • Mai nap: Mai napon esedékes feladatok megjelenítése.

  • Hét: Az elkövetkező hét napban esedékes feladatok megjelenítése.

  • Hónap: Az elkövetkező 30 napban esedékes feladatok megjelenítése.

  • Összes aktív: A jelenleg aktív feladatok megjelenítése.

  • Összes bejegyzés: Az összes feladat megjelenítése.

  • Hozzárendelt feladatok: Csak felhasználói munkaterületen érhető el. Megjelenít minden, a munkaterület tulajdonosához társított feladatot, függetlenül attól, hogy ezek a feladatok hol találhatók a Teaming alkalmazáson belül.

  • Forrásmappa: Csak felhasználói munkaterületen érhető el. Megjelenít minden, az aktuális mappában lévő feladatot.
