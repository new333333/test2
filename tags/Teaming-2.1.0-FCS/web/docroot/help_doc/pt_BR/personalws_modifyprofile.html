Modificar Perfil

Use essa ferramenta para mudar as informações de contato, como senha, nome, telefone, endereço de e-mail e o nome de usuário do Novell Conferencing.

Também é possível fazer upload de imagens, reorganizá-las usando os ícones de seta para cima e para baixo e apagá-las. Se você fizer upload de mais de uma imagem, o Teaming mostrará a primeira imagem na lista na seção Foto, e versões menores das outras imagens na barra abaixo das informações de contato. Quando você passa o mouse sobre uma das imagens menores, o Teaming exibe uma versão maior na seção Foto.

Para obter mais informações, consulte Informações legais.
