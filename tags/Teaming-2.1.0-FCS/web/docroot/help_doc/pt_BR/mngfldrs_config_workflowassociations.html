Associações de Fluxo de Trabalho

Esta seção permite associar fluxos de trabalho a tipos de entrada ou de comentário (resposta). Quando você associa um fluxo de trabalho a um tipo de entrada ou de comentário, o fluxo de trabalho é automaticamente iniciado assim que a entrada é criada ou o comentário é feito.

Associando fluxos de trabalho a entradas

Esta seção contém todos os tipos de entrada que você selecionou na seção Tipos de Entrada Padrão nesta página. Ao lado de cada tipo de entrada na seção Associações de Fluxo de Trabalho, o Teaming exibe uma lista suspensa com todos os processos de fluxo de trabalho que você pode associar ao tipo de entrada.

Para associar um fluxo de trabalho a determinado tipo de entrada:

  1. Verifique se você habilitou a entrada para ser criada nesta pasta, selecionando o tipo de entrada na seção Tipos de Entrada Padrão desta página.

    Após selecionar um tipo de entrada na seção Tipos de Entrada Padrão e clicar em Aplicar, ele será exibido na seção Associações de Fluxo de Trabalho.

  2. Na seção Associações de Fluxo de Trabalho, na seção Tipos de Entrada, selecione o fluxo de trabalho na lista suspensa ao lado do tipo de entrada ao qual deseja aplicar o fluxo de trabalho.

  3. Clique em Aplicar > Fechar.

    Agora, quando alguém criar uma entrada desse tipo nesta pasta, o Teaming a colocará automaticamente no estado inicial do processo de fluxo de trabalho.

Quando você cria um fluxo de trabalho em uma pasta, esse fluxo de trabalho é automaticamente adicionado a esta lista suspensa e, a partir disso, você pode associá-lo a entradas nesta pasta. Para obter informações sobre como criar fluxos de trabalho, consulte Creating and Managing Workflows (Criando e gerenciando fluxos de trabalho) no Novell Teaming 2.1 Advanced User Guide (Guia do Usuário Avançado do Novell Teaming 2.1).

Associando fluxos de trabalho a tipos de comentário (Resposta)

A seção Tipos de Comentário ou Resposta permite determinar que os fluxos de trabalho sejam iniciados quando os usuários inserirem um tipo de resposta específico.

O Teaming exibe todos os tipos de resposta para cada tipo de entrada padrão listado na seção Associações de Fluxo de Trabalho.

Quando um tipo de entrada padrão também é um tipo de resposta, ele não é listado duas vezes.

Para obter mais informações, consulte Informações legais.
