Tela da área de trabalho

Selecione a tela que melhor apresenta a sua área de trabalho.

  • Área de Trabalho de Discussões: Selecione essa tela se quiser criar uma área de trabalho na qual armazenar vários fóruns de discussão.

    Para obter mais informações sobre áreas de trabalho de discussões, consulte Gerenciando uma área de trabalho de discussões no Guia do Usuário do Novell Teaming 2.1.

  • Landing Page: Selecione essa tela se quiser exibir informações importantes aos membros da equipe assim que eles ingressarem na área de trabalho.

    Para obter mais informações sobre landing pages, consulte Criando uma landing page no Guia do Usuário do Novell Teaming 2.1.

  • Área de Trabalho de Gerenciamento do Projeto: Selecione essa tela se quiser rastrear tarefas e etapas de projeto com base na conclusão da tarefa.

    Para obter mais informações sobre áreas de trabalho de gerenciamento do projeto, consulte Gerenciando uma área de trabalho de gerenciamento do projeto no Guia do Usuário do Novell Teaming 2.1.

  • Área de Trabalho de Equipe: Selecione essa tela se a sua área de trabalho de equipe for organizacional por natureza, e não planejada para ter áreas de trabalho de equipe como seus sublocais.

    Para obter mais informações sobre áreas de trabalho de equipe, consulte Gerenciando uma área de trabalho de equipe no Guia do Usuário do Novell Teaming 2.1.

  • Raiz da Área de Trabalho de Equipe: Selecione essa tela se quiser exibir o botão Adicionar uma Área de Trabalho de Equipe em sua área de trabalho de equipe. Isso é recomendável quando você deseja criar áreas de trabalho de equipe adicionais como sublocais dessa área de trabalho.

    Para obter mais informações sobre áreas de trabalho raiz da área de trabalho de equipe, consulte Creating and Managing a Team Workspace Root Workspace (Criando e gerenciando uma área de trabalho raiz da área de trabalho de equipe) no Novell Teaming 2.1 Advanced User Guide (Guia do Usuário Avançado do Novell Teaming 2.1).

  • Área de Trabalho: Uma área de trabalho básica. Você pode escolher quais pastas deseja incluir na área de trabalho.

    Para obter mais informações sobre áreas de trabalho básicas, consulte Creating and Managing a Basic Workspace (Criando e gerenciando uma área de trabalho básica) no Novell Teaming 2.1 Advanced User Guide (Guia do Usuário Avançado do Novell Teaming 2.1).

Se esta for uma área de trabalho pessoal, a tela padrão será Área de Trabalho de Usuário e não poderá ser mudada.

Para obter mais informações, consulte Informações legais.
