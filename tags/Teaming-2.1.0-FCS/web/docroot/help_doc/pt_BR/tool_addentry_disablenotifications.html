Modificando configurações de notificações por e-mail

Selecione essa opção se desejar que suas seleções substituam qualquer configuração de notificação feita pelo administrador ou por um proprietário de pasta.

Se não quiser receber notificações por e-mail, marque essa caixa de seleção e verifique se todos os campos acima não especificam nenhum endereço de e-mail ou número de telefone.

Para obter mais informações, consulte Informações legais.
