Dicas de Ferramentas

Esse link ativa e desativa o recurso Dicas de Ferramentas no Novell Teaming. Clique em Ativar dicas de ferramentas para exibir as dicas de ferramentas. As dicas de ferramentas são exibidas quando você passa o mouse sobre determinadas opções na interface do Teaming. Se não quiser mais ver as dicas de ferramentas, clique em Desativar dicas de ferramentas.

Para obter mais informações, consulte Informações legais.
