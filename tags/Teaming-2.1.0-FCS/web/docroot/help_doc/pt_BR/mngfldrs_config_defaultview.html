Mudando a tela de pasta padrão

  1. Selecione a tela com a aparência que melhor apresente a sua lista de pastas ou área de trabalho.

  2. Clique em Aplicar > Fechar.

    A tela de pasta é modificada.

Para obter informações sobre como adicionar telas a esta seção para que possam ser selecionadas como a tela padrão da pasta, consulte a Habilitando telas para uso como a tela padrão.

Leve em consideração os tipos de entradas contidos na pasta quando for escolher uma tela para a pasta. Por exemplo, uma lista de pasta de entradas de calendário geralmente fica melhor representada como uma tela de calendário.

Para obter mais informações, consulte Informações legais.
