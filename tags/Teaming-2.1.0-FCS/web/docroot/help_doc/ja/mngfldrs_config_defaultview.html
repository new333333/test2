デフォルトのフォルダ表示の変更

  1. フォルダリストまたはワークスペースが最もわかりやすく表示される外観の表示を選択します。

  2. ［適用］>［閉じる］の順にクリックします。

    フォルダ表示が変更されます。

このセクションに表示を追加してフォルダのデフォルト表示として選択できるようにする方法の詳細については、デフォルト表示として使用する表示の有効化を参照してください。

フォルダの表示を選択する際には、フォルダに含まれるエントリのタイプを考慮してください。たとえば、通常、カレンダ表示としては、カレンダエントリのフォルダリストが最もわかりやすい表示です。

詳細については、「保証と著作権」を参照してください。
