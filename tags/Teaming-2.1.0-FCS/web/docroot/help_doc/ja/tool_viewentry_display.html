エントリの表示と他のユーザへの連絡

Novell Teamingの複数のツール(フォルダリストや検索結果など)には、エントリ情報の概要が表示されます。次の項で説明するように、この概要に含まれる情報から、エントリを表示したり、エントリの作成者に連絡したりすることができます。

エントリの表示

エントリを表示するには、エントリのタイトルをクリックするだけです。

Teamingでエントリを表示する方法を設定できます。エントリを開いたときに、オーバーレイ、新しいページ、またはポップアップウィンドウにエントリを表示できます。

Teamingでエントリを表示する方法を変更する

  1. メインサイドバーの［パーソナル初期設定］セクションで、［エントリの表示スタイル］をクリックします。

  2. 目的のエントリ表示方法を選択します。

他のユーザへの連絡

エントリの概要には、作成者名の横に［在席/不在］アイコンが含まれています。この［在席/不在］アイコンは、次のステータスになっている可能性があります。

  • 緑: そのユーザはオンラインでアクティブです。

  • 黄: そのユーザは少し前までアクティブでしたが、現在は席を外しています。

  • 灰色: そのユーザはNovell Conferencingクライアントにログインしていません。

  • 白: そのユーザのステータスは不明です。

そのユーザに連絡するには、［在席/不在］アイコンをクリックしてメニュー項目を選択します。そのユーザとインスタントミーティングを開始したり、ミーティングをスケジュールしたり、電話をかけたり、そのユーザをクリップボードに追加したり、そのユーザのマイクロブログを表示したりすることができます。

詳細については、「保証と著作権」を参照してください。
