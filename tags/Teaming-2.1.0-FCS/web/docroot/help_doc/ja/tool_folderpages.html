リストエントリとページ

これらのツールを使用して、エントリを個別に検索したり、リストページ間を移動したりできます。フォルダリストには、エントリ、またはエントリに関する情報が表示されます。ツールによっては、エントリのリストページが複数ある場合にしか役に立たないものもあります。次に、このツールバーの左から右の順に、使用可能なツールを示します。

  • エントリへ移動: 表示するエントリの番号(［#］カラムに表示されています)を指定して、［移動］をクリックします。

  • 戻る/次へ: フォルダのページをスクロールできます。

  • ページへ移動: このテキストボックスにページ番号を入力して［移動］をクリックすると、Novell Teamingに該当するページのエントリが表示されます。

詳細については、「保証と著作権」を参照してください。
