［お気に入り］パネル

［お気に入り］をクリックすると、Novell Teamingでは、［お気に入り］パネルが表示されます。このパネルにお気に入りのワークスペースやフォルダを追加しておくと、以降はより簡単にアクセスできます。

次の項では、［お気に入り］パネルの変更方法について説明します。

［お気に入り］パネルへの場所の追加

  1. ナビゲーションバーまたはワークスペースツリーを使用して、ワークスペースまたはフォルダを表示します。

  2. お気に入りの場所の1つに移動したら、［お気に入り］をクリックします。

  3. ［この場所を追加］をクリックします。

［お気に入り］パネルの編集

［お気に入り］パネルには、お気に入りの場所へのリンクを整理するためのツールが含まれています。

  1. ［お気に入り］>［お気に入りを編集］の順にクリックします。

  2. 編集する項目を選択します。

  3. ［下へ］、［上へ］、または［削除］をクリックします。

    ［下へ］および［上へ］ボタンをクリックすると、リンクがお気に入りリストの上または下へ移動します。

  4. ［OK］をクリックします。

詳細については、「保証と著作権」を参照してください。
