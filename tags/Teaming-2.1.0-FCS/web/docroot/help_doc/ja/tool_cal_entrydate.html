カレンダでのイベントの表示

［表示］ドロップダウンリストでは、カレンダで表示するカレンダイベントの種類、またその表示方法を選択できます。Teamingユーザのパーソナルワークスペースで作業していない限り、現在のフォルダに配置されているカレンダイベントしか表示することはできません。

  • 割り当て済みイベント: ユーザワークスペースでのみ表示されます。イベントがTeamingのどこに配置されているかにかかわらず、ワークスペースの所有者に割り当てられているすべてのイベントを表示します。

  • フォルダから - イベント: 現在のフォルダにある、イベントがスケジュール設定されている日のすべてのカレンダイベントが表示されます。

  • フォルダから - 作成別にすべて: イベントがスケジュール設定されている日ではなく、イベントが作成された日のすべてのカレンダイベントが表示されます。

  • フォルダから - アクティビティ別にすべて: イベントがスケジュール設定されている日ではなく、イベントが最後に変更された日のすべてのカレンダイベントが表示されます。

詳細については、「保証と著作権」を参照してください。
