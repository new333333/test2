［表示］ツールバー

［表示］ツールバーでは、タスクリストに表示するタスクの種類を選択できます。Teamingユーザのパーソナルワークスペースで作業していない限り、現在のフォルダに配置されているタスクしか表示することはできません。

  • 終了: ステータスが「終了」のタスクを表示します。

  • 今日: 今日が期日のタスクを表示します。

  • 週: 今後7日間のいずれかの日が期日のタスクを表示します。

  • 月: 今後30日間のいずれかの日が期日のタスクを表示します。

  • すべてアクティブ: 現在アクティブなタスクを表示します。

  • すべてのエントリ: すべてのタスクを表示します。

  • 割り当て済みタスク: ユーザワークスペースでのみ表示されます。タスクがTeamingのどこに配置されているかにかかわらず、ワークスペースの所有者に割り当てられているすべてのタスクを表示します。

  • フォルダから: ユーザワークスペースでのみ表示されます。現在のフォルダにあるすべてのタスクを表示します。

詳細については、「保証と著作権」を参照してください。
