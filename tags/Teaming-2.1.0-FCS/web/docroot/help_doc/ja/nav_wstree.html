ワークスペースツリーのナビゲーション

ワークスペースツリーには、現在表示している場所が含まれるワークスペースとフォルダすべてのパスが表示されます。

さらにこのツリーから、Novell Teaming内にある他の目的の場所へ移動することもできます。その際、現在のページから離れる必要はありません。Teamingでは、ツリー内のワークスペースまたはフォルダリンクをクリックすると、新しい場所が表示されます。直前に使用していたページに戻るには、メインサイドバーの［最近使った場所］セクションにあるタイトルのリンクをクリックします。

項目のサブワークスペースまたはサブフォルダを表示するにはプラス記号(+)をクリックします。表示を縮小するにはマイナス記号(-)をクリックします。

場所のタイトルのリンクをクリックすると、該当するページに移動します。

作業を開始するには、組織の最上位レベルのワークスペースの横にあるプラス記号をクリックします。グローバル、パーソナル、およびチームワークスペースへのリンクが表示されます。これらは特定の作業カテゴリ用のコンテナで、デフォルトのワークスペースになります。

詳細については、「保証と著作権」を参照してください。
