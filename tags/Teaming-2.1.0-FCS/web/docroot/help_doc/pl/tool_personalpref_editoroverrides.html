Wymuszenia edytora

Do edycji załączników w witrynie Novell Teaming używane są domyślne programy zdefiniowane w tym systemie. Jednakże tabela ta umożliwia zastąpienie programów domyślnych, używanych do edycji załączników o określonych rozszerzeniach nazwy pliku w witrynie Teaming.

Można zastąpić domyślny program do edycji jednego typu pliku albo domyślne programy do wielu typów plików.

Wymuszenie domyślnego programu dla jednego typu pliku rozszerzenia

Aby zastąpić domyślny program dla jednego typu pliku, należy:

  1. Kliknij przycisk Dodaj.

  2. Wybrać z listy rozwijanej rozszerzenie typu pliku, dla którego trzeba zmienić domyślny program do edycji.

  3. W polu Aplikacja określić nazwę programu, który ma być używany do edycji załączników o rozszerzeniach nazw pliku wybranych w krok 2.

  4. Proszę nacisnąć przycisk OK.

Wymuszenie domyślnego programu dla wszystkich typów plików rozszerzeń

Aby zastąpić domyślny program dla wszystkich typów plików, należy:

  1. Kliknąć przycisk Użyj.

  2. Wybrać z listy rozwijanej program, który ma być używany do edycji wszystkich typów załączników w systemie Teaming.

  3. Proszę nacisnąć przycisk OK.

Aby uzyskać więcej informacji, patrz sekcja Informacje prawne.
