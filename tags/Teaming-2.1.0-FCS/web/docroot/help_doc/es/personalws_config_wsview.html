Vista del área de trabajo

Seleccione la vista que mejor presente su área de trabajo.

  • Área de trabajo de discusiones: seleccione esta vista si desea crear un área de trabajo en la que se puedan almacenar varios foros de discusión.

    Para obtener más información sobre las áreas de trabajo de discusiones, consulte Gestión de un área de trabajo de discusiones en la Guía del usuario de Novell Teaming 2.1.

  • Página de entrada: seleccione esta vista si desea mostrar información importante a miembros del equipo en cuanto entren al área de trabajo.

    Para obtener más información sobre las páginas de entrada, consulte Gestión de una página de entrada en la Guía del usuario de Novell Teaming 2.1.

  • Área de trabajo de gestión de proyectos: seleccione esta vista si desea rastrear las tareas del proyecto y los hitos según finalicen las tareas.

    Para obtener más información sobre las áreas de trabajo de gestión de proyectos, consulte Gestión de un área de trabajo de gestión de proyectos en la Guía del usuario de Novell Teaming 2.1.

  • Área de trabajo del equipo: seleccione esta vista si el área de trabajo del equipo es por naturaleza para toda la organización y no está destinada a tener áreas de trabajo de equipo como sublugares.

    Para obtener más información sobre las áreas de trabajo del equipo, consulte Gestión de un área de trabajo del equipo en la Guía del usuario de Novell Teaming 2.1.

  • Áreas de trabajo raíz del equipo: seleccione esta vista si desea mostrar el botón Añadir un área de trabajo de equipo al área de trabajo de su equipo. Esto es aconsejable si desea crear otras áreas de trabajo de equipo como sublugares de esta área de trabajo.

    Para obtener más información sobre las áreas de trabajo raíz del equipo, consulte Creating and Managing a Team Workspace Root Workspace (Creación y gestión de un área de trabajo raíz del equipo) en la Novell Teaming 2.1 Advanced User Guide (Guía del usuario avanzado de Novell Teaming 2.1).

  • Área de trabajo: se trata de un área de trabajo básica. Puede elegir las carpetas que desee incluir en el área de trabajo.

    Para obtener más información sobre las áreas de trabajo básicas, consulte Creating and Managing a Basic Workspace (Creación y gestión de un área de trabajo básica) en la Novell Teaming 2.1 Advanced User Guide (Guía del usuario avanzado de Novell Teaming 2.1).

Si se trata de un área de trabajo personal, la vista por defecto es Área de trabajo del usuario y no se puede cambiar.

Para obtener más información, consulte Información legal.
