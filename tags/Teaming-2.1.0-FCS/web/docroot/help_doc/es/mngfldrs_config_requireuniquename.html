Requerir que los archivos tengan un nombre exclusivo

Si marca esta casilla de verificación, algunas funciones del producto tratarán esta carpeta como una carpeta de archivos. Por ejemplo, el applet de carga de archivos crea nuevas entradas para los nombres de archivo exclusivos y crea nuevas versiones de los archivos adjuntos que ya existen en la carpeta.

Para obtener más información, consulte Información legal.
