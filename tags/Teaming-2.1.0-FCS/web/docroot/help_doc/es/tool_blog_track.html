Rastreo de blog

La barra de herramientas Rastreo de blog ofrece formas rápidas de:

  • Añadir un comentario: permite realizar un comentario sobre una entrada de un blog. Antes de realizar un comentario, es buena idea ver los comentarios que se hayan efectuado anteriormente. Cuando se añade un comentario a un blog donde ya hay comentarios anteriores, el nuevo comentario se añade automáticamente al final de la lista de comentarios anidados.

  • Ver comentarios: permite ver todos los comentarios de una entrada en particular, sin abrir la entrada.

  • Enviar esta entrada a amigos: permite enviar el contenido de la entrada por correo electrónico a cualquier usuario que tenga derechos de acceso adecuados para ver la entrada.

  • Suscribir: lanza un formulario que permite suscribirse a la entrada. Cuando se suscribe a una entrada, configura Teaming para que le envíe notificaciones de correo electrónico que le informarán sobre las actualizaciones de esa entrada.

    Para obtener más información sobre la suscripción a una entrada, consulte Suscripción a una entrada en la Guía del usuario de Novell Teaming 2.1.
