Barra de herramientas de pie de página

El contenido de la barra del pie de página cambia dependiendo de si está en un área de trabajo, una carpeta o una entrada:

Barra de herramientas de pie de página de área de trabajo

  • Enlaces permanentes de área de trabajo: permite hacer referencia al área de trabajo. Estos enlaces no cambian con el tiempo y se pueden utilizar como marcadores o se pueden enviar por correo electrónico a otros usuarios. Cópielos y péguelos según le convenga.

Barra de herramientas de pie de página de carpeta

  • Enlaces permanentes de carpeta: permite hacer referencia a la carpeta. Estos enlaces no cambian con el tiempo y se pueden utilizar como marcadores o se pueden enviar por correo electrónico a otros usuarios. Cópielos y péguelos según le convenga.

  • Direcciones de correo electrónico: si una carpeta ya se ha configurado para que tenga una URL sencilla, la barra de herramientas del pie de página muestra esta URL al hacer clic en el enlace del pie.

    Para obtener más información sobre cómo configurar una carpeta para que reciba entradas a través del correo electrónico, consulte Configuración de una carpeta para que reciba entradas mediante el correo electrónico en la Guía del usuario de Novell Teaming 2.1.

  • URL de RSS: permite acceder a la carpeta como si fuera un lector de noticias RSS. Sólo tiene que copiar la URL de RSS y pegarla en el lector RSS.

  • URL de ATOM: permite acceder a la carpeta como si fuera un lector de noticias ATOM. Sólo tiene que copiar la URL de ATOM y pegarla en el lector ATOM.

  • URL de WebDAV: si utiliza Internet Explorer 7 o posterior como navegador Web, puede utilizar la URL de WebDAV para acceder a los archivos de Novell Teaming directamente desde Microsoft Office y aplicaciones similares.

  • URL de iCal: permite sincronizar las carpetas de tareas y calendario de Teaming con la aplicación de calendario del escritorio.

    Cuando se ve una carpeta o una entrada de calendario, se puede utilizar este enlace para exportar los datos del calendario de Teaming a otro calendario que admita el formato iCal (por ejemplo, Google o Outlook). Haga clic en el enlace, guarde el archivo con la extensión .ics y, a continuación, importe el archivo en el otro calendario (consulte la ayuda del otro calendario para obtener más información).

  • Ver esta carpeta como carpeta WebDAV: permite ver la carpeta como si fuera una carpeta WebDAV.

    Para obtener más información, consulte Acceso a archivos de WebDAV desde Teaming en la Guía del usuario de Novell Teaming 2.1.

Barra de herramientas de pie de página de entrada

  • Enlace permanente de entrada: permite hacer referencia a la entrada. Estos enlaces no cambian con el tiempo y se pueden utilizar como marcadores o se pueden enviar por correo electrónico a otros usuarios. Cópielos y péguelos según le convenga.

  • iCal: (sólo entradas de calendario) permite sincronizar las carpetas de tareas y calendario de Teaming con la aplicación de calendario del escritorio.

    Cuando se ve una carpeta o una entrada de calendario, se puede utilizar este enlace para exportar los datos del calendario de Teaming a otro calendario que admita el formato iCal (por ejemplo, Google o Outlook). Haga clic en el enlace, guarde el archivo con la extensión .ics y, a continuación, importe el archivo en el otro calendario (consulte la ayuda del otro calendario para obtener más información).

  • Enviar correo electrónico: lanza un formulario de correo electrónico que permite enviar un mensaje a cualquier usuario que desee. La dirección de correo electrónico contiene la ubicación de la entrada en Teaming, así como parte del contenido de la entrada.

  • Suscribirse a esta entrada: lanza un formulario que permite suscribirse a la entrada. Cuando se suscribe a una entrada, configura Teaming para que le envíe notificaciones de correo electrónico que le informarán sobre las actualizaciones de esa entrada.

    Para obtener más información sobre la suscripción a una entrada, consulte Uso de las notificaciones por correo electrónico.

  • Exportar esta entrada: sólo para administradores de Teaming. Permite exportar la entrada. A continuación puede importarla en otro sitio de Teaming.

Para obtener más información, consulte Información legal.
