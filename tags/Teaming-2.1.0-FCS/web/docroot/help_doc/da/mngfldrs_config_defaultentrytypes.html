Standardposttyper

I dette afsnit kan du vælge de typer poster, brugere skal have lov til at oprette i denne mappe. Du kan konfigurere mappen, så yderligere posttyper tillades, eller du kan ændre den aktuelle posttype. Hvis du tilføjer yderligere posttyper, kan brugere, som sender poster til denne mappe, vælge, hvilken posttype de vil tilføje, f.eks. en opgavepost eller kalenderpost.

  1. Vælg de posttyper, der skal aktiveres på mappen.

  2. Klik på Anvend > Luk.

Hvis du aktiverer yderligere posttyper, kan brugere, som sender poster til denne mappe, klikke på Tilføj på værktøjslinjen Mappepost og derefter bruge rullelisten til at vælge, hvilken posttype de vil tilføje, alt efter de posttyper du aktiverede på mappen.

Webstedsadministratorer og mappeejere kan bruge funktionaliteten Formular- og visningsdesignere til at oprette nye posttyper (brugerdefinerede formularer), som derefter vises på denne liste. (Når mappeejere opretter nye posttyper, vises disse typer i afsnittet Lokale definitioner på denne liste).

Yderligere oplysninger om oprettelse af brugerdefinerede posttyper finder du i Designing Custom Folder Entry Forms i Novell Teaming 2.1 Advanced User Guide (Novell Teaming 2.1 Avanceret brugervejledning).

Yderligere oplysninger finder du under Juridiske meddelelser.
