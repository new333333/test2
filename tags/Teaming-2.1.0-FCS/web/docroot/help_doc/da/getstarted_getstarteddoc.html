Dokumentation

Linkene i dette afsnit viser introduktionsvejledninger i et nyt browservindue. Du kan læse disse vejledninger online i HTML- og PDF-format. PDF-formatet er også optimeret til udskrivning.

Vi vil gerne høre dine kommentarer til og forslag om disse vejledninger og den øvrige dokumentation, der følger med dette produkt. Brug venligst funktionen Brugerkommentarer nederst på hver side i onlinedokumentationen, eller gå til www.novell.com/documentation/feedback.html, og indtast dine kommentarer der.

Alle Teaming-brugere

Følgende vejledninger har interesse for alle Novell Teaming-brugere:

  • Novell Teaming 2.1 User Guide (Novell Teaming 2.0 Avanceret brugervejledning)

    Denne vejledning indeholder detaljerede oplysninger om, hvordan man gennemfører forskellige opgaver i Teaming. Den omfatter oversigtsbeskrivelser, navigationsanvisninger, tilpasningsanvisninger og grafik.

  • Novell Teaming 2.1 Advanced User Guide (Novell Teaming 2.0 Avanceret brugervejledning)

    Denne vejledning er beregnet til ejere af arbejdsområder og mapper, som er ansvarlige for at gøre deres arbejdsområder brugbare og effektive for dem, der bruger dem. Eksempler på personer, som kan være interesseret i disse oplysninger, er teamledere og projektledere.

Teaming-administratorer

Følgende vejledninger har interesse for Novell Teaming-administratorer:

  • Kablink Teaming 2.1 Administration Guide (Kablink Teaming 2.1 Administrationsvejledning)

    I denne vejledning beskrives de opgaver, som en administrator skal bruge til at opsætte, konfigurere og drive et Teaming-websted.

  • Novell Teaming 2.1 Advanced User Guide (Novell Teaming 2.0 Avanceret brugervejledning)

    Denne vejledning er beregnet til ejere af arbejdsområder og mapper, som er ansvarlige for at gøre deres arbejdsområder brugbare og effektive for dem, der bruger dem. Eksempler på personer, som kan være interesseret i disse oplysninger, er teamledere og projektledere.

  • Novell Teaming 2.1 Installation Guide (Novell Teaming 2.0 Installationsvejledning)

    I denne vejledning beskrives de opgaver, der skal gennemføres for at installere Teaming, så alle standardfunktioner virker efter hensigten. Den beskriver også, hvordan man konfigurerer en Teaming-bruger.

  • Novell Teaming 2.1 Developer Guide (Novell Teaming 2.0 Udviklingsvejledning)

    Denne vejledning er primært skrevet til udviklere, som vil udvide funktionaliteten af Teaming eller programmere andre programmer til at interagere med det.

Yderligere oplysninger finder du under Juridiske meddelelser.
