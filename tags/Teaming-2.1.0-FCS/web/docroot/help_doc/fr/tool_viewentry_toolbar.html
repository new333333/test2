Barre d'outils Entrée

La barre d'outils Entrée propose uniquement les outils que vous êtes autorisé à utiliser. La barre d'outils apparaît à la fois pour les entrées et les commentaires ; elle peut comprendre les boutons et menus suivants :

  • Commentaire : ajoute un commentaire à l'entrée. Vous pouvez rédiger un commentaire relatif à un commentaire (il est alors question de commentaires imbriqués).

  • Modifier : permet de modifier l'entrée ou d'ajouter une autre pièce jointe. (Vous pouvez également utiliser le lien Modifier dans la section inférieure droite de l'entrée, juste au-dessus de la section Pièces jointes.)

  • Verrouiller : empêche d'autres utilisateurs de modifier l'entrée pendant que vous l'éditez. Il est judicieux de verrouiller une entrée lorsque vous y apportez des changements importants. Une fois les modifications terminées, cliquez sur Déverrouiller.

  • Déplacer : permet de déplacer l'entrée vers un autre dossier.

  • Copier : permet de copier l'entrée et de l'enregistrer à un autre emplacement.

  • Supprimer : supprime l'entrée du dossier.

  • Workflow : lorsqu'un workflow est activé sur un dossier, cet outil permet de démarrer le workflow s'il n'est pas encore lancé sur l'entrée ou d'arrêter le workflow s'il est en cours sur l'entrée.

  • Définir la page d'accueil du Wiki : disponible uniquement pour les entrées Wiki. Cette option permet de configurer Teaming pour afficher n'importe quelle page Wiki comme page d'accueil Wiki. Il vous suffit d'accéder à la page que vous voulez définir comme page d'accueil Wiki, puis de cliquer sur cette option.

  • Rapports : permet de générer les rapports suivants :

    • Rapport d'activité : affiche dans Microsoft Excel un fichier dont les valeurs sont séparées par des virgules et qui fournit des informations sur les utilisateurs et leurs activités liées à cette entrée.

    • Historique des modifications : ouvre une fenêtre Historique des modifications dans laquelle vous pouvez comparer différentes versions de l'entrée en voyant ce qui a été ajouté ou supprimé.

    • Historique du workflow : ouvre une fenêtre Historique du workflow dans laquelle vous pouvez afficher l'historique du workflow, y compris son nom, la date de création de l'entrée et son auteur, la date à laquelle le workflow est passé à un état donné, qui a modifié l'entrée lorsque le workflow a atteint cet état, etc.

  • Partager cette entrée : permet de partager une entrée avec d'autres utilisateurs Teaming. Lorsque vous partagez une entrée, un message de notification est envoyé à l'utilisateur que vous avez choisi pour le partage de l'entrée. Par ailleurs, dans leur espace de travail privé, les utilisateurs peuvent sélectionner l'onglet Micro-Blogues et éléments partagés et, dans la colonne Éléments partagés, Teaming affiche tous les éléments que des personnes ont partagés avec eux.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
