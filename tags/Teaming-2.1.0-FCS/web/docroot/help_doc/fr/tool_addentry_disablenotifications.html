Modification des paramètres de notifications par messagerie électronique

Sélectionnez cette option si vous souhaitez que vos sélections remplacent tout paramètre de notification défini par l'administrateur ou le propriétaire du dossier.

Si vous ne souhaitez recevoir aucune notification par message électronique, cochez cette case et assurez-vous que les champs au-dessus ne renseignent aucune adresse électronique ni aucun numéro de téléphone.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
