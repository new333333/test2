Workflows autorisés

Utilisez cette section pour permettre à un workflow d'être utilisé sur toutes les entrées d'un dossier spécifique. Lorsque l'utilisateur affiche une entrée dans un dossier où vous autorisez un workflow, il peut lancer n'importe quel workflow sélectionné dans cette section en cliquant sur Workflow > Démarrer le workflow : nom_workflow dans la barre d'outils Entrée.

Cette opération démarre le processus de workflow sur cette entrée.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
