Affichage des événements dans un agenda

La liste déroulante Afficher permet de choisir les types d'événements d'agenda que vous souhaitez afficher et la façon dont vous souhaitez qu'ils apparaissent. À moins que vous ne soyez dans l'espace de travail personnel d'un utilisateur de Teaming, vous ne pouvez consulter que les événements d'agenda situés dans le dossier actuel.

  • Événements assignés : disponible uniquement dans les espaces de travail utilisateur. Affiche tous les événements d'agenda assignés au propriétaire de l'espace de travail, quel que soit leur emplacement dans Teaming.

  • Dossier - Événements : affiche tous les événements d'agenda contenus dans le dossier actuel selon les jours pour lesquels ils sont planifiés.

  • Dossier - Tout par date de création : affiche tous les événements d'agenda selon les jours où ils ont été créés, plutôt que ceux pour lesquels ils ont été planifiés.

  • Dossier - Tout par activité : affiche tous les événements d'agenda selon les jours où ils ont été modifiés pour la dernière fois, plutôt que ceux pour lesquels ils ont été planifiés.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
