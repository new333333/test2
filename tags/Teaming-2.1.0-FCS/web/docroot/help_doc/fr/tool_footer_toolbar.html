Barre d'outils Pied de page

Le contenu de la barre d'outils Pied de page change selon que vous soyez dans un espace de travail, un dossier ou une entrée :

Barre d'outils Pied de page d'un espace de travail

  • Liens permanents d'espaces de travail : permet de faire référence à l'espace de travail. Ces liens restent inchangés au fil du temps et peuvent être utilisés comme signets ou envoyés par courrier électronique à d'autres utilisateurs. Vous pouvez les copier-coller selon vos besoins.

Barre d'outils Pied de page d'un dossier

  • Liens permanents de dossiers : permet de faire référence au dossier. Ces liens restent inchangés au fil du temps et peuvent être utilisés comme signets ou envoyés par courrier électronique à d'autres utilisateurs. Vous pouvez les copier-coller selon vos besoins.

  • Adresses électroniques : si un dossier a déjà été configuré pour disposer d'une URL conviviale, la barre d'outils Pied de page affiche cette dernière lorsque vous cliquez sur le lien du pied de page.

    Pour plus d'informations sur la configuration d'un dossier pour la réception d'entrées par le biais de messages électroniques, reportez-vous à la section Configuration d'un dossier pour la réception d'entrées par message électronique du Guide de l'utilisateur de Novell Teaming 2.0.

  • URL RSS : permet d'accéder au dossier en tant que flux RSS. Il suffit de copier l'URL RSS et de la coller dans votre lecteur de flux RSS.

  • URL ATOM : permet d'accéder au dossier en tant que flux ATOM. Il suffit de copier l'URL ATOM et de la coller dans votre lecteur de flux ATOM.

  • URL WebDAV : si vous utilisez Internet Explorer 7 ou une version ultérieure comme navigateur Web, vous pouvez utiliser l'URL WebDAV pour accéder aux fichiers Novell Teaming directement depuis Microsoft Office et d'autres applications similaires.

  • URL iCal : permet de synchroniser vos dossiers d'agenda et de tâches Teaming avec l'application d'agenda de votre bureau.

    Lorsque vous affichez un dossier ou une entrée d'agenda, vous pouvez utiliser ce lien pour exporter les données d'agenda Teaming afin de les inclure dans un autre agenda qui prend en charge le format iCal (par exemple, Google et Outlook). Cliquez sur le lien, enregistrez le fichier avec l'extension .ics, puis importez le fichier dans votre autre agenda (reportez-vous à l'aide de ce dernier pour plus d'informations).

  • Afficher ce dossier sous forme de dossier WebDAV : permet d'afficher le dossier en tant que dossier WebDAV.

    Pour plus d'informations, reportez-vous à la section Accès aux fichiers WebDAV à partir de Teaming du Guide de l'utilisateur de Novell Teaming 2.1.

Barre d'outils Pied de page d'une entrée

  • Lien permanent de l'entrée : permet de faire référence à l'entrée. Ces liens restent inchangés au fil du temps et peuvent être utilisés comme signets ou envoyés par courrier électronique à d'autres utilisateurs. Vous pouvez les copier-coller selon vos besoins.

  • iCal : (entrées d'agenda uniquement) permet de synchroniser vos dossiers d'agenda et de tâches Teaming avec l'application d'agenda de votre bureau.

    Lorsque vous affichez un dossier ou une entrée d'agenda, vous pouvez utiliser ce lien pour exporter les données d'agenda Teaming afin de les inclure dans un autre agenda qui prend en charge le format iCal (par exemple, Google et Outlook). Cliquez sur le lien, enregistrez le fichier avec l'extension .ics, puis importez le fichier dans votre autre agenda (reportez-vous à l'aide de ce dernier pour plus d'informations).

  • Envoyer un message électronique : ouvre un formulaire qui permet d'envoyer un message électronique à la personne de votre choix. Ce message électronique contient l'emplacement de l'entrée dans Teaming et une partie du contenu de l'entrée.

  • S'abonner à cette entrée : ouvre un formulaire qui permet de vous abonner à l'entrée. Lorsque vous vous abonnez à une entrée, vous configurez Teaming pour qu'il vous envoie des notifications par message électronique vous informant des mises à jour de cette entrée.

    Pour plus d'informations sur l'abonnement à une entrée, reportez-vous à la section Utilisation de notifications par message électronique.

  • Exporter cette entrée : administrateurs Teaming uniquement. Permet d'exporter l'entrée. Vous pouvez ensuite l'importer vers n'importe quel autre site Teaming.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
