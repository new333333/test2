Documentation

The links in this section display getting started manuals in a new browser window. You can read these manuals online in HTML and PDF format. The PDF format is also optimized for printing.

We want to hear your comments and suggestions about these manuals and the other documentation included with this product. Please use the User Comments feature at the bottom of each page of the online documentation, or go to www.novell.com/documentation/feedback.html and enter your comments there.

All Teaming Users

The following guides are of interest to all Kablink Teaming users:

  • Kablink Teaming 2.1 User Guide

    This guide provides detailed information that describes how to accomplish various tasks in Teaming. It includes overview descriptions, navigation instructions, customization instructions, and graphics.

  • Kablink Teaming 2.1 Advanced User Guide

    This guide is intended for workspace and folder owners who are responsible for making their workspaces usable and effective for those that consume them. Examples of people who might be interested in this information are team leaders and project managers.

Teaming Administrators

The following guides are of interest to Kablink Teaming administrators:

  • Kablink Teaming 2.1 Administration Guide

    This guide describes the tasks required for an administer to set up, configure, and operate a Teaming site.

  • Kablink Teaming 2.1 User Guide

    This guide is intended for workspace and folder owners who are responsible for making their workspaces usable and effective for those that consume them. Examples of people who might be interested in this information are team leaders and project managers.

  • Kablink Teaming 2.1 Installation Guide

    This guide describes the tasks required to install Teaming so that all default features operate as intended. It also describes how to set up a Teaming user.

  • Novell Teaming 2.1 Developer Guide

    This guide is written primarily for developers who want to extend the functionality of Teaming or program other applications to interact with it.
