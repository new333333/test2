显示条目和联系他人

Novell Teaming 中的各种工具（例如，文件夹列表和搜索结果）显示了条目信息摘要。此摘要中的信息使您能够显示条目或联系条目的创建者，如以下部分所述。

显示条目

要显示条目，只需单击条目的标题。

您可以配置 Teaming 显示条目的方式。打开条目时，Teaming 可以将条目作为叠加图层显示，或者在新页面或弹出窗口中显示此条目。

更改条目在 Teaming 中的显示方式：

  1. 在主侧栏的个人自选设置部分中，单击条目视图样式。

  2. 选择查看条目的所需方式。

联系他人

条目摘要包括一个存在状态图标，位于作者的姓名旁边。此“存在状态”图标可能具有以下状态：

  • 绿色： 此人处于联机和活动状态。

  • 黄色： 此人最近有过活动，但目前处于离开状态。

  • 灰色： 此人未登录到 Novell Conferencing 客户端。

  • 白色： 此人的状态未知。

单击存在状态图标并选择某菜单项可联系此人。您可以与此人开始即时会议或安排会议、呼叫此人、将此人添加到剪贴板或查看此人的微博客。

有关更多信息，请参见 法律声明。
