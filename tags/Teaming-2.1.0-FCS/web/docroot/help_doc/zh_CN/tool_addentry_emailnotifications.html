使用电子邮件通知

可以配置 Novell Teaming 以向您发送有关文件夹及其子文件夹或个别条目的电子邮件通知（这称为“订阅”文件夹或条目）。文件夹拥有者也可以配置 Teaming 以向您发送电子邮件通知。但是，由于用户的个人订阅设置将覆盖文件夹拥有者的设置，因此您可以使用此表单来覆盖任何更高级别的配置。

选择通知类型和发送目标

通过此表单，您可以选择希望接收的通知类型以及希望将通知发送到的电子邮件地址或电话号码目标。

选择希望将通知发送到的电子邮件地址或电话号码。可以在任何提供的代表电子邮件通知类型的部分中选择这些发送目标：

  • 摘要

  • 个人讯息

  • 不带附件的个人讯息

  • 适合文本讯息交换的讯息

有关各种通知类型的更多信息，请参见确定希望接收的电子邮件通知的类型。

可以为每个通知类型选择多个电子邮件或电话号码发送目标，只需按住 Ctrl 并选择电子邮件地址或电话号码即可。

添加和修改发送目标

您可以在个人资料中添加和修改电子邮件及电话号码联系信息。访问个人资料页：

  1. 单击我的工作空间 > 修改个人资料。

  2. 在电子邮件地址、移动电子邮件地址和文本讯息交换电子邮件地址字段中指定相应的发送目标信息。

  3. 单击确定。

现在，在选择通知类型和发送目标时，将显示新的或修改后的联系信息。

确定希望接收的电子邮件通知的类型

可以接收不同类型的电子邮件通知：

  • 摘要： Teaming 发送一条讯息，其中概述了文件夹及其子文件夹中的所有活动（仅对文件夹可用）。Teaming 按由站点管理员建立的日程表发送摘要通知。

  • 个人讯息： Teaming 对于每个新的或更改过的条目都发送一条讯息，且讯息包括附加文件。

  • 不带附件的个人讯息： Teaming 对于每个新的或更改过的条目都发送一条讯息。电子邮件讯息不包括可能附加到新的或更改过的条目的文件。Teaming 每五分钟发送一次所有类型的个人通知。

  • 适合文本讯息交换的讯息： Teaming 会发送一条适合作为文本讯息查看的讯息。讯息内容仅包括新的或更改过的条目的链接标题。Teaming 每五分钟发送一次文本讯息通知。

覆盖预设置的通知

当在此表单中选择设置并选择使用这些选择覆盖预设置的通知时，您的选择将覆盖管理员或文件夹拥有者所做的选择。

有关更多信息，请参见 法律声明。
