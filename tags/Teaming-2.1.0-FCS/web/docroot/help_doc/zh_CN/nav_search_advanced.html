使用高级搜索

通过高级搜索，您可以根据以下任意或所有准则缩小搜索范围：

  • 文本： 提供要搜索的特定文本。

  • 位置： 通过此部分，您可以将搜索限制在某个特定位置。单击主工作空间旁边的加号 (+) 展开树，然后导航至要搜索的工作空间或文件夹并将其选定。

    如果希望 Novell Teaming 搜索您在工作空间树中所选位置下的每个位置，请选择搜索所有子位置。

  • Case Sensitive: 选中此复选框可使得您在文本字段中输入的文本区分大小写。例如，“tree”与“Tree”和“TREE”等都不匹配。

  • 搜索删除的项目： 搜索已删除的项目。搜索仅返回已删除的项目。

  • 呈现选项： 选择希望在每页“搜索结果”上显示的结果数，以及每个结果显示的字数。

  • 作者： 基于作者搜索项目。

  • 标记： 搜索已标记（无论是社区标记还是个人标记）的项目。有关标记的更多信息，请参见标记。

  • 工作流程： 搜索应用了特定工作流程的项目。

  • 条目属性： 选择要搜索的条目的类型，如博客条目、日历条目或评论。

  • 上次活动（以天为单位）： 选择此准则可搜索过去 1、3、7、30 或 90 天内活动的项目。

  • 创建日期： 根据项目的创建日期进行搜索。在提供的字段中，指定所需间隔的开始日期和结束日期。

  • 修改日期： 根据项目的上次修改日期进行搜索。在提供的字段中，指定所需间隔的开始日期和结束日期。

  • 项目类型： 选择要搜索的项目的类型。

以下部分提供了有关使用高级搜索的更多信息：

添加准则

如果您的搜索需要多个准则，请单击添加其他，Novell Teaming 便会提供其他字段供您使用。

单击准则旁边的 X 图标可将其从搜索中去除。

访问高级搜索

您可以从导航栏或搜索结果页访问高级搜索：

导航栏

  1. 单击高级。（位于搜索字段上方。）

搜索结果页

  1. 单击高级。

  2. 如果要提供除初始准则外的更多准则，请单击搜索准则摘要部分中的修改“高级搜索”查询。

有关更多信息，请参见 法律声明。
