工作空间视图

选择能最好地呈现您的工作空间的视图。

  • 讨论工作空间： 如果希望创建一个可储存多个讨论论坛的工作空间，请选择此视图。

    有关讨论工作空间的更多信息，请参见 Novell Teaming 2.1 User Guide 中的管理讨论工作空间。

  • 登录页： 如果希望在团队成员进入工作空间时即对他们显示重要信息，请选择此视图。

    有关登录页的更多信息，请参见 Novell Teaming 2.1 User Guide 中的创建登录页。

  • 项目管理工作空间： 如果希望根据任务完成情况来跟踪项目任务和大事记，请选择此视图。

    有关项目管理工作空间的更多信息，请参见 Novell Teaming 2.1 User Guide 中的管理项目管理工作空间。

  • 团队工作空间： 如果您的团队工作空间实际上是一种组织结构，且不想包含团队工作空间作为其子位置，请选择此视图。

    有关团队工作空间的更多信息，请参见 Novell Teaming 2.1 User Guide 中的管理团队工作空间。

  • 团队工作空间根目录： 如果希望在团队工作空间中显示添加团队工作空间按钮，请选择此视图。如果希望创建其他团队工作空间作为此工作空间的子位置，则这将是所需的选择。

    有关团队工作空间根目录工作空间的更多信息，请参见 Novell Teaming 2.1 Advanced User Guide 中的创建和管理团队工作空间根目录工作空间。

  • 工作空间： 这是基本工作空间。您可以选择希望工作空间包含的文件夹。

    有关基本工作空间的更多信息，请参见 Novell Teaming 2.1 Advanced User Guide 中的创建和管理基本工作空间。

如果是个人工作空间，则默认视图是用户工作空间且无法更改。

有关更多信息，请参见 法律声明。
