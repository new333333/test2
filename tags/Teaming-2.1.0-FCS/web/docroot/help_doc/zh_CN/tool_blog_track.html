博客跟踪

博客跟踪工具栏为以下操作提供了快速方式：

  • 添加评论： 使您能够对博客条目发表评论。在发表评论前，最好先单击查看评论查看以前发表的评论。向已有评论的博客添加评论时，您的评论将自动添加为嵌套评论列表中的最后一条评论。

  • 查看评论： 使您能够查看某个特定条目的所有评论，而无需打开此条目。

  • 将条目发送给朋友： 使您能够通过电子邮件将条目内容发送给具有此条目的相应查看访问权限的任何人。

  • 订阅： 起动一张表单，使您可以订阅条目。订阅条目时，即会配置 Teaming 以向您发送有关此条目更新信息的电子邮件通知。

    有关订阅条目的更多信息，请参见 Novell Teaming 2.1 User Guide 中的订阅条目。
