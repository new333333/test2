发送电子邮件通知

作为文件夹拥有者，您可以配置 Novell Teaming 以向其他用户发送电子邮件通知，通知他们此文件夹及其子文件夹中的任何活动。但是，用户的个人设置会覆盖或取消文件夹拥有者建立的通知。

  1. 在选择电子邮件通知的类型部分中，选择要发送的电子邮件通知的类型。

    有关各种类型的电子邮件通知的信息，请参见确定希望接收的电子邮件通知的类型。

  2. 指定希望其收到电子邮件通知的用户。

    可以指定特定的电子邮件地址、用户、组或向所有团队成员发送通知。

  3. 单击应用 > 关闭。

有关电子邮件通知的更多信息，请参见使用电子邮件通知。

有关更多信息，请参见 法律声明。
