编辑器覆盖

在 Novell Teaming 中编辑附件时，会使用 Teaming 系统的默认应用程序编辑附件。但通过此表，您可以覆盖 Teaming 中用于编辑带特定文件扩展名的附件的默认应用程序。

可以为单个或多个文件扩展名类型覆盖默认的编辑应用程序。

为单个文件扩展名覆盖默认的编辑应用程序

为单个文件扩展名类型覆盖默认应用程序：

  1. 单击添加。

  2. 从下拉列表中选择要对其覆盖默认编辑应用程序的文件扩展名类型。

  3. 在应用程序字段中，指定要用于编辑带有在步骤 2中所指定文件扩展名的附件的应用程序名称。

  4. 单击确定。

为所有文件扩展名覆盖默认的编辑应用程序

覆盖所有文件扩展名类型：

  1. 单击使用。

  2. 从下拉列表中选择要在 Teaming 中用于编辑所有文件扩展名类型的附件的应用程序。

  3. 单击确定。
