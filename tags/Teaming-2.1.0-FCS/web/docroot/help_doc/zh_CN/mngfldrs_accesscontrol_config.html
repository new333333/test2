配置访问控制

使用此页可更改此工作空间或文件夹的访问控制设置。

默认情况下，工作空间和文件夹继承其父代的访问控制设置。仅当您需要不同的访问设置时才使用此页面。

必须先确保文件夹或工作空间未从父目录继承用户和角色，才可更改其访问控制表。有关如何操作的信息，请参见继承角色成员资格。

除了此部分及以下子部分中所含的访问控制信息外，要查看更多详细信息，请参见 Novell Teaming 2.1 Advanced User Guide 中的控制用户访问。

以下部分说明了如何修改访问控制表以适应您的访问需要。

添加用户和组

  1. 单击表左边列中的添加组或添加用户链接。

  2. 在字段中输入用户名称或组的开头几个字母，然后在下拉列表中单击链接名称。

    Novell Teaming 会在表中为此用户或组添加一行。

查看角色任务

在访问控制表中，角色名称用作列标题。

  1. 单击列标题（参与者、团队成员等）可查看与此角色关联的权限列表。

更改用户或组的角色

一个角色就是一组特权；当将某人指派给某角色时，他们就自动具有与此角色关联的所有权限。

  1. 通过选择和取消选择复选框可以调整哪些人可以担当哪些角色。

  2. 单击确定。

在其他位置中调整访问控制

  1. 转到此页顶部的工作空间树，导航至要调整访问控制的位置，然后在树中单击此位置的链接。

    此时会显示此位置的访问控制页。

有关更多信息，请参见 法律声明。
