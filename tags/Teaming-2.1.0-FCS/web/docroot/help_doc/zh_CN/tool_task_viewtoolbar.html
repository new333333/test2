视图工具栏

“视图”工具栏使您能够选择要在任务列表中显示的任务类型。除非在 Teaming 用户的个人工作空间中，否则只能查看那些位于当前文件夹中的日历事件任务。

  • 已关闭： 显示状态为已关闭的任务。

  • 今天： 显示今天到期的任务。

  • 周： 显示将在未来七天内到期的任务。

  • 月： 显示将在未来 30 天内到期的任务。

  • 所有活动的： 显示当前活动的任务。

  • 所有条目： 显示所有任务。

  • 指派的任务： 仅在用户工作空间中可用。显示指派给工作空间拥有者的所有任务，无论这些任务在 Teaming 中的位置。

  • 从文件夹： 仅在用户工作空间中可用。显示在当前文件夹中的所有任务。

有关更多信息，请参见 法律声明。
