文件夹管理工具

单击文件夹工具栏中的管理时，Novell Teaming 将仅显示您有权使用的管理和配置工具。站点管理员或者此工作空间或文件夹的拥有者可以使用这些工具和授予对工具的访问权限。

如果您具有不受限制的访问权限，则会看到以下选项：

  • 访问控制： 使您能够控制哪些人可以查看和参与此位置。

  • 活动报告： 使您能够查看工作空间中的活动。

  • 配置： 使您能够配置文件夹，包括设置默认视图、默认条目类型、允许的视图以及文件夹 URL。

  • 复制此文件夹： 如果您具有相应权限，则能够将当前文件夹复制到工作空间树中所需的任何目标位置。请参见工作空间树导航。

  • 删除此文件夹： 删除当前文件夹。

  • 电子邮件设置： 使您能够配置文件夹以向其他 Teaming 用户自动发送电子邮件通知。

  • 导出/导入： 仅限 Teaming 管理员。使您能够导出当前文件夹，或将文件夹或工作空间导入当前文件夹。

    有关在 Teaming 中导出和导入工作空间和文件夹的更多信息，请参见 Kablink Teaming 2.1 Administration Guide 中的导出/导入工作空间、文件夹和条目。

  • 表单和视图设计程序： 使您能够创建自定义条目表单和自定义工作流程，可供以后应用于文件夹及其子文件夹。

  • 修改此文件夹： 使您能够修改文件夹，包括文件夹名称、说明、固定设计和图标。

  • 移动此文件夹： 如果您具有相应权限，则能够将当前文件夹移动到工作空间树中所需的任何目标位置。请参见工作空间树导航。

  • 添加新文件夹： 使您能够向工作空间添加新文件夹。

有关更多信息，请参见 法律声明。
