Gebeurtenissen weergeven in een agenda

Via de keuzelijst Weergeven kunt u selecteren welke gebeurtenissen moeten worden weergegeven in de agenda en hoe u ze wilt weergeven. U kunt alleen de agendagebeurtenissen weergeven die zich in de huidige map bevinden, tenzij u zich in de persoonlijke workspace van een Teaming-gebruiker bevindt.

  • Toegewezen gebeurtenissen: Alleen beschikbaar in gebruikersworkspaces. Geeft alle agendagebeurtenissen weer die zijn toegewezen aan de eigenaar van de workspace, ongeacht of de gebeurtenissen zich binnen Teaming bevinden.

  • Uit map - Gebeurtenissen: Geeft alle agendagebeurtenissen uit de huidige map weer op de dagen dat de gebeurtenissen gepland zijn.

  • Uit map - Alles per maakdatum: Geeft alle agendagebeurtenissen weer op de dagen dat ze gemaakt zijn in plaats van de dagen waarop de gebeurtenissen gepland zijn.

  • Uit map - Alles per activiteit: Geeft alle agendagebeurtenissen weer op de dagen dat ze het laatst zijn gewijzigd, in plaats van de dagen waarop de gebeurtenissen zijn gepland.

Voor meer informatie zie: Juridische kennisgevingen.
