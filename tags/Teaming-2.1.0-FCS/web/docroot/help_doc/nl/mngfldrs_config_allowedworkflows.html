Toegestane workflows

Via deze sectie kunt u instellen dat een workflow wordt toegepast op alle vermeldingen in een specifieke map. Als gebruikers een vermelding bekijken in een map waarvoor u een workflow toestaat, kunnen zij elke workflow starten die in deze sectie is geselecteerd door in de vermeldingsbalk te klikken op Workflow > Start workflow: wf_name.

Dit start het workflowproces op deze vermelding.

Voor meer informatie zie: Juridische kennisgevingen.
