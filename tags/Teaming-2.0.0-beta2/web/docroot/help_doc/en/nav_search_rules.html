Search String Rules

The rules for search strings are as follows:

  1. Case Sensitivity: If you specify all lowercase letters, then Kablink Teaming returns matches in any case ("tree" matches "tree", "Tree", "TReE", and so on). If you specify one or more uppercase letters, Kablink Teaming returns only exact matches ("Tree" does not match "tree").

  2. More Than One Word: Kablink Teaming returns items that contain all words in the search string.

  3. Quoted Phrase: Kablink Teaming returns items that match all words in order. Case sensitivity rules apply as described previously.

  4. Exclusions: To exclude a word (do not return matches for items that contain this word), place a minus sign (-) in front of it. For example, "-tree" excludes all items containing "tree", "Tree", "TREE", and so on.
