Novell Vibe 3.4

1.0  Introdução

     Os requisitos completos do sistema e as instruções detalhadas de instalação
     do software são fornecidos para o administrador do Novell Vibe no "Vibe 3.4
     Installation Guide (Guia de Instalação do Vibe 3.4)." A configuração do site
     do Vibe, bem como as instruções de administração e manutenção do site em
     funcionamento, estão disponíveis no "Vibe 3.4 Administration Guide" (Guia de
     Administração do Vibe 3.4).

     Depois que seu site do Vibe estiver configurado, os usuários do Vibe poderão
     obter instruções no "Guia do Usuário do Vibe 3.4" e no "Vibe 3.4 Advanced
     User Guide" (Guia do Usuário Avançado do Vibe 3.4). (Uma "Inicialização
     Rápida do Vibe 3.4" é fornecida nos idiomas em que o "Guia do Usuário do
     Vibe 3.4" não está disponível.)

     Você pode exibir a ajuda online do Vibe ao efetuar login no seu site do
     Vibe.
