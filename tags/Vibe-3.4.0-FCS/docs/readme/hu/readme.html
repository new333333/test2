Novell Vibe 3.4

1.0 Első lépések

A rendszerkövetelmények teljes körű ismertetését és a részletes szoftvertelepítési útmutatást a Novell Vibe rendszergazdája számára a Vibe 3.4 telepítési kézikönyv tartalmazza. A Vibe webhely beállításával, valamint a folyamatos webhely-adminisztrációval és -karbantartással kapcsolatos útmutatók a Vibe 3.4 adminisztrátori kézikönyvben találhatók.

A Vibe webhely telepítése után a Vibe felhasználói a Vibe 3.4 felhasználói kézikönyv és a Vibe 3.4 haladó felhasználói kézikönyv tartalmában találhatnak útmutatást. (Vibe 3.4 Gyorskalauz azokon a nyelveken jár a rendszerhez, amelyeken a Vibe 3.4 felhasználói kézikönyv nem érhető el.)

A Vibe webhelyre való bejelentkezés után megjelenítheti a Vibe rendszer online súgóját.

2.0 Ismert problémák

Az ehhez a kiadáshoz kapcsolódó ismert problémák naprakész listáját megtalálja a Novell Vibe 3.4 termék teljes Readme ismertetőjében, amely a Novell Vibe 3.4 dokumentációs webhelyén található.

3.0 Dokumentáció

A Novell Vibe 3.4 termék összes dokumentációja elérhető a Novell Vibe 3.4 dokumentációs webhelyén:

  • A Vibe 3.4 termék teljes Readme ismertetője

  • Telepítési kézikönyv

  • Adminisztrátori kézikönyv

  • Felhasználói kézikönyv

  • Haladó felhasználói kézikönyv

  • Fejlesztői kézikönyv

  • Vibe gyorskalauz

  • Vibe Desktop for Windows – bevezetés

  • Vibe Desktop for Mac – bevezetés

  • Vibe Desktop – gyakori kérdések

  • Vibe beépülő modul gyorskalauz

A Novell Vibe termék dokumentációja mellett további forrásokból is elérhető tájékoztatás a Vibe termékkel kapcsolatban.

  • A Novell támogatási tudásbázisa

  • Novell támogatási fórum

  • Novell Cool Solutions

  • Novell Resource Library

A Vibe termékkel kapcsolatban további tájékoztatást a Kablink Open Collaboration Project (Kablink nyílt együttműködési projekt) webhelye tartalmaz.
