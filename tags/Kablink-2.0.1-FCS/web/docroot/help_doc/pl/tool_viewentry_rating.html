Ocenianie wpisów

Podobnie jak wiele systemów społecznościowych i zarządzania wiedzą, witryna Kablink Teaming umożliwia użytkownikom określenie, które wpisy są najbardziej popularne. System Teaming zawiera narzędzie oceniania w pięciogwiazdkowej skali.

Wyświetlanie ocen wpisów

Podczas przeglądania wpisu wyświetlane są żółte gwiazdki, które sygnalizują średnią ocenę wpisu. Jedna gwiazdka oznacza najniższą ocenę, a pięć gwiazdek najwyższą. Jeśli wszystkie gwiazdki są szare, wpis nie został oceniony.

Wysyłanie ocen wpisu

Aby wysłać swoją ocenę, należy kliknąć gwiazdkę. (Na przykład, aby wysłać ocenę trzygwiazdkową, należy kliknąć trzecią gwiazdkę od lewej.) Można zmieniać ocenę tak często, jak tylko ma się na to ochotę, ale do obliczania średniej używane są tylko najnowsze oceny.

Aby uzyskać więcej informacji, patrz sekcja Informacje prawne.
