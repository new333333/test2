Dziedziczenie ustawień definicji

Ta część strony pozwala określić, czy ustawienia mają być dziedziczone z folderu nadrzędnego.

  1. Aby dziedziczyć ustawienia definicji, należy kliknąć opcje Tak > Zastosuj

    lub

    kliknąć opcje Nie > Zastosuj, aby nie dziedziczyć ustawień definicji.

Aby uzyskać więcej informacji, patrz sekcja Informacje prawne.
