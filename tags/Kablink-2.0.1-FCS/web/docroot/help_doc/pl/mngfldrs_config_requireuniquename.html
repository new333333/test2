Wymagaj niepowtarzalnych nazw plików

Jeśli to pole wyboru zostanie zaznaczone, niektóre funkcje produktu będą traktować ten folder jako folder plików. Na przykład, aplet do wysyłania plików tworzy nowe wpisy dla niepowtarzalnych nazw plików i nowe wersje załączników do plików już istniejących w tym folderze.

Aby uzyskać więcej informacji, patrz sekcja Informacje prawne.
