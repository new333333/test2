Modify Profile

Use this tool to change contact information, such as password, name, phone number, e-mail address, and the Kablink Conferencing user name.

You can also upload pictures, rearrange them by using the up-arrow and down-arrow icons, and delete them. If you upload more than one picture, Teaming displays the first picture in the list in the Photo section, and it displays smaller versions of other pictures in a bar below the contact information. When you mouse over one of the smaller pictures, Teaming displays a larger version in the Photo section.
