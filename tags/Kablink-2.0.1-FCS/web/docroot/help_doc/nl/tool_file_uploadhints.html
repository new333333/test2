Bestandsupload hints

Als u problemen heeft met het uploaden van bestanden omdat het bestand te groot is kunt u dit misschien oplossen door de hoeveelheid geheugen te vergroten voor gebruik door Java applets. Dit proces is beschreven in de volgende secties.

De standaard hoeveelheid geheugen verhogen onder Windows

  1. Ga naar het Configuratiescherm (klik Start > Instellingen > Configuratiescherm).

  2. Dubbelklik op Java.

    Het Java configuratiescherm wordt geopend.

  3. Selecteer het Java tabblad.

  4. In de Java applet runtime instellingen sectie klikt u op Weergeven.

  5. Voor elke Java runtime versie voegt u -Xmx256m toe als Java runtime parameter.

  6. Klik op OK.

    Nadat u het standaard geheugen voor Java applets heeft verhoogd dient u alle browsers af te sluiten en opnieuw te starten voor de aanpassingen effect hebben.

Deze instelling zorgt er voor dat de applet die gebruikt wordt voor het uploaden van bestanden naar Teaming mappen wordt gestart met 256 MB geheugen. Als de applet nog steeds aangeeft dat er onvoldoende geheugen beschikbaar is dient u deze instelling misschien te verlagen.

De standaard hoeveelheid geheugen verhogen onder Linux

  1. Lokaliseer de JRE of JDK die gebruikt wordt door de browser plugin.

    De lokatie van de JRE of JDK verschilt afhankelijk van uw systeem.

  2. In de bin map van de JRE of JDK start u het ControlPanel bestand.

    Het configuratiescherm wordt geopend.

  3. Selecteer het Java tabblad.

  4. In de Java applet runtime instellingen sectie klikt u op Weergeven.

  5. Voor elke Java runtime versie voegt u -Xmx256m toe als Java runtime parameter.

  6. Klik op OK.

    Nadat u het standaard geheugen voor Java applets heeft verhoogd dient u alle browsers af te sluiten en opnieuw te starten voor de aanpassingen effect hebben.

Deze instelling zorgt er voor dat de applet die gebruikt wordt voor het uploaden van bestanden naar Teaming mappen wordt gestart met 256 MB geheugen. Als de applet nog steeds aangeeft dat er onvoldoende geheugen beschikbaar is dient u deze instelling misschien te verlagen.
