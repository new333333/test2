Panelen Favoriter

När du klickar på ikonen Favoriter visas panelen Favoriter i Kablink Teaming. Du kan lägga till dina favoritarbetsytor och favoritmappar på den här panelen så att du snabbt kan nå dem.

Följande avsnitt innehåller information om hur du ändrar panelen Favoriter.

Lägga till platser till panelen Favoriter

  1. Gå till en arbetsyta eller mapp med hjälp av navigeringsfältet eller trädet med arbetsytor.

  2. När du har kommit till en av dina favoritplatser klickar du på Favoriter.

  3. Klicka på Lägg till den här platsen.

Redigera panelen Favoriter

Panelen Favoriter innehåller verktyg som du kan använda för att ordna länkarna till dina favoritplatser.

  1. Klicka på Favoriter > Redigera favoriter.

  2. Markera det objekt som du vill ändra.

  3. Klicka på Ned, Upp eller Ta bort.

    Knapparna Ned och Upp flyttar upp eller ned länken i listan Favoriter.

  4. Klicka på OK.
