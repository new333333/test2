Min arbetsyta

Klicka på ikonen Min arbetsyta om du vill visa din personliga arbetsyta i Kablink Teaming. Din personliga arbetsyta kan bland annat innehålla:

  • din bild och andra favoritfoton

  • kontaktinformation

  • din blogg

  • din personliga kalender

  • en gästbok som andra kan signera.

Ett bra sätt att komma igång med Teaming är att bekanta dig med din personliga arbetsyta. Du kan till exempel skriva en introduktion för arbetsytan, överföra några bilder och lägga till en bloggpost (journalpost). De som besöker din personliga arbetsyta kan se din kontaktinformation, visa dina bloggposter och kommentera dem.

Mer information finns i Juridisk information.
