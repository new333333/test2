［日付］ツールバー

Kablink Teamingの［日付］ツールバーでは、カレンダフォルダ表示をさまざまな方法でカスタマイズできます。

  • 終日の時間を表示するか、それとも通常の業務時間のみを表示するかを指定します。

  • 1つのフォルダページに表示する時間の量(1日、3日、5日、1週間全体、2週間、または月)を指定します。

  • 左矢印および右矢印アイコンを使用して、次または前のフォルダページを指定します。たとえば、現在3日分の情報を表示している場合、矢印をクリックすると、前または次の3日分の情報が表示されます。

  • ［カレンダ設定］アイコンをクリックして、次のオプションを指定します。

    • カレンダの開始曜日(土曜日、日曜日、または月曜日)

    • 勤務日の始業時間

  • ［カレンダ］アイコンをクリックして、新しい月にすばやく移動します。カレンダアプレットに表示する新しい月を選択したら、［OK］をクリックしてフォルダページを更新してください。

詳細については、「保証と著作権」。
