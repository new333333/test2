ミラーリングされたフォルダ

ミラーリングされたフォルダは一種のライブラリフォルダで、そのファイルは外部システム(ファイルシステム内のディレクトリまたはSharePoint Server上のフォルダ)に存在します。Teamingから外部フォルダをミラーリングすると、外部ファイルにKablink Teamingのツール(検索やワークフローなど)を適用できます。

この機能を有効にするには、サーバ上にあるファイルを操作する必要があります。

ミラーリングされたフォルダの作成の詳細については、『Kablink Teaming 2.0上級ユーザガイド』の「ミラーリングされたフォルダの操作」を参照してください。

詳細については、「保証と著作権」。
