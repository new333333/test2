エントリのレーティング

多くのナレッジ管理アプリケーションやソーシャルネットワーキングアプリケーションと同様に、Kablink Teamingにも、最も人気のあるエントリをユーザが判断する方法が用意されています。Teamingでは、このために5つ星のレーティングツールが提供されています。

エントリのレーティングの表示

エントリの表示中、このエントリの平均レーティングを示す黄色い星が表示されます。お気に入りレーティングが最も低いものは1つ星で、お気に入りレーティングが最も高いものは5つ星です。すべての星が灰色の場合、そのエントリにはまだレーティングがありません。

エントリのレーティングの送信

レーティングを送信するには、星をクリックします(たとえば、3つ星のレーティングを送信するには、左から3番目の星をクリックします)。レーティングは何度でも変更できますが、Teamingで平均の算出時に使用されるのは、最も最近のレーティングのみです。

詳細については、「保証と著作権」。
