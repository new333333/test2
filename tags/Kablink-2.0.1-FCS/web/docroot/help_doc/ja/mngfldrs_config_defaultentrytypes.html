デフォルトエントリタイプ

このセクションでは、このフォルダ内にユーザが作成できるエントリのタイプを選択します。追加のエントリタイプを許可するようにフォルダを設定することも、現在のエントリタイプを変更することもできます。追加のエントリタイプを追加した場合、このフォルダにエントリを送信したユーザは、タスクエントリやカレンダエントリなど、ユーザが追加したいエントリのタイプを選択できるようになります。

  1. フォルダで有効にするエントリタイプを選択します。

  2. ［適用］>［閉じる］の順にクリックします。

追加のエントリタイプを有効にすると、このフォルダにエントリを送信したユーザは、［フォルダエントリ］ツールバーの［追加］をクリックし、フォルダで有効になっているエントリのタイプに応じて、ドロップダウンリストを使用して追加するエントリのタイプを選択できます。

サイト管理者およびフォルダ所有者は、［フォームと表示デザイナ］機能を使用して新しいエントリタイプ(カスタムフォーム)を作成し、このリストに表示することができます(フォルダ所有者が新しいエントリタイプを作成した場合、それらのエントリタイプは、このリストの［ローカル定義］セクションに一覧表示されます)。

カスタムエントリタイプの作成方法の詳細については、『Kablink Teaming 2.0上級ユーザガイド』の「カスタムフォルダエントリフォームのデザイン」を参照してください。

詳細については、「保証と著作権」。
