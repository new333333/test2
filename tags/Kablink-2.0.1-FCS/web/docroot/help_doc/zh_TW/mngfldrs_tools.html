資料夾管理工具

如果在「資料夾」工具列中按一下「管理」，Kablink Teaming 將僅顯示允許您使用的管理和組態工具。站台管理員或此工作區或資料夾的擁有者可以使用這些工具，也可以授予對這些工具的存取權限。

如果您擁有不受限制的存取權限，可以看到下列選項：

  • 存取控制︰ 可讓您控制可以檢視和參與此位置的人員。

  • 活動報告： 可讓您檢視工作區中的活動。

  • 設定︰ 可讓您設定資料夾，包括設定預設檢視窗、預設項目類型、允許的檢視窗和資料夾 URL。

  • 複製此資料夾： 可讓您在擁有適當權限的情況下，將目前資料夾複製到工作區樹狀結構中所需的任何目的地。請參閱工作區樹狀結構導覽。

  • 刪除此資料夾： 刪除目前的資料夾。

  • 電子郵件設定︰ 可讓您將資料夾設定為自動傳送電子郵件通知給其他 Teaming 使用者。

  • 表單和檢視窗設計工具： 可讓您建立隨後可套用至資料夾及其子資料夾的自訂項目表單和自訂工作流程。

  • 修改此資料夾： 可讓您修改資料夾，包括資料夾名稱、描述、品牌和圖示。

  • 移動此資料夾： 可讓您在擁有適當權限的情況下，將目前資料夾移動到工作區樹狀結構中所需的任何目的地。請參閱工作區樹狀結構導覽。

  • 新增資料夾： 可讓您在工作區中新增資料夾。

有關更詳細的資訊，請參閱 法律聲明.
