允許的檢視窗

因為只有資料夾擁有者有權設定資料夾的預設檢視窗，所以如果資料夾擁有者對預設檢視窗不滿意，或者更想使用其他檢視窗，可以使用「允許的檢視窗」區段選取允許資料夾的外部使用者使用的檢視窗。例如，如果您設定的是「行事曆」資料夾並在此區段中選取「任務」，則檢視「行事曆」資料夾的使用者可選擇在「任務」資料夾檢視窗中檢視該資料夾。

啟用要對資料夾使用的檢視窗

  1. 在「允許的檢視窗」區段中，選取要用於正在設定之資料夾的檢視窗。

  2. 按一下「套用」>「關閉」。

  3. 在主側邊列的「資料夾選項」區段中，按一下「資料夾檢視窗」。

    此時會顯示該資料夾所有允許的檢視窗。(即您在步驟 1的「允許的檢視窗」區段中選取的檢視窗。)資料夾使用者可以根據自己要檢視資料夾的方式從此清單中選擇檢視窗。

啟用要用做預設檢視窗的檢視窗

在「允許的檢視窗」區段中選取檢視窗，會將該檢視窗新增至「預設檢視窗」區段。這可讓資料夾擁有者選擇該檢視窗做為預設檢視窗。

  1. 在￼「允許的檢視窗」區段中，選取要新增至「預設檢視窗」區段的檢視窗。

  2. 按一下「套用」。

    所選檢視會新增至「預設檢視窗」區段。

如需設定資料夾之預設檢視窗的相關資訊，請參閱變更預設資料夾檢視窗。
