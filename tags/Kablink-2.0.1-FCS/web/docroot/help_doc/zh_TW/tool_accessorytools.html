附屬應用程式工具

您可以在 Kablink Teaming 頁面中新增附屬應用程式區段，以增強現有資訊的可視性和可存取性。它們能提供附加功能，增強團隊工作和通訊。下面是附屬應用程式的一些範例：

  • 搜尋結果 (可能僅適用於目前位置)

  • 好友清單 (可能用於本地主題專家)

  • 工作區樹狀結構 (在階層中開啟到此位置)

  • 留言簿

如需 Teaming 附屬應用程式及其描述的完整清單，請參閱《Kablink Teaming 2.0 使用者指南》中的「附屬應用程式類型」。

下列主題說明如何使用「附屬應用程式面板」：

新增附屬應用程式

若要將附屬應用程式新增至「附屬應用程式面板」：

  1. 按一下「附屬應用程式面板」。

  2. 在「附屬應用程式範圍」區段中，選取所需的附屬應用程式範圍。

    若要瞭解此區段中各選項的詳細資訊，請參閱設定可以檢視附屬應用程式的人員。

  3. 在「附屬應用程式類型」區段中，選取要新增的附屬應用程式類型。

  4. 按一下「確定」。

  5. 提供附屬應用程式的標題，然後填寫其他相關資訊。

    如需填寫每種附屬應用程式之相關資訊的詳細資訊，請參閱《Kablink Teaming 2.0 使用者指南》中的「設定附屬應用程式」。

  6. 按一下「套用」>「關閉」。

設定可以檢視附屬應用程式的人員

新增附屬應用程式時，可以選擇附屬應用程式範圍。在「附屬應用程式範圍」區段中提供下列選項：

  • 僅用於此頁面的個人附屬應用程式： 附屬應用程式僅適用於目前位置，且只有您可以檢視。

  • 用於所有工作區和資料夾的個人附屬應用程式： 附屬應用程式適用於所有工作區和資料夾，且只有您可以檢視。

  • 用於此頁面的社群附屬應用程式： 附屬應用程式僅適用於此頁面，並且對頁面擁有存取權限的所有 Teaming 使用者均可檢視。

變更附屬應用程式的順序

Teaming 可讓您重新排列有多個附屬應用程式的指定頁面上的附屬應用程式。

  1. 在附屬應用程式標題列的右上角，按一下向上箭頭或向下箭頭 ，根據需要向上或向下移動附屬應用程式。

重新設定附屬應用程式

Teaming 可讓您在建立附屬應用程式之後重新對其進行設定。

  1. 在附屬應用程式標題列的右上角，按一下「修改」圖示 。

刪除附屬應用程式

  1. 在附屬應用程式標題列的右上角，按一下「刪除」圖示 。

隱藏附屬應用程式

Teaming 可讓您隱藏附屬應用程式，只顯示附屬應用程式的標題列。

  1. 在附屬應用程式標題列的右上角，按一下「隱藏」圖示 。

您還可以同時隱藏「附屬應用程式面板」中的所有附屬應用程式。

  1. 按一下「附屬應用程式面板」旁邊的「隱藏」圖示。

    此時便會從檢視窗中隱藏目前頁面上的所有附屬應用程式。

    再次按一下「隱藏」圖示可以顯示附屬應用程式。
