所有 Teaming 使用者

下列指南適用於所有 Kablink Teaming 使用者：

  • Kablink Teaming 2.0 使用者指南

    此指南詳細介紹了如何在 Teaming 中完成各種任務。它包含綜覽描述、導覽指示、自定指示以及圖形。

  • Kablink Teaming 2.0 進階使用者指南

    此指南適用於工作區和資料夾擁有者，他們負責保證其工作區可供相關的使用者使用且有效。團隊領導者和專案經理便是此類資訊的需求者。

Teaming 管理員

下列指南適用於 Kablink Teaming 管理員：

  • Kablink Teaming 2.0 管理指南

    此指南說明管理員在建立、設定以及操作 Teaming 站台時需執行的任務。

  • Kablink Teaming 2.0 進階使用者指南

    此指南適用於工作區和資料夾擁有者，他們負責保證其工作區可供相關的使用者使用且有效。團隊領導者和專案經理便是此類資訊的需求者。

  • Kablink Teaming 2.0 安裝指南

    此指南說明為了讓所有預設功能按預期操作，安裝 Teaming 時需執行的任務。它還介紹了如何設定 Teaming 使用者。

  • Kablink Teaming 2.0 開發人員指南

    本指南主要適用於想要延伸 Kablink Teaming 功能，或編寫其他可與該軟體互動之應用程式的開發人員。
