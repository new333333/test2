預設項目類型

此區段可讓您選取允許使用者在此資料夾中建立的項目類型。您可以將資料夾設定為允許其他項目類型，也可以變更目前的項目類型。如果新增其他項目類型，則將項目提交到此資料夾的使用者可以選擇他們要新增的項目類型，例如任務項目或行事曆項目。

  1. 選取要對資料夾啟用的項目類型。

  2. 按一下「套用」>「關閉」。

如果啟用其他項目類型，則將項目提交到此資料夾的使用者可以在「資料夾項目」工具列中按一下「新增」，然後根據您對資料夾啟用的項目類型，使用下拉式清單選擇他們要新增的項目類型。

站台管理員和資料夾擁有者可以使用「表單和檢視窗設計工具」功能建立新項目類型 (自定表單)，這些類型會出現在此清單中。(資料夾擁有者建立新項目類型時，這些項目類型將列在此清單的「本地定義」區段中。)

如需如何建立自定項目類型的詳細資訊，請參閱《Kablink Teaming 2.0 進階使用者指南》中的《設計自定資料夾項目表單》。

有關更詳細的資訊，請參閱 法律聲明.
