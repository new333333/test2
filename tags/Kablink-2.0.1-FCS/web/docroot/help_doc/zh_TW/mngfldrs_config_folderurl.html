定義資料夾或工作區的 URL

此區段用於為目前所在的資料夾或工作區定義易記的 URL。定義此 URL 可讓您從瀏覽器直接導覽至資料夾或工作區。

為資料夾定義 URL 可讓您透過電子郵件或文字訊息將項目新增至該資料夾。

  1. 在「定義 URL」下拉式清單中，選取其中一個預先定義的全域關鍵字。

    除非您是站台管理員，否則 URL 必須以某個預先定義的全域關鍵字開頭。

  2. 在欄位中，為資料夾或工作區指定所需的 URL。

  3. 按一下「新增」。

  4. (視情況而定) 如果是為資料夾定義 URL，必須透過選取 URL 來啟用剛剛定義的 URL。需要選取的 URL 位於「新增」按鈕的下方。

  5. (視情況而定) 如果是為資料夾定義 URL，請按一下「套用」。

  6. 按一下「關閉」。

若要刪除已定義的 URL，請在「目前定義的 URL」區段中選取 URL，然後按一下「刪除選定的 URL」。

有關更詳細的資訊，請參閱 法律聲明.
