帮助模式入门

页面左上角的面板称为“帮助模式控制面板”。它通常用于访问目录或入门 PDF 手册。要最小化“帮助模式控制面板”，请单击位于控制面板顶部的最小化图标 。再次单击此图标可将面板恢复到其实际大小。

帮助模式说明了您当前正在查看的页面。单击信息图标 ，Kablink Teaming 即会显示说明那部分页面的帮助面板。要关闭帮助面板，请单击帮助面板右上角的关闭图标 (X)。或者，可以单击其他信息图标继续阅读。除使用“信息”图标外，您还可以单击“帮助模式控制面板”中的帮助主题，然后单击感兴趣的主题的链接标题。

有关更完整的 Teaming 文档，您可以单击“帮助模式控制面板”中的查看手册。

使用完帮助模式后，您可以通过单击控制面板右上角的关闭图标 (X) 或单击页面中变暗部分的任意位置退出帮助模式。

在您退出帮助模式并开始使用工具后，Teaming 会在需要附加说明的表单上提供信息图标 。单击表单部分旁边的信息图标可阅读更多有关此部分的说明。

有关更多信息，请参见 法律声明.
