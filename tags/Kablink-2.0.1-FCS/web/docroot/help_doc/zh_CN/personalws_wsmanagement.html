工作空间管理工具

单击工作空间工具栏中的管理时，Kablink Teaming 将仅显示那些您有权使用的管理和配置工具。站点管理员或工作空间拥有者可以授予对这些工具的访问权限。

如果您具有不受限制的访问权限，则会看到以下选项：

  • 访问控制： 使您能够控制哪些人可以查看和参与此工作空间。

  • 活动报告： 使您能够查看工作空间中已发生的活动。

  • 配置： 使您能够配置工作空间，包括设置默认视图和工作空间 URL。

  • 复制此工作空间： 如果您具有相应权限，则能够将当前工作空间复制到工作空间树中所需的任何目标位置。请参见工作空间树导航。

  • 删除此工作空间： 删除当前工作空间。

  • 表单和视图设计程序： 使您能够创建自定义条目表单和自定义工作流程，可供以后应用于工作空间及其子文件夹。

  • 修改此工作空间： 使您能够修改工作空间，包括工作空间名称、说明、登录页、固定设计和图标。

  • 移动此工作空间： 如果您具有相应权限，则能够将当前工作空间移动到工作空间树中所需的任何目标位置。请参见工作空间树导航。

  • 添加新工作空间： 使您能够在当前所在工作空间下添加新工作空间。

  • 添加新文件夹： 使您能够向工作空间添加新文件夹。

有关更多信息，请参见 法律声明.
