Quickinfos

Über diesen Link können Sie die Quickinfo-Funktion von Kablink Teaming aktivieren bzw. deaktivieren. Klicken Sie auf Quickinfo aktivieren, wenn die Quickinfos angezeigt werden sollen. Quickinfos werden angezeigt, wenn Sie den Mauszeiger auf bestimmte Optionen der Teaming-Schnittstelle setzen. Wenn Sie die Quickinfos nicht mehr anzeigen möchten, klicken Sie auf Quickinfo deaktivieren.
