Eintragslisten und Listenseiten

Mit diesen Werkzeugen können Sie nach einzelnen Einträgen suchen oder durch die Seiten von Eintragslisten navigieren. Eine Ordnerliste zeigt die Einträge bzw. die Informationen zu den Einträgen des jeweiligen Ordners an. Einige dieser Werkzeuge eignen sich ausschließlich für Eintragslisten, die mehrere Seiten lang sind. Diese Symbolleiste stellt von links nach rechts die folgenden Werkzeuge bereit:

  • Zu einem Eintrag wechseln: Geben Sie die Nummer (in der Spalte Nr. angezeigt) des Eintrags an, den Sie anzeigen möchten, und klicken Sie dann auf Los.

  • Weiter/Zurück: Blättert vorwärts oder rückwärts durch die Seiten der Ordnerliste.

  • Gehe zu Seite: Wenn Sie in dieses Textfeld eine Seitenzahl eingeben und auf Los klicken, zeigt Kablink Teaming die entsprechende Seite mit Ordnereinträgen an.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
