Symbolleiste "Datum"

Über die Symbolleiste "Datum" von Kablink Teaming können Sie die Kalenderansicht anpassen.

  • Sie können angeben, ob Teaming alle Stunden des Tages oder nur die üblichen Arbeitszeiten anzeigen soll.

  • Sie können den auf einer Ordnerseite angezeigten Zeitraum (1 Tag, 3 Tage, 5 Tage, 1 volle Woche, zwei Wochen oder ein Monat) angeben.

  • Mit dem Links- bzw. Rechtspfeil können Sie die vorherige bzw. nächste Ordnerseite anzeigen. Wenn Sie zum Beispiel gerade eine Kalenderseite mit drei Tagen anzeigen und auf eines der beiden Pfeilsymbole klicken, wird die Kalenderseite mit den vorherigen oder nächsten drei Tagen angezeigt.

  • Über das Symbol Kalendereinstellungen können Sie die folgenden Einstellungen vornehmen:

    • Der Wochentag, an dem eine Kalenderwoche beginnt (Samstag, Sonntag oder Montag).

    • Die Uhrzeit, zu der Ihr Arbeitstag beginnt.

  • Über das Symbol Kalender können Sie schnell einen anderen Monat anzeigen. Danach sollten Sie auf OK klicken, um die Ordnerseite zu aktualisieren.
