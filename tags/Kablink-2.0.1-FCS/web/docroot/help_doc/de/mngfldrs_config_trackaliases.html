Beobachten von Aliasnamen

Wenn Ordnereigentümer oder Website-Administratoren in ihren Email-Einstellungen eine Aliasadresse für Email-Postings an den jeweiligen Ordner angeben, werden in dieser Spalte Informationen zur Konfiguration angezeigt. Anhand dieser Informationen können Sie vorhandene Aliasnamen für das Email-Eingangskonto der Installation beobachten.
