Zulassen von Email

Die Optionen dieser Seite werden in den folgenden Abschnitten beschrieben.

Aktivieren des Email-Ausgangs

Für den Email-Ausgang können Sie folgende Konfigurationseinstellungen ändern:

  • Email-Ausgang aktivieren: Aktivieren Sie dieses Kontrollkästchen, um sämtliche abgehenden Email-Nachrichten aus Teaming zu aktivieren. Wenn Sie nicht möchten, dass Benutzer Email-Nachrichten aus Teaming senden, deaktivieren Sie dieses Kontrollkästchen.

    Das Kontrollkästchen muss aktiviert sein, wenn Sie den Benutzern das Versenden von Digest-Benachrichtigungen ermöglichen wollen.

  • Email-Benachrichtigungen senden (Ausgang): In diesem Abschnitt legen Sie die Standardeinstellungen für Digest-Benachrichtigungen fest. Teaming-Benutzer, die via Email Digest-Benachrichtigungen senden oder empfangen möchten, müssen die hier festgelegten Einstellungen für Digest-Benachrichtigungen verwenden. Sie können hier festlegen, dass Digest-Benachrichtigungen jeden Tag, an bestimmten Wochentagen, jeden Tag um eine bestimmte Uhrzeit oder in stündlichen Intervallen gesendet werden.

Aktivieren des Email-Eingangs mithilfe des eingebetteten Email-Servers und einfacher URLs

Aktivieren Sie das Kontrollkästchen Email-Eingang mithilfe des eingebetteten Email-Servers und einfacher URLs aktivieren, wenn Sie zulassen möchten, dass Einträge via Email an Ordner gesendet werden, für die eine einfache URL eingerichtet wurde.
