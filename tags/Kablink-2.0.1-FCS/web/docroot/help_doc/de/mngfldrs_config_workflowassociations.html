Workflow-Zuordnungen

In diesem Abschnitt können Sie Eintragsarten oder Kommentar- bzw. Antworttypen Workflows zuordnen. Wenn einer Eintragsart oder einem Kommentar- bzw. Antworttyp ein Workflow zugeordnet ist, wird der Workflow automatisch gestartet, sobald ein Eintrag oder ein Kommentar/eine Antwort dieser Art erstellt bzw. abgesendet wird.

Zuordnen von Workflows zu Einträgen

Dieser Abschnitt enthält alle Eintragsarten, die im Abschnitt Standardeintragsarten dieser Seite ausgewählt sind. Neben jeder Eintragsart im Abschnitt Workflow-Zuordnungen wird eine Dropdown-Liste mit allen Workflow-Prozessen angezeigt, die der jeweiligen Eintragsart zugeordnet werden können.

So ordnen Sie einer Eintragsart einen Workflow zu:

  1. Stellen Sie sicher, dass die Eintragsart für diesen Ordner zugelassen ist. Falls nicht, wählen Sie sie im Abschnitt Standardeintragsarten dieser Seite aus.

    Nur wenn Sie die Eintragsart im Abschnitt Standardeintragsarten ausgewählt und auf Anwenden geklickt haben, wird die Eintragsart im Abschnitt Workflow-Zuordnungen angezeigt.

  2. Wählen Sie im Abschnitt Workflow-Zuordnungen unter Eintragsarten in der Dropdown-Liste neben der Eintragsart, der Sie einen Workflow zuordnen möchten, den gewünschten Workflow aus.

  3. Klicken Sie auf Anwenden > Schließen.

    Wenn nun ein Benutzer einen Eintrag dieser Art in diesem Ordner erstellt, versetzt Teaming den Eintrag automatisch in den Anfangsstatus des Workflows.

Wenn Sie innerhalb eines Ordners einen neuen Workflow erstellen, wird dieser Workflow automatisch in der Dropdown-Liste neben den Eintragsarten angezeigt. Der Workflow kann dann den Einträgen dieses Ordners zugeordnet werden. Informationen zum Erstellen von Workflows finden Sie im Kablink Teaming 2.0-Handbuch für Fortgeschrittene im Abschnitt Erstellen von Workflows.

Zuordnen von Workflows zu Kommentar- bzw. Antworttypen

Im Abschnitt Kommentar- oder Antworttypen können Sie Kommentar- oder Antworttypen Workflows zuordnen, die gestartet werden, sobald ein Benutzer den jeweiligen Antworttyp eingibt.

Teaming zeigt für jede im Abschnitt Workflow-Zuordnungen aufgeführte Standardeintragsart sämtliche möglichen Antworttypen an.

Wenn eine Standardeintragsart gleichzeitig ein Antworttyp ist, wird dieser nicht nochmals aufgeführt.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
