Tip til filoverførsel

Hvis du har problemer med at overføre en fil, fordi filen er for stor, kan du muligvis løse dette problem ved at forøge standardmængden af hukommelse, der anvendes af Java-applets. Processen er beskrevet i de følgende afsnit.

Forøge standardmængden af hukommelse i Windows

  1. Navigér til Kontrolpanel (klik på Start > Indstillinger > Kontrolpanel).

  2. Dobbeltklik på Java.

    Java-kontrolpanelet startes.

  3. Vælg fanen Java.

  4. I afsnittet Java Applet Runtime Settings skal du klikke på View.

  5. For hver af dine Java Runtime-versioner skal du tilføje -Xmx256m som en Java-runtimeparameter.

  6. Klik på OK.

    Når størrelsen på standardhukommelsen er øget for Java-applets, skal du lukke alle browsere og genstarte dem, før ændringen træder i kraft.

Denne indstilling får den applet, der bruges til at overføre filer til en Teaming-mappe, til at starte op med 256 MB hukommelse. Hvis appletten stadig angiver, at der ikke er tilstrækkelig hukommelse til at køre den, kan det være nødvendigt at formindske dette tal.

Forøge standardmængden af hukommelse på Linux

  1. Find den JRE eller JDK, der bruges til browserens plug-in.

    Placeringen af JRE eller JDK varierer afhængigt af systemet.

  2. I mappen bin i JRE eller JDK skal du køre filen ControlPanel.

    Kontrolpanelet startes.

  3. Vælg fanen Java.

  4. I afsnittet Java Applet Runtime Settings skal du klikke på View.

  5. For hver af dine Java Runtime-versioner skal du tilføje -Xmx256m som en Java-runtimeparameter.

  6. Klik på OK.

    Når størrelsen på standardhukommelsen er øget for Java-applets, skal du lukke alle browsere og genstarte dem, før ændringen træder i kraft.

Denne indstilling får den applet, der bruges til at overføre filer til en Teaming-mappe, til at starte op med 256 MB hukommelse. Hvis appletten stadig angiver, at der ikke er tilstrækkelig hukommelse til at køre den, kan det være nødvendigt at formindske dette tal.
