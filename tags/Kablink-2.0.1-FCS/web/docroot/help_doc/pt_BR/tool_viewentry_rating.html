Classificando entradas

Como em muitos aplicativos de gerenciamento de conhecimento e rede social, o Kablink Teaming oferece maneiras para as pessoas determinarem quais entradas são mais populares. O Teaming dispõe de uma ferramenta de classificação de cinco estrelas para esse objetivo.

Vendo as classificações de entradas

Quando você vê uma entrada, o Teaming exibe estrelas amarelas que indicam a classificação média dessa entrada. Uma estrela é a classificação menos favorável e cinco estrelas, a mais favorável. Se todas as estrelas forem cinzas, a entrada não foi classificada.

Enviando classificações de entradas

Para enviar a sua classificação, clique em uma estrela. (Por exemplo, para enviar uma classificação três estrelas, clique na terceira estrela da esquerda para a direita.) É possível mudar a classificação sempre que quiser, mas o Teaming usa apenas a sua classificação mais recente ao calcular a média.
