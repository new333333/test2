Minhas Equipes

Clique no ícone Minhas Equipes para exibir os links para todas as áreas de trabalho globais e de equipe das quais você é membro de equipe. Para visitar uma área de trabalho de equipe, clique no link do nome do título da equipe que deseja visitar.

Para fechar a lista, clique no ícone Fechar (X), localizado no canto superior direito.

Para obter mais informações, consulte Informações legais.
