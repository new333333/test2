Barra de herramientas Entrada

La barra de herramientas Entrada ofrece sólo las herramientas que tiene permiso para utilizar. La barra de herramientas se muestra tanto para las entradas como para los comentarios y puede incluir los siguientes botones y menús:

  • Comentario: añade un comentario para la entrada. Puede hacer un comentario sobre un comentario (esto se denomina realizar comentarios anidados).

  • Modificar: permite editar la entrada o añadir otro archivo adjunto. (También se pueden utilizar el enlace Editar en la parte inferior derecha de la entrada, justo encima de la sección Datos adjuntos).

  • Reservar: evita que otros usuarios puedan editar la entrada mientras usted la modifica. Una sugerencia que puede resultarle útil es reservar una entrada si está realizando cambios importantes en ella. Cuando haya realizado todos los cambios, haga clic en Anular reserva.

  • Mover: permite mover la entrada a otra carpeta.

  • Copiar: permite copiar la entrada y guardarla en otra ubicación.

  • Suprimir: elimina la entrada de la carpeta.

  • Flujo de trabajo: cuando un flujo de trabajo se habilita en una carpeta, esta herramienta permite iniciar un flujo si aún no se ha iniciado en la entrada, o detener un flujo de trabajo si está en proceso en la entrada.

  • Definir página de inicio wiki (sólo está disponible con entradas wiki): utilice esta opción para configurar Teaming para mostrar una página wiki como página de inicio wiki. Sólo tiene que desplazarse a la página que desea definir como página de inicio wiki y hacer clic en esta opción.

  • Informes: permite generar los siguientes informes:

    • Informe de actividad: muestra un archivo de valores separados por comas en Microsoft Excel que proporciona información acerca de los usuarios y sus actividades relacionadas con esta entrada.

    • Editar historial: lanza una ventana del mismo nombre en la que podrá comparar las distintas versiones de la entrada y comprobar qué se ha añadido o suprimido.

    • Historial de flujo de trabajo: lanza una ventana del mismo nombre en la que podrá ver el historial del flujo de trabajo, incluido el nombre del flujo de trabajo, cuándo se creó la entrada y quién lo hizo, cuándo entró el flujo en un estado determinado, quién modificó la entrada cuando entró en ese estado, y mucho más.

  • Compartir esta entrada: permite compartir una entrada con otros usuarios de Teaming. Cuando se comparte una entrada, se envía una notificación de correo electrónico a los usuarios seleccionados. Asimismo, en sus áreas de trabajo de inicio, los usuarios podrán seleccionar la pestaña Microblogs y elementos compartidos. En la columna Elementos compartidos, Teaming muestra todos los elementos que los usuarios han compartido con ellos.

Para obtener más información, consulte Información legal.
