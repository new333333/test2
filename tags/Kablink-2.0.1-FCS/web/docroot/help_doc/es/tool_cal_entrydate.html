Tipo de fecha de entrada

Esta herramienta es especialmente útil si utiliza la vista de calendario para revisar la información que se ha diseñado para otro tipo de carpeta. Por ejemplo, puede utilizar un wiki o una discusión con su vista por defecto. Después, si el propietario de la carpeta o el administrador la configura para que sea compatible con esta acción, puede utilizar la vista de calendario para ver qué entradas tienen la actividad más reciente (por ejemplo, qué entradas se han editado más recientemente). También puede ver las entradas según su fecha de creación.

Para obtener más información, consulte Información legal.
