Documentación introductoria

Los enlaces de esta sección dirigen a manuales de introducción que se abren en una nueva ventana del navegador. Puede leer estos manuales en línea en formato HTML y PDF. El formato PDF también está optimizado para la impresión.

Todos los usuarios de Teaming

Las guías siguientes son de interés para todos los usuarios de Kablink Teaming:

  • Kablink Teaming 2.0 User Guide (Guía del usuario de Kablink Teaming 2.0)

    En esta guía se ofrece información detallada que describe el procedimiento para realizar varias tareas en Teaming. Incluye descripciones generales, instrucciones de navegación y personalización y gráficos.

  • Kablink Teaming 2.0 Advanced User Guide (Guía del usuario avanzado de Kablink Teaming 2.0)

    Esta guía está dirigida a los propietarios de las carpetas y del área de trabajo responsables de adaptar el área de trabajo para que resulte útil y eficaz para los usuarios que la utilizan. Ejemplos de personas que podrían estar interesadas en esta información son los jefes de equipos y los administradores de proyectos.

Administradores de Teaming

Las guías siguientes son de interés para los administradores de Kablink Teaming:

  • Kablink Teaming 2.0 Administration Guide (Guía de administración de Kablink Teaming 2.0)

    En esta guía se describen las tareas necesarias para que los administradores instalen, configuren y trabajen con un sitio de Teaming.

  • Kablink Teaming 2.0 Advanced User Guide (Guía del usuario avanzado de Kablink Teaming 2.0)

    Esta guía está dirigida a los propietarios de las carpetas y del área de trabajo responsables de adaptar el área de trabajo para que resulte útil y eficaz para los usuarios que la utilizan. Ejemplos de personas que podrían estar interesadas en esta información son los jefes de equipos y los administradores de proyectos.

  • Kablink Teaming 2.0 Installation Guide (Guía de instalación de Kablink Teaming 2.0)

    En esta guía se describen las tareas necesarias para instalar Teaming para que todas las funciones por defecto funcionen normalmente. También se describe cómo configurar un usuario de Teaming.

  • Kablink Teaming 2.0 Developer Guide (Guía del desarrollador de Kablink Teaming 2.0)

    Esta guía está orientada principalmente a los desarrolladores que desean ampliar la funcionalidad de Kablink Teaming o programar otras aplicaciones para interactuar con Teaming.
