Vista del área de trabajo

Seleccione la vista que mejor presente su área de trabajo.

  • Área de trabajo de discusiones: seleccione esta vista si desea crear un área de trabajo en la que se puedan almacenar varios foros de discusión.

    Para obtener más información sobre las áreas de trabajo de discusiones, consulte Managing a Discussions Workspace (Gestión de un área de trabajo de discusiones) en la Kablink Teaming 2.0 Advanced User Guide (Guía del usuario avanzado de Kablink Teaming 2.0).

  • Página de entrada: seleccione esta vista si desea mostrar información importante a miembros del equipo en cuanto entren al área de trabajo.

    Para obtener más información sobre las páginas de entrada, consulte Creating a Landing Page (Creación de una página de entrada) en la Kablink Teaming 2.0 Advanced User Guide (Guía del usuario avanzado de Kablink Teaming 2.0).

  • Área de trabajo de gestión de proyectos: seleccione esta vista si desea rastrear las tareas del proyecto y los hitos según finalicen las tareas.

    Para obtener más información sobre las áreas de trabajo de gestión de proyectos, consulte Managing a Project Management Workspace (Gestión de un área de trabajo de gestión de proyectos) en la Kablink Teaming 2.0 Advanced User Guide (Guía del usuario avanzado de Kablink Teaming 2.0).

  • Área de trabajo del equipo: seleccione esta vista si el área de trabajo del equipo es por naturaleza para toda la organización y no está destinada a tener áreas de trabajo de equipo como sublugares.

    Para obtener más información sobre las áreas de trabajo del equipo, consulte Managing a Team Workspace (Gestión de un área de trabajo del equipo) en la Kablink Teaming 2.0 User Guide (Guía del usuario de Kablink Teaming 2.0).

  • Áreas de trabajo raíz del equipo: seleccione esta vista si desea mostrar el botón Añadir un área de trabajo de equipo al área de trabajo de su equipo. Esto es aconsejable si desea crear otras áreas de trabajo de equipo como sublugares de esta área de trabajo.

    Para obtener más información sobre las áreas de trabajo raíz del equipo, consulte Managing a Team Workspace Root (Gestión de un área de trabajo raíz del equipo) en la Kablink Teaming 2.0 Advanced User Guide (Guía del usuario avanzado de Kablink Teaming 2.0).

  • Área de trabajo: Se trata de un área de trabajo básica. Puede elegir las carpetas que desee incluir en el área de trabajo.

    Para obtener más información sobre las áreas de trabajo básicas, consulte Managing a Basic Workspace (Gestión de un área de trabajo básica) en la Kablink Teaming 2.0 Advanced User Guide (Guía del usuario avanzado de Kablink Teaming 2.0).

Si se trata de un área de trabajo personal, la vista por defecto es Área de trabajo del usuario y no se puede cambiar.
