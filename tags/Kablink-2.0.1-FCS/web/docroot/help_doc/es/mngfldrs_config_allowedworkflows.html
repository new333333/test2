Flujos de trabajo permitidos

Cuando el usuario ve una entrada en esta carpeta, puede iniciar cualquier flujo de trabajo que esté seleccionado en esta sección haciendo clic en Flujo de trabajo > Iniciar flujo de trabajo: nombre_del_flujo en la barra de herramientas Entrada.

De esta forma se inicia el proceso de flujo de trabajo en la entrada.

Para obtener más información, consulte Información legal.
