Инструменты аксессуаров

Аксессуары — это разделы, которые Вы можете добавлять на свои страницы Kablink Teaming, делающие существующую информацию более наглядной и доступной. Они предоставляют дополнительные функции и расширяют возможности командной работы и связи. Ниже приводятся примеры аксессуаров.

  • Результаты поиска (возможно, применяются только к текущему месту)

  • Список друзей (возможно, для местных экспертов по определенным темам)

  • Дерево рабочих областей (открытое для данного места в иерархии)

  • Гостевая книга

Полный список аксессуаров Teaming и их описание см. в разделе Типы аксессуаров в Руководстве пользователя Kablink Teaming 2.0.

В следующих разделах описаны способы использования панели аксессуаров.

Добавление аксессуара

Чтобы добавить аксессуар на панель аксессуаров, выполните следующие действия.

  1. Щелкните Панель аксессуаров.

  2. В разделе Область действия аксессуара выберите нужную область действия аксессуара.

    Дополнительно о параметрах этого раздела см. Настройка доступа для просмотра аксессуаров.

  3. В разделе Тип аксессуара выберите тип добавляемого аксессуара.

  4. Нажмите кнопку ОК.

  5. Введите заголовок аксессуара и другую необходимую информацию.

    Дополнительно об информации для каждого типа аксессуара см. в разделе Настройка аксессуара в Руководстве пользователя Kablink Teaming 2.0.

  6. Щелкните Применить > Закрыть.

Настройка доступа для просмотра аксессуаров

При добавлении аксессуара можно выбрать область действия аксессуара. В разделе Область действия аксессуара доступны следующие параметры.

  • Персональный аксессуар только для этой страницы Аксессуар применяется только к текущей странице и является видимым только для Вас.

  • Персональный аксессуар для всех рабочих областей и папок Аксессуар применяется ко всем рабочим областям и папкам и является видимым только для Вас.

  • Общий аксессуар для этой страницы Аксессуар применяется только к текущей странице и является видимым для всех пользователей Teaming, имеющих права доступа к этой странице.

Изменение порядка аксессуаров

Teaming позволяет изменять порядок аксессуаров на любой странице, где имеется несколько аксессуаров.

  1. В правом верхнем углу строки заголовка аксессуара щелкните стрелку вверх или вниз , если требуется переместить аксессуар вверх или вниз.

Изменение конфигурации аксессуара

Teaming позволяет изменить конфигурацию аксессуара после его создания.

  1. В правом верхнем углу строки заголовка аксессуара щелкните значок Изменить .

Удаление аксессуара

  1. В правом верхнем углу строки заголовка аксессуара щелкните значок Удалить .

Скрытие аксессуаров

Teaming позволяет скрыть аксессуары, чтобы оставалась видимой только строка заголовка аксессуара.

  1. В правом верхнем углу строки заголовка аксессуара щелкните значок Скрыть .

Можно также скрыть все аксессуары на панели аксессуаров.

  1. Щелкните значок Скрыть рядом с панелью аксессуаров.

    Все аксессуары на текущей странице будут скрыты от просмотра.

    Щелкните значок Скрыть еще раз, чтобы сделать аксессуары видимыми.

Дополнительную информацию см. в Юридическая информация.
