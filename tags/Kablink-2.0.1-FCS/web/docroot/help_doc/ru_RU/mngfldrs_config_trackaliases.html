Отслеживание псевдонимов

Когда владельцы папок или администраторы в настройке электронной почты для отправки в папку указывают адрес, являющийся псевдонимом, информация о настройке отображается в этом столбце. Эту информацию используют для отслеживания существующих псевдонимов для входящей электронной почты всей установки.

Дополнительную информацию см. в Юридическая информация.
