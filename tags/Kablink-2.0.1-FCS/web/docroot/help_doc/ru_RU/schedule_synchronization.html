Синхронизация по расписанию

Эта страница позволяет включить и настроить ведение расписания синхронизации для отраженных папок. Страница содержит следующие параметры:

Включить синхронизацию по расписанию: Установите этот флажок для проведения синхронизации по расписанию.

Каждый день: Выберите этот параметр, если хотите выполнять синхронизацию ежедневно.

Еженедельно: Выберите этот параметр, чтобы задать конкретные дни недели для выполнения синхронизации.

В: Укажите конкретное время для проведения синхронизации.

Повторять каждые nn часов: Укажите интервал между синхронизациями.

Дополнительную информацию см. в Юридическая информация.
