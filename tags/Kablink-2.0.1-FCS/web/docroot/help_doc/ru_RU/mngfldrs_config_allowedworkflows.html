Разрешенные рабочие потоки

Когда пользователь просматривает запись в этой папке, он может запустить любой рабочий поток, щелкнув Рабочий поток > Запустить рабочий поток: имя_рп на панели инструментов записи.

Будет запущен рабочий поток для этой записи.

Дополнительную информацию см. в Юридическая информация.
