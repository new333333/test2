Ссылка на запись в другой папке

  1. Щелкните ссылку Изменить.

  2. В предоставленное поле начните вводить имя папки, содержащей запись, на которую устанавливается ссылка.

  3. Когда имя папки появится в списке поиска вводящегося текста, щелкните его.

  4. В поле Имя страницы для ссылки укажите полное имя записи, на которую устанавливается ссылка, или нажмите кнопку Найти, чтобы использовать функцию поиска вводящегося текста.

  5. Закончив ввод имени или выбрав имя в раскрывающемся списке поиска вводящегося текста, нажмите кнопку Вставить.

    Ссылка с заголовком выбранной записи вставляется в поле Описание.

Дополнительную информацию см. в Юридическая информация.
