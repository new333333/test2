Отправка оповещений по электронной почте

Как владелец папки, Вы можете настроить Kablink Teaming на отправку другим пользователям оповещений по электронной почте, чтобы информировать их о любых действиях в этой папке и вложенных в нее папках. Однако индивидуальные настройки пользователя переопределяют или отменяют оповещения, заданные владельцами папок.

  1. В разделе Выберите тип оповещения по электронной почте выберите тип отправляемых оповещений.

    О различных типах оповещений по электронной почте см. Выбор типов оповещений по электронной почте для получения.

  2. Укажите пользователей, которые должны получать оповещения по электронной почте.

    Можно указать конкретные адреса электронной почты, пользователей, группы или отправлять оповещения всем участникам команды.

  3. Нажмите кнопки Применить > Закрыть.

Дополнительную информацию об оповещениях по электронной почте см. в разделе Использование оповещений по электронной почте.

Дополнительную информацию см. в Юридическая информация.
