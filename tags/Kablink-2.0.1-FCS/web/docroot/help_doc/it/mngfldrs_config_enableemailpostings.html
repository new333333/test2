Abilitazione delle pubblicazioni e-mail

A prescindere dalle altre impostazioni di questa pagina, attivare questa casella di controllo per abilitare l'inserimento dei messaggi come voci di cartella. Gli alias di e-mail sono utilizzati per determinare i messaggi dell'account di posta in arrivo da indirizzare a cartelle specifiche.

Per ulteriori informazioni, vedere Note legali.
