Modifica

Il collegamento Modifica consente di modificare solo la sezione Descrizione di una voce.

Per modificare altre parti della voce o del commento, incluso il titolo, oppure se si desidera effettuare l'upload di un file allegato o cancellare i file attualmente allegati, fare clic su Modifica nella barra degli strumenti Voce.

Per ulteriori informazioni, vedere Note legali.
