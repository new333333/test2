Documentazione introduttiva

I collegamenti presenti nella sezione visualizzano i manuali introduttivi all'interno di una nuova finestra del browser. È possibile consultare i manuali online in formato HTML e PDF. Il formato PDF è ottimizzato anche per la stampa.

Tutti gli utenti di Teaming

Le seguenti guide sono rivolte a tutti gli utenti di Kablink Teaming:

  • Kablink Teaming 2.0 User Guide (in lingua inglese)

    Questa guida fornisce informazioni dettagliate su come eseguire i vari task di Teaming. Sono incluse le descrizioni generali, le istruzioni per la navigazione e la personalizzazione, nonché immagini grafiche.

  • Kablink Teaming 2.0 Advanced User Guide (in lingua inglese)

    Questa guida è destinata ai proprietari di spazi di lavoro e cartelle, che li devono rendere funzionali ed efficaci per gli utenti finali, come ad esempio i team leader o i project manager.

Amministratori di Teaming

Le seguenti guide sono rivolte agli amministratori di Kablink Teaming:

  • Kablink Teaming 2.0 Administration Guide (in lingua inglese)

    Questa guida descrive i task necessari all'amministratore per la configurazione e il funzionamento di un sito Teaming.

  • Kablink Teaming 2.0 Advanced User Guide (in lingua inglese)

    Questa guida è destinata ai proprietari di spazi di lavoro e cartelle, che li devono rendere funzionali ed efficaci per gli utenti finali, come ad esempio i team leader o i project manager.

  • Kablink Teaming 2.0 Installation Guide (in lingua inglese)

    Questa guida descrive i task necessari per installare Teaming in modo che le funzioni di default operino correttamente. Descrive inoltre come configurare un utente Teaming.

  • Kablink Teaming 2.0 Developer Guide (in lingua inglese)

    Questa guida è rivolta principalmente agli sviluppatori che desiderano estendere le funzionalità di Kablink Teaming o programmare altre applicazioni per l'interazione con questa piattaforma.
