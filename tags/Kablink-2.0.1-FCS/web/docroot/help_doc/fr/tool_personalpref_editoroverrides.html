Paramètres de l'éditeur

Lorsque vous modifiez des pièces jointes dans Kablink Teaming, les applications par défaut du système Teaming sont utilisées pour éditer les pièces jointes. Cependant, cette table permet de remplacer les applications par défaut utilisées pour modifier des pièces jointes avec certaines extensions de fichier dans Teaming.

Vous pouvez remplacer l'application d'édition par défaut pour un seul type d'extension de fichier ou plusieurs.

Remplacement de l'application d'édition par défaut pour des extensions de fichier spécifiques

Pour remplacer l'application par défaut pour un type d'extension de fichier spécifique :

  1. Cliquez sur Ajouter.

  2. Dans la liste déroulante, sélectionnez le type d'extension de fichier pour lequel vous voulez remplacer l'application d'édition par défaut.

  3. Dans le champ Application, indiquez le nom de l'application que vous voulez utiliser pour modifier les pièces jointes présentant l'extension de fichier spécifiée à l'Étape 2.

  4. Cliquez sur OK.

Remplacement de l'application d'édition par défaut pour toutes les extensions de fichier

Pour remplacer l'application d'édition pour tous les types d'extension de fichier :

  1. Cliquez sur Utiliser.

  2. Dans la liste déroulante, sélectionnez l'application que vous voulez utiliser pour modifier les pièces jointes de n'importe quel type d'extension de fichier dans Teaming.

  3. Cliquez sur OK.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
