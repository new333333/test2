Modification de la vue par défaut d'un dossier

  1. Sélectionnez la vue dont l'aspect représente le mieux votre liste de dossiers ou votre espace de travail.

  2. Cliquez sur Appliquer > Fermer.

    La vue du dossier est modifiée.

Pour plus d'informations sur la procédure d'ajout de vues à cette section de manière à pouvoir les sélectionner comme vue par défaut pour le dossier, reportez-vous à la section Activation des vues à utiliser comme vues par défaut.

Lorsque vous choisissez une vue pour un dossier, vous devez prendre en compte les types d'entrées que le dossier contient. Par exemple, une liste d'entrées d'agenda est généralement mieux présentée sous l'aspect d'une vue d'agenda.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
