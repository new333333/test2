Mes équipes

Cliquez sur l'icône Mes équipes pour afficher les liens de tous les espaces de travail globaux et d'équipe dont vous êtes membre. Pour se rendre sur un espace de travail d'équipe, cliquez sur le titre de l'équipe que vous souhaitez visiter.

Pour fermer la liste, cliquez sur l'icône Fermer (X), située dans le coin supérieur droit.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
