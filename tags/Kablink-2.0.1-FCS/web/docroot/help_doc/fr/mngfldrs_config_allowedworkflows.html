Workflows autorisés

Lorsque l'utilisateur affiche une entrée dans ce dossier, il peut lancer n'importe quel workflow qui est sélectionné dans cette section en cliquant sur Workflow > Démarrer le workflow : nom_workflow dans la barre d'outils Entrée.

Cette opération démarre le processus de workflow sur cette entrée.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
