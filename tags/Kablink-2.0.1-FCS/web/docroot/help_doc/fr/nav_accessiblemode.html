Info-bulles

Ce lien permet d'activer et de désactiver la fonction Info-bulles de Kablink Teaming. Cliquez sur Activer les info-bulles pour afficher les info-bulles. Des info-bulles s'affichent alors quand le pointeur de la souris passe sur certaines options dans l'interface Teaming. Si vous ne souhaitez plus voir les info-bulles, cliquez sur Désactiver les info-bulles.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
