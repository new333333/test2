Application récursive des paramètres de définition

Utilisez cette fonctionnalité pour appliquer les paramètres de définition aux entrées ou dossiers déjà créés.

Application des paramètres de définition à tous les sous-dossiers

Vous pouvez appliquer tous les paramètres de définition d'un dossier parent à l'ensemble de ses sous-dossiers.

  1. Sélectionnez Modifier les dossiers inclus de façon à hériter des définitions de ce dossier.

  2. Cliquez sur Appliquer.

Tous les paramètres de définition qui figurent sur la page Configurer les paramètres par défaut sont appliqués à l'ensemble des sous-dossiers.

Application des paramètres de définition aux entrées existantes

Si vous avez précédemment ajouté des entrées dans ce dossier mais que leur type de définition ne correspond pas à celui des autres entrées du dossier (par exemple, vous pouvez posséder un dossier Blogue contenant essentiellement des types de définitions Blogue, tandis que certaines entrées apparaissent en tant que types de définitions Discussion), vous pouvez utiliser cette section pour modifier les paramètres de définition de façon à ce que toutes les entrées du dossier soient identiques.

  1. Sélectionnez Modifier les dossiers inclus de façon à hériter des définitions de ce dossier, puis Modifier les entrées incluses pour qu'elles soient du type suivant.

  2. Dans la liste déroulante, sélectionnez le type d'entrée à appliquer à toutes les entrées existantes du dossier.

    Pour ajouter des types d'entrées à cette liste déroulante, sélectionnez-les dans la section Types d'entrée par défaut, puis cliquez sur Appliquer.

  3. Cliquez sur Appliquer.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
