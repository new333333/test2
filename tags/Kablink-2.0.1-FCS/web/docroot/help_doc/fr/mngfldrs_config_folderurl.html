Définition d'une URL pour un dossier ou un espace de travail

Cette section permet de définir une URL conviviale pour le dossier ou l'espace dans lequel vous êtes actuellement. Cette URL permet alors d'accéder directement au dossier ou à l'espace à partir de votre navigateur.

La définition d'une URL pour un dossier permet d'ajouter des entrées à ce dossier par courrier électronique ou message de texte.

  1. Dans la liste déroulante Définir l'URL, sélectionnez l'un des mots-clés globaux prédéfinis.

    Sauf si vous êtes un administrateur de site, l'URL doit commencer par l'un de ces mots-clés globaux prédéfinis.

  2. Dans le champ, indiquez l'URL de votre choix pour le dossier ou l'espace de travail.

  3. Cliquez sur Ajouter.

  4. (Facultatif) Si vous définissez l'URL d'un dossier, vous devez activer celle-ci en la sélectionnant. L'URL que vous devez sélectionner est située sous le bouton Ajouter.

  5. (Facultatif) Si vous définissez l'URL d'un dossier, cliquez sur Appliquer.

  6. Cliquez sur Fermer.

Pour supprimer une URL définie, sélectionnez-la dans la section URL actuellement définies, puis cliquez sur Supprimer les URL sélectionnées.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
