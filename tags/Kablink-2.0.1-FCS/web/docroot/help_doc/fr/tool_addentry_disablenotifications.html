Désactivation des notifications par message électronique

Sélectionnez cette option pour désactiver toutes les notifications qui ont été précédemment définies par l'administrateur et les propriétaires de dossiers. Si vous ne souhaitez recevoir aucune notification par message électronique, cochez cette case et assurez-vous que les champs au-dessus ne renseignent aucune adresse électronique ni aucun numéro de téléphone.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
