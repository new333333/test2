Vue de l'espace de travail

Sélectionnez la vue qui présente le mieux votre espace de travail.

  • Espace de travail réservé aux discussions : sélectionnez cette vue si vous voulez créer un espace de travail dans lequel vous pouvez stocker plusieurs forums de discussion.

    Pour plus d'informations sur les espaces de travail réservés aux discussions, reportez-vous à la section Managing a Discussions Workspace (Gestion d'un espace de travail réservé aux discussions) du manuel Kablink Teaming 2.0 Advanced User Guide (Guide de l'utilisateur avancé de Kablink Teaming 2.0).

  • Page de renvoi : sélectionnez cette vue si vous souhaitez présenter des informations importantes aux membres de l'équipe dès qu'ils entrent dans l'espace de travail.

    Pour plus d'informations sur les pages de renvoi, reportez-vous à la section Creating a Landing Page (Création d'une page de renvoi) du manuel Kablink Teaming 2.0 Advanced User Guide (Guide de l'utilisateur avancé de Kablink Teaming 2.0).

  • Espace de travail de gestion de projet : sélectionnez cette vue si vous souhaitez un suivi des tâches de projet et des réalisations sur la base de l'exécution des tâches.

    Pour plus d'informations sur les espaces de travail de gestion de projet, reportez-vous à la section Managing a Project Management Workspace (Administration d'un espace de travail de gestion de projet) du manuel Kablink Teaming 2.0 Advanced User Guide (Guide de l'utilisateur avancé de Kablink Teaming 2.0).

  • Espace de travail d'équipe : sélectionnez cette vue si votre espace de travail d'équipe est de nature organisationnelle et non destiné à avoir des espaces de travail d'équipe comme sous-emplacements.

    Pour plus d'informations sur les espaces de travail d'équipe, reportez-vous à la section Managing a Team Workspace (Gestion d'un espace de travail d'équipe) du manuel Kablink Teaming 2.0 User Guide (Guide de l'utilisateur avancé de Kablink Teaming 2.0).

  • Racine de l'espace de travail d'équipe : sélectionnez cette vue si vous souhaitez afficher le bouton Ajouter un espace de travail d'équipe dans votre espace de travail d'équipe. Cela est utile si vous voulez créer d'autres espaces de travail d'équipe en tant que sous-emplacements de cet espace de travail.

    Pour plus d'informations sur les espace de travail racines des espaces de travail d'équipe, reportez-vous à la section Managing a Team Workspace Root Workspace (Gestion d'un espace de travail racine d'un espace de travail d'équipe) du manuel Kablink Teaming 2.0 Advanced User Guide (Guide de l'utilisateur avancé de Kablink Teaming 2.0).

  • Espace de travail : il s'agit d'un espace de travail de base. Vous pouvez choisir quels sont les dossiers que l'espace de travail doit contenir.

    Pour plus d'informations sur les espaces de travail de base, reportez-vous à la section Managing a Basic Workspace (Gestion d'un espace de travail de base) du manuel Kablink Teaming 2.0 Advanced User Guide (Guide de l'utilisateur avancé de Kablink Teaming 2.0).

S'il s'agit d'un espace de travail personnel, la vue par défaut est Espace de travail utilisateur et ne peut pas être modifiée.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
