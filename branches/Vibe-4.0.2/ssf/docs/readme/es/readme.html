Novell Vibe 3.2

1.0 Inicio

Los administradores de Novell Vibe encontrarán los requisitos del sistema e instrucciones detalladas para la instalación del software en la Guía de instalación de Vibe 3.2. En la Vibe 3.2 Administration Guide (Guía de administración de Vibe OnPrem 3) se proporcionan instrucciones para configurar el sitio y para realizar las tareas de administración y mantenimiento habituales.

Después de instalar el sitio de Vibe, los usuarios del programa podrán obtener información en la Guía del usuario de Vibe 3.2 y en la Vibe 3.2 Advanced User Guide (Guía del usuario avanzado de Novell Vibe OnPrem 3). (Hay disponible una guía de Inicio rápido de Vibe 3.2 en los idiomas para los que no hay disponible una Guía del usuario de Vibe 3.2).

Se puede mostrar la ayuda en línea de Vibe después de entrar al sitio de Vibe.
