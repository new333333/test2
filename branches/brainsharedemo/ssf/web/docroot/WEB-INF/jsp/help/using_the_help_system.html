The Aspen Help System

ssHelpSpot

To add a help spot, enclose the target item with <ssHelpSpot> tags. The browser will automatically find these help spots and add them to the table of contents.
Include the following attributes in each help spot:
  • helpId - The name of the jsp to be shown. All help jsps must come from the jsp/help directory.
  • align - Specifies how to horitontally align the help spot in the target area. values can be "left", "center" or "right".
  • valign - Specifies how to vertically align the help spot in the target area. values can be "top", "middle" or "bottom".
  • title - The title line for the help spot.
  • offsetX - the number of pixels to offset the help spot horizontally (can be negative).
  • offsetY - the number of pixels to offset the help spot vertically (can be negative).


ss_helpSystem functions

There are various functions that can be used when writing help pages. Here are the most common:

  • ss_helpSystem.showHelpPanel(id, panelId, x, y)
    id - the help jsp to be shown
    panelId - the id of the panel div to use. A div will be created automatically if none by that name exists.
    x - the horizontal placement of the panel. if x = "", then leave the panel where it was. if x = "left", "center" or "right" than position the panel horizontally at the left, right or center of the window.
    y - the vertical placement of the panel. if y = "", then leave the panel where it was. if y = "top", "middle" or "bottom" than position the panel vertically at the top, bottom or middle of the window.
  • ss_helpSystem.highlight(id)
    id - the id of the div on the original page to highlight.
