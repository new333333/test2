部落格追蹤

「部落格追蹤」工具列可用於快速執行以下操作：

  • 新增意見： 可讓您提出對部落格項目的意見。在提出意見之前，最好先檢視之前提出的意見。如果對已有其他意見的部落格新增意見，則您的意見會自動新增為巢狀意見清單中的最新意見。

  • 檢視意見： 可讓您檢視特定項目的所有意見，不需要開啟項目。

  • 將項目傳送給朋友： 可讓您透過電子郵件將項目的內容傳送給擁有適當存取權限來檢視項目的任何人。

  • 訂閱︰ 啟動表單，可用於訂閱項目。訂閱項目後，您可以設定 Teaming 在該項目有更新時給您傳送電子郵件通知。

    如需訂閱項目的詳細資訊，請參閱《Novell Teaming 2.1 使用者指南》中的「訂閱項目」。

有關更詳細的資訊，請參閱 法律聲明。
