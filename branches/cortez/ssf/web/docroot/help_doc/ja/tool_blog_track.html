ブログのトラッキング

［ブログのトラッキング］ツールバーを使用すると、次の操作をすばやく実行できます。

  • コメントを追加: ブログエントリに対してコメントすることができます。コメントする前に、すでに作成されているコメントを表示することをお勧めします。すでにコメントされているブログにコメントを追加すると、コメントは、ネストされたコメントのリストに自動的に最新のコメントとして追加されます。

  • コメントを表示: エントリを開かずに、特定のエントリのコメントをすべて表示できます。

  • エントリを友人に送信: エントリを表示するための適切なアクセス権を持っているユーザにエントリの内容を電子メールで送信できます。

  • 購読: エントリを購読できるようにするフォームを起動します。エントリを購読するときには、そのエントリの更新を知らせる電子メール通知を送信するようにTeamingを設定します。

    エントリの購読の詳細については、『Novell Teaming 2.1 User Guide』の「Subscribing to an Entry」を参照してください。

詳細については、「保証と著作権」を参照してください。
