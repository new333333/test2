異なるフォルダ内にあるエントリへのリンク

  1. ［変更］をクリックします。

  2. 表示されるフィールドに、リンク先のエントリが含まれるフォルダの名前を入力し始めます。

  3. ドロップダウンリストに目的のフォルダが表示されたら、そのフォルダを選択します。

  4. ［リンク先のページ名］で、リンク先のエントリの完全な名前を指定するか、［検索］をクリックしてインクリメンタルサーチ機能を使用します。

  5. エントリの名前を入力し終わるか、またはインクリメンタルサーチドロップダウンリストで名前をクリックしたら、［挿入］をクリックします。

    選択したエントリのタイトルのリンクが［説明］ボックスに挿入されます。

詳細については、「保証と著作権」を参照してください。
