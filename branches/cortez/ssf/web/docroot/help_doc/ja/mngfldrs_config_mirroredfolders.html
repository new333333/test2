ミラーリングされたフォルダ

ミラーリングされたフォルダは一種のライブラリフォルダで、そのファイルは外部システム(ファイルシステム内のディレクトリまたはSharePoint Server上のフォルダ)に存在します。Teamingから外部フォルダをミラーリングすると、外部ファイルにNovell Teamingのツール(検索やワークフローなど)を適用できます。

この機能を有効にするには、サーバ上にあるファイルを操作する必要があります。

ミラーリングされたフォルダの作成の詳細については、『Novell Teaming 2.1 User Guide』の「Working with Mirrored Folders」を参照してください。

詳細については、「保証と著作権」を参照してください。
