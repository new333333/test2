［フッタ］ツールバー

［フッタ］ツールバーの内容は、ワークスペース、フォルダ、またはエントリのいずれで作業しているかに応じて変わります。

ワークスペースの［フッタ］ツールバー

  • ワークスペースのパーマリンク: ワークスペースを参照できます。これらのリンクは時間が経っても変わることはなく、ブックマークとして使用したり、他のユーザに電子メールで送信したりすることができます。必要に応じて、コピーして貼り付けます。

フォルダの［フッタ］ツールバー

  • フォルダのパーマリンク: フォルダを参照できます。これらのリンクは時間が経っても変わることはなく、ブックマークとして使用したり、他のユーザに電子メールで送信したりすることができます。必要に応じて、コピーして貼り付けます。

  • 電子メールアドレス: わかりやすいURLを使用するようにすでにフォルダが設定されている場合、［フッタ］リンクをクリックすると、［フッタ］ツールバーにこのURLが表示されます。

    電子メール形態でエントリを受信するフォルダを設定する方法の詳細については、『Novell Teaming 2.1 User Guide』の「Setting Up a Folder to Receive Entries Via E-Mail」を参照してください。

  • RSS URL: フォルダにRSSフィードとしてアクセスできます。RSSのURLをコピーしてRSSリーダに貼り付けるだけです。

  • ATOM URL: フォルダにATOMフィードとしてアクセスできます。ATOM URLをコピーしてATOMリーダーに簡単に貼り付けます。

  • WebDAV URL: WebブラウザとしてInternet Explorer 7以上を使用している場合は、［WebDAV URL］を使用して、Microsoft Officeおよび同様のアプリケーションからNovell Teamingのファイルに直接アクセスできます。

  • iCal URL: Teamingタスクおよびカレンダフォルダをデスクトップカレンダアプリケーションと同期できます。

    カレンダフォルダまたはエントリを表示する際に、このリンクを使用してTeamingのカレンダデータをエクスポートし、iCal形式をサポートする別のカレンダ(Google、Outlookなど)に含めることができます。リンクをクリックして、ファイルにファイル拡張子.icsを付けて保存し、他のカレンダにファイルをインポートします(詳細については、他のカレンダのヘルプを参照してください)。

  • フォルダをWebDAVフォルダとして表示: フォルダをWebDAVフォルダとして表示できます。

    詳細については、『Novell Teaming 2.1 User Guide』の「Accessing WebDAV Files from within Teaming」を参照してください。

エントリの［フッタ］ツールバー

  • エントリのパーマリンク: エントリを参照できます。これらのリンクは時間が経っても変わることはなく、ブックマークとして使用したり、他のユーザに電子メールで送信したりすることができます。必要に応じて、コピーして貼り付けます。

  • iCal: (カレンダエントリのみ)Teamingタスクおよびカレンダフォルダをデスクトップカレンダアプリケーションに同期できます。

    カレンダフォルダまたはエントリを表示する際に、このリンクを使用してTeamingのカレンダデータをエクスポートし、iCal形式をサポートする別のカレンダ(Google、Outlookなど)に含めることができます。リンクをクリックして、ファイルにファイル拡張子.icsを付けて保存し、他のカレンダにファイルをインポートします(詳細については、他のカレンダのヘルプを参照してください)。

  • 電子メールを送信: 電子メールフォームを起動し、目的のユーザに電子メールを送信できます。電子メールには、Teaming内でのエントリの場所と、エントリの内容の一部が含まれます。

  • このエントリを購読: フォームを起動し、エントリを購読できます。エントリを購読するときには、そのエントリの更新を知らせる電子メール通知を送信するようにTeamingを設定します。

    エントリの購読の詳細については、電子メール通知の使用を参照してください。

  • このエントリのエクスポート: Teaming管理者のみです。エントリをエクスポートできます。その後、エントリを別のTeamingサイトにインポートできます。

詳細については、「保証と著作権」を参照してください。
