すべてのTeamingユーザ向け

次のガイドには、すべてのNovell Teamingユーザにとって役立つ情報が記載されています。

  • Novell Teaming 2.1 User Guide

    このガイドには、Teamingでさまざまなタスクを実行する方法について説明した詳しい情報が記載されています。概要の説明、ナビゲーション方法、カスタマイズ方法、およびグラフィックが含まれます。

  • Novell Teaming 2.1 Advanced User Guide

    このガイドは、ワークスペース所有者およびフォルダ所有者を対象に、ワークスペースを使用するユーザがワークスペースを効果的に使用できるように設定する方法について説明しています。たとえば、チームリーダーやプロジェクトマネージャなどが対象読者となります。

Teaming管理者向け

次のガイドには、Novell Teamingの管理者にとって役立つ情報が記載されています。

  • Kablink Teaming 2.1 Administration Guide

    このガイドでは、管理者がTeamingサイトをセットアップ、設定、および運用するために必要なタスクについて説明しています。

  • Novell Teaming 2.1 Advanced User Guide

    このガイドは、ワークスペース所有者およびフォルダ所有者を対象に、ワークスペースを使用するユーザがワークスペースを効果的に使用できるように設定する方法について説明しています。たとえば、チームリーダーやプロジェクトマネージャなどが対象読者となります。

  • Novell Teaming 2.1 Installation Guide

    このガイドでは、すべてのデフォルト機能が正しく動作するように、Teamingをインストールするために必要なタスクについて説明しています。さらに、Teamingユーザのセットアップ方法についても説明しています。

  • Novell Teaming 2.1 Developer Guide

    このガイドは、他のアプリケーションで相互作用できるように、 Teamingまたはプログラムの機能を拡張することを望んでいる開発者を主に対象としています。

詳細については、「保証と著作権」を参照してください。
