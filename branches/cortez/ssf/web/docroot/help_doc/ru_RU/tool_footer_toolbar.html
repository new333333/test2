Нижняя панель инструментов

Содержимое нижней панели инструментов изменяется в зависимости от того, используется ли рабочая область, папка или запись

Нижняя панель инструментов рабочей области

  • Постоянные ссылки на рабочую область Позволяют ссылаться на рабочую область. Такие ссылки не изменяются со временем и могут использоваться как закладки или отправляться по электронной почте другим пользователям. Их можно копировать и вставлять по мере надобности.

Нижняя панель инструментов папки

  • Постоянные ссылки на папку Позволяют ссылаться на папку. Такие ссылки не изменяются со временем и могут использоваться как закладки или отправляться по электронной почте другим пользователям. Их можно копировать и вставлять по мере необходимости.

  • Адреса электронной почты Если папка уже настроена с удобным для пользователя URL-адресом, то при выборе ссылки нижнего колонтитула на нижней панели инструментов отображается этот URL-адрес.

    Дополнительные сведения о настройке папки на получение записей через электронную почту см. в разделе Настройка папки на получение записей по электронной почте в Руководстве пользователя Novell Teaming 2.1.

  • URL-адрес RSS Позволяет получать доступ к папке в виде RSS-канала. Скопируйте URL-адрес RSS-канала и вставьте его в средство чтения RSS.

  • URL-адрес ATOM Позволяет получать доступ к папке в виде канала АТОМ. Скопируйте URL-адрес канала АТОМ и вставьте его в cредство для чтения АТОМ.

  • WebDAV URL Если используется веб-навигатор Internet Explorer 7 или более поздней версии, можно использовать URL-адрес WebDAV для доступа к файлам Novell Teaming непосредственно из Microsoft Office и аналогичных приложений.

  • URL-адрес iCal Позволяет синхронизировать задачу Teaming и папки календаря с приложением календаря на рабочей станции.

    При просмотре папки календаря или записи эту ссылку можно использовать, чтобы экспортировать данные календаря Teaming для включения в другой календарь, поддерживающий формат iCal (например, Google или Outlook). Щелкните файл, сохраните его с расширением имени .ics, а затем импортируйте файл в другой календарь (дополнительно см. в справке другого календаря).

  • Просмотреть папку как папку WebDAV Позволяет просматривать папку как папку WebDAV.

    Более подробные сведения см. в разделе Accessing WebDAV Files from within Teaming документа Руководство пользователя Novell Teaming 2.1.

Нижняя панель инструментов записи

  • Постоянные ссылки на запись Позволяют ссылаться на запись. Такие ссылки не изменяются со временем и могут использоваться как закладки или отправляться по электронной почте другим пользователям. Их можно копировать и вставлять по мере необходимости.

  • iCal (только для записей календаря) Позволяет синхронизировать папки задач и календаря Teaming с приложением календаря, которое используется на рабочей станции.

    При просмотре папки календаря или записи эту ссылку можно использовать, чтобы экспортировать данные календаря Teaming для включения в другой календарь, поддерживающий формат iCal (например, Google или Outlook). Щелкните файл, сохраните его с расширением имени .ics, а затем импортируйте файл в другой календарь (дополнительно см. в справке другого календаря).

  • Отправить электронную почту Запускает форму электронной почты, позволяющую отправлять сообщения любому пользователю. В сообщение электронной почты включается расположение записи и некоторое содержимое записи.

  • Подписаться на запись Запускает форму, позволяющую подписаться на запись. При подписке на запись Teaming настраивается на отправку по электронной почте оповещений об обновлениях этой записи.

    Дополнительно о подписке на запись см. Использование оповещений по электронной почте.

  • Экспортировать эту запись Только для администраторов Teaming. Позволяет экспортировать запись. Впоследствии можно импортировать запись на другой сайт Teaming.

Дополнительную информацию см. в Юридическая информация.
