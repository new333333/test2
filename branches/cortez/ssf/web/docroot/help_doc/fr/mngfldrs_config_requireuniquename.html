Exiger que tous les fichiers chargés dans ce dossier possèdent un nom unique

Si vous cochez cette case, certaines fonctions du produit considèrent ce dossier comme un dossier de fichiers. Par exemple, l'applet de téléchargement de fichiers crée des nouvelles entrées pour les noms de fichier uniques et des nouvelles versions des pièces jointes pour les fichiers existant déjà dans le dossier.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
