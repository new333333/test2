Autorisation des messages électroniques

Les sections suivantes décrivent les options de cette page.

Activation de la messagerie sortante

Vous pouvez modifier les options de configuration suivantes pour votre courrier électronique sortant :

  • Activer la messagerie sortante : cochez cette case pour activer tous les messages électroniques sortants de Teaming. Si vous ne souhaitez pas autoriser les utilisateurs à envoyer des messages à partir de Teaming, désélectionnez cette case.

    Cochez cette case si vous souhaitez autoriser les utilisateurs à envoyer des notifications digest.

  • Envoyer des notifications par message électronique (sortant) : utilisez cette section pour définir les valeurs par défaut de configuration des notifications digest. Si les utilisateurs Teaming veulent envoyer ou recevoir des notifications par message digest, ils doivent utiliser les paramètres de notification digest que vous sélectionnez ici. Vous pouvez définir que des notifications digest soient envoyées chaque jour, certains jours de la semaine, à un certain moment de la journée ou encore à une fréquence horaire spécifique.

Activation de la messagerie électronique entrante à l'aide du serveur de messagerie intégré et des URL simples

Cochez la case Activer la messagerie entrante à l'aide du serveur de messagerie intégré et des URL simples si vous voulez autoriser la soumission d'entrées par courrier électronique pour les dossiers qui ont été préconfigurés avec une URL simple.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
