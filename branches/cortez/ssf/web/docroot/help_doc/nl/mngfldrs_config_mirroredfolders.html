Gespiegelde mappen

Een gespiegelde map is een map waarvan de bestanden zich op een externe lokatie bevinden (of een map op een bestandssysteem of een map op een Sharepoint server). Als u externe mappen spiegelt vanuit Teaming kunt Novell Teaming hulpmiddelen (bijvoorbeeld zoeken en workflow) toepassen op externe bestanden.

U moet met een bestand op de server werken om deze functie te activeren.

Voor meer informatie over het aanmaken van gespiegelde mappen, zie Working with Mirrored Folders in de Novell Teaming 2.1 User Guide.
