Werkruimteweergave

Selecteer de weergave die het beste bij uw werkruimte past.

  • Discussiewerkruimte: Selecteer deze weergave als u een werkruimte wilt maken waar u verschillende discussieforums in kan plaatsen.

    Voor meer informatie over discussieworkspaces, zie Managing a Discussions Workspace in de Novell Teaming 2.1 User Guide.

  • Landingspagina: Selecteer deze weergave als u belangrijke informatie voor team voor teamleden wilt weergeven zodra zij de werkruimte betreden.

    Voor meer informatie, zie Managing a Landing Page in de Novell Teaming 2.1 User Guide.

  • Werkruimte voor projectbeheer: Selecteer deze weergave als u projecttaken en mijlpalen gebaseerd op het voltooien van taken wilt bijhouden.

    Voor meer informatie over workspaces voor projectmanagement, zie Managing a Project Management Workspace in de Novell Teaming 2.1 User Guide.

  • Teamwerkruimte: Selecteer deze weergave als uw teamwerkruimte organisatorisch van aard is en niet bedoeld om teamwerkruimtes als subplaatsen te bevatten.

    Voor meer informatie over teamwerkruimtes, zie Een teamwerkruimte beheren in de Novell Teaming 2.1 User Guide.

  • Teamwerkruimte hoofdstructuur: Selecteer deze weergave als u de Een teamwerkruimte toevoegen knop op uw teamwerkruimte wilt weergeven. Dit heeft de voorkeur als u aanvullende teamwerkruimtes wilt toevoegen aan deze werkruimte.

    Voor meer informatie over de hoofdstructuur van een teamworkspace, zie Creating and Managing a Team Workspace Root Workspace in de Novell Teaming 2.1 Advanced User Guide.

  • Werkruimte: Dit is een basiswerkruimte. U kunt kiezen welke mappen deze werkruimte moet bevatten.

    Voor meer informatie over basisworkspaces, zie Creating and Managing a Basic Workspace in de Novell Teaming 2.1 Advanced User Guide.

Als dit een persoonlijke werkruimte is is de standaard weergave Persoonlijke werkruimte en dit kan niet gewijzigd worden.

Voor meer informatie zie: Juridische kennisgevingen.
