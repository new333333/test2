Arbetsytvy

Välj den vy som bäst återger arbetsytan.

  • Diskussionsarbetsyta: Välj den här vyn om du vill skapa en arbetsyta där du kan lagra flera diskussionsforum.

    Mer information om diskussionsarbetsytor finns i avsnittet om att hantera ett diskussionsforum i Novell Teaming 2.1 User Guide.

  • Landningssida: Välj den här vyn om du vill visa viktig information för teammedlemmarna när de öppnar arbetsytan.

    Mer information om landningssidor finns i avsnittet om att hantera en landningssida i Novell Teaming 2.1 User Guide.

  • Projektledningsarbetsyta: Välj den här vyn om du vill spåra projektaktiviteter och milstolpar baserat på aktivitetsförloppet.

    Mer information om projektledningsarbetsytor finns i avsnittet om att hantera en projektledningsarbetsyta i Novell Teaming 2.1 User Guide.

  • Teamarbetsyta: Välj den här vyn om teamarbetsytan har en organisationsstruktur, och om den inte ska ha några teamarbetsytor som underordnade platser.

    Mer information om teamarbetsytor finns i avsnittet om att hantera en teamarbetsyta i Novell Teaming 2.1 User Guide.

  • Rot-teamarbetsyta: Välj den här vyn om du vill att knappen Lägg till en teamarbetsyta ska visas på teamarbetsytan. Det här är praktiskt om du vill skapa fler teamarbetsytor som underordnade platser till arbetsytan.

    Mer information om rot-teamarbetsytor finns i avsnittet om att skapa och hantera en rot-teamarbetsyta i Novell Teaming 2.1 Advanced User Guide.

  • Arbetsyta: Det här är en enkel arbetsyta. Du kan välja vilka mappar som arbetsytan ska innehålla.

    Mer information om enkla arbetsytor finns i avsnittet om att skapa och hantera en enkel arbetsyta i Novell Teaming 2.1 Advanced User Guide.

Om det här är en personlig arbetsyta används vyn Användararbetsyta som standard. Vyn kan inte ändras.

Mer information finns i Juridisk information.
