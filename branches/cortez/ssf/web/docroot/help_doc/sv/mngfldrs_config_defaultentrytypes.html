Standardposttyper

I det här området kan du välja vilka typer av poster som användarna ska kunna skapa i den här mappen. Du kan konfigurera mappen så att ytterligare posttyper tillåts, eller ändra den aktuella posttypen. Om du lägger till fler posttyper kan användare som skickar poster till mappen välja vilken typ av post de vill lägga till, t.ex. en uppgiftspost eller en kalenderpost.

  1. Välj de posttyper som du vill tillåta för mappen.

  2. Klicka på Verkställ > Stäng.

Om du aktiverar fler posttyper kan användare som skickar poster till den här mappen klicka på Lägg till i verktygsfältet Mappost och välja den typ av post som de vill lägga till från listrutan, som innehåller de typer av poster som du har aktiverat för mappen.

Webbplatsadministratörer och mappägare kan använda funktionen Formulär- och vydesigners för att skapa nya posttyper (anpassade formulär), som sedan visas i den här listan. (När mappägare skapar nya posttyper visas dessa under Lokala definitioner i listan.)

Mer information om hur du skapar anpassade posttyper finns i avsnittet om att skapa inmatningsformulär för anpassade mappar i Novell Teaming 2.1 Advanced User Guide.
