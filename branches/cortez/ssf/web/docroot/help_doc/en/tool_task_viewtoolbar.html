View Toolbar

The View toolbar enables you to choose the kinds of tasks that you want to display in the task list. Unless you are in the personal workspace of a Teaming user, you can view only those tasks that are located in the current folder.

  • Closed: Displays tasks that have the status of Closed.

  • Today: Displays tasks that have a due date of today.

  • Week: Displays tasks that have a due date sometime in the next seven days.

  • Month: Displays tasks that have a due date sometime in the next 30 days.

  • All Active: Displays tasks that are currently active.

  • All Entries: Displays all tasks.

  • Assigned Tasks: Available only in user workspaces. Displays all tasks assigned to the owner of the workspace, regardless of where the tasks are located in Teaming.

  • From Folder: Available only in user workspaces. Displays all tasks that are in the current folder.
