Definiowanie adresu URL dla folderu albo obszaru roboczego

Części tej należy użyć, aby zdefiniować adres URL bieżącego folderu lub obszaru roboczego w formie przyjaznej dla użytkownika. Zdefiniowanie tego adresu URL pozwala na bezpośrednie przejście do folderu lub obszaru roboczego z przeglądarki.

Zdefiniowanie adresu URL dla folderu pozwala dodawać wpisy do tego folderu poprzez pocztę elektroniczną lub wiadomość tekstową.

  1. Z listy rozwijanej Określ URL należy wybrać jedno z predefiniowanych globalnych słów kluczowych.

    Adres URL musi zaczynać się od jednego z tych predefiniowanych globalnych słów kluczowych, chyba że jest się administratorem.

  2. W tym polu należy określić żądany adres URL folderu lub obszaru roboczego.

  3. Kliknij przycisk Dodaj.

  4. (Warunkowo) W przypadku definiowania adresu URL dla folderu należy włączyć nowo zdefiniowany adres URL przez jego zaznaczenie. Adres URL, który należy zaznaczyć, znajduje się pod przyciskiem Dodaj.

  5. (Warunkowo) W przypadku definiowania adresu URL dla folderu należy kliknąć przycisk Zastosuj.

  6. Kliknij przycisk Zamknij.

Aby usunąć zdefiniowany adres URL, należy go zaznaczyć w części Obecnie zdefiniowane adresy URL, a następnie kliknąć Usuń zaznaczone adresy URL.

Aby uzyskać więcej informacji, patrz sekcja Informacje prawne.
