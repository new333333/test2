Barra degli strumenti nel piè di pagina

Il contenuto della barra degli strumenti nel piè di pagina cambia a seconda che si utilizzi uno spazio di lavoro, una cartella o una voce.

Barra degli strumenti nel piè di pagina dello spazio di lavoro

  • Collegamenti permanenti spazio di lavoro: consente di fare riferimento allo spazio di lavoro. Questi collegamenti rimangono costanti nel tempo ed è possibile utilizzarli come segnalibri o inviarli tramite e-mail ad altri utenti. Copiarli e incollarli secondo necessità.

Barra degli strumenti nel piè di pagina della cartella

  • Collegamenti permanenti cartella: consente di fare riferimento alla cartella. Questi collegamenti rimangono costanti nel tempo ed è possibile utilizzarli come segnalibri o inviarli tramite e-mail ad altri utenti. Copiarli e incollarli secondo necessità.

  • Indirizzi e-mail: se una cartella è già stata configurata con un URL descrittivo, questo viene visualizzato nella barra degli strumenti nel piè di pagina quando si fa clic sul collegamento al piè di pagina.

    Per ulteriori informazioni su come configurare una cartella per consentire tipi di voce alternativi, consultare Setting Up a Folder to Receive Entries Via E-Mail nella Novell Teaming 2.1 User Guide (in lingua inglese).

  • URL RSS: consente di accedere alla cartella come feed RSS. Copiare semplicemente l'URL RSS e incollarlo nel lettore RSS.

  • URL ATOM: consente di accedere alla cartella come feed ATOM. Copiare semplicemente l'URL ATOM e incollarlo nel lettore ATOM.

  • URL WebDAV: se si utilizza Internet Explorer 7 o una versione successiva come browser Web, è possibile utilizzare l'URL WebDAV per accedere ai file di Novell Teaming direttamente da Microsoft Office o applicazioni simili.

  • URL iCal: consente di sincronizzare le cartelle Task e Calendario di Teaming con l'applicazione calendario del desktop.

    Quando si visualizza una cartella Calendario o una voce, è possibile utilizzare questo collegamento per esportare i dati del calendario di Teaming per includerli in un altro calendario che supporti il formato iCal (ad esempio, Google e Outlook). Fare clic sul collegamento, salvare il file con estensione .ics, quindi importarlo nell'altro calendario (consultare la Guida per ulteriori informazioni sull'altro calendario).

  • Visualizza cartella come cartella WebDAV: consente di visualizzare la cartella come una cartella WebDAV.

    Per ulteriori informazioni, consultare Accessing WebDAV Files from within Teaming nella Novell Teaming 2.1 User Guide (in lingua inglese).

Barra degli strumenti nel piè di pagina della voce

  • Collegamento permanente voce: consente di fare riferimento alla voce. Questi collegamenti rimangono costanti nel tempo ed è possibile utilizzarli come segnalibri o inviarli tramite e-mail ad altri utenti. Copiarli e incollarli secondo necessità.

  • iCal: consente di sincronizzare le cartelle Task e Calendario di Teaming con l'applicazione calendario del desktop (solo per le voci del calendario).

    Quando si visualizza una cartella Calendario o una voce, è possibile utilizzare questo collegamento per esportare i dati del calendario di Teaming per includerli in un altro calendario che supporti il formato iCal (ad esempio, Google e Outlook). Fare clic sul collegamento, salvare il file con estensione .ics, quindi importarlo nell'altro calendario (consultare la Guida per ulteriori informazioni sull'altro calendario).

  • Invia e-mail: avvia un modulo e-mail che consente di inviare un'e-mail a chiunque. L'e-mail contiene l'ubicazione della voce all'interno di Teaming e parte del contenuto della voce.

  • Sottoscrivi la voce specificata: avvia un modulo che consente di effettuare la sottoscrizione della voce. Quando si effettua la sottoscrizione di una voce, è necessario configurare Teaming in modo da ricevere le notifiche e-mail sugli aggiornamenti della voce.

    Per ulteriori informazioni sulla sottoscrizione di una voce, vedere Utilizzo delle notifiche e-mail.

  • Esporta la voce: solo per amministratori di Teaming. Permette di esportare la voce. In seguito è possibile importare la voce in un altro sito Teaming.

Per ulteriori informazioni, vedere Note legali.
