Applicazione ricorsiva delle impostazioni della definizione

Utilizzare questa funzione per applicare le impostazioni della definizione a cartelle o voci create precedentemente.

Applicazione delle impostazioni della definizione a tutte le sottocartelle

È possibile applicare tutte le impostazioni della definizione di una cartella superiore a tutte le sottocartelle.

  1. Selezionare Modifica le cartelle contenute affinché ereditino le definizioni di questa cartella.

  2. Fare clic su Applica.

Tutte le impostazioni della definizione sulla pagina Configura impostazioni di default vengono applicate a tutte le sottocartelle.

Applicazione delle impostazioni della definizione alle voci esistenti

Se precedentemente sono state aggiunte voci alla cartella e il tipo di definizione delle voci non corrisponde a quello delle altre voci della cartella (ad esempio, è possibile che una cartella Blog contenga principalmente tipi di definizioni Blog, ma che alcune voci siano attualmente visualizzate come Discussione), questa sezione consente di modificare le impostazioni della definizione per uniformare tutte le voci della cartella.

  1. Selezionare Modifica le cartelle contenute affinché ereditino le definizioni di questa cartella, quindi selezionare Modifica le voci contenute perché siano del seguente tipo di voce:.

  2. Nell'elenco a discesa selezionare il tipo di voce da applicare a tutte le voci esistenti nella cartella.

    Per aggiungere i tipi di voce all'elenco a discesa, selezionare i tipi di voce nella sezione Tipi di voci di default, quindi fare clic su Applica.

  3. Fare clic su Applica.
