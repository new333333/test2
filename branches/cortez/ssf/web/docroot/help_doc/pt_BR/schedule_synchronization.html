Programando a sincronização

Esta página permite ativar e configurar uma programação de sincronização para as pastas espelhadas. Esta página contém as seguintes opções:

Habilitar Sincronização Programada: Você deve marcar essa caixa de seleção para realizar a sincronização programada.

Todos os Dias: Selecione essa opção se quiser que a sincronização seja feita todos os dias.

Semanal: Selecione essa opção se quiser especificar determinados dias da semana para realizar a sincronização.

No Horário: Especifique o horário certo para realizar a sincronização.

Repetir a Cada nn Horas: Especifique o intervalo entre as sincronizações.

Para obter mais informações, consulte Informações legais.
