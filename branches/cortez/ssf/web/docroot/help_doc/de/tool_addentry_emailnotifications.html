Verwenden von Email-Benachrichtigungen

Sie können Novell Teaming so konfigurieren, dass Sie Email-Benachrichtigungen für Ordner und deren Unterordner oder für einzelne Einträge erhalten. Dies bezeichnet man auch als "Abonnieren". Auch die Eigentümer von Ordnern können Teaming so konfigurieren, dass Sie Email-Benachrichtigungen erhalten. Allerdings haben die Abonnementeinstellungen der Benutzer Vorrang vor den Einstellungen der Ordnereigentümer. Daher können Sie mit diesem Formular alle übergeordneten Konfigurationen außer Kraft setzen.

Auswahl von Benachrichtigungstypen und Zustellungszielen

In diesem Formular können Sie angeben, welche Art von Benachrichtigungen Sie erhalten möchten und an welche Email-Adresse bzw. an welche Telefonnummer die Benachrichtigungen gesendet werden sollen.

Wählen Sie die Email-Adresse oder Telefonnummer aus, an die die Benachrichtigungen gesendet werden sollen. Das Zustellungsziel können Sie in jedem der folgenden Abschnitte (in denen die einzelnen Benachrichtigungstypen eingestellt werden) gesondert angeben:

  • Übersicht

  • Einzelne Nachrichten

  • Einzelne Nachrichten ohne Anlagen

  • Für Text-Messaging geeignete Nachricht

Weitere Informationen zu den Benachrichtigungstypen finden Sie im Abschnitt Auswahl der zu empfangenden Email-Benachrichtigungstypen.

Für jeden Benachrichtigungstyp können Sie auch mehrere Email-Adressen oder Telefonnummern angeben, indem Sie bei der Auswahl die Strg-Taste drücken.

Hinzufügen und Ändern von Zustellungszielen

Die Email-Adressen und Telefonnummern Ihrer Kontaktinformationen können Sie in Ihrem persönlichen Profil hinzufügen und ändern. So öffnen Sie Ihr persönliches Profil:

  1. Klicken Sie auf Mein Arbeitsbereich > Profil ändern.

  2. Geben Sie in den Feldern Email-Adresse, Mobile Email-Adresse und Email-Adresse für Text-Messaging die entsprechenden Zustellungsinformationen ein.

  3. Klicken Sie auf OK.

Wenn Sie nun die zu empfangenden Benachrichtigungstypen und Zustellungsziele auswählen, werden die neuen bzw. geänderten Kontaktinformationen angezeigt.

Auswahl der zu empfangenden Email-Benachrichtigungstypen

Sie können verschiedene Arten von Email-Benachrichtigungen empfangen:

  • Übersicht: Teaming sendet eine Nachricht mit einer Zusammenfassung der Aktivitäten, die im Ordner und seinen Unterordnern stattgefunden haben. Diese Option steht nur bei Ordnern zur Verfügung. Die Digest-Benachrichtigungen werden nach dem vom Website-Administrator festgelegten Zeitplan zugesendet.

  • Einzelne Nachrichten: Teaming sendet bei jedem neuen oder geänderten Eintrag eine Nachricht. Die Nachrichten enthalten jeweils auch Anlagen.

  • Einzelne Nachrichten ohne Anlagen: Teaming sendet bei jedem neuen oder geänderten Eintrag eine Nachricht. Etwaige Anlagen des Eintrags sind in den Email-Nachrichten nicht enthalten. Alle Arten von Einzelnachrichten werden jeweils alle fünf Minuten gesendet.

  • Nachricht für Text-Messaging geeignet: Teaming sendet eine Nachricht, die als Textnachricht angezeigt werden kann. Die Nachricht enthält nur den Link des neuen oder geänderten Eintrags. Textnachrichten werden in einem Zeitintervall von bis zu fünf Minuten gesendet.

Außerkraftsetzen vordefinierter Benachrichtigungen

Wenn Sie in diesem Formular Einstellungen vornehmen und Vordefinierte Benachrichtigungen durch diese Auswahl außer Kraft setzen auswählen, setzen Ihre eigenen Einstellungen diejenigen des Administrators bzw. der Ordnereigentümer außer Kraft.
