Symbolleiste "Ordnereintrag"

In der Symbolleiste "Ordnereintrag" zeigt Novell Teaming nur Optionen an, die für den aktuell angezeigten Ordner relevant sind. Allerdings können Sie die Symbolleiste "Ordnereintrag" auch so konfigurieren, dass sie die Optionen für mehrere Eintragsarten enthält. Wenn Sie zum Beispiel möchten, dass Ihr Wiki-Ordner Fotoeinträge enthält, können Sie den Ordner so konfigurieren, dass er auch Fotoeinträge zulässt.

Optionen in der Symbolleiste

Jeder Ordnertyp verfügt über Standardoptionen, die in der Symbolleiste "Ordnereintrag" automatisch angezeigt werden. Nachfolgend finden Sie eine Aufstellung der einzelnen Ordnertypen mit den Optionen, die in der Symbolleiste eines Eintrags des entsprechenden Ordners automatisch angezeigt werden.

Blog

  • Blog-Eintrag hinzufügen: Fügt Ihrem Blog-Ordner einen neuen Eintrag hinzu. Sie können dem Eintrag einen Titel, eine Beschreibung und Anlagen hinzufügen. Sie können den Eintrag auch abonnieren und bestimmten Benutzern eine Email senden, durch die sie benachrichtigt werden, dass Sie einen Eintrag hinzugefügt haben.

  • Neue Blog-Seite: Fügt dem vorhandenen Blog-Ordner einen neuen Blog-Ordner hinzu. Dieser neue Ordner ist Ihrem Haupt-Blog-Ordner untergeordnet. Durch neue Blog-Seiten können Sie Ihren Blog-Ordner in sinnvolle Themen unterteilen.

  • Sortieren nach: Sie können Ihre Blog-Einträge nach Nummer, Aktivitätsdatum, Autor, Status oder Titel sortieren.

  • Dateien zu Ordner hinzufügen: Stellt ein Symbol bereit, über das Sie Dateien mittels Ziehen und Ablegen oder Kopieren und Einfügen in Ihren Blog-Ordner verschieben können. Durch diese Aktion wird die Datei dem Ordner als neuer Blog-Eintrag hinzugefügt.

Kalender

  • Kalendereintrag hinzufügen: Fügt Ihrem Aufgabenordner einen neuen Eintrag hinzu. Sie können dem Eintrag einen Titel, eine Beschreibung und Anlagen hinzufügen. Sie können den Eintrag auch abonnieren, bestimmten Benutzern eine Email senden, durch die sie benachrichtigt werden, dass Sie einen Eintrag hinzugefügt haben, und andere Aufgaben durchführen.

  • Dateien zu Ordner hinzufügen: Stellt ein Symbol bereit, über das Sie Dateien mittels Ziehen und Ablegen oder Kopieren und Einfügen in Ihren Kalenderordner verschieben können. Durch diese Aktion wird die Datei dem Ordner als neuer Kalendereintrag hinzugefügt.

  • Anzeigen: In dieser Dropdown-Liste können Sie auswählen, welche Ereignisse im Kalender angezeigt werden sollen. Sie können zwischen folgenden Kategorien wählen:

    • Zugewiesene Ereignisse: Steht nur in Benutzerarbeitsbereichen zur Verfügung. Hiermit werden sämtliche Kalenderereignisse/-termine angezeigt, die dem Eigentümer des Arbeitsbereichs zugewiesen sind, unabhängig vom Speicherort der Ereignisse in Teaming.

    • In Ordner - Eigenschaften: Hiermit werden sämtliche Kalenderereignisse angezeigt, die sich an den Tagen, an denen die Ereignisse geplant sind, im aktuellen Ordner befinden.

    • In Ordner - Alle nach Erstellung: Hiermit werden sämtliche Kalenderereignisse an dem Datum angezeigt, an dem sie erstellt wurden.

    • In Ordner - Alle nach Aktivität: Hiermit werden sämtliche Kalenderereignisse an dem Datum angezeigt, an dem sie zuletzt geändert wurden.

Fotoalbum

  • Fotoeintrag hinzufügen: Fügt Ihrem Fotoalbumordner einen neuen Eintrag hinzu. Sie können Ihren Fotos und anderen Grafiken Namen geben, sie beschreiben und hochladen. Sie können bestimmten Benutzern auch eine Email senden, durch die sie benachrichtigt werden, dass Sie einen Eintrag hinzugefügt haben.

  • Neues Fotoalbum: Fügt dem aktuellen Fotoalbumordner einen neuen Ordner hinzu. Dieser neue Ordner ist Ihrem Haupt-Fotoalbumordner untergeordnet.

  • Sortieren nach: Sie können Ihre Bilder nach Aktivitätsdatum oder Titel sortieren:

    Wenn Sie nach Aktivitätsdatum sortieren, werden die zuletzt hochgeladenen Fotos am Anfang der Liste angezeigt und die Fotos, die sich am längsten in Ihrem Fotoalbum befinden, am Ende der Liste.

    Wenn Sie nach Titel sortieren, werden die Fotos in alphabetischer Reihenfolge nach ihren Namen sortiert.

  • Dateien zu Ordner hinzufügen: Stellt ein Symbol bereit, über das Sie Dateien mittels Ziehen und Ablegen oder Kopieren und Einfügen in Ihren Fotoalbumordner verschieben können. Durch diese Aktion wird die Datei dem Ordner als neuer Dateieintrag hinzugefügt.

Sie können bestimmten Benutzern auch eine Email senden, durch die sie benachrichtigt werden, dass Sie der Wiki-Seite einen Eintrag hinzugefügt haben.

  • Neues Wiki-Thema: Fügt dem aktuellen Wiki-Ordner einen neuen Ordner hinzu, den Sie benennen können. Dieser neue Ordner ist Ihrem Haupt-Wiki-Ordner untergeordnet.

  • Sortieren nach: Sie können die Wiki-Einträge nach Aktivitätsdatum oder Titel sortieren:

    Wenn Sie nach Aktivitätsdatum sortieren, werden die neuesten Einträge im Wiki-Ordner am Anfang der Liste und die ältesten Einträge am Ende der Liste angezeigt.

    Wenn Sie nach Titel sortieren, werden die Wiki-Einträge in alphabetischer Reihenfolge nach ihren Titeln sortiert.

  • Dateien zu Ordner hinzufügen: Stellt ein Symbol bereit, über das Sie Dateien mittels Ziehen und Ablegen oder Kopieren und Einfügen in Ihren Wiki-Ordner verschieben können. Durch diese Aktion wird die Datei dem Ordner als neuer Wiki-Eintrag hinzugefügt. Sie wird im Abschnitt Wiki-Seiten von Seitenleiste "Wiki" angezeigt.

Zulassen alternativer Eintragsarten in einem Ordner

  1. Navigieren Sie zu dem Ordner, dem Sie die alternative Eintragsart hinzufügen möchten.

  2. Klicken Sie auf Verwalten > Konfigurieren.

  3. Wählen Sie im Abschnitt Standardeintragsarten die Ordner aus, die den Eintragsarten entsprechen, die Sie in diesem Ordner zulassen möchten.

  4. Klicken Sie auf Anwenden > Schließen.

  5. Klicken Sie in der Symbolleiste "Ordnereintrag" auf Hinzufügen und wählen Sie in der Dropdown-Liste die Eintragsart aus, die Sie hinzufügen möchten.
