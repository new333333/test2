Erstellen Ihrer Marke

Wenn Sie eine Grafik mit Text einfügen möchten, können Sie im Markenbearbeitungsbereich eine aus zwei Spalten und einer Zeile bestehende Tabelle einfügen. Die Grafik fügen Sie in der linken Zelle, den Text in der rechten Zelle ein.

So fügen Sie eine Tabelle ein:

  1. Klicken Sie auf das Symbol Tabelle einfügen .

  2. Geben Sie die Tabelleneigenschaften an und klicken Sie auf Einfügen.

Ein optimales Ergebnis erhalten Sie, wenn die Bildhöhe nicht größer als 100 Pixel ist.
