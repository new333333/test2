Übernehmen von Definitionseinstellungen

In diesem Abschnitt können Sie angeben, ob Sie Definitionseinstellungen von einem übergeordneten Ordner übernehmen möchten.

  1. Klicken Sie auf Ja > Anwenden, wenn Sie die Definitionseinstellungen übernehmen möchten.

    oder

    Klicken Sie auf Nein > Anwenden, wenn Sie die Definitionseinstellungen nicht übernehmen möchten.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
