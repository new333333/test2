Verwaltungswerkzeuge für Arbeitsbereiche

Wenn Sie in der Symbolleiste eines Arbeitsbereichs auf Verwalten klicken, zeigt Novell Teaming nur die Verwaltungs- und Konfigurationswerkzeuge an, die zu verwenden Sie berechtigt sind. Die Zugriffsberechtigungen für diese Werkzeuge werden von den Administratoren der Website bzw. vom Eigentümer des jeweiligen Arbeitsbereichs erteilt.

Bei uneingeschränktem Zugriff werden die folgenden Optionen angezeigt:

  • Zugriffssteuerung: Damit können Sie festlegen, wer diesen Arbeitsbereich anzeigen und dazu beitragen darf.

  • Aktivitätsberichte: Damit können Sie die Aktivitäten anzeigen, die im Arbeitsbereich stattgefunden haben.

  • Konfigurieren: Damit können Sie den Arbeitsbereich konfigurieren. Unter anderem können Sie die Standardansicht und die URL des Arbeitsbereichs festlegen.

  • Diesen Arbeitsbereich kopieren: Damit können Sie den aktuellen Arbeitsbereich, sofern Sie über die entsprechenden Berechtigungen verfügen, an jedes beliebige Ziel innerhalb des Arbeitsbereichsbaums kopieren. Weitere Informationen hierzu finden Sie unter Navigation im Arbeitsbereichsbaum.

  • Diesen Arbeitsbereich löschen: Damit können Sie den aktuellen Arbeitsbereich löschen.

  • Exportieren/Importieren: (Nur Teaming-Administratoren) Hiermit können Sie den aktuellen Arbeitsbereich exportieren bzw. einen Ordner oder Arbeitsbereich in den aktuellen Arbeitsbereich importieren.

    Weitere Informationen zum Exportieren und Importieren von Arbeitsbereichen und Ordnern in Teaming finden Sie im Kablink Teaming 2.1 Administration Guide im Abschnitt zum Exportieren/Importieren von Arbeitsbereichen, Ordnern und Einträgen.

  • Formular- und Ansichtsdesigner: Damit können Sie benutzerdefinierte Eintragsformulare und Workflows erstellen, die Sie auf den Arbeitsbereich und seine Unterordner anwenden können.

  • Diesen Arbeitsbereich ändern: Damit können Sie den Arbeitsbereich ändern. Unter anderem können Sie den Namen, die Beschreibung, die Landeseite, die Marke und das Symbol des Arbeitsbereichs ändern.

  • Diesen Arbeitsbereich verschieben: Damit können Sie den aktuellen Arbeitsbereich, sofern Sie über die entsprechenden Berechtigungen verfügen, an jedes beliebige Ziel innerhalb des Arbeitsbereichsbaums verschieben. Weitere Informationen hierzu finden Sie unter Navigation im Arbeitsbereichsbaum.

  • Neuen Arbeitsbereich hinzufügen: Damit können Sie unter dem aktuellen Arbeitsbereich einen neuen Arbeitsbereich hinzufügen.

  • Neuen Ordner hinzufügen: Damit können Sie dem Arbeitsbereich einen neuen Ordner hinzufügen.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
