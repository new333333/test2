Erstellen eines Zeitplans zur Synchronisierung

Auf dieser Seite können Sie einen Synchronisierungszeitplan für Ihre gespiegelten Ordner aktivieren und einrichten. Diese Seite enthält folgende Optionen:

Geplante Synchronisierung aktivieren: Sie müssen dieses Kontrollkästchen aktivieren, damit die Synchronisierung geplant ausgeführt wird.

Jeden Tag: Wählen Sie diese Option aus, wenn die Synchronisierung jeden Tag durchgeführt werden soll.

Wöchentlich: Wählen Sie diese Option aus, wenn Sie bestimmte Wochentage angeben möchten, an denen die Synchronisierung durchgeführt werden soll.

Uhrzeit: Geben Sie eine bestimmte Uhrzeit an, zu der die Synchronisierung durchgeführt werden soll.

Alle nn Stunden wiederholen: Geben Sie das Intervall zwischen den Synchronisierungen an.
