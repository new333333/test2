Novell Teaming 1.0.3 Installation Guide

This guide covers the installation of Novell® Teaming. The term “ICEcore” appears in some file and directory names and refers to the application technology that enables the Novell Teaming software. In this guide, the term “Novell Teaming” applies to all versions of the Novell Teaming software unless otherwise noted.

This guide provides information about Installing Novell Teaming.

Audience

This guide is intended for Novell Teaming administrators.

Software and Documentation Version

This guide describes features in Novell Teaming Version 1.0.3. This is the first version of this guide.

Conventions

This guide uses the following conventions:

A greater-than symbol (>) is used to separate actions within a step and items in a cross-reference path.

A trademark symbol (®, ™, etc.) denotes a Novell trademark. An asterisk (*) denotes a third-party trademark.

When a single pathname can be written with a backslash for some platforms or a forward slash for other platforms, the pathname is presented with a backslash. Users of platforms that require a forward slash, such as Linux* or UNIX*, should use forward slashes as required by your software.

A trademark symbol (®, TM, etc.) denotes a Novell trademark. An asterisk (*) denotes a third-party trademark. For more information, see Legal Notices.
