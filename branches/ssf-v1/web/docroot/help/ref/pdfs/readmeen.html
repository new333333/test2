Novell Teaming 1.0

1.0 Getting Started

Complete system requirements and detailed installation instructions for the Teaming administrator are provided in the Teaming 1.0 Installation Guide. Ongoing Teaming site administration and maintenance instructions are available in the Teaming 1.0 Administration Guide.

After your Teaming site is set up, Teaming users can obtain instructions in the Teaming 1.0 Quick Start and the Teaming 1.0 User Guide.

You can display the Teaming documentation, as well as online help, when you log in to your Teaming site.
