Редактирование прикрепленных файлов

Kablink Teaming позволяет изменять файлы, прикрепленные к записям.

Если Вы не можете изменять файлы, то, возможно, Teaming не настроен на поддержку установленного на Вашем компьютере приложения, которое используется для редактирования. Например, если Teaming настраивается на использование Microsoft Word как редактора текстовых файлов по умолчанию, но на компьютере вместо Microsoft Word установлен OpenOffice, Teaming возвращает ошибку и не позволяет редактировать документ.

Чтобы устранить эту проблему, Teaming позволяет переопределить настройку редактора по умолчанию.

Изменение настройки редактора по умолчанию для одного типа файлов

  1. На главной боковой панели в разделе Личные настройки щелкните Определить переопределения редактора.

  2. Нажмите кнопку Добавить.

  3. В раскрывающемся списке выберите расширение имени файла, для которого требуется сменить приложение редактирования по умолчанию.

  4. В поле Приложение укажите имя приложения, которое следует использовать для редактирования файлов, выбранных в Шаг 3.

  5. Установите флажок и нажмите кнопку ОК.

Изменение настройки редактора по умолчанию для всех типов файлов

Можно настроить Teaming на использование одного приложения для редактирования всех типов файлов.

  1. На главной боковой панели в разделе Личные настройки щелкните Определить переопределения редактора.

  2. Щелкните Использовать и выберите приложение, которое Teaming будет использовать для редактирования всех типов файлов.

  3. Установите флажок Расширение и нажмите кнопку OK.

Дополнительную информацию см. в Юридическая информация.
