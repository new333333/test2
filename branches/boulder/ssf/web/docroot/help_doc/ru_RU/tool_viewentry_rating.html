Оценка записей

Как и многие приложения для управления знаниями и социальных сетей, Kablink Teaming предоставляет пользователям возможность определить наиболее популярные записи. Для этого Teaming предлагает пятизвездный инструмент рейтинга.

Просмотр рейтингов записей

При просмотре записи Teaming отображает желтые звезды, показывающие средний рейтинг записи. Одна звезда представляет самый низкий, а пять — самый высокий рейтинг. Если все звезды являются серыми, запись не оценена.

Отправка рейтингов записи

Чтобы отправить свой рейтинг, щелкните звезду. (Например, чтобы отправить рейтинг "три звезды", щелкните третью слева звезду.) Свой рейтинг Вы можете изменять сколь угодно часто, но при расчете среднего Teaming использует только самый последний рейтинг.

Дополнительную информацию см. в Юридическая информация.
