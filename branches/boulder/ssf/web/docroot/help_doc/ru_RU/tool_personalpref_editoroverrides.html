Переопределения редактора

Для редактирования вложения в Kablink Teaming используются приложения по умолчанию системы Teaming. Однако настоящая таблица позволяет переопределить приложения, используемые по умолчанию для редактирования в Teaming файлов с определенными расширениями имени.

Можно переопределить приложение по умолчанию для файлов одного типа или переопределить приложение по умолчанию для нескольких типов файлов.

Переопределение приложения по умолчанию для одного типа расширений

Чтобы переопределять приложение по умолчанию для одного типа расширений имени файла, выполните следующие действия.

  1. Нажмите кнопку Добавить.

  2. В раскрывающемся списке выберите тип расширения имени файла, для которого необходимо переопределить приложение для редактирования по умолчанию.

  3. В поле Приложение укажите имя приложения, которое следует использовать для редактирования вложений с расширением имени файла, выбранным в Шаг 2.

  4. Нажмите кнопку ОК.

Переопределение приложения по умолчанию для всех типов расширений

Чтобы переопределить все типы расширений, выполните следующие действия.

  1. Нажмите кнопку Использовать.

  2. В раскрывающемся списке выберите приложение, которое следует использовать для редактирования вложений с любыми расширениями имени файла.

  3. Нажмите кнопку ОК.

Дополнительную информацию см. в Юридическая информация.
