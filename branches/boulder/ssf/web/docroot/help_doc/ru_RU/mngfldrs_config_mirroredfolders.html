Отраженные папки

Отраженная папка является библиотечной папкой, файлы которой размещаются во внешней системе (в каталоге файловой системы или в папке на сервере SharePoint). Когда внешние папки отражаются из Teaming, к внешним файлам можно применять инструменты Kablink Teaming (например, поиск и рабочие потоки).

Для включения этой функции необходимо работать с файлом, размещенным на сервере.

Дополнительно о создании отраженных папок см. в разделе Работа с отраженными папками в Расширенном руководстве пользователя Kablink Teaming 2.0.

Дополнительную информацию см. в Юридическая информация.
