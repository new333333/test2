Документация по началу работы

Ссылки в этом разделе открывают руководства по началу работы в новом окне навигатора. Эти руководства можно читать в электронном виде в формате HTML или PDF. Формат PDF также оптимизирован для печати.

Все пользователи Teaming

Следующие руководства будут интересны всем пользователям Kablink Teaming:

  • Руководство пользователя Kablink Teaming 2.0

    В этом руководстве содержится подробная информация о выполнении различных задач в Teaming. В него включено общее описание, инструкции по навигации, инструкции по настройке и иллюстрации.

  • Расширенное руководство пользователя Kablink Teaming 2.0

    Настоящее руководство предназначено для владельцев рабочих областей и папок, которые являются ответственными за то, чтобы рабочие области и папки оказались доступными и полезными для тех, кто их использует. Примерами пользователей, которым может быть интересна эта информация, могут служить руководители команд и менеджеры проектов.

Администраторы Teaming

Следующие руководства представляют интерес для администраторов Kablink Teaming:

  • Руководство по администрированию Kablink Teaming 2.0

    Руководство содержит описание задач, необходимых для установки, настройки и поддержания сайта Teaming.

  • Расширенное руководство пользователя Kablink Teaming 2.0

    Настоящее руководство предназначено для владельцев рабочих областей и папок, которые являются ответственными за то, чтобы рабочие области и папки оказались доступными и полезными для тех, кто их использует. Примерами пользователей, которым может быть интересна эта информация, могут служить руководители команд и менеджеры проектов.

  • Руководство по установке Kablink Teaming 2.0

    Руководство содержит описание задач, необходимых для установки Teaming с заданными функциями по умолчанию. В нем также описываются способы настройки пользователей Teaming.

  • Руководство разработчика Kablink Teaming 2.0

    Руководство предназначено в первую очередь для разработчиков, стремящихся расширить функциональные возможности портала Kablink Teaming или запрограммировать другие приложения для взаимодействия с ним.
