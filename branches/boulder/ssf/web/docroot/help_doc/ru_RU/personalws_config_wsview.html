Представление рабочей области

Выберите наилучшее представление рабочей области.

  • Рабочая область обсуждений Выберите это представление, если требуется создать рабочую область, в которой можно сохранять несколько дискуссионных форумов.

    Дополнительные сведения см. в разделе Управление рабочей областью обсуждений в Расширенном руководстве пользователя Kablink Teaming 2.0.

  • Целевая страница Выберите это представление, если требуется отображать важную информацию для участников команды, как только они входят в рабочую область.

    Дополнительные сведения см. в разделе Создание целевой страницы в Расширенном руководстве пользователя Kablink Teaming 2.0.

  • Рабочая область управления проектами Выберите это представление, если требуется отслеживать задачи проекта и этапы на основании завершения выполнения задач.

    Дополнительные сведения см. в разделе Управление рабочей областью управления проектами в Расширенном руководстве пользователя Kablink Teaming 2.0.

  • Рабочая область команды Выберите это представление, если рабочая область команды является организационной по своей структуре и не должна содержать вложенные рабочие области команд.

    Дополнительные сведения см. в разделе Управление рабочей областью команды в Расширенном руководстве пользователя Kablink Teaming 2.0.

  • Корень рабочей области команды Выберите это представление, если требуется отображать кнопку Добавить рабочую область команды в Вашей рабочей области команды. Это полезно, если требуется создавать дополнительные рабочие области команд, вложенные в эту рабочую область.

    Дополнительные сведения см. в разделе Управление корневой областью рабочей области команды в Расширенном руководстве пользователя Kablink Teaming 2.0.

  • Рабочая область Это базовая рабочая область. Вы можете выбирать, какие папки должна содержать рабочая область.

    Дополнительные сведения см. в разделе Управление базовой рабочей областью в Расширенном руководстве пользователя Kablink Teaming 2.0.

Если это персональная рабочая область, по умолчанию используется представление Рабочая область пользователя, и оно не может быть изменено.

Дополнительную информацию см. в Юридическая информация.
