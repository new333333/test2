Использование оповещений по электронной почте

Можно настроить Kablink Teaming на отправку Вам оповещений по электронной почте для папок, вложенных папок или отдельных записей (это называется "подпиской" на папку или запись). Кроме того, владельцы папок могут настраивать Teaming на отправку оповещений по электронной почте другим пользователям. Однако, поскольку индивидуальная настройка подписки пользователя переопределяет настройку владельца папки, Вы можете использовать эту форму, чтобы переопределить любые конфигурации верхнего уровня.

Выбор типов оповещений и мест доставки

Эта форма позволяет выбрать тип получаемых оповещений, а также адрес электронной почты или телефон, на которые будут отправляться оповещения.

Выберите адрес электронной почты или телефон, на которые следует отправлять оповещения. Места доставки оповещений можно выбрать в любом из разделов, представляющих типы оповещений:

  • Сводка

  • Индивидуальные сообщения

  • Индивидуальные сообщения без вложений

  • Сообщение подходит для текстовой отправки

Дополнительно о типах оповещений см. Выбор типов оповещений по электронной почте для получения.

Можно выбрать несколько мест доставки по электронной почте или телефону для каждого типа оповещений, удерживая нажатой клавишу Ctrl и выбирая почтовые адреса или телефоны.

Добавление и изменение мест доставки

В своем персональном профиле можно изменять контактную информацию об электронной почте и телефонном номере. Чтобы получить доступ к своему персональному профилю, выполните следующие действия.

  1. Щелкните Моя рабочая область > Изменить профиль.

  2. Укажите места доставки в полях Адрес электронной почты, Мобильный адрес электронной почты и Электронный адрес текстовых сообщений.

  3. Нажмите кнопку ОК.

После выбора типов оповещений и мест доставки отображается новая или измененная контактная информация.

Выбор типов оповещений по электронной почте для получения

Вы можете получать различные типы оповещений по электронной почте.

  • Сводка Teaming отправляет одно сообщение, содержащее сводку всех действий в папке и вложенных в нее папках (доступно только для папок). Teaming отправляет оповещения со сводками по расписанию, установленному администратором сайта.

  • Индивидуальные сообщения с вложениями Teaming отправляет одно сообщение для каждой новой или измененной записи; сообщения содержат вложенные файлы.

  • Индивидуальные сообщения без вложений Teaming отправляет одно сообщение для каждой новой или измененной записи. Сообщения электронной почты не включают файлы, которые могут быть прикреплены к новым или измененным записям. Teaming отправляет все типы индивидуальных оповещений каждые пять минут.

  • Сообщение подходит для текстовой отправки Teaming отправляет сообщение, подходящее для просмотра в виде текстового сообщения. В сообщение включается только ссылка в виде заголовка новой или измененной записи. Teaming отправляет текстовые оповещения каждые пять минут.

Переопределение настройки, установленной владельцем папки

Если выбрать параметры в этой форме и щелкнуть Отключить оповещения, заданные администратором, Ваши параметры переопределяют параметры, заданные администратором или владельцами папок.

Дополнительную информацию см. в Юридическая информация.
