编辑附加的文件

Kablink Teaming 允许您编辑附加到条目的文件。

如果无法编辑文件，可能是因为未将 Teaming 配置为支持您计算机上安装的编辑应用程序。例如，如果已将 Teaming 配置为使用 Microsoft Word 作为所有文本文件的默认编辑器，但您计算机上安装的是 OpenOffice 而非 Microsoft Word，则 Teaming 会显示错误而不会编辑文档。

要解决此问题，Teaming 允许您覆盖默认编辑器设置。

更改单个文件类型的默认编辑器设置

  1. 在主侧栏的个人自选设置部分中，单击定义编辑器覆盖。

  2. 单击添加。

  3. 从下拉列表中选择要更改默认编辑应用程序的文件扩展名。

  4. 在应用程序字段中，指定要用于编辑在步骤 3中所选文件类型的应用程序名称。

  5. 选择此复选框，然后单击确定。

更改所有文件类型的默认编辑器设置

可以将 Teaming 配置为对所有文件类型都使用相同的编辑应用程序。

  1. 在主侧栏的个人自选设置部分中，单击定义编辑器覆盖。

  2. 单击使用，然后选择希望 Teaming 用于编辑所有文件类型的应用程序。

  3. 选中扩展名左边的复选框，然后单击确定。
