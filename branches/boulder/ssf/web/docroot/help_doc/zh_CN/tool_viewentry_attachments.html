对条目附加文件

在条目的附件部分中，Kablink Teaming 提供了多种选择用于对条目添加附件：

拖放文件

可以通过将文件从其他位置拖放到条目中来对条目附加文件。

  1. 单击拖放文件图标。

    此时会起动一个带有文件夹图标的窗口。

  2. 请将要上载的文件拖放到此文件夹图标上。

    Teaming 会显示一个打开又关闭的文件夹，表示您拖放的文件正在作为附件上载。

现在已将文件上载至条目。

使用 WebDAV 访问附件（仅适用于 Internet Explorer）

Teaming 允许使用 WebDAV 访问储存在远程 Web 服务器上的文件并将它们附加到条目。但 Teaming 仅对 Internet Explorer 浏览器支持 WebDAV。如果您使用的浏览器不是 Internet Explorer，则不会显示此选项。

  1. 单击使用 WebDAV 访问附件。

  2. 导航至要附加的文件并将其选定。

有关在 Teaming 中使用 WebDAV 的更多信息，请参见通过 WebDAV 访问附件（仅适用于 Internet Explorer）![我并不完全确定它仅适用于 IE。]，此部分内容包含在《Kablink Teaming 2.0 用户指南》中。

上载单个文件

  1. 在要上载文件的条目中，单击附件部分中的上载单个文件图标。

  2. 单击浏览，然后浏览至要附加的文件并将其选定。

  3. 单击确定。

有关更多信息，请参见 法律声明.
