为文件夹或工作空间定义 URL

使用此部分可为当前所在的文件夹或工作空间定义用户友好的 URL。通过定义此 URL，您可以从浏览器直接导航至文件夹或工作空间。

通过为文件夹定义 URL，您可以通过电子邮件或文本讯息向此文件夹添加条目。

  1. 在定义 URL 下拉列表中，选择某个预定义的全局关键字。

    除非是站点管理员，否则 URL 必须以这些预定义全局关键字中的一个开头。

  2. 在字段中，为文件夹或工作空间指定所需的 URL。

  3. 单击添加。

  4. （视条件而定）如果是为文件夹定义 URL，则必须通过选择刚才定义的 URL 来启用此 URL。需要选择的 URL 位于添加按钮下方。

  5. （视条件而定）如果是为文件夹定义 URL，请单击应用。

  6. 单击关闭。

要删除定义的 URL，请在当前定义的 URL 部分中选择此 URL，然后单击删除选定的 URL。

有关更多信息，请参见 法律声明.
