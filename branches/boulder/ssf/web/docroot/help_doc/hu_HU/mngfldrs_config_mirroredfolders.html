Tükrözött mappák

A tükrözött mappa egy könyvtár mappa, amelynek fájljai egy külső rendszerben helyezkednek el, (vagy egy fájlrendszer könyvtárában, vagy egy SharePoint szerver mappájában). Amikor külső mappákat tükröz a Teamingbe, akkor a külső fájlok esetében is használhatja a Kablink Teaming eszközöket, (például keresés és munkafolyamat).

A szolgáltatás engedélyezéséhez a kiszolgálón található fájllal kell dolgoznia.

Tükrözött mappák létrehozásáról további információt talál a Kablink Teaming 2.0 haladó felhasználói kézikönyv Tükrözött mappák használata című részében.
