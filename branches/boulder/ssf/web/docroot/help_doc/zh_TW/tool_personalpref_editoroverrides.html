編輯器置換

在 Kablink Teaming 中編輯附件時，將使用 Teaming 系統的預設應用程式來編輯附件。不過，此表格可讓您置換 Teaming 中具有特定副檔名之附件的預設編輯應用程式。

您可以置換單一副檔名類型的預設編輯應用程式，也可以置換多個副檔名類型的預設應用程式。

置換個別副檔名的預設編輯應用程式

若要置換個別副檔名類型的預設應用程式：

  1. 按一下「新增」。

  2. 從下拉式清單中，選取要置換預設編輯應用程式的副檔名類型。

  3. 在「應用程式」欄位中，指定要用於編輯具有指定副檔名 (在步驟 2中指定) 之附件的應用程式名稱。

  4. 按一下「確定」。

置換所有副檔名的預設編輯應用程式

若要置換所有副檔名類型：

  1. 按一下「使用」。

  2. 從下拉式清單中，選取在 Teaming 中要用於編輯各種檔案類型附件的應用程式。

  3. 按一下「確定」。
