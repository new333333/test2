傳送電子郵件通知

如果您是資料夾擁有者，可以將 Kablink Teaming 設定為向其他使用者傳送電子郵件通知，以就此資料夾或其子資料夾中的任何活動通知他們。不過，使用者的個別設定會置換或取消資料夾擁有者建立的通知。

  1. 在「選取電子郵件通知的類型」區段中，選取要傳送的電子郵件通知類型。

    如需各種電子郵件通知類型的相關資訊，請參閱決定要接收哪些類型的電子郵件通知。

  2. 指定要接收電子郵件通知的使用者。

    您可以指定特定的電子郵件地址、使用者和群組，也可以將通知傳送給所有團隊成員。

  3. 按一下「套用」>「關閉」。

如需電子郵件通知的詳細資訊，請參閱使用電子郵件通知。

有關更詳細的資訊，請參閱 法律聲明.
