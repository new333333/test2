Bewerten von Einträgen

Wie viele andere Knowledge Management- und Social Networking-Anwendungen bietet Kablink Teaming seinen Benutzern die Möglichkeit, den Beliebtheitsgrad seiner Einträge anzuzeigen. Zu diesem Zweck stellt Teaming ein Bewertungswerkzeug zur Verfügung, mit dem bis zu fünf Sterne vergeben werden können.

Anzeigen von Eintragsbewertungen

Teaming kennzeichnet jeden angezeigten Eintrag durch eine bestimmte Anzahl gelber Sterne, die die Durchschnittsbewertung dieses Eintrags angeben. Ein Stern steht dabei für die schlechteste Bewertung, fünf Sterne für die beste. Wenn alle Sterne grau angezeigt werden, wurde der Eintrag noch nicht bewertet.

Einsenden von Eintragsbewertungen

Wenn Sie eine Bewertung abgeben möchten, klicken Sie auf den entsprechenden Stern, zum Beispiel auf den dritten Stern von links, wenn Sie drei Sterne vergeben möchten. Sie können Ihre Bewertung jederzeit ändern. Zur Berechnung der Durchschnittsbewertung verwendet Teaming jedoch nur Ihre letzte Bewertung.
