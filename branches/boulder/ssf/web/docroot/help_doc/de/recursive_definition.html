Rekursives Anwenden von Definitionseinstellungen

Verwenden Sie diese Funktion, um Definitionseinstellungen auf Ordner oder Einträge anzuwenden, die bereits erstellt wurden.

Anwenden von Definitionseinstellungen auf alle Unterordner

Sie können alle Definitionseinstellungen eines übergeordneten Ordners auf alle Unterordner anwenden.

  1. Wählen Sie die Option Enthaltene Ordner ändern, um die Definitionen des Ordners zu übernehmen aus.

  2. Klicken Sie auf Anwenden.

Alle Definitionseinstellungen auf der Seite "Standardwerte konfigurieren" werden auf alle Unterordner angewendet.

Anwenden von Definitionseinstellungen auf vorhandene Einträge

Wenn Sie diesem Ordner bereits vorher Einträge hinzugefügt haben und der Eintragsdefinitionstyp nicht mit dem der anderen Ordnereinträge übereinstimmt (Sie könnten beispielsweise einen Blog-Ordner haben, der vorwiegend Blog-Definitionstypen enthält, doch einige Ihrer Einträge werden zurzeit mit dem Diskussions-Definitionstyp angezeigt), so können Sie diesen Abschnitt dazu verwenden, die Definitionseinstellungen zu ändern, um für alle Ordnereinträge den gleichen Wert festzulegen.

  1. Wählen Sie Enthaltene Ordner ändern, um die Definitionen des Ordners zu übernehmen und wählen Sie anschließend die Option Enthaltene Einträge in folgender Eintragsart ändern aus.

  2. Wählen Sie in der Dropdown-Liste die Eintragsart aus, die auf alle vorhandenen Einträge im Ordner angewendet werden soll.

    Wählen Sie zum Hinzufügen von Eintragsarten zu dieser Dropdown-Liste die Eintragsarten im Abschnitt Standardeintragsarten aus und klicken Sie anschließend auf Anwenden.

  3. Klicken Sie auf Übernehmen.
