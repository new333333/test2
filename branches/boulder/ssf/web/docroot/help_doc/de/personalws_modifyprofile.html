Profil ändern

Mit diesem Werkzeug können Sie Kontaktinformationen wie Passwort, Name, Telefonnummer, Email-Adresse und den für Kablink-Konferenzen verwendeten Benutzernamen ändern.

Außerdem können Sie mit diesem Werkzeug Bilder hochladen und sie mit den Pfeilen "Nach oben" und "Nach unten" verschieben oder sie löschen. Wenn Sie mehrere Bilder hochladen, zeigt Teaming das erste Bild in der Liste im Abschnitt Foto an. Alle weiteren Bilder werden verkleinert in einer Leiste unterhalb der Kontaktinformationen angezeigt. Wenn Sie den Mauszeiger auf eines der kleinen Bilder in der Leiste setzen, wird das Bild vergrößert im Abschnitt Foto angezeigt.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
