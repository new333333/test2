Barra degli strumenti Voce cartella

In Kablink Teaming la barra degli strumenti Voce cartella è popolata con opzioni specifiche alla cartella visualizzata. Tuttavia, è possibile configurare la barra degli strumenti Voce cartella in modo che contenga vari tipi di voci. Se si desidera ad esempio che la cartella Wiki contenga anche voci per le foto, è possibile configurare la cartella in modo da consentire anche le voci Foto.

Opzioni della barra degli strumenti

Per ciascun tipo di cartella esistono delle opzioni di default che vengono visualizzate automaticamente sulla barra degli strumenti Voce cartella. Di seguito è riportato un elenco di ogni tipo di cartella e le opzioni che vengono visualizzate automaticamente sulle rispettive barre degli strumenti Voce cartella.

Blog

  • Aggiungi voce blog: aggiunge una nuova voce alla cartella Blog. È possibile aggiungere un titolo, una descrizione e un allegato alla voce. È inoltre possibile sottoscrivere la voce e inviare un'e-mail a utenti specifici notificandoli della sottoscrizione della voce effettuata.

  • Nuova pagina blog: aggiunge una nuova cartella Blog a quella esistente. La nuova cartella è subordinata alla cartella Blog principale. È possibile utilizzare nuove pagine blog per organizzare meglio la cartella Blog in argomenti utili.

  • Ordina per: consente di ordinare i blog per numero, data di attività, autore, stato o titolo.

  • Aggiungi file a cartella: fornisce un'icona che consente di trascinare e rilasciare i file o copiarli e incollarli nella cartella Blog. Mediante questa azione il file viene aggiunto come nuova voce blog.

Album fotografico

  • Aggiungi voce foto: aggiunge una nuova voce alla cartella Album fotografico. È possibile assegnare un nome, descrivere ed effettuare l'upload di fotografie o altre immagini grafiche. È inoltre possibile inviare un'e-mail a utenti specifici notificandoli della sottoscrizione della voce effettuata.

  • Nuovo album fotografico: aggiunge una nuova cartella all'attuale cartella Album fotografico. La nuova è una sottocartella della cartella Album fotografico principale.

  • Ordina per: È possibile ordinare le immagini per Data attività o Titolo:

    Mediante l'ordinamento per Data attività le fotografie di cui si è effettuato l'upload più di recente vengono collocate in cima all'elenco, mentre quelle presenti già da tempo nell'album fotografico passano in fondo.

    Mediante l'ordinamento per Titolo le fotografie vengono organizzate in ordine alfabetico.

  • Aggiungi file a cartella: fornisce un'icona che consente di trascinare e rilasciare i file o copiarli e incollarli nella cartella Album fotografico. Mediante questa azione il file viene aggiunto come nuova voce file.

  • Aggiungi file a cartella: fornisce un'icona che consente di trascinare e rilasciare i file o copiarli e incollarli nella cartella Sondaggio. Mediante questa azione il file viene aggiunto come nuovo sondaggio non definito.

Wiki

  • Aggiungi voce wiki: aggiunge una nuova voce alla cartella Wiki. È possibile aggiungere un titolo, una descrizione e allegati. È inoltre possibile inviare un'e-mail a utenti specifici notificandoli dell'invio della voce effettuato sulla pagina wiki.

  • Nuovo argomento wiki: consente di assegnare un nome e di aggiungere una nuova cartella all'attuale cartella Wiki. La nuova è una sottocartella della cartella Wiki principale.

  • Ordina per: è possibile ordinare le voci wiki per Data attività o Titolo:

    Mediante l'ordinamento per Data attività le voci più recenti vengono collocate in cima all'elenco, mentre quelle da più tempo nella cartella Wiki passano in fondo.

    Mediante l'ordinamento per Titolo le voci wiki vengono organizzate in ordine alfabetico.

  • Aggiungi file a cartella: fornisce un'icona che consente di trascinare e rilasciare i file o copiarli e incollarli nella cartella Wiki. Mediante questa azione il file viene aggiunto come nuova voce wiki. È visibile nella sezione Pagine wiki di Barra laterale del wiki.

Configurazione di una cartella per consentire tipi di voci alternativi

  1. Individuare la cartella in cui si desidera aggiungere il tipo di voce alternativo.

  2. Fare clic su Gestisci > Configura.

  3. Nella sezione Tipi di voci di default, selezionare le cartelle che rappresentano i tipi di voci da consentire nella cartella.

  4. Fare clic su Applica > Chiudi.

  5. Fare clic su Aggiungi nella barra degli strumenti Voce cartella, quindi selezionare nell'elenco a discesa il tipo di voce da aggiungere.
