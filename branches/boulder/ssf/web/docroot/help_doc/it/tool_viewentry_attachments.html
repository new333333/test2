Come allegare i file a una voce

Nella sezione Allegati di una voce, in Kablink Teaming sono disponibili vari modi per aggiungere gli allegati a una voce:

Trascinamento e rilascio dei file

È possibile allegare un file a una voce trascinandolo da un'altra ubicazione e rilasciandolo nella voce.

  1. Fare clic sull'icona Trascina e rilascia file.

    In tal modo viene avviata una finestra con un'icona della cartella.

  2. Trascinare il file di cui si desidera effettuare l'upload e rilasciarlo sull'icona della cartella.

    In Teaming viene visualizzata una cartella di apertura e di chiusura a indicare che viene effettuato l'upload dei file rilasciati come allegati.

Nella voce è stato effettuato l'upload del file.

Accesso agli allegati mediante WebDAV (solo Internet Explorer)

In Teaming è possibile utilizzare WebDAV per accedere ai file memorizzati su server Web remoti e allegarli alle voci. Tuttavia, Teaming supporta WebDAV solo sul browser Internet Explorer. Se si utilizza un browser diverso da Internet Explorer, l'opzione non è disponibile.

  1. Fare clic su Accedi agli allegati mediante WebDAV.

  2. Individuare e selezionare il file da allegare.

Per ulteriori informazioni su come utilizzare WebDAV con Teaming, vedere Accessing Attachments Through WebDAV (Internet Explorer only)![I’m not 100% positive that this only works on IE.] nella Kablink Teaming 2.0 User Guide (in lingua inglese).

Upload di un file singolo

  1. Per effettuare l'upload di un file singolo nella voce, fare clic sull'icona Upload di un singolo file nella sezione Allegati.

  2. Fare clic su Sfoglia, quindi individuare e selezionare il file da allegare.

  3. Fare clic su OK.

Per ulteriori informazioni, vedere Note legali.
