Cartelle in copia speculare

Una cartella in copia speculare è una cartella Libreria i cui file sono ubicati in un sistema esterno (una directory in un file system o una cartella su un server SharePoint). Quando si esegue la copia speculare di cartelle esterne a Teaming, è possibile applicare gli strumenti di Kablink Teaming (ad esempio, la funzione di ricerca e il workflow) a file esterni.

È necessario utilizzare un file sul server per abilitare questa funzione.

Per ulteriori informazioni su come creare cartelle in copia speculare, vedere Working With Mirrored Folders nella Kablink Teaming 2.0 Advanced User Guide (in lingua inglese).
