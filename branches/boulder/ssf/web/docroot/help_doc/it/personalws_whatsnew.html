Novità

Quando si fa clic su Novità all'interno di uno spazio di lavoro o di una cartella, in Kablink Teaming vengono elencate tutte le voci nuove (per le quali si dispone dei diritti di accesso appropriati) presenti nell'attuale spazio di lavoro o cartella e in tutte le posizioni a essi subordinate. Nell'elenco sono incluse solo le voci create o modificate negli ultimi 30 giorni.

Per ulteriori informazioni, vedere Note legali.
