Tipi di voci di default

Questa sezione consente di selezionare i tipi di voci che gli utenti possono creare nella cartella. È possibile configurare la cartella per consentire tipi di voci aggiuntivi o modificare il tipo di voce attuale. Se si aggiungono ulteriori tipi di voci, gli utenti che inviano le voci nella cartella possono scegliere il tipo di voce da aggiungere, come una voce task o calendario.

  1. Selezionare i tipi di voci abilitare nella cartella.

  2. Fare clic su Applica > Chiudi.

Se si abilitano tipi di voci aggiuntive, gli utenti che inviano le voci nella cartella possono fare clic su Aggiungi nella barra degli strumenti Voce cartella e scegliere nell'elenco a discesa il tipo di voce da aggiungere, in base a quelli abilitati nella cartella.

La funzionalità Progettazioni modulo e vista consente agli amministratori del sito e ai proprietari delle cartelle di creare nuovi tipi di voci (moduli personalizzati), che vengono visualizzati nell'elenco. (Quando i proprietari delle cartelle creano nuovi tipi di voci, questi vengono elencati nella sezione Definizioni locali dell'elenco).

Per ulteriori informazioni su come creare tipi di voci personalizzate, vedere Designing Custom Folder Entries nella Kablink Teaming 2.0 Advanced User Guide (in lingua inglese).
