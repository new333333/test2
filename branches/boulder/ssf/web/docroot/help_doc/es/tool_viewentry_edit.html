Editar

El enlace Editar permite modificar sólo la sección Descripción de una entrada.

Si desea modificar otras partes de la entrada o del comentario, por ejemplo si quiere cambiar el título, cargar un archivo adjunto o suprimir los archivos adjuntos actuales, haga clic en Modificar en la barra de herramientas Entrada.

Para obtener más información, consulte Información legal.
