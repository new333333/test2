Redefiniciones del editor

Cuando se modifican archivos adjuntos en Kablink Teaming, se utilizan las aplicaciones por defecto del sistema Teaming para editarlos. Sin embargo, esta tabla permite anular las aplicaciones por defecto que se utilizan para editar los archivos adjuntos con extensiones de archivo específicas en Teaming.

Es posible sustituir la aplicación de edición por defecto para un único tipo de extensión de archivo, o bien puede sustituir las aplicaciones por defecto para diversos tipos de extensión de archivo.

Sustitución de la aplicación de edición por defecto para extensiones de archivo individuales

Para sustituir la aplicación por defecto para una extensión de archivo concreta:

  1. Haga clic en Añadir.

  2. En la lista desplegable, seleccione la extensión de archivo para la que desea sustituir la aplicación de edición por defecto.

  3. En el campo Aplicación, especifique el nombre de la aplicación que desea utilizar para editar los adjuntos con la extensión de archivo que ha especificado en Paso 2.

  4. Haga clic en Aceptar.

Sustitución de la aplicación de edición por defecto para todas las extensiones de archivo

Para sustituir todos los tipos de extensión de archivo:

  1. Haga clic en Utilizar.

  2. En la lista desplegable, seleccione la aplicación que desea utilizar para editar los archivos adjuntos de todos los tipos de extensión de archivo en Teaming.

  3. Haga clic en Aceptar.
