Podpowiedzi przekazywania pliku

Jeśli występują problemy z wysyłaniem pliku, ponieważ jest zbyt duży, można naprawić ten problem zwiększając ilość pamięci używanej przez aplety Java. Proces opisano w kolejnych rozdziałach.

Zwiększanie domyślnej ilości pamięci w systemie Windows

  1. Należy przejść do panelu sterowania (kliknąć przycisk Start > Ustawienia > Panel sterowania).

  2. Następnie należy kliknąć dwukrotnie ikonę Java.

    Uruchomiony zostanie Panel sterowania maszyny Java.

  3. Proszę wybrać kartę Java.

  4. W części Java Applet Runtime Settings proszę kliknąć przycisk View.

  5. Dla każdej z wersji środowiska uruchomieniowego Java proszę dodać -Xmx256m jako parametr środowiska uruchomieniowego Java.

  6. Następnie należy kliknąć przycisk OK.

    Aby zwiększenie domyślnego rozmiaru pamięci dla apletów Java odniosło skutek, należy zakończyć działanie wszystkich przeglądarek i uruchomić je ponownie.

To ustawienie powoduje uruchamianie apletu używanego do wysyłania plików do folderu Teaming z 256 MB pamięci. Jeśli aplet wciąż wskazuje, że nie ma wystarczającej pamięci do uruchomienia apletu, konieczne może być zmniejszenie tej liczby.

Zwiększanie domyślnej ilości pamięci w systemie Linux

  1. Proszę zlokalizować środowisko JRE lub JDK używane przez dodatek typu plug-in przeglądarki.

    Położenie JRE i JDK różni się w zależności od systemu.

  2. W katalogu bin środowiska JRE lub JDK proszę uruchomić plik ControlPanel.

    Uruchomiony zostanie Panel sterowania.

  3. Proszę wybrać kartę Java.

  4. W części Java Applet Runtime Settings proszę kliknąć przycisk View.

  5. Dla każdej z wersji środowiska uruchomieniowego Java proszę dodać -Xmx256m jako parametr środowiska uruchomieniowego Java.

  6. Następnie należy kliknąć przycisk OK.

    Aby zwiększenie domyślnego rozmiaru pamięci dla apletów Java odniosło skutek, należy zakończyć działanie wszystkich przeglądarek i uruchomić je ponownie.

To ustawienie powoduje uruchamianie apletu używanego do przekazywania plików do folderu witryny Teaming z 256 MB pamięci. Jeśli aplet wciąż wskazuje, że nie ma wystarczającej pamięci do uruchomienia apletu, konieczne może być zmniejszenie tej liczby.

Aby uzyskać więcej informacji, patrz sekcja Informacje prawne.
