Suivi de blogue

La barre d'outils Suivi de blogue permet d'effectuer rapidement les opérations suivantes :

  • Ajouter un commentaire : permet de commenter une entrée de blogue. Avant de rédiger un commentaire, il est judicieux d'afficher les commentaires qui ont déjà été formulés. Lorsque vous ajoutez un commentaire à un blogue qui a déjà été commenté, votre commentaire est automatiquement ajouté à la fin de la liste des commentaires imbriqués.

  • Afficher les commentaires : permet d'afficher tous les commentaires d'une entrée spécifique sans ouvrir cette dernière.

  • Envoyer l'entrée à un ami : permet d'envoyer le contenu de l'entrée, par courrier électronique, à quiconque dispose des droits d'accès appropriés pour afficher l'entrée.

  • S'abonner : ouvre un formulaire qui permet de vous abonner à l'entrée. Lorsque vous vous abonnez à une entrée, vous configurez Teaming pour qu'il vous envoie des notifications par message électronique vous informant des mises à jour de cette entrée.

    Pour plus d'informations sur l'abonnement à une entrée, reportez-vous à la section Subscribing to an Entry (Abonnement à une entrée) du manuel Kablink Teaming 2.0 User Guide (Guide de l'utilisateur de Kablink Teaming 2.0).

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
