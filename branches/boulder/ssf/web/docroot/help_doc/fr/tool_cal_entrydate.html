Type de date d'entrée

Cet outil est particulièrement utile lorsque vous utilisez la vue Agenda pour examiner des informations qui sont conçues pour un autre type de dossier. Par exemple, vous pouvez utiliser un wiki ou une discussion avec sa vue par défaut. Puis, si l'administrateur ou le propriétaire du dossier le configure pour prendre en charge cette opération, vous pouvez utiliser la vue Agenda pour voir quelles sont les entrées présentant l'activité la plus récente (par exemple, quelles sont les entrées qui ont été modifiées en dernier). Vous pouvez également afficher les entrées selon leur date de création.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
