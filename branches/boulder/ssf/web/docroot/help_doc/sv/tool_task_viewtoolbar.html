Verktygsfältet Visa

Med verktygsfältet Visa kan du välja vilka uppgifter som ska visas i uppgiftslistan baserat på följande kategorier:

  • Stängd: Visar uppgifter som har statusen Stängd.

  • I dag: Visar uppgifter som förfaller i dag.

  • Vecka: Visar uppgifter som förfaller inom sju dagar.

  • Månad: Visar uppgifter som förfaller inom 30 dagar.

  • Alla aktiva: Visar uppgifter som är aktiva.

  • Alla poster: Visar alla uppgifter.

Mer information finns i Juridisk information.
