フォルダテーブル

フォルダテーブルを操作する場合、各行には1つのエントリの情報が含まれています。このテーブルの操作にあたっては次のヒントを参考にしてください。

  • カラムのサイズ変更: テーブルの最上部で、2つの矢印が付いた線をクリックしたまま、左右にドラッグします。

  • テーブルの内容のソート: カラムの内容に従ってテーブルを順番にソートするには、カラムヘッダをクリックします。ソート順序を逆にするには、ヘッダを再度クリックします。カラムヘッダの横にある矢印アイコンは、テーブルが現在そのカラムを基準にソートされていることを示します(上矢印は降順を、下矢印は昇順をそれぞれ示します)。

  • セルにマウスを合わせた場合の動作: たとえば、タイトルが長すぎてテーブルのセルに収まらない場合、セルにマウスを合わせると、セルの内容全体が表示されます。

詳細については、「保証と著作権」。
