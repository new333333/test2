ワークフローの関連付け

このセクションでは、ワークフローをエントリタイプまたはコメント(返信)タイプに関連付けることができます。ワークフローをエントリタイプまたはコメントタイプに関連付けると、エントリが作成されるか、コメントが投稿されると、ワークフローが直ちに自動的に開始されます。

エントリへのワークフローの関連付け

このセクションには、このページの［デフォルトエントリタイプ］セクションで選択したエントリタイプがすべて含まれます。［ワークフローの関連付け］セクションにある各エントリタイプの横に、エントリタイプに関連付けることができるワークフロープロセスがすべて含まれたドロップダウンリストが表示されます。

ワークフローを特定のタイプのエントリに関連付ける

  1. このページの［デフォルトエントリタイプ］セクションでエントリタイプを選択して、このフォルダにエントリを作成できるようになっていることを確認します。

    ［デフォルトエントリタイプ］セクションでエントリタイプを選択して［適用］をクリックすると、［ワークフローの関連付け］セクションにそのエントリタイプが表示されます。

  2. ［ワークフローの関連付け］セクションの［エントリタイプ］セクションで、ワークフローを適用するエントリタイプの横にあるドロップダウンリストからワークフローを選択します。

  3. ［適用］>［閉じる］の順にクリックします。

    これで、誰かがこのフォルダ内に該当するタイプのエントリを作成すると、そのエントリは自動的にワークフローの初期状態に配置されるようになります。

フォルダ内にワークフローを作成すると、そのワークフローは自動的にこのドロップダウンリストに追加され、このフォルダ内のエントリに関連付けることができます。ワークフローを作成する方法の詳細については、『Kablink Teaming 2.0上級ユーザガイド』の「ワークフローの作成」を参照してください。

コメント(返信)タイプへのワークフローの関連付け

［コメントまたは返信のタイプ］セクションでは、ワークフローを割り当てて、ユーザが特定の返信タイプを入力した際に開始されるようにすることができます。

［ワークフローの関連付け］セクションに一覧表示されている各デフォルトエントリタイプに対して、すべての返信タイプが表示されます。

デフォルトエントリタイプが同時に返信タイプであっても、リストに複数回表示されることはありません。

詳細については、「保証と著作権」。
