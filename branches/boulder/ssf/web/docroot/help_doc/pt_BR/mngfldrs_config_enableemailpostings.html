Habilitando postagens de e-mail

Para habilitar postagens de e-mail para todas as pastas na instalação, especifique uma programação para todo o site. Em seguida, o proprietário da pasta deve especificar um nome e uma senha de conta de e-mail ou apenas um endereço de álias de e-mail. Essas informações são usadas para determinar as mensagens na conta que serão roteadas para a pasta.

Para obter mais informações, consulte Informações legais.
