Configurando controle de acesso

Use esta página para mudar as configurações de controle de acesso desta área de trabalho ou pasta.

Por padrão, as áreas de trabalho e pastas herdam as configurações de controle de acesso de seus pais. Use esta página apenas se quiser configurações de acesso diferentes.

Antes de fazer alterações na tabela de controle de acesso, você deve primeiro garantir que a pasta ou a área de trabalho não esteja herdando os usuários e as funções do diretório pai. Para obter informações sobre como fazer isso, consulte a Herdar Associação de Função.

Para ver informações sobre controle de acesso mais detalhadas do que as contidas nesta seção e nas seções seguintes, consulte Controlando acesso do usuário no Guia do Usuário Avançado do Kablink Teaming 2.0.

As seções a seguir descrevem como modificar a tabela de controle de acesso para atender às suas necessidades.

Adicionando usuários e grupos

  1. Clique no link Adicionar um Grupo ou Adicionar um Usuário na coluna esquerda da tabela.

  2. No campo, digite as primeiras letras do nome do usuário ou do grupo e clique no link do nome na lista suspensa.

    O Kablink Teaming adiciona uma linha à tabela para esse usuário ou grupo.

Vendo tarefas de função

Na tabela de controle de acesso, os nomes de função servem como cabeçalhos de coluna.

  1. Clique em um cabeçalho de coluna (Participante, Membro da Equipe etc) para ver uma lista dos direitos associados à função.

Mudando as funções de usuários ou grupos

Função é um conjunto de privilégios; quando você atribui uma função a uma pessoa, ela recebe automaticamente todos os direitos associados à função.

  1. Marque e desmarque as caixas de seleção para definir quem pode atuar em qual função.

  2. Clique em OK.

Ajustando controles de acesso em outro local

  1. Vá para a árvore da área de trabalho na parte superior desta página, navegue até o local onde deseja ajustar os controles de acesso e clique no link desse local na árvore.

    É exibida a página Controle de Acesso desse local.

Para obter mais informações, consulte Informações legais.
