Aplicando recursivamente as configurações de definição

Use esse recurso para aplicar as configurações de definição a pastas ou entradas que já foram criadas.

Aplicando as configurações de definição a todas as subpastas

É possível aplicar todas as configurações de definição de uma pasta-mãe a todas as subpastas.

  1. Selecione Mudar pastas contidas para herdar as definições desta pasta.

  2. Clique em Aplicar.

Todas as configurações de definição na página Definir Configurações Padrão serão aplicadas a todas as subpastas.

Aplicando as configurações de definição a entradas existentes

Se você já adicionou entradas a esta pasta e o tipo de definição de entrada não corresponde ao tipo das outras entradas de pasta (por exemplo, se você tiver uma pasta Blog que tenha basicamente tipos de definição de Blog, mas algumas das entradas são exibidas como tipos de definição de Discussão), poderá usar esta seção para mudar as configurações de definição de forma que todas as entradas de pasta sejam do mesmo tipo.

  1. Selecione Mudar pastas contidas para herdar as definições desta pasta e depois selecione Mudar entradas contidas para o seguinte tipo de entrada.

  2. Na lista suspensa, selecione o tipo de entrada que deseja aplicar a todas as entradas existentes na pasta.

    Para adicionar tipos de entrada a esta lista suspensa, selecione os tipos de entrada na seção Tipos de Entrada Padrão e clique em Aplicar.

  3. Clique em Aplicar.
