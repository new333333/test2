Vinculando a uma entrada em uma pasta diferente

  1. Clique em Mudar.

  2. No campo apresentado, comece a digitar o nome da pasta que contém a entrada à qual deseja vincular.

  3. Quando a pasta desejada aparecer na lista suspensa, selecione-a.

  4. No campo Nome da página para a qual vincular, especifique o nome completo da entrada à qual deseja vincular, ou clique em Localizar para usar a funcionalidade Digitar para Encontrar.

  5. Após terminar de digitar o nome da entrada, ou após clicar no nome na lista suspensa Digitar para Encontrar, clique em Inserir.

    O link do título da entrada que você escolheu é inserido na caixa Descrição.

Para obter mais informações, consulte Informações legais.
