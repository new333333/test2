Defining a URL for a Folder or Workspace

Use this section to define a user-friendly URL for the folder or workspace where you currently are. Defining this URL enables you to navigate directly to the folder or workspace from your browser.

Defining a URL for a folder enables you to add entries to that folder via e-mail or text message.

  1. In the Define URL drop-down list, select one of the predefined global keywords.

    Unless you are a site administrator, your URL must begin with one of these predefined global keywords.

  2. In the field, specify the desired URL for the folder or workspace.

  3. Click Add.

  4. (Conditional) If you are defining the URL for a Folder, you must enable the URL that you just defined by selecting the URL. The URL that you need to select is located below the Add button.

  5. (Conditional) If you are defining the URL for a Folder, click Apply.

  6. Click Close.

To delete a defined URL, select the URL in the Currently defined URLs section, then click Delete the selected URLs.
