Novell Vibe 3.2

1.0 Komma igång

Fullständiga systemkrav och installationsanvisningar för Novell Vibe-administratörer finns i installationshandboken för Vibe 3.2. Anvisningar för installation av Vibe-webbplatsen samt för löpande administration och underhåll finns i administrationsguiden för Vibe 3.2.

När Vibe-webbplatsen har installerats finns anvisningar för användarna i användarhandboken för Vibe 3.2 och användarhandboken för avancerade Vibe 3.2-användare. (På språk där användarhandboken för Vibe 3.2 inte är tillgänglig tillhandahålls i stället en snabbstart för Vibe 3.2.)

Du kan visa onlinehjälpen för Vibe när du loggar in på Vibe-webbplatsen.
