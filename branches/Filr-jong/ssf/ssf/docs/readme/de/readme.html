Novell Vibe 3.2

1.0 Einführung

Umfassende Systemanforderungen und detaillierte Anweisungen zur Installation der Software für den Novell Vibe-Administrator finden Sie im Vibe 3.2-Installationshandbuch. Anweisungen zum Einrichten der Vibe-Site sowie zur fortlaufenden Verwaltung und Wartung der Site finden Sie im Vibe 3.2-Verwaltungshandbuch.

Nach der Einrichtung der Vibe-Site erhalten die Vibe-Benutzer Anweisungen im Vibe 3.2-Benutzerhandbuch und im Erweiterten Benutzerhandbuch zu Vibe 3.2. (Ein Vibe 3.2-Schnellstart ist in Sprachen verfügbar, in denen das Vibe 3.2-Benutzerhandbuch nicht verfügbar ist.)

Sie können die Vibe-Onlinehilfe anzeigen, indem Sie sich auf Ihrer Vibe-Site anmelden.
