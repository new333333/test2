Novell Vibe 3.2

1.0 Guida introduttiva

I requisiti di sistema completi e istruzioni dettagliate per l'installazione del software sono fornite per l'amministratore di Novell Vibe nella Vibe 3.2 Installation Guide (in lingua inglese). La configurazione del sito di Vibe e le istruzioni sull'amministrazione e la manutenzione Vibe 3.2 Administration Guide (in lingua inglese).

Dopo aver configurato il sito di Vibe, gli utenti di Vibe possono ottenere istruzioni nella Guida dell'utente di Vibe 3.2 e nella Vibe 3.2 Advanced User Guide (in lingua inglese). (Viene fornito un Riferimento rapido per Vibe 3.2 per le lingue in cui non è disponibile la Guida dell'utente di Vibe 3.2.)

Dopo aver effettuato il login al sito Web di Vibe è possibile visualizzare la Guida online di Vibe.
