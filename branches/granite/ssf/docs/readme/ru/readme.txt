Novell Vibe 3.4

1.0  Приступая к работе

     Полный перечень требований к системе и подробные инструкции по установке
     программного обеспечения для администратора Novell Vibe см. в "Руководстве
     по установке Vibe 3.4." Настройка сайта Vibe, а также инструкции по
     последующему администрированию и обслуживанию сайта описаны в "Руководстве
     по администрированию Vibe 3.4".

     После настройки сайта Vibe пользователи Vibe могут найти необходимые
     инструкции в "Руководстве пользователя Vibe 3.4" и "Руководстве для опытных
     пользователей Vibe 3.4". (Если для какого-то языка "Руководство пользователя
     Vibe 3.4" отсутствует, для него предоставляется "Краткое руководство по Vibe
     3.4".)

     После регистрации на сайте Vibe можно просматривать электронную справку
     Vibe.
