Novell Vibe 3.2

1.0 入門

針對 Novell Vibe 管理員的完整系統要求及詳細的軟體安裝指示會在《Vibe 3.2 安裝指南》中提供。Vibe 站台設定以及持續的站台管理與維護指示可在《Vibe 3.2 管理指南》中找到。

Vibe 站台設定完成後，Vibe 使用者便可獲取《Vibe 3.2 使用者指南》和《Vibe 3.2 進階使用者指南》中的指示。(如果某些語言的《Vibe 3.2 使用者指南》未提供，可以使用該語言的《Vibe 3.2 快速入門》。)

您可以在登入 Vibe 站台時顯示 Vibe 線上說明。
