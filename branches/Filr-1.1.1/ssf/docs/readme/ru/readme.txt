Novell Vibe 3.2

1.0  Начало работы

     Полный перечень требований к системе и подробные инструкции по установке
     программного обеспечения для администратора Novell Vibe см. в "Руководстве
     по установке Vibe 3.2." Настройка сайта Vibe, а также инструкции по
     последующему администрированию и обслуживанию сайта описаны в "Руководстве
     по администрированию Vibe 3.2."

     После настройки сайта Vibe пользователи Vibe могут найти необходимые
     инструкции в "Руководстве пользователя Vibe 3.2" и "Расширенном руководстве
     пользователя Vibe 3.2". (Если для какого-то языка отсутствует "Руководство
     пользователя Vibe 3.2", то для него предоставляется "Краткое справочное
     руководство по Vibe 3.2".)

     После регистрации на сайте Vibe можно просматривать электронную справку
     Vibe.
