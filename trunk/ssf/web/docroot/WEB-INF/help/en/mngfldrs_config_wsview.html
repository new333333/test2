Workspace View

Select the view whose look best presents your folder list or workspace.

When choosing for a workspace, you can select the default Workspace definition. If you want to track project tasks and milestones based on task completion, you can select the Project management workspace definition. Finally, when choosing for a team workspace, select Team workspace root for workspaces that you want to display the Add a team workspace button. If your team workspace is organizational in nature and not intended to have team workspaces as its sub-workspaces, select the Team workspace definition (which omits the button for team creation).

For more information, see Understanding SiteScape ICEcore Enterprise Teams.
