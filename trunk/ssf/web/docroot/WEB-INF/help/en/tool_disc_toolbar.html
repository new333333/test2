Discussion Toolbar

The Discussion toolbar provides the following options that enable you to manage your Discussion folder:

  • Add Discussioin entry: Add a new entry to your Discussion folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry and send an e-mail to specific users notifying them that you have submitted an entry.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Discussion folder. This action adds the file as a new discussion entry.
