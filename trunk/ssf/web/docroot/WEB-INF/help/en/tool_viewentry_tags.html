Tags

Tags are keywords that categorize a workspace, folder, or entry. These categorical labels can assist you when using SiteScape ICEcore Enterprise search.

A personal tag is a keyword that only you can see and use. A community tag is a keyword created by the owner of a workspace or folder for use by everyone participating in that place.

To create a tag, click the Tags icon. In a text box, type a keyword, and click Add. The new tag appears in a list above the text box. Click the Close icon (X) in the upper right corner of the Add/Delete Tags panel to exit.

Tags must be a single identifier, and you can use underscores (_) to separate words (for example, marketing_plans). You cannot use quoted phrases (for example, "marketing plans") or dashes (for example, marketing-plans).

To delete a tag from a workspace or folder or entry, view the item, click the Tags icon, and click the Close icon (X) to the left of the tag you want to delete.
