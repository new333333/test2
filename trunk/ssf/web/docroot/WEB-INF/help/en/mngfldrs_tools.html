Folder Management Tools

These are only those management and configuration tools you are allowed to use. Site administrators, or the owner of this workspace or folder, can use and grant access to these tools.

The Manage menu allows you to add sub-workspaces, folders, and sub-folders; generate reports; configure views for the page; or delete, modify, or move this place

The Manage > Access control menu item alters who can view and participate in this place. The Team menu allows you to manage team membership for this place and to communicate rapidly with team members.

The Subscriptions menu allows you to recieve e-mail notifications about new or changed entries, and allows an RSS Reader to retrieve those entries.
