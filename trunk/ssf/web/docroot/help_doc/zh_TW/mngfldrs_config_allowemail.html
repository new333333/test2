允許的電子郵件

以下幾部分介紹此頁面上的選項。

啟用外送電子郵件

您可以修改針對外送電子郵件的下列組態選項：

  • 啟用外送電子郵件： 選取此核取方塊可以啟用 Teaming 中的所有外送電子郵件。如果不想讓使用者從 Teaming 傳送外送電子郵件，請不選此核取方塊。

    如果要允許使用者傳送摘要通知，則必須選中此核取方塊。

  • 傳送電子郵件通知 (外送)： 此區段用於設定針對摘要通知的組態預設值。如果 Teaming 使用者要傳送或接收摘要電子郵件通知，他們必須使用您在此處選取的摘要通知設定。您可以將摘要通知的傳送設定為每天、一週中的某幾天、在每天的特定時間或每隔幾小時。

允許使用內嵌式電子郵件伺服器與簡易 URL 內送電子郵件

如果要允許透過電子郵件將項目提交至已預先設定簡易 URL 的資料夾，請選取「允許使用內嵌式電子郵件伺服器與簡易 URL 內送電子郵件」核取方塊。

有關更詳細的資訊，請參閱 法律聲明。
