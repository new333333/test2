檔案上載提示

如果上載檔案時因檔案太大而發生問題，可以增大 Java Applet 預設使用的記憶體量來解決此問題。該程序將在以下幾部分中予以說明。

在 Windows 上增加預設記憶體量

  1. 導覽至「控制台」(按一下「開始」>「設定」>「控制台」)。

  2. 連按兩下「Java」。

    此時會啟動「Java 控制面板」。

  3. 選取「Java」索引標籤。

  4. 在「Java Applet 執行階段設定」區段中，按一下「檢視」。

  5. 對於每個 Java 執行階段版本，新增 -Xmx256m 做為 Java 執行階段參數。

  6. 按一下「確定」。

    增大 Java Applet 的預設記憶體大小之後，必須結束所有瀏覽器，然後重新啟動這些瀏覽器，這樣才能使變更生效。

此設定會導致將檔案上載至 Teaming 資料夾所使用的 Applet，在啟動時使用 256 MB 的記憶體。如果 Applet 仍然指示因記憶體不足而無法執行，則可能需要減小此數字。

在 Linux 上增大預設記憶體量

  1. 找到用於瀏覽器外掛程式的 JRE 或 JDK。

    JRE 或 JDK 的位置因系統而異。

  2. 在 JRE 或 JDK 的 bin 目錄中，執行 ControlPanel 檔案。

    此時會啟動「控制台」。

  3. 選取「Java」索引標籤。

  4. 在「Java Applet 執行階段設定」區段中，按一下「檢視」。

  5. 對於每個 Java 執行階段版本，新增 -Xmx256m 做為 Java 執行階段參數。

  6. 按一下「確定」。

    增大 Java Applet 的預設記憶體大小之後，必須結束所有瀏覽器，然後重新啟動這些瀏覽器，這樣才能使變更生效。

此設定會導致將檔案上載至 Teaming 資料夾所使用的 Applet，在啟動時使用 256 MB 的記憶體。如果 Applet 仍然指示因記憶體不足而無法執行，可能必須減小此數字。
