標記

標記是對工作區、資料夾或項目進行分類的關鍵字。Novell Teaming 中的標記功能可讓您為位置與項目建立虛擬容器。與電子郵件資料夾一樣，您可以分類資訊，日後再回來查看資訊。

標記工作區或資料夾

  1. 導覽至要標記的工作區或資料夾。

  2. 在主側邊列的「標記此工作區」 (或「標記此資料夾」) 區段，於「個人標記」或「社群標記」區段的欄位中輸入標記名稱。

    是建立個人標記還是社群標記，請參閱在個人標記和社群標記之間進行選擇。

    此欄位使用「輸入以尋找」功能，因此，如果使用所需名稱的標記已存在，它將出現在下拉式清單中。如果該名稱未出現在下拉式清單中，則要建立新標記。

  3. 輸入標記名稱或從下拉式清單中選取名稱後，按一下「新增」。

此時，工作區或資料夾便已加上了所指定的標記。

標記項目

  1. 在要標記的項目中，按一下「標記此」。

  2. 在「個人標記」或「社群標記」區段的欄位中輸入標記的名稱。

    是建立個人標記還是社群標記，請參閱在個人標記和社群標記之間進行選擇。

    此欄位使用「輸入以尋找」功能，因此，如果使用所需名稱的標記已存在，它將出現在下拉式清單中。如果該名稱未出現在下拉式清單中，則要建立新標記。

  3. 輸入標記名稱或從下拉式清單中選取名稱後，按一下「新增」。

此時，項目便已加上了所指定的標記。

在個人標記和社群標記之間進行選擇

  • 個人標記： 個人標記僅供您使用。建立個人標記後，只有您可以使用和檢視該標記。

  • 社群標記： 有權檢視已加上社群標記的工作區、資料夾或項目的任何人均可使用和檢視社群標記。例如，您在團隊工作區的「檔案」資料夾上建立社群標記，則對「檔案」資料夾擁有「檢視」權限的任何人，均可使用您建立的標記以同樣的標記名稱標記其他位置。他們還可以使用 Teaming 的搜尋和尋找功能來尋找擁有該標記名稱的項目。

    如需使用搜尋功能來尋找標記項目的詳細資訊，請參閱使用進階搜尋。

    如需使用尋找功能來尋找標記項目的詳細資訊，請參閱尋找特定的人員、位置或標記項目。

    僅當您擁有適當的權限時，才可以使用社群標記來標記位置和項目。

有關更詳細的資訊，請參閱 法律聲明。
