工作區檢視窗

選取能最好地呈現出您工作區的檢視窗。

  • 討論工作區： 如果要建立可儲存多個論壇的工作區，請選取此檢視窗。

    如需討論工作區的詳細資訊，請參閱《Novell Teaming 2.1 使用者指南》中的「管理討論工作區」。

  • 抵達頁面： 如果要在團隊成員進入工作區時向他們顯示重要資訊，請選取此檢視窗。

    如需抵達頁面的詳細資訊，請參閱《Novell Teaming 2.1 使用者指南》中的「管理抵達頁面」。

  • 專案管理工作區： 如果要根據任務完成情況追蹤專案任務和重要事件，請選取此檢視窗。

    如需專案管理工作區的詳細資訊，請參閱《Novell Teaming 2.1 使用者指南》中的「管理專案管理工作區」。

  • 團隊工作區： 如果您對團隊工作區原本的組織很滿意，不打算在下面再建立團隊工作區，請選取此檢視窗。

    如需團隊工作區的詳細資訊，請參閱《Novell Teaming 2.1 使用者指南》中的「管理團隊工作區」。

  • 團隊工作區的根： 如果要在團隊工作區中顯示「新增團隊工作區」按鈕，請選取此檢視窗。如果要在此工作區中建立其他團隊工作區做為子位置，則需要選取此項。

    如需團隊工作區的根工作區的詳細資訊，請參閱《Novell Teaming 2.1 Advanced User Guide》(Novell Teaming 2.1 進階使用者指南) 中的「Creating and Managing a Team Workspace Root Workspace」(建立和管理團隊工作區的根工作區)。

  • 工作區： 這是基本工作區。您可以選擇工作區要包含的資料夾。

    如需基本工作區的詳細資訊，請參閱《Novell Teaming 2.1 Advanced User Guide》(Novell Teaming 2.1 進階使用者指南) 中的「Creating and Managing a Basic Workspace」(建立和管理基本工作區)。

如果這是個人工作區，則預設檢視窗為「使用者工作區」且無法變更。

有關更詳細的資訊，請參閱 法律聲明。
