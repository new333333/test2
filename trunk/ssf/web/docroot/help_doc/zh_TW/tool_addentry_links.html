在描述欄位中使用連結

撰寫項目的「描述」區段時，可以包含指向 Kablink Teaming 站台中其他項目或指向網際網路上其他位置的連結。

連結至目前資料夾中的項目

  1. 在「新增項目」表單的「描述」區段中，按一下 HTML 編輯器中按鈕列末尾的「套用連結」圖示 。

    Teaming 將顯示一個視窗，可在其中指定要連結的項目。

  2. 在「要連結的頁面名稱」欄位中，執行下列動作之一：

    • 準確輸入頁面標題。

    • 按一下欄位旁邊的「尋找」，開始輸入頁面的名稱，並從下拉式清單中選取合適的標題。

    • 使用每個 Kablink Teaming 頁面底部的永久連結功能。執行此操作時，按一下要連結之頁面底部「頁尾」工具列中的「永久連結」連結，複製 URL，然後貼上到「要連結的頁面名稱」欄位中。

  3. 按一下「插入」。

連結至其他資料夾中的項目

  1. 在「新增項目」表單的「描述」區段中，按一下 HTML 編輯器中按鈕列末尾的「套用連結」圖示 。

    Teaming 將顯示一個視窗，可在其中指定要連結的項目。

  2. 按一下「變更」連結。

  3. 在提供的欄位中，開始輸入要連結的項目所在之資料夾的名稱。

  4. 當所需資料夾出現在「輸入以尋找」清單中時，按一下。

  5. 在「要連結的頁面名稱」欄位中，指定要連結之項目的完整名稱，或按一下「尋找」以使用「輸入以尋找」功能。

  6. 輸入名稱或在「輸入以尋找」下拉式清單中按一下名稱之後，按一下「插入」。

    所選項目的連結標題即會插入「描述」方塊中。

連結至 Web 上的外部頁面

  1. 在「新增項目」表單的「描述」區段中，選取要轉換為連結的文字。

  2. 按一下「插入/編輯連結」圖示 。

    此時會顯示「插入/編輯連結」視窗。

  3. 在「連結 URL」欄位中，指定要連結之頁面的 URL。

  4. 填寫下列欄位：

    • 目標︰ 指定是在同一個視窗中開啟連結，還是在新視窗中開啟連結。

    • 標題︰ 指定連結的標題。

    • 類別： 指定希望連結顯示的大小。

      如果此欄位保留空白，則會自動選取 12 像素。這是標準設定。

  5. 按一下「插入」。
