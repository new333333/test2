頁尾工具列

您目前是在工作區中、資料夾中還是項目中將決定「頁尾」工具列的內容。

工作區頁尾工具列

  • 工作區永久連結： 可讓您參考工作區。這些連結長期保持不變，可用作書籤或以電子郵件方式傳送給其他人。還可以根據需要對它們進行複製和貼上操作。

資料夾頁尾工具列

  • 資料夾永久連結： 可讓您參考資料夾。這些連結長期保持不變，可用作書籤或以電子郵件方式傳送給其他人。還可以根據需要對它們進行複製和貼上操作。

  • 電子郵件地址︰ 如果資料夾已設定為使用易記的 URL，則在按一下頁尾連結時，「頁尾」工具列將顯示此 URL。

    如需如何將資料夾設定為透過電子郵件接收項目的詳細資訊，請參閱《Novell Teaming 2.1 使用者指南》中的「將資料夾設定為透過電子郵件接收項目」。

  • RSS URL： 可讓您以 RSS 摘要的方式存取資料夾。只需複製 RSS URL 並將其貼到您的 RSS 讀取器中即可。

  • ATOM URL： 可讓您以 ATOM 摘要的方式存取資料夾。只需複製 ATOM URL 並將其貼到您的 ATOM 讀取器中即可。

  • WebDAV URL： 如果您使用的網頁瀏覽器為 Internet Explorer 7 或更新版本，可以使用 WebDAV URL 直接從 Microsoft Office 或類似的應用程式存取 Novell Teaming 檔案。

  • iCal URL： 可讓您將 Teaming 任務和行事曆資料夾同步到您的桌面行事曆應用程式。

    檢視行事曆資料夾或項目時，可以使用此連結輸出 Teaming 行事曆資料，以併入支援 iCal 格式的其他行事曆 (例如，Google 和 Outlook)。按一下連結，以 .ics 副檔名儲存檔案，然後將檔案輸入到其他行事曆中 (如需詳細資訊，請參閱其他行事曆的說明)。

  • 將資料夾做為 WebDAV 資料夾檢視： 可讓您將資料夾做為 WebDAV 資料夾檢視。

    如需詳細資訊，請參閱《Novell Teaming 2.1 使用者指南》中的「在 Teaming 中存取 WebDAV 檔案」。

項目頁尾工具列

  • 項目永久連結： 可讓您參考項目。這些連結長期保持不變，可用作書籤或以電子郵件方式傳送給其他人。還可以根據需要對它們進行複製和貼上操作。

  • iCal: (僅限行事曆項目) 可讓您將 Teaming 任務和行事曆資料夾同步到您的桌面行事曆應用程式。

    檢視行事曆資料夾或項目時，可以使用此連結輸出 Teaming 行事曆資料，以併入支援 iCal 格式的其他行事曆 (例如，Google 和 Outlook)。按一下連結，以 .ics 副檔名儲存檔案，然後將檔案輸入到其他行事曆中 (如需詳細資訊，請參閱其他行事曆的說明)。

  • 傳送電子郵件： 啟動電子郵件表單，可透過其將電子郵件傳送給所需的任何人。電子郵件包含 Teaming 中項目的位置，以及項目的部分內容。

  • 訂閱此項目： 啟動表單，可用於訂閱項目。訂閱項目後，您可以設定 Teaming 在該項目有更新時給您傳送電子郵件通知。

    如需訂閱項目的詳細資訊，請參閱使用電子郵件通知。

  • 輸出此項目： 僅限 Teaming 管理員。可讓您輸出項目。之後即可將該項目輸入到其他 Teaming 站台。

有關更詳細的資訊，請參閱 法律聲明。
