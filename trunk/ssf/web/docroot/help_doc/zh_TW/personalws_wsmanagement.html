工作區管理工具

如果在「工作區」工具列中按一下「管理」，Novell Teaming 將僅顯示允許您使用的管理和組態工具。站台管理員或工作區擁有者可以授予對這些工具的存取權限。

如果您擁有不受限制的存取權限，可以看到下列選項：

  • 存取控制︰ 可讓您控制可以檢視和參與此工作區的人員。

  • 活動報告： 可讓您檢視工作區中發生的活動。

  • 設定︰ 可讓您設定工作區，包括設定預設檢視窗和工作區 URL。

  • 複製此工作區： 可讓您在擁有適當權限的情況下，將目前工作區複製到工作區樹狀結構中所需的任何目的地。請參閱工作區樹狀結構導覽。

  • 刪除此工作區： 刪除目前的工作區。

  • 輸出/輸入： (僅限 Teaming 管理員) 可讓您輸出目前工作區，或將資料夾或工作區輸入到目前工作區。

    如需在 Teaming 中輸出和輸入工作區與資料夾的詳細資訊，請參閱《Kablink Teaming 2.1 Administration Guide》(Kablink Teaming 2.1 管理指南) 中的「Exporting/Importing Workspaces, Folders, and Entries」(輸出/輸入工作區、資料夾和項目)。

  • 表單和檢視窗設計工具： 可讓您建立隨後可套用至工作區及其子資料夾的自訂項目表單和自訂工作流程。

  • 修改此工作區： 可讓您修改工作區，包括工作區名稱、描述、抵達頁面、品牌和圖示。

  • 移動此工作區： 可讓您在擁有適當權限的情況下，將目前工作區移動到工作區樹狀結構中所需的任何目的地。請參閱工作區樹狀結構導覽。

  • 新增工作區： 可讓您在目前工作區的下面新增工作區。

  • 新增資料夾： 可讓您在工作區中新增資料夾。

有關更詳細的資訊，請參閱 法律聲明。
