顯示項目與聯絡人員

Novell Teaming 中有多個工具 (例如，資料夾清單和搜尋結果) 可顯示項目資訊的摘要。此摘要中的資訊可讓您顯示項目或聯絡項目的建立者，如以下幾部分中所述。

顯示項目

若要顯示項目，只需按一下項目的標題。

您可以設定 Teaming 顯示項目的方式。開啟項目時，Teaming 可以將項目顯示為覆疊、在新頁面中顯示或在快顯視窗中顯示。

若要變更在 Teaming 中顯示項目的方式：

  1. 在主側邊列的「個人優先設定」區段中，按一下「項目檢視樣式」。

  2. 選取所需的項目檢視方法。

聯絡人員

項目摘要包含「目前狀態」圖示，該圖示位於原著者姓名的旁邊。此「目前狀態」圖示可能有下列幾種狀態：

  • 綠色︰ 對方在線上且活動。

  • 黃色︰ 對方最近曾活動，但現在已離開。

  • 灰色︰ 對方未登入 Novell Conferencing 用戶端。

  • 白色︰ 對方的狀態未知。

按一下「目前狀態」圖示並選取功能表項目可與對方聯絡。您可以啟動與對方之間的即時會議或排程會議、呼叫對方、將對方新增至剪貼簿或檢視對方的微型部落格。

有關更詳細的資訊，請參閱 法律聲明。
