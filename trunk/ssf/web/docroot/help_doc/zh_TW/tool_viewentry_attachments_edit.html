編輯附加檔案

Novell Teaming 可讓您編輯附加至項目的檔案。

如果無法編輯檔案，可能是未將 Teaming 設定為支援電腦上安裝的編輯應用程式。例如，如果將 Teaming 設定為使用 Microsoft Word 做為所有文字檔案的預設編輯器，但您電腦上安裝的是 OpenOffice 而非 Microsoft Word，Teaming 將顯示錯誤，並且不會編輯文件。

為了解決此問題，Teaming 允許您置換預設編輯器設定。

變更單一檔案類型的預設編輯器設定

  1. 在主側邊列的「個人優先設定」區段中，按一下「定義編輯器置換」。

  2. 按一下「新增」。

  3. 從下拉式清單中，選取要變更其預設編輯應用程式的副檔名。

  4. 在「應用程式」欄位中，指定編輯步驟 3中選取的檔案類型所使用之應用程式的名稱。

  5. 選取核取方塊，然後按一下「確定」。

變更所有檔案類型的預設編輯器設定

您可以將 Teaming 設定為對所有檔案類型使用同一個編輯應用程式。

  1. 在主側邊列的「個人優先設定」區段中，按一下「定義編輯器置換」。

  2. 按一下「使用」，然後選取要讓 Teaming 用於編輯所有檔案類型的應用程式。

  3. 選取「副檔名」左邊的核取方塊，然後按一下「確定」。
