將檔案附加至項目

在項目的「附件」區段中，Kablink Teaming 提供了將附件新增至項目的各種選擇。

拖放檔案

從其他位置拖曳檔案並將其放入項目中，藉以將檔案附加至項目。

  1. 按一下「拖放檔案」圖示。

    此時會啟動包含資料夾圖示的視窗。

  2. 拖曳要上載的檔案，並將其放在資料夾圖示上。

    Teaming 將顯示不斷開合的資料夾，表示正在將所拖曳的檔案上載為附件。

現在已將檔案上載至項目。

使用 WebDAV 存取附件 (僅限 Internet Explorer)

Teaming 可讓您使用 WebDAV 存取儲存在遠端 Web 伺服器上的檔案，並將它們附加至項目。不過，Teaming 僅在 Internet Explorer 瀏覽器上支援 WebDAV。如果您正在使用 Internet Explorer 以外的瀏覽器，則不會顯示此選項。

  1. 按一下「使用 WebDAV 存取附件」。

  2. 導覽至要附加的檔案並加以選取。

如需 WebDAV 與 Teaming 搭配使用的詳細資訊，請參閱使用 WebDAV 存取附件 (僅限 Internet Explorer)！[I’m not 100% positive that this only works on IE.] (在《Kablink Teaming 2.0 使用者指南》中)。

上載單一檔案

  1. 在要上載檔案的項目中，按一下「附件」區段中的「上載單一檔案」圖示。

  2. 按一下「瀏覽」，然後瀏覽至要附加的檔案並加以選取。

  3. 按一下「確定」。

有關更詳細的資訊，請參閱 法律聲明.
