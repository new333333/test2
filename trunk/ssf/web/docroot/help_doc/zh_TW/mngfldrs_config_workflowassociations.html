工作流程關聯

此區段可讓您將工作流程與項目類型或意見 (回覆) 類型關聯起來。將工作流程與項目類型或意見類型相關聯後，建立項目或提出意見時，會自動啟動工作流程。

將工作流程與項目相關聯

此區段包含在此頁面的「預設項目類型」區段中選取的所有項目類型。Teaming 會在「工作流程關聯」區段中每種項目類型的旁邊顯示一個下拉式清單，內含可與項目類型關聯的所有工作流程程序。

若要使工作流程與特定的項目類型相關聯：

  1. 在此頁面上的「預設項目類型」區段中選取項目類型，藉以確定已啟用要在此資料夾中建立的項目。

    在「預設項目類型」區段中選取項目類型並按一下「套用」之後，該項目類型即會出現在「工作流程關聯」區段中。

  2. 在「工作流程關聯」區段的「項目類型」區段中，從要套用工作流程之項目類型旁的下拉式清單中選取工作流程。

  3. 按一下「套用」>「關閉」。

    現在，當有人在這個資料夾中建立該類型的項目時，Teaming 會自動將項目設為工作流程程序的初始狀態。

在資料夾中建立工作流程時，會自動將該工作流程新增至此下拉式清單，然後便可以將其與此資料夾中的項目相關聯。如需如何建立工作流程的相關資訊，請參閱《Novell Teaming 2.1 Advanced User Guide》(Novell Teaming 2.1 進階使用者指南) 中的「Creating and Managing Workflows」(建立和管理工作流程)。

將工作流程與意見 (回覆) 類型相關聯

「意見或回覆類型」區段可讓您指定當使用者輸入特定回覆類型時要啟動的工作流程。

Teaming 顯示「工作流程關聯」區段中列出的每種預設項目類型的所有回覆類型。

如果某預設項目類型也是回覆類型，則不會再次列出。

有關更詳細的資訊，請參閱 法律聲明。
