Gespiegelde mappen

Een gespiegelde map is een map waarvan de bestanden zich op een externe lokatie bevinden (of een map op een bestandssysteem of een map op een Sharepoint server). Als u externe mappen spiegelt vanuit Teaming kunt Kablink Teaming hulpmiddelen (bijvoorbeeld zoeken en workflow) toepassen op externe bestanden.

U moet met een bestand op de server werken om deze functie te activeren.

Voor meer informatie over het aanmaken van gespiegelde mappen, zie Werken met gespiegelde mappen in de Kablink Teaming 2.0 geavanceerde gebruikershandleiding.

Voor meer informatie zie: Juridische kennisgevingen.
