De standaard mapweergave aanpassen

  1. Selecteer de weergave die uw mappenlijst of werkruimte het beste doet uitkomen.

  2. Klik op Toepassen > Sluiten.

    De mapweergave is aangepast.

Voor informatie over hoe u een weergave kunt toevoegen aan deze sectie zodat deze geselecteerd kan worden als de standaardweergave, zie Weergaven instellen voor gebruik als standaard weergave.

Als u een weergave selecteert voor een map houdt u rekening met de vermeldingen die de map bevat. Een vermeldingslijst van agendavermeldingen wordt bijvoorbeeld normaal gesproken het beste weergegeven in een agendaweergave.

Voor meer informatie zie: Juridische kennisgevingen.
