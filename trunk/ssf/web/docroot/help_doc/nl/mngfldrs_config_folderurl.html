Een URL voor een map of werkruimte instellen

Gebruik deze sectie om een gebruiksvriendelijke URL voor de map of werkruimte waar u nu bent in te stellen. Het instellen van deze URL stelt u in staat direct naar de map of werkruimte te navigeren binnen uw browser.

Het instellen van een URL stelt u in staat vermeldingen te maken via e-mail- of SMS-berichten.

  1. In het Definieer URL afrollijstje selecteert u één van de vooraf gedefinieerde sleutelwoorden.

    Tenzij u een sitebeheerder bent moet uw URL beginnen met één van deze vooraf gedefinieerde sleutelwoorden.

  2. Geef in het veld de gewenste URL op voor de map of werkruimte.

  3. Klik op Toevoegen.

  4. (Optioneel) Als u een URL instelt voor een map moet u deze URL activeren door deze te selecteren. De URL die u moet selecteren wordt weergegeven onder de Toevoegen knop.

  5. (Optioneel) Als u een URL instelt voor een map klikt u op Toepassen.

  6. Klik op Sluiten.

Om een ingesteld URL te verwijderen selecteert u de URL in de Huidige URL's sectie en klikt u op Verwijder de geselecteerde URL's.
