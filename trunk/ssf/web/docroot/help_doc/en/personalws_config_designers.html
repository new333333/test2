Understanding Designers

Using the Form and View Designers, you can alter default definitions or create new ones.

To use the form and view designer:

  1. Navigate to the folder or workspace where you want to modify or create the definitions.

  2. Click Manage > Form and View Designers.

Kablink Teaming offers the following designers:

  • Entry designer: Definitions for entries.

  • Folder designer: Definitions for folders.

  • Workflow designer: Definitions for workflow processes.

  • Profile listing designer: Definitions for the personal workspaces page.

  • Profile designer: Definitions for contact information (alter using the button).

  • Workspace designer: Definitions for global and team workspaces.

  • User workspace designer: Definitions for personal workspaces.

When you use the non-workflow designers, click the plus sign (+) to expand the tree until you can see the view and form. Click a title of an element you want to modify or that is near the location of a new element, then use the tools presented to the right of the tree to add, modify, delete, or move elements. To guide your work, the designer displays a preview of the form near the bottom of the page.

Site administrators can use all of the designers; workspace and folder owners can use only the entry, folder, and workflow designers.
