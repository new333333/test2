Understanding Designers

Using the form and view designers, you can alter default definitions or create new ones.

To use these tools, click the plus sign (+) next to Form and view designers in the administration portlet, and then click the category within which you want to work.

Kablink Teaming offers these designers:

  • Entry designer: Definitions for entries.

  • Folder designer: Definitions for folders.

  • Workflow designer: Definitions for workflow processes.

  • Profile listing designer: Definitions for the Personal workspaces page.

  • Profile designer: Definitions for contact information (alter using the button).

  • Workspace designer: Definitions for global and team workspaces.

  • User workspace designer: Definitions for personal workspaces.

When using the non-workflow designers, click the plus sign (+) to expand the tree until you can see the view and form. Click a title of an element you want to modify or that is near the location of a new element, and then use the tools presented to the right of the tree to add, modify, delete, or move elements. To guide your work, the designer displays a preview of the form toward the bottom of the page.

Site administrators can use all of the designers; workspace and folder owners can use only the entry and workflow designers.
