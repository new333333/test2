Folder Toolbar

The folder toolbar is populated with options specific to the folder you are viewing. However, you can configure the folder toolbar to contain options that are specific to other folders. For example, if you are viewing the File folder, you can configure the File folder to allow you to add a Discussion entry.

Possible Toolbar Options

You can configure any folder to display any of the following options on the folder toolbar:

  • Add Blog entry: Add a new entry to your Blog folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry and send an e-mail to specific users notifying them that you have submitted an entry.

  • New Blog Page: Add a new blog folder to your existing blog folder. This new folder is subordinate to your main blog folder.

  • Sort by: Sort your blogs by Number, Activity Date, Author, State, or Title.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Blog folder. This action adds the file as a new blog entry.

  • Add Calendar entry: Add a new entry to your Calendar folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry, send an e-mail to specific users notifying them that you have submitted an entry, and more.

  • Add files to folders: Provides an icon whereby you can drag and drop files, or copy and paste files into your Calendar folder. This action adds the file as a new calendar entry.

  • Add Discussion entry: Add a new entry to your Discussion folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry and send an e-mail to specific users notifying them that you have submitted an entry.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Discussion folder. This action adds the file as a new discussion entry.

  • Add File entry: Add a new entry to your File folder. You can add an attachment and description to your entry. You can also subscribe to the entry and send an e-mail to specific users notifying them that you have submitted an entry.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your File folder. This action adds the file as a new file entry.

  • Add Sign the guestbook: Add an entry to your Guestbook folder. You can add a title, description, and attachment to your entry.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into the Guestbook folder. This action adds the file as a new file entry.

  • Add Milestone: Add a new entry to your Milestone folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry, send an e-mail to specific users notifying them that you have submitted an entry, and more.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Milestone folder. This action adds the file as a new file entry.

  • Add MiniBlog: Launches the MiniBlog dialog box, which enables you to compose and post a new miniature blog.

  • Add Photo entry: Add a new entry to your Photo Album folder. You can name, describe, and upload your photographs or other graphics. You can also send an e-mail to specific users notifying them that you have submitted an entry.

  • New Photo Album: Add a new folder to your current Photo Album folder. This new folder is a sub-folder of your main Photo Album folder.

  • Sort by: You can sort your pictures either by Activity date or Title:

    Sorting by Activity date places the most recently uploaded photographs first in the list, while the photographs that have been in your photo album the longest are last.

    Sorting by Title organizes your photographs into alphabetical order.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Photo Album folder. This action adds the file as a new file entry.

  • Add Survey: Add a new entry to your Survey folder. You can add a title, description, and attachments to your entry. You can also send an e-mail to yourself or other users notifying them that an entry has been submitted, and more. Launches the Survey dialog box, which enables you to create a new survey by adding a title, description, and attachments to the survey. You can also set up other survey parameters.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Survey folder. This action adds the file as a new, undefined survey.

  • Add Task entry: Add a new entry to your Calendar folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry, send an e-mail to specific users notifying them that you have submitted an entry, and more.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Task folder. This action adds the file as a new, undefined task.

  • Add Wiki entry: Add a new entry to your Wiki folder. You can add a title, description, and attachments. You can also send an e-mail to specific users notifying them that you have submitted an entry on the Wiki page.

  • New Wiki Topic: Name and add a new folder to your current Wiki folder. This new folder is a sub-folder of your main wiki folder.

  • Sort by: You can sort wiki entries either by Activity date or Title:

    Sorting by Activity date places the most recent entries first in the list, while the entries that have been in the wiki folder the longest are last.

    Sorting by Title organizes the wiki entries into alphabetical order.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Wiki folder. This action adds the file as a new wiki entry. It is visible in the Wiki Pages section of the Wiki Sidebar.
