Folder Entry ToolBar

Kablink Teaming populates the Folder Entry toolbar with options specific to the folder you are viewing. However, you can configure the Folder Entry toolbar to contain various entry types. For example, if you have a Wiki folder, and you want that folder to also contain photo entries, you can configure the folder to also allow Photo entries.

Toolbar Options

Every folder type has default options that automatically display on the Folder Entry toolbar. Following is a list of each type of folder, and the options that are automatically displayed on the respective Folder Entry toolbars.

Blog

  • Add Blog entry: Adds a new entry to your Blog folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry and send an e-mail to specific users notifying them that you have submitted an entry.

  • New Blog Page: Adds a new blog folder to your existing blog folder. This new folder is subordinate to your main blog folder. You can use new blog pages to better organize your blog folder into meaningful topics.

  • Sort by: Sort your blogs by Number, Activity Date, Author, State, or Title.

  • Add files to folder: Provides an icon that enables you to drag and drop files, or copy and paste files into your Blog folder. This action adds the file as a new blog entry.

Calendar

  • Add Calendar entry: Adds a new entry to your Calendar folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry, send an e-mail to specific users notifying them that you have submitted an entry, and more.

  • Add files to folders: Provides an icon that enables you to drag and drop files, or copy and paste files into your Calendar folder. This action adds the file as a new calendar entry.

Discussion

  • Add Discussion entry: Adds a new entry to your Discussion folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry and send an e-mail to specific users notifying them that you have submitted an entry.

  • Add files to folder: Provides an icon that enables you to drag and drop files, or copy and paste files into your Discussion folder. This action adds the file as a new discussion entry.

File

  • Add File entry: Adds a new entry to your File folder. You can add an attachment and description to your entry. You can also subscribe to the entry and send an e-mail to specific users notifying them that you have submitted an entry.

  • Add files to folder: Provides an icon that enables you to drag and drop files, or copy and paste files into your File folder. This action adds the file as a new file entry.

Guestbook

  • Add Guestbook signature: Adds an entry to your Guestbook folder. You can add a title, description, and attachment to your entry.

  • Add files to folder: Provides an icon that enables you to drag and drop files, or copy and paste files into the Guestbook folder. This action adds the file as a new file entry.

Milestone

  • Add Milestone: Adds a new entry to your Milestone folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry, send an e-mail to specific users notifying them that you have submitted an entry, and more.

  • Add files to folder: Provides an icon that enables you to drag and drop files, or copy and paste files into your Milestone folder. This action adds the file as a new file entry.

Micro-Blog

  • Add Micro-Blog: Launches the Micro-Blog dialog box, which enables you to compose and post a new miniature blog.

Photo Album

  • Add Photo entry: Adds a new entry to your Photo Album folder. You can name, describe, and upload your photographs or other graphics. You can also send an e-mail to specific users notifying them that you have submitted an entry.

  • New Photo Album: Adds a new folder to your current Photo Album folder. This new folder is a sub-folder of your main Photo Album folder.

  • Sort by: You can sort your pictures either by Activity date or Title:

    Sorting by Activity date places the most recently uploaded photographs first in the list, while the photographs that have been in your photo album the longest are last.

    Sorting by Title organizes your photographs into alphabetical order.

  • Add files to folder: Provides an icon that enables you to drag and drop files, or copy and paste files into your Photo Album folder. This action adds the file as a new file entry.

Survey

  • Add Survey: Adds a new entry to your Survey folder. You can add a title, description, and attachments to your entry. You can also send an e-mail to yourself or other users notifying them that an entry has been submitted, and more. You can also set up other survey parameters.

  • Add files to folder: Provides an icon that enables you to drag and drop files, or copy and paste files into your Survey folder. This action adds the file as a new, undefined survey.

Task

  • Add Task entry: Adds a new entry to your Calendar folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry, send an e-mail to specific users notifying them that you have submitted an entry, and more.

  • Add files to folder: Provides an icon that enables you to drag and drop files, or copy and paste files into your Task folder. This action adds the file as a new, undefined task.

Wiki

  • Add Wiki entry: Adds a new entry to your Wiki folder. You can add a title, description, and attachments. You can also send an e-mail to specific users notifying them that you have submitted an entry on the Wiki page.

  • New Wiki Topic: Lets you name and add a new folder to your current Wiki folder. This new folder is a sub-folder of your main wiki folder.

  • Sort by: You can sort wiki entries either by Activity date or Title:

    Sorting by Activity date places the most recent entries first in the list, while the entries that have been in the wiki folder the longest are last.

    Sorting by Title organizes the wiki entries into alphabetical order.

  • Add files to folder: Provides an icon that enables you to drag and drop files, or copy and paste files into your Wiki folder. This action adds the file as a new wiki entry. It is visible in the Wiki Pages section of the Wiki Sidebar.

Configuring A Folder To Allow Alternate Entry Types

  1. Navigate to the folder where you want to add the alternate entry type.

  2. Click Manage > Configure.

  3. In the Default Entry Types section, select the folders that represent the entry types that you want to enable on the folder.

  4. Click Apply > Close.

  5. Click Add in the Folder Entry toolbar, then from the drop-down list, select the entry type that you want to add.
