Allowing E-Mail

The following sections describe the options on this page.

Enabling Outgoing E-Mail

You can modify the following configuration options for outgoing e-mail:

  • Enable Outgoing E-Mail: Select this check box to enable all outgoing e-mail from Teaming. If you donâ€™t want to allow users to send e-mail messages from Teaming, deselect this check box.

    This check box must be selected if you want to allow users to send digest notifications.

  • Send E-Mail Notifications (outgoing): Use this section to set the configuration defaults for digest notifications. If Teaming users want to send digest e-mail notifications, they must use the digest notification settings that you select here. You can configure digest notifications to be sent every day, on certain days of the week, at a certain time of the day, or at hourly intervals.

Enabling Incoming E-Mail Using the Embedded E-Mail Server and Simple URLs

Select the Enable Incoming E-Mail Using the Embedded E-Mail Server and Simple URLs check box if you want to allow entries to be submitted via e-mail to folders that have been preconfigured with a simple URL.
