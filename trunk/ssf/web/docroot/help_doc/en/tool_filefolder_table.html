Folder Table

When working with the folder table, each row contains information about one entry. Here are tips for using this table:

  • At the top of the table, click and hold a line with two arrow heads, and drag it to the left or right to resize the column.

  • Click a column header to sort the table in order according to the contents of that column. Click it again to reverse the order. The Arrow icon next to the column header indicates that the table is currently sorted by that column (the up arrow means descending order, and the down arrow means ascending order).

  • Mouse over a cell (for example, a title too long for the table cell), and Kablink Teaming displays the entire contents of the cell.

  • Click the down-arrow next to the entry title, select a viewing method (using the default method or in a new browser window), and view the entry.
