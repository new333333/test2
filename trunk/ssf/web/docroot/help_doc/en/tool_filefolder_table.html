Folder Table

When working with the folder table, each row contains information about one entry. Use the following tips for working with this table:

  • Resizing a Column: At the top of the table, click and hold a line with two arrowheads, and drag it to the left or right.

  • Sorting Contents of the Table: Click a column header to sort the table in order according to the contents of that column. Click it again to reverse the order. The arrow icon next to the column header indicates that the table is currently sorted by that column (the up-arrow indicates descending order, and the down-arrow indicates ascending order).

  • Cell Mouse-Over: Mouse over a cell (for example, a title too long for the table cell), and Kablink Teaming displays the entire contents of the cell.
