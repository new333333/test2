Include Accessibility Enhancements

When you include accessibility enhancements, Kablink Teaming presents a slightly altered user interface that conforms to the United States Section 508 and W3C international standards for accessibility. You can choose to include these enhancements when using Teaming with assistive devices such as page readers.

After you click this link, Teaming displays the Exclude Accessibility Enhancements link, which returns you to the default user interface.
