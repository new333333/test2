Allowed Views

Because only folder owners have rights to set the default view of a folder, the Allowed Views section enables the folder owner to select views that external users of the folder are allowed to use, if they are not satisfied with the default view, or feel more comfortable using another view. For example, if you are configuring your Calendar folder and you select Task in this section, then users who view your Calendar folder have the option to view the folder in the Task folder view.

Enabling Views to Be Used On Your Folder

  1. In the Allowed Views section, select the views that you want to be allowed for the folder that you are configuring.

  2. Click Apply > Close.

  3. In the main sidebar, in the Folder Options section, click Folder View.

    All of the allowed views for the folder are displayed. (These are the views that you selected in the Allowed Views section in Step 1.) Users of the folder are able to choose views from this list depending on how they want to view the folder.

Enabling Views to Be Used As the Default View

Selecting a view in the Allowed Views section adds the view to the Default View section. This makes it possible for the folder owner to choose that view as the default view.

  1. In the Allowed Views section, select the view that you want to add to the Default View section.

  2. Click Apply.

    The view that you selected is added to the Default View section.

For information on how to set the default view of the folder, see Changing the Default Folder View.
