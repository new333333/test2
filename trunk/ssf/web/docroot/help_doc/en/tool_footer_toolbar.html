Footer Toolbar

The Footer toolbar can include the following information:

  • Folder or Workspace Permalink: Enables you to reference the workspace or folder. These links remain constant over time, and can be used as bookmarks or e-mailed to other users. Copy and paste them as needed.

  • Folder E-Mail Address: If you have configured a folder to have a user-friendly URL, then the Footer toolbar displays this URL when you click the footer link.

  • RSS Link: Enables you to access the folder as an RSS feed. Simply copy the RSS URL and paste it into your RSS reader.

  • WebDAV URL: If you are using Internet Explorer 7 or later as your web browser, you can use the WebDAV URL to access Kablink Teaming files directly from Microsoft Office, and similar applications.

  • iCal URL: Enables you to synchronize your Teaming task and calendar folders to your desktop calendar application.

    When you view a calendar folder or entry, you can use this link to export Teaming calendar data for inclusion in another calendar that supports the iCal format (for example, Google and Outlook). Click the link, save the file with .ics file extension, then import the file into your other calendar (see the Help for your other calendar for more information).

  • View Folder as a WebDAV Folder: Enables you to view the folder as a WebDAV folder.

    For more information, see Using WebDAV.
