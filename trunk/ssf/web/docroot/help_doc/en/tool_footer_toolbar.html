Footer Toolbar

The contents of the Footer toolbar change, depending on whether you are in a workspace, folder, or entry:

Workspace Footer Toolbar

  • Workspace Permalink: Enables you to reference the workspace. These links remain constant over time, and can be used as bookmarks or e-mailed to other users. Copy and paste them as needed.

Folder Footer Toolbar

  • Folder Permalinks: Enables you to reference the folder. These links remain constant over time, and can be used as bookmarks or e-mailed to other users. Copy and paste them as needed.

  • E-Mail Addresses: If a folder has already been configured to have a user-friendly URL, the Footer toolbar displays this URL when you click the footer link.

    For more information on how to configure a folder to receive entries by way of e-mail, see Setting Up a Folder to Receive Entries Via E-Mail in the Kablink Teaming 2.0 User Guide.

  • RSS Link: Enables you to access the folder as an RSS feed. Simply copy the RSS URL and paste it into your RSS reader.

  • WebDAV URL: If you are using Internet Explorer 7 or later as your Web browser, you can use the WebDAV URL to access Kablink Teaming files directly from Microsoft Office, and similar applications.

  • iCal URL: Enables you to synchronize your Teaming task and calendar folders to your desktop calendar application.

    When you view a calendar folder or entry, you can use this link to export Teaming calendar data for inclusion in another calendar that supports the iCal format (for example, Google and Outlook). Click the link, save the file with a .ics file extension, then import the file into your other calendar (see the Help for your other calendar for more information).

  • View Folder as a WebDAV Folder: Enables you to view the folder as a WebDAV folder.

Entry Footer Toolbar

  • Entry Permalink: Enables you to reference the entry. These links remain constant over time, and can be used as bookmarks or e-mailed to other users. Copy and paste them as needed.

  • Send E-Mail: Launches an e-mail form that enables you to send an e-mail to anyone that you desire. The e-mail contains the location of the entry in Teaming, and some of the content of the entry.

  • Subscribe to This Entry: Launches a form that enables you to subscribe the entry. When you subscribe to an entry, you configure Teaming to send you e-mail notifications informing you about updates to that entry.

    For more information about subscribing to an entry, see Using E-Mail Notifications.
