Linking to an Entry in a Different Folder

If you want to link to an entry in another folder:

  1. Click Change.

  2. In the provided field, start typing the name of the folder where the entry is that you want to link to.

  3. When the desired folder appears in the type-to-find list, select it.

  4. In the Page name to link to field, specify the complete name of the entry that you want to link to, or click Find to use the type-to-find functionality.

  5. After you have finished typing the name of the entry, or after you have clicked the name in the type-to-find drop-down list, click Insert.

    The linked title of your chosen entry is inserted into the Description box.
