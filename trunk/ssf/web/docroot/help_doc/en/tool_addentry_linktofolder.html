Link To Folder

If you want to link to an entry in another folder:

  1. In the Description section of the Add entry form, click the Apply link icon, located at the end of the row of buttons in the HTML editor.

    Kablink Teaming displays a window that allows you to specify the entry to which you want to link.

  2. Click the Change link.

  3. In the provided field, start typing the name of the folder where the entry is that you want to link to.

  4. When the desired folder appears in the type-to-find list, click it.

  5. In the Page name to link to field, enter the complete name of the entry that you want to link to, or click Find to use the type-to-find functionality.

  6. After you have finished typing the name, or after you have clicked on the name in the type-to-find drop-down list, click Insert.

    The linked title of your chosen entry is inserted into the Description box.
