Understanding E-Mail Notifications

A folder owner can select a type of notification and a list of people who receive the notification. People's individual subscription settings override those of the folder owner.

Available in folders only, digest notifications are single e-mail messages that contain a summary of all new and changed entries in the folder since the last notification. Kablink Teaming reviews recent activity at a schedule determined by the site administrator; if changes occurred, Kablink Teaming sends a digest message. Individual notifications are separate e-mail messages for every new or changed entry, and Kablink Teaming reviews recent activity every five minutes. Text-messaging notifications are individual notifications that are suitable for display as text messages; these text messages include only the titles of the new or changed entries.

To set up text-message notifications, view your personal workspace, click , and specify an e-mail address to which Kablink Teaming sends the text-message notifications. Then, use the subscription form and its Message suitable for text messaging select box to select the e-mail address you just placed in your profile.

People's profiles can contain more than one e-mail address of any type (general e-mail, mobile, and text messaging). Also, Kablink Teaming allows people to receive notifications at any number of e-mail addresses of any type. For example, a person can receive two digest notifications and an individual message: one digest going to a work address, one digest going to a home address, and text message notifications going to a cell phone address. When someone subscribes to a folder or entry, under each category of notification (digest, individual, text message), that person uses a select box to select any number of e-mail addresses. Finally, to turn off all notifications (including those sent by folder owners using distribution lists), a person must leave all select boxes blank and must select the Disable notifications... check box.
