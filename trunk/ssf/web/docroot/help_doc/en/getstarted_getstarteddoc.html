Getting Started Documentation

The links that follow this paragraph display getting started manuals in a new browser window. These manuals are formatted as PDF files and are optimized for printing; however, you can also read them online.

All Teaming Users

The following guide is of interest to all Kablink Teaming users:

  • Novell Teaming 2.0 User Guide

    This document provides detailed information that describes how to accomplish various tasks in Teaming. It includes overview descriptions, navigation instruction, customization instructions, and more graphics.

Teaming Administrators

The following guides are of interest to Kablink Teaming administrators:

  • Novell Teaming 2.0 Administration Guide

    This document describes the tasks required for an administer to set up, configure, and operate a Teaming site.

  • Novell Teaming 2.0 Installation Guide

    This document describes the tasks required to install Teaming so that all default features operate as intended. It also describes how to set up a Teaming user.

  • Novell Teaming 2.0 Web Services Guide

    This document is written primarily for developers.
