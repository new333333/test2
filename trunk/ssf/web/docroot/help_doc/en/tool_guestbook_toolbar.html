Guestbook Toolbar

The Guestbook toolbar provides the following options that enable you to manage your Guestbook folder:

  • Add Sign the guestbook: Add an entry to your Guestbook folder. You can add a title, description, and attachment to your entry.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into the Guestbook folder. This action adds the file as a new file entry.
