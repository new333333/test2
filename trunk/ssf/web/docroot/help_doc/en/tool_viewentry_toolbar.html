Entry Toolbar

The Entry toolbar provides only those tools that you have permission to use. The toolbar appears for both entries and comments, and it can include the following buttons and menus:

  • Comment: Adds a comment to the entry. You can make a comment about a comment (this is called making nested comments).

  • Modify: Enables you to edit the entry or to add another attachment. (You can also use the Edit link in the lower right section of the entry, just above the Attachments section.)

  • Reserve: Prevents others from editing the entry while you are editing it. It is a good idea to reserve an entry if you are making significant changes to the entry. When you are finished making the changes, click Unreserve.

  • Move: Enables you to move the entry to another folder.

  • Copy: Enables you to copy the entry and save it to another location.

  • Delete: Removes the entry from the folder.

  • Workflow: When a workflow is enabled on a folder, this tool enables you to start a workflow if the workflow has not been started on the entry, or stop a workflow if the workflow is currently in process on the entry.

  • Set Wiki Home Page (Only Available With Wiki Entries): Use this option to configure Teaming to display any wiki page as the wiki home page. Simply navigate to the page that you want to set as the wiki home page, then click this option.

  • Reports: Enables you to generate the following reports:

    • Activity Report: Displays a comma-separated values file in Microsoft Excel, which provides information about users and their activities related to this entry.

    • Edit History: Launches an Edit History window, where you can compare different versions of the entry by seeing what was added and what was deleted.

    • Workflow History: Launches a Workflow History window where you can view the history of the workflow, including the workflow name, when the entry was created and who created it, when the workflow entered a certain state, who modified the entry when it entered that state, and more.

  • Share This Entry: Enables you to share an entry with other Teaming users. When you share an entry, a notification e-mail is sent to the users you selected to share the entry with. Also, in their home workspaces, users can select the MiniBlogs and Shared Items tab, and in the Shared Items column, Teaming displays all the items that people have shared with them.
