Default Entry Types

This section enables you to select the types of entries you want to allow users to create in this folder. You can configure your folder to allow additional entry types, or you can change the current entry type. If you add additional entry types, users who submit entries into this folder are able to choose which type of entry they want to add, whether it be a task entry, calendar entry, and so forth.

  1. Select the entry types that you want to enable on the folder.

  2. Click Apply > Close.

If you enable additional entry types, users who submit entries into this folder are able to click Add in the Folder Entry toolbar, then choose from the drop-down list which type of entry they want to add, depending on which types of entries you enabled on the folder.

Site administrators can use the Form and View Designers functionality to create new entry types (custom forms), which then appear in this list.
