Display Entry and Contact People

Several tools in Kablink Teaming (for example, folder listings and search results) display a summary of entry information. In the summary, use the Display entry icon to quickly view the entry. This is the Display entry icon:

Using the Display entry icon, you can view the entry using the default folder setting (show entry) or you can view it in a new browser window.

The entry summary also includes a Presence icon, located next to the author's name. This Presence icon can appear as follows:

The Presence icon indicates if the person is active at the computer (green), recently active but now away (yellow), not logged into the Pidgin client of SiteScape ICEcore Conference (gray), or unknown status (white). Click the Presence icon and select a menu item to contact the person (for example, send e-mail, send an instant message using SiteScape ICEcore Conference, add the person to your clipboard, and so on).
