Workspace View

Select the view that best presents your workspace.

  • Discussions Workspace: Select this view if you want to create a workspace where you can store multiple discussion forums.

    For more information about discussions workspaces, see Managing a Discussions Workspace in the Kablink Teaming 2.0 Advanced User Guide.

  • Landing Page: Select this view if you want to display important information to team members as soon as they enter the workspace.

    For more information on landing pages, see Creating a Landing Page in the Kablink Teaming 2.0 Advanced User Guide.

  • Project Management Workspace: Select this view if you want to track project tasks and milestones based on task completion.

    For more information about project management workspaces, see Managing a Project Management Workspace in the Kablink Teaming 2.0 Advanced User Guide.

  • Team Workspace: Select this view if your team workspace is organizational in nature, and not intended to have team workspaces as its sub-places.

    For more information about team workspaces, see Managing a Team Workspace in the Kablink Teaming 2.0 User Guide.

  • Team Workspace Root: Select this view if you want to display the Add a team workspace button in your team workspace. This is desirable if you want to create additional team workspaces as sub-places to this workspace.

    For more information on team workspace root workspaces, see Managing a Team Workspace Root Workspace in the Kablink Teaming 2.0 Advanced User Guide.

  • Workspace: This is a basic workspace. You can choose which folders you want the workspace to contain.

    For more information on basic workspaces, see Managing a Basic Workspace in the Kablink Teaming 2.0 Advanced User Guide.

If this is a personal workspace, the default view is User Workspace, and it cannot be changed.
