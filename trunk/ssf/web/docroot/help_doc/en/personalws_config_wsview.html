Workspace View

Select the view whose look best presents your workspace.

  • Discussions Workspace: Select this view if you want to create a workspace where you can store multiple discussion forums.

  • Landing Page: Select this view if you want to display important information to team members as soon as they enter the workspace. For more information on team landing pages and how to create one, see

  • Project Management: Select this view if you want to track project tasks and milestones based on task completion.

  • Team Workspace: Select this view if your team workspace is organizational in nature, and not intended to have team workspaces as its sub-places.

  • Team Workspace Root :Select this view if you want to display the Add a team workspace button in your team workspace. This is desirable if you want to create additional team workspaces as sub-places to this workspace.

  • Workspace: This is a basic workspace. You can choose which folders you want the workspace to contain.

For more information, see Understanding Kablink Teaming Teams.
