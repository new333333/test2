Applying Configuration Settings

To configure a workspace or folder, view the place, and click Manage this... > Configure.

When configuring a workspace, you can inherit the workspace definition from the parent workspace, and you can select a default view. You cannot alter the default view of your organization's top workspace.

You can establish these configuration settings for folders:

  • Allowable views: Users can select from this list.

  • Default view: The view when first visiting the folder.

  • Default entry types: Types of entries a user can create.

  • Workflow associations: Workflow processes that begin automatically as soon as a user creates the specified type of entry.

  • Workflows: Workflow processes that you can apply to an entry while viewing it, using a drop-down list.
