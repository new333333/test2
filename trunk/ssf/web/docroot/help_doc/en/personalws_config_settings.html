Configuring a Workspace or Folder

  1. Navigate to the workspace or folder that you want to configure.

  2. In the Workspace or Folder toolbar, click Manage > Configure.

Following are the configuration settings that you can choose, depending on if you are configuring a workspace or a folder:

Configuring A Workspace

When configuring a workspace, you can inherit the workspace definition from the parent workspace, and you can select a default view. You cannot alter the default view of your organization's top workspace.

Configuring a Folder

You can establish the following configuration settings for folders:

  • Allowed Views: Users can select from this list.

  • Default View: The view when first visiting the folder.

  • Default Entry Types: Types of entries a user can create.

  • Workflow Associations: Workflow processes that begin automatically as soon as a user creates the specified type of entry.

  • Allowed Workflows: Workflow processes that you can apply to an entry while viewing it, by using Workflow drop-down list on the Entry toolbar.
