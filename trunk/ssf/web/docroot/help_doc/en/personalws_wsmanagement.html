Workspace Management Tools

When you click Manage in the Workspace toolbar, Kablink Teaming displays only those management and configuration tools that you are allowed to use. Site administrators, or the owner of the workspace, can grant access to these tools.

If you have unrestricted access, you see the following options:

  • Add New Folder: Enables you to add a new folder to the workspace.

  • Add New Workspace: Enables you to add a new workspace underneath the workspace you are currently in.

  • Access Control: Enables you to regulate who can view and participate in this place.

  • Activity Reports: Enables you to view activities that have been taking place in the workspace.

  • Configure: Enables you to configure the workspace, including setting the default view and the workspace URL.

  • Copy This Workspace: Enables you to copy the current workspace into whatever destination in the Workspace tree (see Workspace Tree Navigation) you desire, if you have the appropriate rights.

  • Delete This Workspace: Deletes the current workspace.

  • Form and View Designers: Enables you to create custom entry forms and custom workflows that you can later apply to the workspace and its sub-folders.

  • Modify This Workspace: Enables you to modify the workspace, including the workspace name, description, landing page, brand, and icon.

  • Move This Workspace: Enables you to move the current workspace into whatever destination in the Workspace tree (see Workspace Tree Navigation) that you desire, if you have the appropriate rights.
