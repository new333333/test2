Receiving E-Mail Notifications

Tell your Kablink Teaming site administrator that you want to enable e-mail posting for your folder. The administrator then provides you with either one or two pieces of information to place in these text boxes. If given only one piece of information, place it in the first text box and leave the Account password... text box empty.

The link that follows provides detailed background information about the values you are given to complete these text boxes. It is most helpful for Kablink Teaming site administrators.
