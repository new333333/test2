Editing Attached Files

Kablink Teaming enables you to edit files that are attached to entries.

If you are unable to edit a file, Teaming might not be configured to support the editing application that is installed on your computer. For example, if Teaming is configured to use Microsoft Word as the default editor for all text files, but you have OpenOffice instead of Microsoft Word installed on your computer, Teaming displays an error, and wonâ€™t edit the document.

To fix this problem, Teaming enables you to override the default editor setting.
