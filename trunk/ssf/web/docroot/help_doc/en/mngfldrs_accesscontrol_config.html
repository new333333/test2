Configure Access Control

Use this page to assign people or groups to roles for this workspace or folder. A role is a set of privileges; when you assign someone to a role, they automatically have all the rights associated with that role. In the table that follows, role names serve as column headers. Click a column header (Participant, Site administration, and so on) to see a list of rights associated with the role.

By default, workspaces and folders inherit access control settings from their parents. Use this page only if you want different access settings.
