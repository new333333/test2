Editor Overrides

When you edit attachments in Kablink Teaming, the Teaming systemâ€™s default applications are used to edit the attachments. However, this table enables you to override the default applications that are used to edit attachments with specific file extensions within Teaming.

You can override the default editing application for a single file extension type, or you can override the default applications for multiple file extension types.

Overriding the Default Editing Application for Individual File Extensions

To override the default application for an individual file extension type:

  1. Click Add.

  2. From the drop-down list, select the file extension type for which you want to override the default editing application.

  3. In the Application field, specify the name of the application that you want to use to edit attachments with the file extension that you specified in Step 2.

  4. Click OK.

Overriding the Default Editing Application for All File Extensions

To override all file extension types:

  1. Click Use.

  2. From the drop-down list, select the application you want to use to edit attachments with all file extension types in Teaming.

  3. Click OK.
