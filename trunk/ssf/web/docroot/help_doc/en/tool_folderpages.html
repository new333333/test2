Listing Entries and Pages

These tools enable you to search for entries individually, or navigate among pages of listings. A folder listing displays entries or information about entries. Some of these tools are useful only if there is more than one listing page of entries. From left to right on this toolbar, the available tools are:

  • Go to this Entry: Specify the number (displayed in the # column) of the entry that you want to view, then click Go.

  • Previous/Next: These buttons enable you to scroll through the pages of the folder.

  • Go to page: When you type a page number in this text box and then click Go, Kablink Teaming displays the entries on that listing page.
