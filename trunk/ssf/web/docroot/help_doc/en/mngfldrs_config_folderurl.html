Defining a URL for a Folder or Workspace

You can define a user-friendly URL for a folder or workspace, enabling you to navigate directly to that folder or workspace from your browser.

Defining a URL for a folder enables you to add entries to that folder via e-mail or text message.

Unless you are a site administrator, your URL must begin with either your login name or one of the pre-defined global keywords provided in the drop-down list.
