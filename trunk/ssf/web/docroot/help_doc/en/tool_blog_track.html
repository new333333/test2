Blog Tracking

The Blog Tracking toolbar provides quick ways to:

  • Add a Comment: Enables you to make a comment on a blog entry. Before you make a comment, it is a good idea to view the comments that have previously been made. When you add a comment to a blog that has already been commented on, your comment is automatically added as the last comment in the list of nested comments.

  • View Comments: Enables you to view all the comments of a particular entry, without opening the entry.

  • Send Entry to a Friend: Enables you to send the contents of the entry, via e-mail, to anyone who has appropriate access rights to view the entry.

  • Subscribe: Launches a form that enables you to subscribe the entry. When you subscribe to an entry, you configure Teaming to send you e-mail notifications informing you about updates to that entry.

    For more information about subscribing to an entry, see Using E-Mail Notifications.
