Task Folder Toolbar

The Task Folder toolbar provides the following options that enable you to manage your Task Folder:

  • Add Task entry: Add a new entry to your Calendar folder. You can add a title, description, and attachment to your entry. You can also subscribe to the entry, send an e-mail to specific users notifying them that you have submitted an entry, and more.

  • Add files to folder: Provides an icon whereby you can drag and drop files, or copy and paste files into your Task folder. This action adds the file as a new, undefined task.
