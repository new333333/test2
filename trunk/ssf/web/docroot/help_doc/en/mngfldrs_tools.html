Folder Management Tools

When you click Manage, Kablink Teaming displays only those management and configuration tools you are allowed to use. Site administrators, or the owner of this workspace or folder, can use and grant access to these tools.

If you have unrestricted access, you see the following options:

  • Add new folder: Enables you to add a new folder to the workspace.

  • Add new workspace: Enables you to add a new workspace underneath the workspace you are currently in.

  • Access Control: Enables you to regulate who can view and participate in this place. access to the workspace.

  • Activity Reports: Enables you to view activities that have been happening in the workspace.

  • Configure: Enables you to configure the workspace, including setting the default view and the workspace URL.

  • Copy this workspace: Enables you to copy the current workspace into whatever destination in the Workspaces tree (see Workspace Tree Navigation) you desire, provided you have the appropriate rights.

  • Delete this workspace: Deletes the current workspace.

  • Form and View Designers: Enables you to form and view the definitions for the workspace.

  • Modify this workspace: Enables you to modify the workspace, including the workspace name, description, branding, and icon.

  • Move this workspace: Enables you to move the current workspace into whatever destination in the Workspaces tree (see Workspace Tree Navigation) you desire, provided you have the appropriate rights.

The Manage menu allows you to add sub-workspaces, folders, and sub-folders; generate reports; configure views for the page; or delete, modify, or move this place

The Manage > Access control menu item alters who can view and participate in this place. The Team menu allows you to manage team membership for this place and to communicate rapidly with team members.

The Subscriptions menu allows you to receive e-mail notifications about new or changed entries, and allows an RSS Reader to retrieve those entries.
