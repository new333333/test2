Folder Management Tools

When you click Manage in the Folder toolbar, Kablink Teaming displays only those management and configuration tools you are allowed to use. Site administrators, or the owner of this workspace or folder, can use and grant access to these tools.

If you have unrestricted access, you see the following options:

  • Add new folder: Enables you to add a new folder to the workspace.

  • Add new workspace: Enables you to add a new workspace underneath the workspace you are currently in.

  • Access Control: Enables you to regulate who can view and participate in this place.

  • Activity Reports: Enables you to view activities that have been happening in the workspace.

  • Configure: Enables you to configure the folder, including setting the default view, default entry types, allowed views, and the folder URL.

  • Copy This Folder: Enables you to copy the current folder into whatever destination in the Workspace tree (see Workspace Tree Navigation) you desire, if you have the appropriate rights.

  • Delete This Folder: Deletes the current folder.

  • E-Mail Settings: Enables you to configure the folder to automatically send e-mail notifications to other Teaming users.

  • Form and View Designers: Enables you to form and view the definitions for the workspace.

  • Modify This Folder: Enables you to modify the folder, including the folder name, description, branding, and icon.

  • Move This Folder: Enables you to move the current folder into whatever destination in the Workspace tree (see Workspace Tree Navigation) you desire, if you have the appropriate rights.
