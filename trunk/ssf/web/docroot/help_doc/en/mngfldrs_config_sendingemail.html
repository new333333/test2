Sending E-Mail Notifications

As a folder owner, you can configure Kablink Teaming to send other users e-mail notifications, to inform them of any activity in this folder and any sub-folders of this folder. However, a userâ€™s individual settings override or cancel notifications established by folder owners.

  1. In the Select the Type of E-Mail Notification section, select the type of e-mail notification that you want to send.

    For information about the various types of e-mail notifications, see Deciding Which Type of E-Mail Notification You Want to Receive.

  2. Specify the users who you want to receive the e-mail notifications.

    You can specify specific e-mail addresses, users, groups, or send the notifications to all team members.

  3. Click Apply > Close.

For more information on e-mail notifications, see Using E-Mail Notifications.
