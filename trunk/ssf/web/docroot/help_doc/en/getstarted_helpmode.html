Getting Started with Help Mode

The panel in the upper left corner of the page is called the Help Mode Control Panel. Most often, you use it to access the table of contents or the getting started PDF manuals. To minimize the Help Mode Control Panel, click the Minimize icon, located at the top of the control panel. Click this icon again to restore the panel to its full size.

Help mode describes the page you are currently viewing. Click an Information icon, and Kablink Teaming displays a Help panel describing that section of the page. To close the help panel, click the Close icon (X) in the upper right corner of the Help panel. Or, you can click another Information icon to keep reading. Instead of using Information icons, you can click Help topics in the Help Mode Control Panel, and click the linked titles of topics that interest you.

For more complete Teaming documentation, you can click View manuals in the Help Mode Control Panel.

When finished using the Help mode, you can exit by clicking the Close icon (X) in the upper right corner of the control panel, or you can click anywhere on the dimmed part of the page.

After you exit Help mode and begin using tools, Teaming provides Information icons on forms that require additional explanation. Click the Information icon next to a form section to read more about it.
