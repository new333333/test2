Displaying Events in a Calendar

The Show drop-down list enables you to choose the kinds of calendar events that you want to display in the calendar, and how you want to display them. Unless you are in the personal workspace of a Teaming user, you can view only those calendar events that are located in the current folder.

  • Assigned Events: Available only in user workspaces. Displays all calendar events assigned to the owner of the workspace, regardless of where the events are located in Teaming.

  • From Folder - Events: Displays all calendar events that are in the current folder on the days the events are scheduled.

  • From Folder - All by Creation: Displays all calendar events on the days the events were created, rather than on the days the events are scheduled.

  • From Folder - All by Activity: Displays all calendar events on the days the events were last modified, rather than on the days the events are scheduled.
