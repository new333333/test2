Attaching Files to an Entry

In the Attachments section of an entry, Kablink Teaming provides various alternatives to add attachments to the entry:

Dragging and Dropping Files

You can attach a file to an entry by dragging it from another location and dropping into the entry.

  1. Click the Drag and Drop Files icon.

    This launches a window with a folder icon.

  2. Drag the file that you want to upload and drop it onto the folder icon.

    Teaming displays an opening and closing folder to signify that your dropped files are being uploaded as attachments.

The file is now uploaded to the entry.

Accessing Attachments Using WebDAV (Internet Explorer only)

Teaming enables you to use WebDAV to access files and attach them to entries. However, WebDAV is only supported on the Internet Explorer browser.

  1. Click Access Attachments Using WebDAV.

  2. Navigate and select the file that you want to attach.

Uploading a Single File

  1. In the entry where you want to upload the file, click the Upload a Single File icon in the Attachments section.

  2. Click Browse, then browse to and select the file that you want to attach.

  3. Click OK.
