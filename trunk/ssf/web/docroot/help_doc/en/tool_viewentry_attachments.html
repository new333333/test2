Attachments

Click one of the Add attachments icons to attach files to the entry. The first icon allows you to drag and drop files, the second one displays a form used to upload files, and the third one allows you to attach results from SiteScape ICEcore Conference meetings.

After you click the first icon, Kablink Teaming displays a Folder icon. Select files in another window, drag them over the Folder icon, and release the mouse button (drop them). Kablink Teaming displays an opening and closing folder to signify that your dropped files are being uploaded as attachments.

If folders are included in the items you drag and drop, Kablink Teaming adds all the files (including files in the folders) but does not include the folder structure.

If the attachment is a picture, Kablink Teaming displays a small representation of the picture (a thumbnail picture). Click the linked filename, small picture, or No picture available icon to view an attachment.

For non-picture attachments, Kablink Teaming displays links you can use when working with the file (if you have permission). Click the HTML link to view the file in HTML (which is helpful if you do not have the source application needed to work with the file directly), and click the Edit link to edit the file.
