Wiki Sidebar

The Wiki sidebar contains the following features:

  • Wiki Home Page: Click this button to navigate to the wiki home page that you have previously defined. By default, Kablink Teaming does not define a wiki home page; a user must do this manually. If you want to configure the wiki folder to display an initial wiki home page, locate and display the entry that you want to set as the home page, then click Set Wiki Home Page in the Entry toolbar.

  • Find a Page: This text box uses the Type-to-Find feature. Simply start typing the name of the wiki page that you are searching for, then click the name when it is displayed.

    Teaming takes you to the desired page.

  • Wiki Topics: This section lists all of the wiki topics that exist in the wiki folder. When you select a topic from this list, all of the wiki pages inside the selected topic are displayed in the Wiki Pages section.

  • Wiki Pages: This section lists all of the wiki pages that are inside of the wiki topic that is currently selected.
