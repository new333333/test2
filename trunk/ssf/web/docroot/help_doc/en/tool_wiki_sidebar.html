Wiki Sidebar

A wiki is a set of related entries coauthored potentially by all folder participants.

For example, the best known wiki on the Internet is Wikipedia*. Other examples of possible wiki content are glossaries, and policies and procedures

By default, when first viewing a wiki folder, Kablink Teaming displays no wiki page; a user must use the wiki sidebar to locate pages. If you want Kablink Teaming to display an initial wiki page, then locate and display the entry, and click Set homepage in the entry tool bar.

The wiki sidebar includes the Find a page text box, which uses the "type to find" feature. Using this text box, Kablink Teaming lists wiki pages whose titles contain a word or phrase that matches what you have typed so far. For example, if you type "administration tasks", Kablink Teaming lists the "Administration tasks" and "Delegating administration tasks" pages.

The Navigation section provides a sorted list of all wiki pages. Click All to sort the list alphabetically by title. Click Recent to sort the list according to most recent activity (creation or modification).
