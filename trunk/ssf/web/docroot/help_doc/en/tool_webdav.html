Using WebDAV

WebDAV (World Wide Web Distributed Authoring and Versioning) is a standard facilitating collaborative editing and file management. When using Internet Explorer* Version 7.0 or higher, Kablink Teaming enables WebDAV tools. You also must be using the Microsoft Windows XP* operating system to use WebDAV tools.

When you click View this folder as a WebDAV folder in the footer tool bar, Kablink Teaming invokes a WebDAV window. If the window is not operable, consult with the folder owner or administrator to ensure that the configuration supports WebDAV (folder filenames must be unique). In addition to the footer-tool-bar link, Kablink Teaming enables another icon for uploading entry attachments using WebDAV (it appears as the second icon in the set).

Using WebDAV, you need to specify your username and password for Kablink Teaming several times when working with files. WebDAV requires several passwords for applications (your browser and Kablink Teaming), and it might require a third login when accessing an individual file.

Note: While running the Novell software iChain or Access Manager (NAM) on your server, your users might still be required to sign in multiple times as they use WebDAV with Kablink Teaming. Multiple sign-ins using WebDAV are triggered by the client, not by server software.
