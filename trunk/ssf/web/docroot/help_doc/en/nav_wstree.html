Workspace Tree Navigation

This workspace tree displays the path of all of the workspaces and folders that contain the place you are currently viewing.

This tree also allows you to navigate to any other place you want to go within Novell TeamingKablink, but it does so without leaving the current page. When you click on a workspace or folder link in the tree, Novell TeamingKablink displays the new place. To return to the page you were previously using, click its linked title in the Recent places section of the tools sidebar.

Click a plus sign (+) to see an item's sub-workspaces or sub-folders. Click a minus sign (-) to collapse it.
