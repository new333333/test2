Modify Profile

Use this tool to change contact information, such as name, phone numbers, e-mail addresses, and Kablink Teaming Conference user name.

You can also upload pictures, order them using Up arrow and Down arrow icons, and delete them. If you upload more than one picture, Kablink Teaming displays the first picture in the list in the Photo section, and it displays smaller versions of other pictures in a bar below the contact information. When you mouse over one of the smaller pictures, Kablink Teaming displays a larger version in the Photo section.
