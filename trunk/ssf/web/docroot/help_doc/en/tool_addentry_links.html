Using Links in the Description Field

When composing the Description section of an entry, you can include a link to another entry anywhere in your Kablink Teaming installation.

In the Description section of the Add entry form, click the Apply link icon, located at the end of the second row of buttons in the HTML editor. After clicking this icon, Kablink Teaming displays a window allowing you to specify the entry to which you want to link.

If you want to link to an entry in another folder, click the Change link. To locate an entry, either type the entry title exactly in the text box or use the Find link to select the appropriate title. The second link use the "type to find" feature (type a few characters, and you see a list of titles that match what you typed so far). When you click the Insert link, the linked title of your chosen entry is inserted into the Description box.

If you want to apply the link to some other text, type the text into the Description box, select it, and click the Apply link icon to display a window allowing you to select an entry. Kablink Teaming applies the link to the selected text.
