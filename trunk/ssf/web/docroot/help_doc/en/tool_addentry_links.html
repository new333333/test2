Using Links in the Description Field

When composing the Description section of an entry, you can include a link to another page or another entry in your Kablink Teaming installation, or to anywhere else on the web.

Linking To A Kablink Teaming Page

  1. In the Description section of the Add entry form, click the Apply link icon, located at the end of the row of buttons in the HTML editor.

    Kablink Teaming displays a window that allows you to specify the entry to which you want to link.

  2. In the Page name to link to field, you can do any of the following:type the exact page title.

    • Type the exact page title.

    • Click Find next to the field, start typing the name of the page, and select the appropriate title.

    • Use the Permalink feature that is found at the bottom of every Kablink Teaming page. To do this, click the Permalink icon at the bottom of the page you want to link to, copy the URL, and paste the URL into the Page name to link to field.

  3. Click Insert.

Linking To An Entry In Another Folder

If you want to link to an entry in another folder,

  1. In the Description section of the Add entry form, click the Apply link icon, located at the end of the row of buttons in the HTML editor.

    Kablink Teaming displays a window that allows you to specify the entry to which you want to link.

  2. Click the Change link.

  3. In the provided field, start typing the name of the folder where the entry is that you want to link to.

  4. When the desired folder appears in the type-to-find list, click it.

  5. In the Page name to link to field, enter the complete name of the entry that you want to link to, or click Find to use the type-to-find functionality.

  6. After you have finished typing the name, or after you have clicked on the name in the type-to-find drop-down list, click Insert.

    The linked title of your chosen entry is inserted into the Description box.

Linking To Another Page On the Web

  1. In the Description section of the Add entry form, select the text that you want to turn into a link.

  2. Click the Inser/edit link icon.

    The Insert/edit link window is displayed.

  3. In the Link URL field, specify the URL of the page you want the link to.

  4. Fill in the following fields:

    • Target: Specify whether you want to open the link in the same window, or open the link in a new window.

    • Title: Specify a title for the link.

    • Class: You can specify a class for the link.

  5. Click Insert.
