在“说明”字段中使用链接

在撰写条目的说明部分时，可以包括指向 Kablink Teaming 站点中其他条目或因特网上任意位置的链接。

链接到当前文件夹中的条目

  1. 在添加条目表单的说明部分中，单击 HTML 编辑器中按钮行末端的应用链接图标 。

    此时，Teaming 会显示一个窗口，允许您指定要链接到的条目。

  2. 在链接到的页面名称字段中，执行以下任何操作：

    • 输入准确的页面标题。

    • 单击字段旁边的查找，开始输入页面名称，然后从下拉列表中选择相应标题。

    • 使用每个 Kablink Teaming 页面底部的永久链接功能。要使用此功能，请单击要链接到的页面底部的页脚工具栏中的永久链接链接，复制 URL，然后将此 URL 粘贴到链接到的页面名称字段中。

  3. 单击插入。

链接到其他文件夹中的条目

  1. 在添加条目表单的说明部分中，单击 HTML 编辑器中按钮行末端的应用链接图标 。

    此时，Teaming 会显示一个窗口，允许您指定要链接到的条目。

  2. 单击更改链接。

  3. 在提供的字段中，开始输入包含要链接到的条目的文件夹名称。

  4. 当所需文件夹显示在“边输入边查找”列表中时，单击它。

  5. 在链接到的页面名称字段中，指定要链接到的条目的完整名称，或单击查找使用“边输入边查找”功能。

  6. 输入完名称或在“边输入边查找”下拉列表中单击名称后，请单击插入。

    所选条目的链接标题便插入到说明框中。

链接到 Web 上的外部页面

  1. 在添加条目表单的说明部分中，选择要转换成链接的文本。

  2. 单击插入/编辑链接图标 。

    此时会显示“插入/编辑链接”窗口。

  3. 在链接 URL 字段中，指定要链接到的页面的 URL。

  4. 填写以下字段：

    • 目标： 指定要在同一窗口还是新窗口中打开链接。

    • 标题： 指定链接的标题。

    • 分类： 指定所需的链接显示大小。

      如果将此字段留空，则会自动选择 12px。这是标准设置。

  5. 单击插入。

有关更多信息，请参见 法律声明.
