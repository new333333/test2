附件工具

附件是可以添加到 Novell Teaming 页面中、使现有信息更便于查看和访问的部分。它提供了附加功能，并增进了团队协作与交流。以下是附件的一些示例：

  • 搜索结果（可能只应用于当前位置）

  • 好友列表（可能用于本地主题相关的专家）

  • 工作空间树（在层次结构中打开到此位置）

  • 留言板

有关 Teaming 附件及其说明的完整列表，请参见 Novell Teaming 2.1 User Guide 中的附件类型。

以下主题说明了如何使用附件面板：

添加附件

向附件面板添加附件：

  1. 单击附件面板。

  2. 在附件范围部分中，选择所需的附件范围。

    要了解有关此部分中的选项的更多信息，请参见配置有权查看附件的用户。

  3. 在附件类型部分中，选择要添加的附件类型。

  4. 单击确定。

  5. 为附件提供一个标题，并填写所有其他相关信息。

    有关如何为每种类型的附件填写相关信息的更多信息，请参见 Novell Teaming 2.1 User Guide 中的设置附件。

  6. 单击应用 > 关闭。

配置有权查看附件的用户

添加附件时，您可以选择附件范围。附件范围部分中有以下选项：

  • 仅适用于此页面的个人附件： 附件仅适用于当前位置，且只有您才能查看此附件。

  • 适用于所有工作空间和文件夹的个人附件： 附件适用于所有工作空间和文件夹，且只有您才能查看此附件。

  • 适用于此页面的社区附件： 附件仅适用于此页面，且有权访问此页面的所有 Teaming 用户都可以查看此附件。

更改附件的顺序

Teaming 允许在存在多个附件的任何给定页面上重新排列附件。

  1. 在附件标题栏的右上角，单击向上箭头或向下箭头 可根据需要上移或下移附件。

重新配置附件

Teaming 允许在创建附件后重新配置此附件。

  1. 在附件标题栏的右上角，单击修改图标 。

删除附件

  1. 在附件标题栏的右上角，单击删除图标 。

隐藏附件

Teaming 允许隐藏附件，这样就只有附件标题栏可见。

  1. 在附件标题栏的右上角，单击隐藏图标 。

此外，您还可以同时隐藏附件面板中的所有附件。

  1. 单击附件面板旁边的隐藏图标。

    当前页面上的所有附件就从视图中隐藏起来。

    再次单击隐藏图标可显示附件。
