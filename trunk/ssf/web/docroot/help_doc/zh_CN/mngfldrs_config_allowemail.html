允许电子邮件

以下部分说明了此页上的选项。

启用寄出电子邮件

可以为寄出电子邮件修改以下配置选项：

  • 启用寄出电子邮件： 选中此复选框可启用所有从 Teaming 寄出的电子邮件。如果不想允许用户从 Teaming 发送电子邮件讯息，请取消选择此复选框。

    如果希望允许用户发送摘要通知，则必须选中此复选框。

  • 发送电子邮件通知（寄出）： 使用此部分可为摘要通知设置配置默认值。如果 Teaming 用户希望发送或接收摘要电子邮件通知，则他们必须使用您在此处选择的摘要通知设置。可以将摘要通知配置为每天、一周中的某天、一天中的某个时间或每小时发送一次。

使用嵌入式电子邮件服务器和简单 URL 启用进来的电子邮件

如果要允许通过电子邮件向已预配置简单 URL 的文件夹提交条目，请选中使用嵌入式电子邮件服务器和简单 URL 启用进来的电子邮件复选框。

有关更多信息，请参见 法律声明。
