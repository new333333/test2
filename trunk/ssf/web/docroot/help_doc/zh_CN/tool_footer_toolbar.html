页脚工具栏

页脚工具栏的内容会根据是位于工作空间、文件夹还是条目中而有所不同：

工作空间页脚工具栏

  • 工作空间永久链接： 使您能够参照工作空间。这些链接将保持不变，可用作书签或通过电子邮件发送给其他用户。可根据需要对它们进行复制和粘贴。

文件夹页脚工具栏

  • 文件夹永久链接： 使您能够参照文件夹。这些链接将保持不变，可用作书签或通过电子邮件发送给其他用户。可根据需要对它们进行复制和粘贴。

  • 电子邮件地址： 如果已将文件夹配置为具有用户友好的 URL，则单击页脚链接时，页脚工具栏会显示此 URL。

    有关如何配置文件夹以通过电子邮件接收条目的更多信息，请参见《Kablink Teaming 2.0 用户指南》中的设置文件夹以通过电子邮件接收条目。

  • RSS 链接： 使您能够将文件夹作为 RSS 源访问。只需复制 RSS URL 并将其粘贴到 RSS 阅读器中即可。

  • WebDAV URL： 如果您使用 Internet Explorer 7 或更高版本作为 Web 浏览器，则可以使用 WebDAV URL 直接从 Microsoft Office 和类似应用程序访问 Kablink Teaming 文件。

  • iCal URL： 使您能够将 Teaming 任务和日历文件夹与桌面日历应用程序同步。

    在查看日历文件夹或条目时，您可以使用此链接导出 Teaming 日历数据，以便在支持 iCal 格式的其他日历（例如，Google 和 Outlook）中包含这些数据。单击此链接，用 .ics 文件扩展名保存文件，然后将此文件导入到其他日历中（请参见相关日历的帮助以了解更多信息）。

  • 将文件夹作为 WebDAV 文件夹查看： 使您能够将文件夹作为 WebDAV 文件夹查看。

    有关更多信息，请参见从 Teaming 中访问 WebDAV 文件![.jsp 参照：workspaces_folders?misc_tools/webdav.jsp]![我还需要研究一下！]![我已从帮助中复制出此部分并将其从帮助中删除了]，此部分内容位于《Kablink Teaming 2.0 用户指南》中。

条目页脚工具栏

  • 条目永久链接： 使您能够参照条目。这些链接将保持不变，可用作书签或通过电子邮件发送给其他用户。可根据需要对它们进行复制和粘贴。

  • 发送电子邮件： 起动一张电子邮件表单，您就可以向所需的任何人发送电子邮件。电子邮件包含条目在 Teaming 中的位置和部分内容。

  • 订阅此条目： 起动一张表单，您就可以订阅此条目。订阅条目时，会配置 Teaming 以向您发送有关此条目更新信息的电子邮件通知。

    有关订阅条目的更多信息，请参见使用电子邮件通知。

有关更多信息，请参见 法律声明.
