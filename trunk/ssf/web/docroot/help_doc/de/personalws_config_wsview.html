Arbeitsbereichsansicht

Wählen Sie eine für Ihren Arbeitsbereich geeignete Ansicht aus.

  • Diskussions-Arbeitsbereich: Wählen Sie diese Ansicht aus, wenn Sie einen Arbeitsbereich erstellen möchten, in dem Sie mehrere Diskussionsforen speichern können.

    Weitere Informationen zu Diskussions-Arbeitsbereichen finden Sie im Kablink Teaming 2.0-Handbuch für Fortgeschrittene im Abschnitt Verwalten eines Diskussions-Arbeitsbereichs.

  • Landeseite: Wählen Sie diese Ansicht aus, wenn Sie Ihren Teammitgliedern wichtige Informationen präsentieren möchten, sobald sie den Arbeitsbereich öffnen.

    Weitere Informationen zu Landeseiten finden Sie im Kablink Teaming 2.0-Handbuch für Fortgeschrittene im Abschnitt Erstellen einer Landeseite.

  • Arbeitsbereich zur Projektverwaltung: Wählen Sie diese Ansicht aus, wenn Sie die Erledigung von Projektaufgaben und Meilensteinen verfolgen möchten.

    Weitere Informationen zu Projektverwaltungs-Arbeitsbereichen finden Sie im Kablink Teaming 2.0-Handbuch für Fortgeschrittene im Abschnitt Verwalten eines Arbeitsbereichs zur Projektverwaltung.

  • Teamarbeitsbereich: Wählen Sie diese Ansicht aus, wenn Sie Ihren Teamarbeitsbereich zu organisatorischen Zwecken verwenden wollen und ihm keine weiteren Teamarbeitsbereiche unterordnen möchten.

    Weitere Informationen zu Teamarbeitsbereichen finden Sie im Kablink Teaming 2.0-Benutzerhandbuch im Abschnitt Verwalten eines Teamarbeitsbereichs.

  • Teamstammarbeitsbereich: Wählen Sie diese Ansicht aus, wenn Ihr Teamarbeitsbereich die Schaltfläche Teamarbeitsbereich hinzufügen enthalten soll. Diese Schaltfläche benötigen Sie, wenn Sie Ihrem Teamarbeitsbereich weitere Teamarbeitsbereiche unterordnen möchten.

    Weitere Informationen zu Teamstammarbeitsbereichen finden Sie im Kablink Teaming 2.0-Handbuch für Fortgeschrittene im Abschnitt Verwalten eines Teamstammarbeitsbereichs.

  • Arbeitsbereich: Dies ist ein Basis-Arbeitsbereich. Sie können selbst die Ordner auswählen, die dieser Arbeitsbereich enthalten soll.

    Weitere Informationen zu Basis-Arbeitsbereichen finden Sie im Kablink Teaming 2.0-Handbuch für Fortgeschrittene im Abschnitt Verwalten eines Basis-Arbeitsbereichs.

Die Standardansicht eines persönlichen Arbeitsbereichs ist die Ansicht Benutzerarbeitsbereich. Diese kann bei einem persönlichen Arbeitsbereich nicht geändert werden.
