Symbolleiste eines Eintrags

In der Symbolleiste eines Eintrags werden nur diejenigen Werkzeuge angezeigt, die zu benutzen Sie berechtigt sind. Die Symbolleiste wird sowohl für Einträge als auch für Kommentare angezeigt. Sie kann folgende Schaltflächen und Menüs enthalten:

  • Kommentar: Fügt einem Eintrag einen Kommentar hinzu. Auch ein Kommentar kann kommentiert werden (dadurch wird ein verschachtelter Kommentar erstellt).

  • Bearbeiten: Damit können Sie den Eintrag bearbeiten oder ihm eine Anlage hinzufügen. (Zu diesem Zweck können Sie auch den Link Bearbeiten verwenden, der sich rechts unten im Eintrag direkt über dem Abschnitt Anlagen befindet.)

  • Reservieren: Verhindert, dass andere Personen den Eintrag bearbeiten, solange er von Ihnen bearbeitet wird. Wenn Sie erhebliche Änderungen an einem Eintrag vornehmen, sollten Sie den Eintrag unbedingt reservieren. Vergessen Sie nicht, nach Beendigung Ihrer Überarbeitung auf Reservierung aufheben zu klicken.

  • Verschieben: Damit können Sie den Eintrag in einen anderen Ordner verschieben.

  • Kopieren: Damit können Sie den Eintrag kopieren und die Kopie in einem anderen Ordner speichern.

  • Löschen: Entfernt den Eintrag aus dem Ordner.

  • Workflow: Wenn für einen Ordner ein Workflow aktiviert ist, können Sie den Workflow mit diesem Werkzeug starten, sofern er noch nicht für den Eintrag gestartet wurde, bzw. Sie können ihn anhalten, wenn er zurzeit für den Eintrag ausgeführt wird.

  • Wiki-Startseite festlegen: Steht nur für Wiki-Einträge zur Verfügung. Damit können Sie Teaming so konfigurieren, dass eine bestimmte Wiki-Seite als Wiki-Startseite angezeigt wird. Navigieren Sie einfach zu der Seite, die Sie als Wiki-Startseite festlegen möchten, und klicken Sie dann auf diese Option.

  • Berichte: Damit können Sie folgende Berichte generieren:

    • Aktivitätsbericht: Öffnet in Microsoft Excel eine CSV-Datei mit Informationen zu Benutzern und deren Aktivitäten hinsichtlich dieses Eintrags.

    • Bearbeitungsverlauf: Öffnet das Fenster "Bearbeitungsverlauf", in dem Sie verschiedene Versionen des Eintrags vergleichen können. Sie sehen in diesem Fenster, was hinzugefügt und was gelöscht wurde.

    • Workflow-Verlauf: Öffnet das Fenster "Workflow-Verlauf", in dem der Verlauf des Workflows einschließlich seines Namens angezeigt wird. Sie sehen in diesem Fenster, wann der Eintrag erstellt wurde, von wem er erstellt wurde, wann der Workflow einen bestimmten Status erreicht hat, wer den Eintrag bei Erreichen dieses Status bearbeitet hat und vieles mehr.

  • Diesen Eintrag freigeben: Damit können Sie einen Eintrag mit anderen Teaming-Benutzern gemeinsam verwenden. Bei der Freigabe eines Eintrags erhalten die Benutzer, für die Sie den Eintrag freigegeben haben, eine Benachrichtigungs-Email. In den persönlichen Arbeitsbereichen dieser Benutzer werden außerdem auf der Registerkarte Mikro-Blogs und freigegebene Elemente in der Spalte Freigegebene Elemente alle Elemente angezeigt, die andere Personen für sie freigegeben haben.
