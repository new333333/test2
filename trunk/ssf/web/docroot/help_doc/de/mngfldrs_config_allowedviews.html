Zulässige Ansichten

Nur die Eigentümer von Ordnern haben das Recht, die Standardansicht ihres Ordners festzulegen. Daher bietet der Abschnitt Zulässige Ansichten dem Ordnereigentümer die Möglichkeit, Ansichten auszuwählen, die externe Benutzer des Ordners verwenden dürfen, wenn sie mit der Standardansicht nicht zufrieden sind oder sich in einer anderen Ansicht wohler fühlen. Wenn Sie zum Beispiel bei der Konfiguration Ihres Kalenderordners in diesem Abschnitt Aufgabe auswählen, dann können Benutzer Ihren Kalenderordner in der Ordneransicht "Aufgabe" anzeigen.

Hinzufügen zulässiger Ansichten für einen Ordner

  1. Wählen Sie im Abschnitt Zulässige Ansichten die Ansichten aus, die Sie für den aktuellen Ordner zulassen möchten.

  2. Klicken Sie auf Anwenden > Schließen.

  3. Klicken Sie in der Hauptseitenleiste im Abschnitt Ordneroptionen auf Ordneransicht.

    Die für den Ordner zulässigen Ansichten werden angezeigt. (Dies sind die gleichen Ansichten, die Sie in Schritt 1 im Abschnitt Zulässige Ansichten ausgewählt haben.) Die Benutzer Ihres Ordners können aus dieser Liste die für sie geeignete Ordneransicht auswählen.

Hinzufügen zulässiger Standardansichten für einen Ordner

Durch Auswahl einer Ansicht im Abschnitt Zulässige Ansichten wird die Ansicht dem Abschnitt Standardansicht hinzugefügt. Der Eigentümer des Ordners kann die Ansicht dann als Standardansicht auswählen.

  1. Wählen Sie im Abschnitt Zulässige Ansichten die Ansicht aus, die Sie dem Abschnitt Standardansicht hinzufügen möchten.

  2. Klicken Sie auf Anwenden.

    Die ausgewählte Ansicht wird dem Abschnitt Standardansicht hinzugefügt.

Informationen zum Ändern der Standardansicht eines Ordners finden Sie im Abschnitt Ändern der Standardansicht eines Ordners.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
