Aktivieren von Email-Postings

Wenn Sie Email-Postings für alle Ordner der Installation zulassen möchten, müssen Sie einen Zeitplan festlegen, der für die gesamte Website gilt. Der Eigentümer eines Ordners muss dann den Namen und das Passwort eines Email-Kontos oder nur eine Email-Aliasadresse angeben. Anhand dieser Informationen wird bestimmt, welche Nachrichten des Kontos an den Ordner weitergeleitet werden.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
