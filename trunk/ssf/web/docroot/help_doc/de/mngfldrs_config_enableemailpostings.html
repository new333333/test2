Aktivieren von Email-Postings

Ungeachtet der anderen Einstellungen auf dieser Seite müssen Sie dieses Feld aktivieren, um die Bereitstellung von E-Mails als Ordnereinträge zu aktivieren. Anhand von E-Mail-Aliasen wird bestimmt, welche Nachrichten im E-Mail-Eingangskonto der Installation zu welchen Ordnern weitergeleitet werden.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
