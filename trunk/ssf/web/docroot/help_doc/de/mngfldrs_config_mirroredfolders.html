Gespiegelte Ordner

Ein gespiegelter Ordner ist ein Archivordner auf einem externen System (ein Dateisystemverzeichnis oder ein Ordner auf einem SharePoint-Server). Wenn Sie externe Ordner in Teaming spiegeln, können Sie auf die externen Dateien Novell Teaming-Werkzeuge wie die Suchfunktion und Workflows anwenden.

Um diese Funktion zu aktivieren, müssen Sie mit einer Datei auf dem Server arbeiten.

Weitere Informationen zum Erstellen gespiegelter Ordner finden Sie im Novell Teaming 2.1-Benutzerhandbuch für Fortgeschrittene im Abschnitt Arbeiten mit gespiegelten Ordnern.
