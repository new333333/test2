Symbolleiste "Fußzeile"

Der Inhalt der Symbolleiste "Fußzeile" ändert sich je nachdem, ob ein Arbeitsbereich, ein Ordner oder ein Eintrag geöffnet ist:

Symbolleiste "Fußzeile" für Arbeitsbereich

  • Permalink auf Arbeitsbereich: Fügt einen Link zum Arbeitsbereich ein. Diese Links sind persistent und können als Lesezeichen verwendet oder per Email an andere Benutzer gesendet werden. Sie können diese Links nach Bedarf kopieren und einfügen.

Symbolleiste "Fußzeile" für Ordner

  • Permalink auf Ordner: Fügt einen Link zum Ordner ein. Diese Links sind persistent und können als Lesezeichen verwendet oder per Email an andere Benutzer gesendet werden. Sie können diese Links nach Bedarf kopieren und einfügen.

  • Email-Adressen: Wenn für den Ordner bereits eine benutzerfreundliche URL konfiguriert ist, wird diese URL in der Symbolleiste "Fußzeile" angezeigt, sobald Sie auf den Fußzeilen-Link klicken.

    Weitere Informationen zur Konfiguration eines Ordners für den Empfang von Einträgen via Email finden Sie im Kablink Teaming 2.0-Benutzerhandbuch im Abschnitt Einrichten eines Ordners für den Empfang von Einträgen via Email.

  • RSS-Link: Ermöglicht Ihnen den Zugriff auf diesen Ordner als RSS-Feed. Kopieren Sie dazu die RSS-URL und fügen Sie sie in Ihren RSS-Reader ein.

  • WebDAV-URL: Wenn Sie Internet Explorer 7 oder höher als Webbrowser verwenden, können Sie mittels der WebDAV-URL aus Microsoft Office und ähnlichen Anwendungen direkt auf Kablink Teaming-Dateien zugreifen.

  • iCal-URL: Ermöglicht Ihnen die Synchronisierung Ihrer Aufgaben- und Kalenderordner aus Teaming mit Ihrer Desktop-Kalenderanwendung.

    Wenn ein Kalenderordner oder -eintrag von Teaming geöffnet ist, können Sie die Kalenderdaten über diesen Link exportieren, um sie in einen anderen Kalender einzufügen, der das iCal-Format unterstützt (z. B. Google und Outlook). Klicken Sie dazu auf diesen Link, speichern Sie die Datei mit der Dateierweiterung .ics und importieren Sie die Datei in die gewünschte Kalenderanwendung (Informationen zum Importieren in den Kalender finden Sie in der Hilfe der jeweiligen Kalenderanwendung).

  • Diesen Ordner als WebDAV-Ordner anzeigen: Ermöglicht Ihnen die Anzeige des Ordners als WebDAV-Ordner.

    Weitere Informationen finden Sie unter Accessing WebDAV Files from within Teaming![.jsp ref: workspaces_folders?misc_tools/webdav.jsp]![I need to research this too!]![I copied this over from the Help and deleted it from the Help] (Verwenden von WebDAV) im Kablink Teaming 2.0-Benutzerhandbuch.

Symbolleiste "Fußzeile" für Eintrag

  • Permalink auf Eintrag: Fügt einen Link zum Eintrag ein. Diese Links sind persistent und können als Lesezeichen verwendet oder per Email an andere Benutzer gesendet werden. Sie können diese Links nach Bedarf kopieren und einfügen.

  • Email senden: Öffnet ein Email-Formular, über das Sie jeder beliebigen Person eine Email senden können. Die Email enthält den Ort des Eintrags innerhalb von Teaming sowie einen Teil seines Inhalts.

  • Diesen Eintrag abonnieren: Öffnet ein Formular, über das Sie den Eintrag abonnieren können. Durch das Abonnieren eines Eintrags konfigurieren Sie Teaming so, dass Sie bei jeder Aktualisierung des Eintrags eine entsprechende Email-Benachrichtigung erhalten.

    Weitere Informationen zum Abonnieren von Einträgen finden Sie im Abschnitt Verwenden von Email-Benachrichtigungen.
