Editor-Overrides

Zur Bearbeitung von Anlagen in Novell Teaming werden in der Regel die Standardanwendungen von Teaming zur Bearbeitung von Anlagen geöffnet. In dieser Tabelle können Sie die Standardanwendungen, die Teaming für die Bearbeitung von Anlagen mit bestimmten Dateitypen verwendet, jedoch überschreiben.

Sie können die Standardbearbeitungsanwendung für nur einen Dateityp oder für alle Dateitypen überschreiben.

Überschreiben der Standardbearbeitungsanwendung für einen Dateityp

So überschreiben Sie die Standardbearbeitungsanwendung für einen bestimmten Dateityp:

  1. Klicken Sie auf Hinzufügen.

  2. Wählen Sie in der Dropdown-Liste den Dateityp aus, dessen Standardbearbeitungsanwendung Sie überschreiben möchten.

  3. Geben Sie im Feld Anwendung den Namen der Anwendung an, die Sie zur Bearbeitung von Anlagen mit dem in Schritt 2 ausgewählten Dateityp verwenden möchten.

  4. Klicken Sie auf OK.

Überschreiben der Standardbearbeitungsanwendung für alle Dateitypen

So überschreiben Sie die Standardbearbeitungsanwendung für alle Dateitypen:

  1. Klicken Sie auf Verwenden.

  2. Wählen Sie in der Dropdown-Liste die Anwendung aus, die Sie in Teaming zur Bearbeitung aller Anlagen unabhängig von ihrem Dateityp verwenden möchten.

  3. Klicken Sie auf OK.
