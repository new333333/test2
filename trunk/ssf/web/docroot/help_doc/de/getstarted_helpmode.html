Hilfemodus - Erste Schritte

Links oben auf der Seite befindet sich eine Kontrollleiste mit dem Namen "Hilfemodus". Meist benutzen Sie diese Leiste, um das Inhaltsverzeichnis oder die Online-Dokumentation aufzurufen. Wenn Sie die Kontrollleiste "Hilfemodus" minimieren möchten, klicken Sie oben in der Kontrollleiste auf das Symbol Minimieren . Zum Wiederanzeigen der Kontrollleiste klicken Sie erneut auf dieses Symbol.

Im Hilfemodus wird die aktuell auf dem Bildschirm angezeigte Seite beschrieben. Wenn Sie auf das Symbol Information klicken, zeigt Novell Teaming ein Hilfefenster mit einer Beschreibung des betreffenden Seitenabschnitts an. Zum Schließen des Hilfefensters klicken Sie rechts oben im Fenster auf das Symbol Schließen (X). Sie können aber auch auf ein anderes Informationssymbol klicken, um weitere Hilfeinformationen anzuzeigen. Nicht nur über die Informationssymbole können Sie die Hilfe aufrufen, sondern auch über die Kontrollleiste "Hilfemodus". Klicken Sie dort auf Hilfethemen und danach auf die Links der Themen, die Sie interessieren.

Eine ausführlichere Dokumentation zu Teaming erhalten Sie über den Link Handbücher anzeigen der Kontrollleiste "Hilfemodus".

Wenn Sie den Hilfemodus nicht mehr benötigen, können Sie die Kontrollleiste "Hilfemodus" schließen, indem Sie rechts oben in der Leiste auf das Symbol Schließen (X) oder auf einen beliebigen abgeblendeten Seitenabschnitt klicken.

Auch wenn Sie den Hilfemodus während der Arbeit in Teaming deaktiviert lassen, können Sie über Informationssymbole , die Teaming auf Formularen mit Erläuterungsbedarf bereitstellt, Hilfe aufrufen. Klicken Sie neben einem Formularabschnitt auf das Symbol Information, wenn Sie Hilfe zu diesem Abschnitt benötigen.
