Konfigurieren der Zugriffssteuerung

Auf dieser Seite können Sie die Einstellungen der Zugriffssteuerung für den aktuellen Arbeitsbereich oder Ordner ändern.

Standardmäßig übernehmen Arbeitsbereiche und Ordner die Einstellungen der Zugriffssteuerung des übergeordneten Arbeitsbereichs oder Ordners. Verwenden Sie diese Seite nur, wenn Sie andere Einstellungen für den Zugriff festlegen möchten.

Bevor Sie Änderungen an der Zugriffssteuerungstabelle vornehmen, müssen Sie sicherstellen, dass der Arbeitsbereich oder Ordner seine Benutzer und Rollen nicht vom übergeordneten Ordner übernimmt. Anweisungen hierzu finden Sie im Abschnitt Rollenmitgliedschaft erben.

Weitere Informationen zur Zugriffssteuerung, die über die Informationen in diesem Abschnitt und den nachfolgenden Unterabschnitten hinausgehen, finden Sie im Kablink Teaming 2.0-Handbuch für Fortgeschrittene im Abschnitt Steuern des Benutzerzugriffs.

In den folgenden Abschnitten wird beschrieben, wie Sie die Zugriffssteuerungstabelle an Ihre Anforderungen anpassen.

Hinzufügen von Benutzern und Gruppen

  1. Klicken Sie in der ersten Spalte der Tabelle auf den Link Gruppe hinzufügen oder Benutzer hinzufügen.

  2. Geben Sie im Feld die ersten Buchstaben des Benutzer- oder Gruppennamens ein und klicken Sie dann in der Dropdown-Liste auf den Link mit dem gewünschten Namen.

    Kablink Teaming fügt der Tabelle eine Zeile für den Benutzer bzw. die Gruppe hinzu.

Anzeigen der Zugriffsrechte einer Rolle

Die Spaltenüberschrift der Zugriffssteuerungstabelle enthält die Namen der einzelnen Rollen.

  1. Klicken Sie in der Spaltenüberschrift auf den Namen einer Rolle (Teilnehmer, Teammitglied usw.), um eine Liste der mit der Rolle verbundenen Zugriffsrechte anzuzeigen.

Ändern der Rollen von Benutzern oder Gruppen

Eine Rolle gibt eine Reihe von Berechtigungen bzw. Zugriffsrechten vor. Wenn Sie einem Benutzer oder einer Gruppe eine Rolle zuweisen, erhält der Benutzer bzw. die Gruppe automatisch alle mit der Rolle verbundenen Rechte.

  1. Legen Sie die Rollen der einzelnen Benutzer und Gruppen fest, indem Sie die entsprechenden Kontrollkästchen aktivieren bzw. deaktivieren.

  2. Klicken Sie auf OK.

Anpassen der Zugriffssteuerung für einen anderen Ort

  1. Navigieren Sie im Arbeitsbereichsbaum am Anfang dieser Seite zu dem Ort, dessen Zugriffssteuerung Sie anpassen möchten, und klicken Sie im Baum auf den Link dieses Ortes.

    Die Seite "Zugriffssteuerung" für diesen Ort wird angezeigt.
