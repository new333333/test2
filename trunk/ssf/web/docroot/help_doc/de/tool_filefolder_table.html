Ordnertabelle

Jede Zeile der Ordnertabelle enthält Informationen zu einem Eintrag. Nachfolgend finden Sie einige Tipps zur Verwendung dieser Tabelle:

  • Ändern einer Spaltenbreite: Setzen Sie den Mauszeiger in der Zeile mit der Tabellenüberschrift auf eine Linie mit einem Doppelpfeil, drücken Sie die Maustaste und ziehen Sie den Mauszeiger nach links oder rechts.

  • Sortieren des Tabelleninhalts: Klicken Sie auf eine Tabellenüberschrift, um die Tabelle nach dem Inhalt dieser Spalte zu sortieren. Klicken Sie nochmals auf die Tabellenüberschrift, um die Tabelle in umgekehrter Reihenfolge zu sortieren. Das Pfeilsymbol neben dieser Spalte weist darauf hin, dass die Tabelle zurzeit nach dem Inhalt dieser Spalte sortiert ist (der Aufwärtspfeil kennzeichnet die absteigende Reihenfolge, der Abwärtspfeil die aufsteigende Reihenfolge).

  • Verschieben des Mauszeigers auf eine Zelle: Wenn Sie den Mauszeiger auf eine Zelle verschieben (z. B. auf einen Titel, der für die Tabellenzelle zu lang ist), blendet Novell Teaming den gesamten Inhalt der Zelle ein.
