Anhängen von Anlagen an einen Eintrag

Im Abschnitt Anlagen eines Eintrags bietet Novell Teaming verschiedene Möglichkeiten, dem Eintrag Anlagen anzuhängen:

Ziehen und Ablegen von Dateien

Sie können einem Eintrag Dateien als Anlagen anhängen, indem Sie die Dateien mit der Maus von einem anderen Ort auf den Eintrag ziehen.

  1. Klicken Sie auf das Symbol Dateien ziehen und ablegen.

    Dadurch wird ein Fenster mit einem Ordnersymbol geöffnet.

  2. Ziehen Sie die Dateien, die Sie heraufladen möchten, auf das Ordnersymbol.

    Teaming zeigt ein Symbol mit einem sich öffnenden und wieder schließenden Ordner an, um anzugeben, dass die auf das Symbol abgelegten Dateien als Anlagen hochgeladen werden.

Dadurch werden die Dateien dem Eintrag als Anlagen angehängt.

Anhängen von Anlagen mittels WebDAV (nur Internet Explorer)

Teaming erlaubt das Aufrufen von Dateien, die auf entfernten Webservern gespeichert sind, und das Anhängen dieser Dateien als Anlagen mittels WebDAV. Allerdings unterstützt Teaming WebDAV nur in Internet Explorer. Bei anderen Webbrowsern steht diese Option nicht zur Verfügung.

  1. Klicken Sie auf Anlagen über WebDAV aufrufen.

  2. Navigieren Sie zu der Datei, die Sie als Anlage anhängen möchten, und wählen Sie sie aus.

Weitere Informationen zur Verwendung von WebDAV mit Teaming finden Sie im Novell Teaming 2.1-Benutzerhandbuch im Abschnitt Zugreifen auf Anlagen über (nur unterstützte Umgebungen).

Anhängen einer einzelnen Datei als Anlage

  1. Klicken Sie in dem Eintrag, an den Sie eine einzelne Datei als Anlage anhängen möchten, im Abschnitt Anlagen auf das Symbol Einzelne Datei hochladen.

  2. Klicken Sie auf Durchsuchen und wählen Sie die gewünschte Datei aus.

  3. Klicken Sie auf OK.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
