Bearbeiten

Über den Link Bearbeiten eines Eintrags können Sie nur den Abschnitt Beschreibung des Eintrags ändern.

Wenn Sie andere Teile des Eintrags oder Kommentars bearbeiten möchten, zum Beispiel den Titel, oder wenn Sie Anlagen hochladen oder löschen möchten, klicken Sie in der Symbolleiste des Eintrags auf Ändern.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
