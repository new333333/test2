Definieren einer URL für einen Ordner oder Arbeitsbereich

In diesem Abschnitt können Sie für den aktuellen Ordner oder Arbeitsbereich eine benutzerfreundliche URL definieren. Mittels dieser URL können Sie den Ordner oder Arbeitsbereich direkt aus Ihrem Browser aufrufen.

Einem Ordner können Sie mittels dieser URL auch Einträge via Email oder Text-Messaging hinzufügen.

  1. Wählen Sie in der Dropdown-Liste URL definieren eines der vordefinierten globalen Schlüsselwörter aus.

    Wenn Sie nicht Administrator der Website sind, muss Ihre URL mit einem dieser vordefinierten globalen Schlüsselwörter beginnen.

  2. Geben Sie im Feld die gewünschte URL für den Ordner oder Arbeitsbereich ein.

  3. Klicken Sie auf Hinzufügen.

  4. (Bedingt) Wenn Sie die URL für einen Ordner definieren, müssen Sie sie nach der Eingabe aktivieren, indem Sie sie auswählen. indem Sie sie unterhalb der Schaltfläche Hinzufügen auswählen.

  5. (Bedingt) Nach der Definition einer Ordner-URL müssen Sie auf Anwenden klicken.

  6. Klicken Sie auf Schließen.

Zum Löschen einer definierten URL wählen Sie die URL im Abschnitt Zurzeit definierte URLs aus und klicken Sie dann auf Ausgewählte URLs löschen.
