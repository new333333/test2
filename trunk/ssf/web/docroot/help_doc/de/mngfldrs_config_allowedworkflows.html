Zulässige Workflows

In diesem Abschnitt können Sie festlegen, dass ein Workflow für sämtliche Einträge in diesem spezifischen Ordner verwendet werden kann. Wenn ein Benutzer einen Eintrag in einem Ordner anzeigt, in dem Sie einen Workflow als zulässig definiert haben, kann er jeden in diesem Abschnitt ausgewählten Workflow starten, indem er in der Symbolleiste des Eintrags auf Workflow > Workflow starten: Workflow_Name klickt.

Dadurch wird der Workflow für diesen Eintrag gestartet.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
