Ändern von Einstellungen für Email-Benachrichtigungen

Wählen Sie diese Option aus, wenn sämtliche vom Administrator bzw. Ordnereigentümer vorgenommenen Benachrichtigungseinstellungen durch die von Ihnen getroffene Auswahl außer Kraft gesetzt werden sollen.

Wenn Sie keine Email-Benachrichtigungen erhalten möchten, aktivieren Sie dieses Kontrollkästchen und stellen Sie sicher, dass in den Feldern darüber keine Email-Adressen und Telefonnummern angegeben sind.

Weitere Informationen hierzu finden Sie unter Rechtliche Hinweise.
