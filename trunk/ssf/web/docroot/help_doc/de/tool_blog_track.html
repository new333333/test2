Blog-Protokollierung

Über die Symbolleiste "Blog-Protokollierung" können Sie die folgenden Aufgaben schnell ausführen:

  • Kommentar hinzufügen: Damit können Sie einen Kommentar zu einem Blog-Eintrag eingeben. Zuvor empfiehlt es sich allerdings, die Kommentare anderer Benutzer zu lesen. Wenn Sie einem Blog, zu dem bereits Kommentare existieren, einen Kommentar hinzufügen, wird Ihr Kommentar automatisch am Ende der Liste der verschachtelten Kommentare angefügt.

  • Kommentare anzeigen: Damit können Sie alle Kommentare zu einem bestimmten Eintrag anzeigen, ohne den Eintrag zu öffnen.

  • An einen Freund senden: Damit können Sie den Inhalt des Eintrags per Email an jede Person senden, die über die Berechtigung zum Anzeigen des Eintrags verfügt.

  • Abonnieren: Öffnet ein Formular, über das Sie den Eintrag abonnieren können. Durch das Abonnieren eines Eintrags konfigurieren Sie Teaming so, dass Sie bei jeder Aktualisierung des Eintrags eine entsprechende Email-Benachrichtigung erhalten.

    Weitere Informationen zum Importieren eines Eintrags finden Sie im Novell Teaming 2.1-Benutzerhandbuch im Abschnitt Abonnieren eines Eintrags.
