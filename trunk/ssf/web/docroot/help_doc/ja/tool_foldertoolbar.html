［フォルダエントリ］ツールバー

Novell Teaming の［フォルダエントリ］ツールバーには、表示しているフォルダに固有のオプションが表示されます。ただし、さまざまなエントリタイプが含まれるように［フォルダエントリ］ツールバーを設定することができます。たとえば、Wikiフォルダがあり、そのフォルダにフォトエントリも含めたい場合は、フォトエントリも許可するようにフォルダを設定できます。

ツールバーオプション

すべてのフォルダタイプには、［フォルダエントリ］ツールバーに自動的に表示されるデフォルトのオプションがあります。次に、各タイプのフォルダ、および［フォルダエントリ］ツールバーそれぞれに自動的に表示されるオプションを示します。

ブログ

  • ブログエントリの追加: ブログフォルダに新しいエントリを追加します。タイトル、説明、および添付ファイルをエントリに追加できます。エントリを購読したり、エントリを送信したことを通知する電子メールを特定のユーザに送信したりすることもできます。

  • 新規ブログページ: 既存のブログフォルダに新しいブログフォルダを追加します。この新しいフォルダは、メインブログフォルダに属します。新しいブログページを使用すると、ブログフォルダをよりわかりやすいトピックに整理することができます。

  • ソート基準: ［番号］、［アクティビティの日付］、［作成者］、［状態］、または［タイトル］でブログをソートします。

  • フォルダへファイルを追加: アイコンが表示され、ブログフォルダにファイルをドラッグアンドドロップしたり、ファイルをコピーして貼り付けたりすることができます。このアクションを実行すると、ファイルは新しいブログエントリとして追加されます。

カレンダ

  • フォルダへファイルを追加: アイコンが表示され、カレンダフォルダにファイルをドラッグアンドドロップしたり、ファイルをコピーして貼り付けたりすることができます。このアクションを実行すると、ファイルは新しいカレンダエントリとして追加されます。

  • 表示: カレンダに表示するイベントを選択するには、このドロップダウンリストを使用します。以下のカテゴリから選択できます。

    • 割り当て済みイベント: ユーザワークスペースでのみ表示されます。イベントがTeamingのどこに配置されているかにかかわらず、ワークスペースの所有者に割り当てられているすべてのカレンダイベントを表示します。

    • フォルダから - イベント: 現在のフォルダにある、イベントがスケジュール設定されている日のすべてのカレンダイベントが表示されます。

    • フォルダから - 作成別にすべて: 作成された日別にすべてのカレンダイベントを表示します。

    • フォルダから - アクティビティ別にすべて: 最後に変更された日別にすべてのカレンダイベントを表示します。

フォトアルバム

  • フォトエントリの追加: フォトアルバムフォルダに新しいエントリを追加します。写真やそれ以外のグラフィックに名前を付けたり、説明を付けたり、アップロードしたりすることができます。エントリを送信したことを通知する電子メールを特定のユーザに送信することもできます。

  • 新規フォトアルバム: 現在のフォトアルバムフォルダに新しいフォルダを追加します。この新しいフォルダは、メインフォトアルバムフォルダのサブフォルダになります。

  • ソート基準: ［アクティビティの日付］または［タイトル］のいずれかで画像をソートできます。

    ［アクティビティの日付］でソートすると、最も最近アップロードされたフォトがリストの最初に配置され、最も長い期間フォトアルバムに保存されているが写真が最後に配置されます。

    ［タイトル］でソートすると、写真はアルファベット順に並べられます。

  • フォルダへファイルを追加: アイコンが表示され、フォトアルバムフォルダにファイルをドラッグアンドドロップしたり、ファイルをコピーして貼り付けたりすることができます。このアクションを実行すると、ファイルは新しいファイルエントリとして追加されます。

  • フォルダへファイルを追加: アイコンが表示され、調査フォルダにファイルをドラッグアンドドロップしたり、ファイルをコピーして貼り付けたりすることができます。このアクションを実行すると、ファイルは未定義の新しい調査として追加されます。

  • 新規Wikiトピック: 新しいフォルダに名前を付けて現在のWikiフォルダに追加できます。この新しいフォルダは、メインWikiフォルダのサブフォルダになります。

  • ソート基準: ［アクティビティの日付］または［タイトル］のいずれかでWikiエントリをソートできます。

    ［アクティビティの日付］でソートすると、最も最近のエントリがリストの最初に配置され、最も長い期間Wikiフォルダに保存されているエントリが最後に配置されます。

    ［タイトル］でソートすると、Wikiエントリはアルファベット順に並べられます。

  • フォルダへファイルを追加: アイコンが表示され、Wikiフォルダにファイルをドラッグアンドドロップしたり、ファイルをコピーして貼り付けたりすることができます。このアクションを実行すると、ファイルは新しいWikiエントリとして追加されます。新しいWikiエントリは、［Wiki］サイドバーの［Wikiページ］セクションに表示されます。

代替エントリタイプを許可するためのフォルダの設定

  1. 代替エントリタイプを追加するフォルダに移動します。

  2. ［管理］>［設定］の順にクリックします。

  3. ［デフォルトエントリタイプ］セクションで、フォルダで有効にするエントリタイプを表すフォルダを選択します。

  4. ［適用］>［閉じる］の順にクリックします。

  5. ［フォルダエントリ］ツールバーで［追加］をクリックし、ドロップダウンリストを使用して、追加するエントリタイプを選択します。

詳細については、「保証と著作権」を参照してください。
