アクセサリツール

アクセサリはNovell Teamingの自分のページに追加できるセクションで、これらを使うことで、既存の情報の視認性とアクセス性が向上します。アクセサリは追加機能を提供し、これによってチームワークとコミュニケーションが向上します。たとえば、次のようなアクセサリがあります。

  • 検索結果(現在の場所にのみ適用できます)

  • バディリスト(多くの場合、対象案件のローカルの専門家用)

  • ワークスペースツリー(階層内のこの場所が開かれます)

  • ゲストブック

Teamingのアクセサリすべてのリストとそれらの説明については、『Novell Teaming 2.1 User Guide』の「Types of Accessories」を参照してください。

次のトピックでは、［アクセサリパネル］の使い方について説明します。

アクセサリの追加

［アクセサリパネル］にアクセサリを追加する

  1. ［アクセサリパネル］をクリックします。

  2. ［アクセサリのスコープ］セクションで、アクセサリの目的のスコープを選択します。

    このセクションのオプションについてより詳しく学ぶには、アクセスを表示可能なユーザの設定を参照してください。

  3. ［アクセサリタイプ］セクションで、追加するアクセサリタイプを選択します。

  4. ［OK］をクリックします。

  5. アクセサリのタイトルを指定して、ほかに関連情報があれば入力します。

    各タイプのアクセサリの関連情報を入力する方法の詳細については、『Novell Teaming 2.1 User Guide』の「Setting Up the Accessory」を参照してください。

  6. ［適用］>［閉じる］の順にクリックします。

アクセスを表示可能なユーザの設定

アクセサリを追加する際に、アクセサリのスコープを選択することができます。［アクセサリのスコープ］セクションには、次のオプションがあります。

  • このページ専用のパーソナルアクセサリ: アクセサリは現在の場所にのみ適用され、自分だけが表示できます。

  • すべてのワークスペースとフォルダ用のパーソナルアクセサリ: アクセサリはすべてのワークスペースとフォルダに適用され、自分だけが表示できます。

  • このページのコミュニティアクセサリ: アクセサリはこのページにのみ適用され、このページに対してアクセス権を持つすべてのTeamingユーザが表示できます。

アクセサリの順序の変更

複数のアクセサリが存在する任意のページのアクセサリを並べ替えることができます。

  1. アクセサリを上へ移動するか、それとも下へ移動するかに応じて、アクセサリのタイトルバー右上隅にある上矢印または下矢印をクリックします。

アクセサリの再設定

アクセサリの作成後、アクセサリを再設定できます。

  1. アクセサリのタイトルバー右上隅にある［変更］アイコンをクリックします。

アクセサリの削除

  1. アクセサリのタイトルバー右上隅にある［削除］アイコンをクリックします。

アクセサリの非表示

アクセサリを隠して、アクセサリのタイトルバーのみを表示することができます。

  1. アクセサリのタイトルバー右上隅にある［隠す］アイコンをクリックします。

［アクセサリパネル］のアクセサリすべてを同時に隠すこともできます。

  1. ［アクセサリパネル］の横にある［隠す］アイコンをクリックします。

    現在のページにあるすべてのアクセサリが隠れ、表示されなくなります。

    アクセサリを再度表示するには、［隠す］アイコンをもう一度クリックします。

詳細については、「保証と著作権」を参照してください。
