エントリへのファイルの添付

エントリの［添付ファイル］セクションでは、エントリに添付ファイルを追加するさまざまな代替方法が提供されています。

ファイルのドラッグアンドドロップ

別の場所からファイルをドラッグしてエントリにドロップすることで、エントリにファイルを添付できます。

  1. ［ファイルのドラッグアンドドロップ］アイコンをクリックします。

    フォルダアイコンの付いたウィンドウが起動します。

  2. アップロードするファイルをドラッグして、フォルダアイコンにドロップします。

    開閉しているフォルダが表示され、ドロップしたファイルが添付ファイルとしてアップロードされていることを示します。

これでファイルがエントリにアップロードされます。

WebDAVを使用した添付ファイルへのアクセス(Internet Explorerのみ)

Teamingでは、リモートWebサーバに保存されているファイルにWebDAVを使用してアクセスして、エントリに添付することができます。ただし、WebDAVがサポートされているブラウザはInternet Explorerのみです。Internet Explorer以外のブラウザを使用している場合、このオプションは表示されません。

  1. ［WebDAVを使用して添付ファイルにアクセス］をクリックします。

  2. 添付するファイルに移動してファイルを選択します。

TeamingでWebDAVを使用する際の詳細については、『Novell Teaming 2.1 User Guide』の「Accessing Attachments Through WebDAV (Supported Environments Only)」を参照してください。

単一のファイルのアップロード

  1. ファイルをアップロードするエントリで、［添付ファイル］セクションの［単一のファイルのアップロード］アイコンをクリックします。

  2. ［参照］をクリックして、添付するファイルを参照して選択します。

  3. ［OK］をクリックします。

詳細については、「保証と著作権」を参照してください。
