フォルダまたはワークスペースのURLの定義

この項では、現在作業しているフォルダやワークスペースに対してわかりやすいURLを定義する方法について説明します。このURLを定義すると、ブラウザからフォルダまたはワークスペースに直接移動できるようになります。

フォルダのURLを定義すると、電子メールまたはテキストメッセージを使用してそのフォルダにエントリを追加できるようになります。

  1. ［URLの定義］ドロップダウンリストで、事前定義されたグローバルキーワードの1つを選択します。

    サイト管理者でない限り、これらの事前定義されたグローバルキーワードの1つをURLの先頭に付ける必要があります。

  2. このフィールドで、フォルダまたはワークスペースの目的のURLを指定します。

  3. ［追加］をクリックします。

  4. (条件付き)フォルダのURLを定義する場合は、定義したURLを選択して、URLを有効にする必要があります。選択する必要があるURLは、［追加］ボタンの下にあります。

  5. (条件付き)フォルダのURLを定義する場合は、［適用］をクリックします。

  6. ［閉じる］をクリックします。

定義したURLを削除するには、［現在定義されているURL］セクションでURLを選択して、［選択したURLを削除］をクリックします。

詳細については、「保証と著作権」。
