フォルダ管理ツール

［フォルダ］ツールバーの［管理］をクリックすると、使用が許可されている管理および設定ツールのみが表示されます。これらのツールを使用したり、アクセスを付与したりできるのは、サイト管理者か、このワークスペースまたはフォルダの所有者です。

無制限アクセスを持っている場合は、次のオプションが表示されます。

  • アクセス制御: この場所を表示したり、この場所に参加したりできるユーザを管理できます。

  • アクティビティレポート: ワークスペース内のアクティビティを表示できます。

  • 設定: デフォルト表示、デフォルトエントリタイプ、許可された表示、およびフォルダURLの設定など、フォルダの設定を行うことができます。

  • このフォルダをコピー: 適切な権限があれば、ワークスペースツリー内の目的の宛先に現在のフォルダをコピーできます。ワークスペースツリーのナビゲーションを参照してください。

  • このフォルダの削除: 現在のフォルダを削除します。

  • 電子メール設定: 他のTeamingユーザに自動的に電子メール通知を送信するようにフォルダを設定できます。

  • エクスポート/インポート: Teaming管理者のみです。現在のフォルダをエクスポートしたり、フォルダやワークスペースを現在のフォルダにインポートしたりできます。

    Teamingでワークスペースやフォルダをエクスポートおよびインポートする際の詳細については、 『Kablink Teaming 2.1 Administration Guide』の「Exporting/Importing Workspaces, Folders, and Entries」を参照してください。

  • フォームと表示デザイナ: カスタムエントリフォーラムやカスタムワークフローを作成して、後でフォルダやそのサブフォルダに適用できます。

  • このフォルダを変更: フォルダ名、説明、ブランディング、およびアイコンなど、フォルダを変更できます。

  • このフォルダを移動: 適切な権限があれば、ワークスペースツリー内の目的の宛先に現在のフォルダを移動できます。ワークスペースツリーのナビゲーションを参照してください。

  • 新規フォルダの追加: ワークスペースに新しいフォルダを追加できます。

詳細については、「保証と著作権」を参照してください。
