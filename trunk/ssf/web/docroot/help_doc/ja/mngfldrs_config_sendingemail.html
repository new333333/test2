電子メール通知の送信

フォルダ所有者は、他のユーザに電子メール通知を送信したり、このフォルダおよびこのフォルダの任意のサブフォルダ内のアクティビティを通知したりするようにKablink Teamingを設定できます。ただし、フォルダ所有者が設定した通知は、ユーザの個々の設定によって上書きまたはキャンセルされます。

  1. ［電子メール通知のタイプを選択］セクションで、送信する電子メール通知のタイプを選択します。

    さまざまなタイプの電子メール通知の詳細については、受信する電子メール通知のタイプの判断を参照してください。

  2. 電子メール通知を受信してもらいたいユーザを指定します。

    特定の電子メールアドレス、ユーザ、またはグループを指定することも、チームメンバー全員に通知を送信することもできます。

  3. ［適用］>［閉じる］の順にクリックします。

電子メール通知の詳細については、電子メール通知の使用を参照してください。

詳細については、「保証と著作権」。
