ワークスペース表示

ワークスペースを表示するのに最適な表示を選択します。

  • ディスカッションワークスペース: この表示は、複数のディスカッションフォーラムを格納可能なワークスペースを作成する場合に選択します。

    ディスカッションワークスペースの詳細については、『Novell Teaming 2.1 User Guide』の「Managing a Discussions Workspace」を参照してください。

  • ランディングページ: この表示は、チームメンバーがワークスペースに入ると同時に重要な情報を表示する場合に選択します。

    ランディングページの詳細については、『Novell Teaming 2.1 User Guide』の「Managing a Landing Page」を参照してください。

  • プロジェクト管理ワークスペース: この表示は、タスクの完了度に基づいてプロジェクトタスクやマイルストーンをトラッキングする場合に選択します。

    プロジェクト管理ワークスペースの詳細については、『Novell Teaming 2.1 User Guide』の「Managing a Project Management Workspace」を参照してください。

  • チームワークスペース: この表示は、自分のチームワークスペースが事実上組織のもので、チームワークスペースをその副次的な場所として使用する目的がない場合に選択します。

    チームワークスペースの詳細については、『Novell Teaming 2.1 User Guide』の「Managing a Team Workspace」を参照してください。

  • チームワークスペースルート: この表示は、自分のチームワークスペースに［チームワークスペースを追加］ボタンを表示する場合に選択します。これは、追加のチームワークスペースをこのワークスペースの副次的な場所として作成する場合に効果的です。

    チームワークスペースのルートワークスペースの詳細については、『Novell Teaming 2.1 Advanced User Guide』の「Creating and Managing a Team Workspace Root Workspace」を参照してください。

  • ワークスペース: これは基本ワークスペースです。ワークスペースに含めるフォルダを選択できます。

    基本ワークスペースの詳細については、『Novell Teaming 2.1 Advanced User Guide』の「Creating and Managing a Basic Workspace」を参照してください。

これがパーソナルワークスペースの場合、［ユーザワークスペース］がデフォルト表示になり、変更することはできません。

詳細については、「保証と著作権」を参照してください。
