［ブログ］サイドバー

Novell Teamingの［ブログ］サイドバーには、ブログエントリを検索および表示するための次のツールが用意されています。

ブログページ

ブログフォルダ内に複数のページがある場合、それらのページが［ブログ］サイドバーの上部に一覧表示されます。フォルダ内の別のページに移動するには、ページ名のリンクをクリックします。

アーカイブ

［ブログ］サイドバーには［アーカイブ］セクションが用意されており、ブログフォルダにエントリを作成したすべての月が一覧表示されます。月をクリックすると、その月に作成されたすべてのブログエントリが新しいものから順に表示され、最新のエントリがリストの先頭に表示されます。

詳細については、「保証と著作権」を参照してください。
