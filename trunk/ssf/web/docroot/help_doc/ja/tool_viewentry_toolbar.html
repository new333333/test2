［エントリ］ツールバー

［エントリ］ツールバーには、ユーザが許可を持っているツールのみが提供されます。このツールバーはエントリとコメントの両方に対して表示され、次のボタンとメニューが含まれる可能性があります。

  • コメント: エントリにコメントを追加します。コメントについてのコメントを作成することができます(これを「ネストされたコメントの作成」と呼びます)。

  • 変更: エントリを編集したり、別の添付ファイルを追加したりすることができます(エントリ右下のセクション、［添付ファイル］セクションの真上にある［編集］を使用することもできます)。

  • 予約: エントリの編集中に他のユーザがエントリを編集できないようにします。エントリを大幅に変更する場合、エントリを予約することをお勧めします。変更し終わったら、［予約解除］をクリックします。

  • 移動: エントリを別のフォルダに移動できます。

  • コピー: エントリをコピーして、別の場所に保存できます。

  • 削除: エントリをフォルダから削除します。

  • ワークフロー: フォルダでワークフローが有効になっている場合、エントリでワークフローが開始されていなければ、このツールからワークフローを開始できます。エントリでワークフローが進行中の場合は、ワークフローを停止できます。

  • Wikiホームページの設定: Wikiエントリでのみ利用できます。このオプションは、任意のWikiページをWikiホームページとして表示するようにTeamingを設定する場合に使用します。Wikiホームページとして設定するページに移動して、このオプションをクリックするだけです。

  • レポート: 次のレポートを生成できます。

    • アクティビティレポート: Microsoft Excelでカンマ区切り値ファイルを表示します。このファイルは、ユーザ、およびこのエントリに関連するそのアクティビティの情報を提供します。

    • 履歴を編集: ［履歴を編集］ウィンドウを起動します。このウィンドウで、追加された内容および削除された内容を確認して、エントリの異なるバージョンを比較できます。

    • ワークフロー履歴: ［ワークフロー履歴］ウィンドウを起動します。このウィンドウで、ワークフローの履歴を参照できます。ワークフロー名、エントリの作成日と作成者、ワークフローが特定の状態になった日時、エントリがその状態になったときにエントリを変更したユーザなどが含まれます。

  • このエントリを共有: 他のTeamingユーザとエントリを共有できます。エントリを共有すると、エントリを共有するよう選択したユーザに通知電子メールが送信されます。また、ユーザのホームワークスペースで［マイクロブログと共有項目］タブを選択すると、［共有項目］カラムに、他のユーザと共有している項目がすべて表示されます。

詳細については、「保証と著作権」を参照してください。
