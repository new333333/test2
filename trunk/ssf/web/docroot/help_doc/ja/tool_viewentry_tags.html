タグ

タグは、ワークスペース、フォルダ、またはエントリを分類するキーワードです。Novell Teamingのタグ機能を使用すると、場所やエントリの仮想コンテナを作成できます。電子メール内のフォルダと同様に、情報を分類しておき、後でその情報に再度アクセスできます。

ワークスペースまたはフォルダへのタグ付け

  1. タグを付けるワークスペースまたはフォルダに移動します。

  2. メインサイドバーの［このワークスペースにタグを付ける］(または［このフォルダにタグを付ける］)セクションで、［パーソナルタグ］または［コミュニティタグ］セクションのいずれかにタグの名前を入力します。

    パーソナルタグとコミュニティタグのどちらを作成するべきかを判断するには、パーソナルタグとコミュニティタグの選択を参照してください。

    このフィールドではインクリメンタルサーチ機能が使用されているため、目的の名前を持つタグがすでに存在している場合は、ドロップダウンリストに表示されます。ドロップダウンリストに名前が表示されない場合は、新しいタグを作成することになります。

  3. タグ名を入力し終わるか、ドロップダウンリストからタグ名を選択したら、［追加］をクリックします。

これで、指定したタグがワークスペースまたはフォルダに付きます。

エントリへのタグ付け

  1. タグを付けるエントリで、［これにタグを付ける］をクリックします。

  2. ［パーソナルタグ］または［コミュニティタグ］セクションのいずれかのフィールドにタグ名を入力します。

    パーソナルタグとコミュニティタグのどちらを作成するべきかを判断するには、パーソナルタグとコミュニティタグの選択を参照してください。

    このフィールドではインクリメンタルサーチ機能が使用されているため、目的の名前を持つタグがすでに存在している場合は、ドロップダウンリストに表示されます。ドロップダウンリストに名前が表示されない場合は、新しいタグを作成することになります。

  3. タグ名を入力し終わるか、ドロップダウンリストからタグ名を選択したら、［追加］をクリックします。

これで、指定したタグがエントリに付きます。

パーソナルタグとコミュニティタグの選択

  • パーソナルタグ: パーソナルタグは自分専用です。パーソナルタグを作成した場合、そのタグを使用および表示できるのは自分のみになります。

  • コミュニティタグ: コミュニティタグは、コミュニティタグが実装されたワークスペース、フォルダ、またはエントリの表示権限を持つユーザならだれでも使用および表示できます。たとえば、チームワークスペース内のファイルフォルダに対してコミュニティタグを作成した場合、そのファイルフォルダに対する表示権限持つユーザならだれでも、作成されたタグを使用して、同じタグ名のタグを他の場所に付けることができます。さらに、Teamingの検索機能を使用して、該当するタグ名のタグが付いたエントリを検索することもできます。

    検索を使用してタグ付き項目を検索する方法の詳細については、詳細検索の使用を参照してください。

    検索を使用してタグ付き項目を検索する方法の詳細については、特定の人、場所、またはタグ付き項目の検索を参照してください。

    場所およびエントリにコミュニティタグを付けることができるのは、適切な権限を持っている場合のみです。

詳細については、「保証と著作権」を参照してください。
