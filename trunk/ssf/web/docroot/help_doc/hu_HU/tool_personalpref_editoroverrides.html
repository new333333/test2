Szerkesztő felülbírálása

A mellékletek szerkesztésekor a Novell Teaming rendszer az alapértelmezetten beállított alkalmazást használja a mellékletek szerkesztésére. Azonban a táblázat segítségével felülbírálhatja azokat az alapértelmezett alkalmazásokat, amelyeket meghatározott fájlkiterjesztések szerkesztésére használ a Teaming.

Felülbírálhatja az alapértelmezett szerkesztő alkalmazást egyetlen fájlkiterjesztés típus esetében, vagy felülbírálhatja az alapértelmezett alkalmazásokat több fájlkiterjesztés típus esetében is.

Az alapértelmezett szerkesztő alkalmazás felülbírálása egyedi fájlkiterjesztések esetében

Az alapértelmezett alkalmazás felülbírálása egyedi fájlkiterjesztés típusok esetében:

  1. Kattintson a Hozzáadás gombra.

  2. A legördülő listából válassza ki azt a fájlkiterjesztés típust, amelyen szeretné felülbírálni az alapértelmezett szerkesztő alkalmazást.

  3. Az Alkalmazás mezőben, határozza meg alkalmazás nevét, amelyet a meghatározott fájlkiterjesztéssel rendelkező mellékletek szerkesztésére szeretne használni Lépés 2.

  4. Kattintson az OK gombra.

Az alapértelmezett szerkesztő alkalmazás felülbírálása az összes fájlkiterjesztés esetében

Az összes fájlkiterjesztés típus felülbírálásához:

  1. Kattintson a Használat menüpontra.

  2. A legördülő listából válassza azt az alkalmazást, amellyel szerkeszteni szeretné az összes fájlkiterjesztés típust a Teamingben.

  3. Kattintson az OK gombra.

További információ a Szerzői jogi megjegyzések.
