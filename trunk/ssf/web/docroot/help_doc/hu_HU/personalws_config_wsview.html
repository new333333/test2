Munkaterület-nézet

Válassza ki azt a nézetet, amely a legjobban illeszkedik a munkaterületéhez.

  • Fórumok munkaterület: Válassza ezt a nézetet, ha olyan munkaterületet több fórum tárolására szeretne létrehozni.

    További tájékoztatás a fórumok munkaterületeiről: Managing a Discussions Workspace című fejezet, Novell Teaming 2.1 User Guide útmutató.

  • Érkezési oldal: Ezt a nézetet akkor válassza, ha szeretne fontos információkat megjeleníteni a csapattagok számára, mihelyst azok belépnek a munkaterületre.

    További tájékoztatás az érkezési oldalakról: Managing a Landing Page című fejezet, Novell Teaming 2.1 User Guide útmutató.

  • Projektvezetési munkaterület: Ezt a nézetet akkor válassza, ha szeretné a projektfeladatokat és mérföldköveket a feladatok elkészültsége szerint figyelemmel kísérni.

    További tájékoztatás a projektvezetési munkaterületekről: Managing a Project Management Workspace című fejezet, Novell Teaming 2.1 User Guide útmutató.

  • Csapat-munkaterület: Ezt a nézetet akkor válassza, ha csapat munkaterülete szervezeti sajátosságokkal rendelkezik és nincs szándékában további csapat-munkaterületeket almunkaterületként használni.

    További tájékoztatás a csapat-munkaterületekről: Managing a Team Workspace című témakör, Novell Teaming 2.1 User Guide útmutató.

  • Csapat munkaterületének gyökere: Válassza ezt a nézetet, ha szeretné, hogy csapat-munkaterületén a Csapat-munkaterület hozzáadása gomb megjelenjen. Erre akkor van szükség, ha további csapat-munkaterületeket szeretne létrehozni, az aktuális munkaterület alhelyeiként.

    További tájékoztatás a csapat-munkaterületen elhelyezett munkaterületekről : Creating and Managing a Team Workspace Root Workspace című fejezet, Novell Teaming 2.1 Advanced User Guide útmutató.

  • Munkaterület: Ez egy alapszintű munkaterület. Választhat, hogy a munkaterület mely mappákat tartalmazza.

    További tájékoztatás az alapszintű munkaterületekről: Creating and Managing a Basic Workspace című fejezet, Novell Teaming 2.1 Advanced User Guide útmutató.

Amennyiben ez egy személyes munkaterület, akkor a Felhasználói munkaterület az alapértelmezett nézet és ez nem változtatható.
