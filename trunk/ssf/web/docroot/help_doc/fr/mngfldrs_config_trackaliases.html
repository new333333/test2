Suivi des alias

Lorsque des propriétaires de dossier ou des administrateurs configurent leurs paramètres de messagerie électronique pour spécifier une adresse alias pour la publication dans le dossier, les informations de configuration apparaissent dans cette colonne. Utilisez ces informations pour suivre les alias existants pour le compte de messagerie entrante de l'installation.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
