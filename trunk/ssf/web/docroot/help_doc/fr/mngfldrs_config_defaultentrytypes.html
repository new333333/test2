Types d'entrée par défaut

Cette section permet de sélectionner les types d'entrée que vous souhaitez permettre aux utilisateurs de créer dans le dossier. Vous pouvez modifier le type d'entrée actuel ou configurer votre dossier de manière à autoriser d'autres types d'entrée. Si vous ajoutez des types d'entrée supplémentaires, les utilisateurs qui soumettent des entrées dans ce dossier peuvent choisir le type d'entrée qu'ils souhaitent ajouter, tel qu'une entrée de tâche ou d'agenda.

  1. Sélectionnez les types d'entrée que vous souhaitez activer pour ce dossier.

  2. Cliquez sur Appliquer > Fermer.

Si vous activez des types d'entrée supplémentaires, les utilisateurs qui soumettent des entrées dans ce dossier peuvent cliquer sur Ajouter dans la barre d'outils Entrée de dossier, puis utiliser la liste déroulante pour sélectionner le type d'entrée qu'ils souhaitent ajouter, selon les types d'entrée que vous avez activés pour le dossier.

Les administrateurs de site et les propriétaires de dossier peuvent utiliser la fonction Concepteurs de formulaires et de vues pour créer de nouveaux types d'entrée (formulaires personnalisés), qui s'affichent ensuite dans cette liste. (Lorsque des propriétaires de dossier créent des nouveaux types d'entrée, ceux-ci sont répertoriés dans la section Définitions locales de cette liste.)

Pour plus d'informations sur la création de types d'entrée personnalisés, reportez-vous à la section Designing Custom Folder Entry Forms (Conception de formulaires d'entrée de dossier personnalisés) du manuel Novell Teaming 2.1 Advanced User Guide (Guide de l'utilisateur avancé de Novell Teaming 2.1).

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
