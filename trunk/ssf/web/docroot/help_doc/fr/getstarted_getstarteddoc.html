Documentation

Les liens de cette section affichent les manuels de mise en route dans une nouvelle fenêtre de navigateur. Vous pouvez lire ces manuels en ligne au format HTML et PDF. Le format PDF est également optimisé pour l'impression.

Nous sommes à l'écoute de vos commentaires et suggestions concernant ces manuels et les autres documentations fournies avec ce produit. Utilisez la fonction Commentaires proposée au bas de chaque page de la documentation en ligne ou accédez à la page Web www.novell.com/documentation/feedback.html (en anglais).

Tous les utilisateurs Teaming

Les guides suivants présentent un intérêt pour tous les utilisateurs Novell Teaming :

  • Novell Teaming 2.1 User Guide (Guide de l'utilisateur avancé de Novell Teaming 2.0)

    Ce guide fournit des informations détaillées expliquant comment effectuer diverses tâches dans Teaming. Il comprend des descriptions globales, des instructions de navigation et de personnalisation, ainsi que des graphiques.

  • Novell Teaming 2.1 Advanced User Guide (Guide de l'utilisateur avancé de Novell Teaming 2.0)

    Ce guide est destiné aux propriétaires d'espaces de travail et de dossiers qui sont chargés de rendre leurs espaces de travail opérationnels et efficaces pour les utilisateurs. Parmi les personnes susceptibles d'être intéressées par ces informations, citons notamment les chefs d'équipe et les responsables de projet.

Administrateurs Teaming

Les guides suivants présentent un intérêt pour les administrateurs Novell Teaming :

  • Kablink Teaming 2.1 Administration Guide (Guide d'administration de Kablink Teaming 2.1)

    Ce guide décrit les tâches qu'un administrateur doit accomplir pour installer, configurer et gérer un site Teaming.

  • Novell Teaming 2.1 Advanced User Guide (Guide de l'utilisateur avancé de Novell Teaming 2.0)

    Ce guide est destiné aux propriétaires d'espaces de travail et de dossiers qui sont chargés de rendre leurs espaces de travail opérationnels et efficaces pour les utilisateurs. Parmi les personnes susceptibles d'être intéressées par ces informations, citons notamment les chefs d'équipe et les responsables de projet.

  • Guide d'installation de Novell Teaming 2.1

    Ce guide décrit les tâches requises pour installer Teaming de manière à ce que toutes les fonctions par défaut fonctionnent comme prévu. Il explique également comment configurer un utilisateur Teaming.

  • Novell Teaming 2.1 Developer Guide (Guide du développeur de Novell Teaming 2.1)

    Le présent guide est destiné principalement aux développeurs qui souhaitent étendre la fonctionnalité de Teaming ou programmer d'autres applications pour qu'elles interagissent avec celui-ci.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
