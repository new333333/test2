Documentation de mise en route

Les liens de cette section affichent les manuels de mise en route dans une nouvelle fenêtre de navigateur. Vous pouvez lire ces manuels en ligne au format HTML et PDF. Le format PDF est également optimisé pour l'impression.

Tous les utilisateurs Teaming

Les guides suivants présentent un intérêt pour tous les utilisateurs Kablink Teaming :

  • Kablink Teaming 2.0 User Guide (Guide de l'utilisateur de Kablink Teaming 2.0)

    Ce guide fournit des informations détaillées expliquant comment effectuer diverses tâches dans Teaming. Il comprend des descriptions globales, des instructions de navigation et de personnalisation, ainsi que des graphiques.

  • Kablink Teaming 2.0 Advanced User Guide (Guide de l'utilisateur avancé de Kablink Teaming 2.0)

    Ce guide est destiné aux propriétaires d'espaces de travail et de dossiers qui sont chargés de rendre leurs espaces de travail opérationnels et efficaces pour les utilisateurs. Parmi les personnes susceptibles d'être intéressées par ces informations, citons notamment les chefs d'équipe et les responsables de projet.

Administrateurs Teaming

Les guides suivants présentent un intérêt pour les administrateurs Kablink Teaming :

  • Kablink Teaming 2.0 Administration Guide (Guide d'administration de Kablink Teaming 2.0)

    Ce guide décrit les tâches qu'un administrateur doit accomplir pour installer, configurer et gérer un site Teaming.

  • Kablink Teaming 2.0 Advanced User Guide (Guide de l'utilisateur avancé de Kablink Teaming 2.0)

    Ce guide est destiné aux propriétaires d'espaces de travail et de dossiers qui sont chargés de rendre leurs espaces de travail opérationnels et efficaces pour les utilisateurs. Parmi les personnes susceptibles d'être intéressées par ces informations, citons notamment les chefs d'équipe et les responsables de projet.

  • Kablink Teaming 2.0 Installation Guide (Guide d'installation de Kablink Teaming 2.0)

    Ce guide décrit les tâches requises pour installer Teaming de manière à ce que toutes les fonctions par défaut fonctionnent comme prévu. Il explique également comment configurer un utilisateur Teaming.

  • Kablink Teaming 2.0 Developer Guide (Guide du développeur de Kablink Teaming 2.0)

    Le présent guide est destiné principalement aux développeurs qui souhaitent étendre la fonctionnalité de Kablink Teaming ou programmer d'autres applications pour qu'elles interagissent avec celui-ci.
