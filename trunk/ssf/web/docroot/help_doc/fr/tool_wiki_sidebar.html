Barre latérale Wiki

La barre latérale Wiki contient les fonctions suivantes :

  • Page d'accueil Wiki : cliquez sur ce bouton pour accéder à la page d'accueil Wiki que vous avez préalablement définie. Par défaut, Novell Teaming ne définit pas de page d'accueil Wiki ; c'est à l'utilisateur de le faire manuellement. Si vous souhaitez configurer le dossier Wiki pour qu'il affiche une page d'accueil Wiki initiale, localisez et affichez l'entrée que vous voulez définir comme page d'accueil, puis cliquez sur Définir la page d'accueil du Wiki dans la barre d'outils Entrée.

  • Rechercher une page : cette zone de texte utilise la fonction Saisir le texte à rechercher. Il suffit de commencer à saisir le nom de la page Wiki que vous recherchez et de cliquer sur le nom lorsqu'il apparaît.

    Teaming vous amène alors sur la page souhaitée.

  • Rubriques Wiki : cette section dresse la liste de toutes les rubriques Wiki qui existent dans le dossier Wiki. Lorsque vous sélectionnez une rubrique dans cette liste, l'ensemble des pages Wiki figurant dans cette rubrique s'affichent dans la section Pages Wiki.

  • Pages Wiki : cette section dresse la liste de toutes les pages Wiki figurant dans la rubrique Wiki actuellement sélectionnée.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
