Listes des entrées et des pages

Ces outils vous permettent de rechercher des entrées individuellement ou naviguer parmi les pages de listes. Une liste de dossiers affiche les entrées ou les informations sur les entrées. Certains de ces outils ne sont utiles que s'il existe plusieurs pages de liste d'entrées. De gauche à droite de cette barre d'outils, les outils disponibles sont les suivants :

  • Aller à une entrée : indiquez le numéro (affiché dans la colonne Numéro) de l'entrée que vous souhaitez afficher, puis cliquez sur Aller.

  • Précédent/Suivant : permet de faire défiler les pages du dossier.

  • Aller à la page : si vous entrez un numéro de page dans cette zone de texte et cliquez sur Aller, Kablink Teaming affiche les entrées de cette page de liste.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
