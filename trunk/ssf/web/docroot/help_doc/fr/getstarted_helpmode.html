Mise en route avec le mode Aide

Le panneau dans le coin supérieur gauche de la page s'intitule Panneau de configuration du mode Aide. Le plus souvent, vous l'utilisez pour accéder à la table des matières ou aux manuels de mise en route au format PDF. Pour réduire au maximum le Panneau de configuration du mode Aide, cliquez sur l'icône Réduire , située en haut du Panneau de configuration. Cliquez sur cette icône à nouveau pour rétablir la taille d'origine du panneau.

Le mode Aide décrit la page que vous voyez actuellement. Cliquez sur une icône Informations et Kablink Teaming affiche un panneau Aide qui décrit la section concernée de la page. Pour fermer le panneau d'aide, cliquez sur l'icône Fermer (X) dans le coin supérieur droit du panneau Aide. Vous pouvez également cliquer sur une autre icône Informations pour lire d'autres explications. Plutôt que d'utiliser les icônes Informations, vous pouvez cliquer sur des rubriques d'aide dans le Panneau de configuration du mode Aide, puis sur les titres des sections qui vous intéressent.

Pour une documentation plus complète concernant Teaming, vous pouvez cliquer sur Afficher les manuels dans le Panneau de configuration du mode Aide.

Lorsque vous avez fini d'utiliser le mode Aide, vous pouvez quitter en cliquant sur l'icône Fermer (X) dans l'angle supérieur droit du Panneau de configuration ou vous pouvez cliquer n'importe où dans la zone grisée de la page.

Une fois que vous avez quitté le mode Aide et que vous commencez à utiliser des outils, Teaming propose des icônes Informations sur les formulaires qui nécessitent des explications supplémentaires. Cliquez sur l'icône Informations en regard d'une section de formulaire pour en savoir plus à son sujet.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
