Envoi de notifications par message électronique

En tant que propriétaire d'un dossier, vous pouvez configurer Kablink Teaming de manière à envoyer à d'autres utilisateurs des notifications par message électronique pour les informer de toute activité dans ce dossier et ses sous-dossiers. Toutefois, les paramètres individuels d'un utilisateur remplacent ou annulent les notifications établies par les propriétaires de dossier.

  1. Dans la section Sélectionnez le type de notification par message électronique, choisissez le type de notification à envoyer.

    Pour plus d'informations sur les différents types de notifications par message électronique, reportez-vous à la section Choix du type de notification par message électronique que vous voulez recevoir.

  2. Spécifiez les utilisateurs qui doivent recevoir les notifications par message électronique.

    Vous pouvez soit spécifier des adresses électroniques, des utilisateurs ou des groupes spécifiques, soit envoyer les notifications à tous les membres de l'équipe.

  3. Cliquez sur Appliquer > Fermer.

Pour plus d'informations sur les notifications par message électronique, reportez-vous à la section Utilisation de notifications par message électronique.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
