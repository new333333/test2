Planification de la synchronisation

Cette page vous permet d'activer et de configurer une planification de synchronisation pour vos dossiers en miroir. Elle contient les options suivantes :

Activer la synchronisation planifiée : cochez cette case pour activer la synchronisation planifiée.

Tous les jours : sélectionnez cette option pour que la synchronisation ait lieu tous les jours.

Toutes les semaines : sélectionnez cette option pour spécifier les jours de la semaine où la synchronisation doit avoir lieu.

À heure fixe : indiquez l'heure précise à laquelle la synchronisation doit avoir lieu.

Répéter toutes les nn heures : indiquez l'intervalle entre deux synchronisations.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
