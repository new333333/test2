Vista spazio di lavoro

Selezionare la vista desiderata per lo spazio di lavoro.

  • Spazio di lavoro di discussione: selezionare questa vista se si desidera creare uno spazio di lavoro in cui memorizzare più forum di discussione.

    Per ulteriori informazioni sugli spazi di lavoro di discussione, vedere Managing a Discussions Workspace nella Kablink Teaming 2.0 Advanced User Guide (in lingua inglese).

  • Pagina di destinazione: selezionare questa vista per visualizzare informazioni importanti sui membri del team non appena accedono allo spazio di lavoro.

    Per ulteriori informazioni sulle pagine di destinazione, vedere Creating a Landing Page nella Kablink Teaming 2.0 Advanced User Guide (in lingua inglese).

  • Spazio di lavoro per la gestione dei progetti: selezionare questa vista per controllare task e milestone del progetto in base al completamento dei task.

    Per ulteriori informazioni sugli spazi di lavoro per la gestione dei progetti, vedere Managing a Project Management Workspace nella Kablink Teaming 2.0 Advanced User Guide (in lingua inglese).

  • Spazio di lavoro del team: selezionare questa vista se lo spazio di lavoro del team è di natura organizzativa e non sono previsti spazi di lavoro del team come rispettive posizioni secondarie.

    Per ulteriori informazioni sugli spazi di lavoro del team, vedere Managing a Team Workspace nella Kablink Teaming 2.0 User Guide (in lingua inglese).

  • Radice spazio di lavoro del team: selezionare questa vista per visualizzare il pulsante Aggiungi spazio di lavoro del team nello spazio di lavoro del team. Questa operazione è necessaria se si desidera creare spazi di lavoro del team aggiuntivi come posizioni secondarie allo spazio di lavoro specificato.

    Per ulteriori informazioni sugli spazi di lavoro radice degli spazi di lavoro del team, vedere Managing a Team Workspace Root Workspace nella Kablink Teaming 2.0 Advanced User Guide (in lingua inglese).

  • Spazio di lavoro: è lo spazio di lavoro di base. È possibile scegliere le cartelle da includere nello spazio di lavoro.

    Per ulteriori informazioni sugli spazi di lavoro di base, vedere Managing a Basic Workspace nella Kablink Teaming 2.0 Advanced User Guide (in lingua inglese).

Se si tratta di uno spazio di lavoro personale, la vista di default è Spazio di lavoro utente e non è possibile modificarla.
