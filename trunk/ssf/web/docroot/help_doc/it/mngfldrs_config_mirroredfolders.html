Cartelle in copia speculare

Una cartella in copia speculare è una cartella Libreria i cui file sono ubicati in un sistema esterno (una directory in un file system o una cartella su un server SharePoint). Quando si esegue la copia speculare di cartelle esterne a Teaming, è possibile applicare gli strumenti di Novell Teaming (ad esempio, la funzione di ricerca e il workflow) a file esterni.

È necessario utilizzare un file sul server per abilitare questa funzione.

Per ulteriori informazioni sulla creazione di cartelle in copia speculare, consultare Working with Mirrored Folders nella Novell Teaming 2.1 User Guide (in lingua inglese).
