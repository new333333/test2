Priorità dell'editor

Quando si modificano gli allegati in Novell Teaming, vengono utilizzate le applicazioni di default del sistema. Tuttavia, questa tabella consente di ignorare le applicazioni di default utilizzate e di sostituirle con estensioni di file specifiche fornite da Teaming.

È possibile ignorare l'applicazione di modifica di default per un singolo tipo di estensione di file oppure è possibile ignorare le applicazioni di default per più tipi di estensioni di file.

Priorità sull'applicazione di modifica di default per estensioni di file singole

Per ignorare l'applicazione di default per un tipo di estensione di file singolo:

  1. Fare clic su Aggiungi.

  2. Nell'elenco a discesa, selezionare il tipo di estensione di file di cui ignorare l'applicazione di modifica di default.

  3. Nel campo Applicazione, specificare il nome dell'applicazione da utilizzare per modificare gli allegati con l'estensione di file specificata in Passaggio 2.

  4. Fare clic su OK.

Priorità sull'applicazione di modifica di default per tutte le estensioni di file

Per ignorare tutti i tipi di estensioni di file:

  1. Fare clic su Utilizza.

  2. Nell'elenco a discesa, selezionare l'applicazione da utilizzare per modificare gli allegati con tutti i tipi di estensioni di file in Teaming.

  3. Fare clic su OK.

Per ulteriori informazioni, vedere Note legali.
