Definizione di un URL per una cartella o spazio di lavoro

Utilizzare questa sezione per definire un URL descrittivo per la cartella o lo spazio di lavoro attualmente in uso. La definizione dell'URL consente di accedere direttamente alla cartella o allo spazio di lavoro dal browser.

La definizione di un URL per una cartella consente di aggiungere voci a tale cartella tramite e-mail o messaggio di testo.

  1. Nell'elenco a discesa Definisci URL, selezionare una delle parole chiave globali predefinite.

    A meno che non si è un amministratore del sito, l'URL deve iniziare con una di queste parole chiave globali predefinite.

  2. Specificare nel campo l'URL desiderato per la cartella o lo spazio di lavoro.

  3. Fare clic su Aggiungi.

  4. (Condizionale) Se si definisce l'URL per una cartella, è necessario abilitarlo selezionandolo. L'URL da selezionare è situato sotto il pulsante Aggiungi.

  5. (Condizionale) Se si definisce l'URL per una cartella, fare clic su Applica.

  6. Fare clic su Chiudi.

Per cancellare l'URL definito, selezionare l'URL nella sezione URL attualmente definiti, quindi fare clic su Cancella gli URL selezionati.

Per ulteriori informazioni, vedere Note legali.
