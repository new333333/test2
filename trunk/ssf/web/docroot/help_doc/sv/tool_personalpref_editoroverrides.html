Åsidosättningar för editor

När du redigerar bifogade filer i Novell Teaming använder du standardprogrammen i Teaming-systemet. Om du vill kan du emellertid åsidosätta standardprogrammen som används för att redigera bifogade filer med särskilda filnamnstillägg i Teaming.

Du kan åsidosätta standardprogrammet för ett eller flera filnamnstillägg.

Åsidosätta standardredigeringsprogrammet för enstaka filnamnstillägg

Så här åsidosätter du standardprogrammet för en enskild typ av filnamnstillägg:

  1. Klicka på Lägg till.

  2. Välj den typ av filnamnstillägg i listrutan som du vill åsidosätta standardredigeringsprogrammet för.

  3. Ange namnet på det program i fältet Program som du vill använda när du redigerar bifogade filer med det filnamnstillägg du valde i Steg 2.

  4. Klicka på OK.

Åsidosätta standardredigeringsprogrammet för alla filnamnstillägg

Så här åsidosätter du alla typer av filnamnstillägg:

  1. Klicka på Använd.

  2. Välj det program i listrutan som du vill använda när du redigerar bifogade filer i Teaming, oavsett filernas filnamnstillägg.

  3. Klicka på OK.

Mer information finns i Juridisk information.
