Schemalägga synkronisering

På den här sidan kan du aktivera och skapa ett synkroniseringsschema för speglade mappar. Sidan innehåller följande alternativ:

Aktivera schemalagd synkronisering: Du måste markera den här kryssrutan för att synkroniseringsschemat ska tillämpas.

Varje dag: Välj det här alternativet om du vill köra synkroniseringen varje dag.

Varje vecka: Välj det här alternativet om du vill ange särskilda dagar i veckan då synkroniseringen ska köras.

Vid tidpunkten: Ange tidpunkten då synkroniseringen ska köras.

Upprepa var nn timme: Ange synkroniseringsintervallet.

Mer information finns i Juridisk information.
