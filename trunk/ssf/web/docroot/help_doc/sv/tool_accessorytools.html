Tillbehörsverktyg

Tillbehör är komponenter som du kan lägga till på sidor i Novell Teaming för att framhäva befintlig information och göra den mer lättåtkomlig. Tillbehören tillför ytterligare funktionalitet och förbättrar gruppsamarbetet och kommunikationen. Här följer några exempel på olika tillbehör:

  • Sökresultat (används förmodligen bara för den aktuella platsen)

  • Kompislista (t.ex. för lokala ämnesexperter)

  • Arbetsyteträd (öppnas till den här platsen i hierarkin)

  • Gästbok

En fullständig lista med Teaming-tillbehör och tillhörande beskrivningar finns i avsnittet om tillbehörstyper i Novell Teaming 2.1 User Guide.

Följande avsnitt innehåller information om hur du använder tillbehörspanelen:

Lägga till ett tillbehör

Så här lägger du till ett tillbehör på tillbehörspanelen:

  1. Klicka på Tillbehörspanel.

  2. Välj en tillbehörsomfattning under Tillbehörsomfattning.

    Mer information om alternativen i det här området finns i avsnittet Ange vem som kan visa tillbehör.

  3. Välj den typ av tillbehör som du vill lägga till under Tillbehörstyp.

  4. Klicka på OK.

  5. Ange ett namn för tillbehöret och fyll i annan relevant information.

    Mer information om hur du fyller i informationen för respektive tillbehörstyp finns i avsnittet om att konfigurera tillbehör i Novell Teaming 2.1 User Guide.

  6. Klicka på Verkställ > Stäng.

Ange vem som kan visa tillbehör

När du lägger till ett tillbehör kan du välja omfattning för tillbehöret. Du kan använda följande alternativ under Tillbehörsomfattning:

  • Personligt tillbehör enbart för den här sidan: Tillbehöret används bara för den aktuella platsen, och bara du kan visa det.

  • Personligt tillbehör för alla arbetsytor och mappar: Tillbehöret används för alla arbetsytor och mappar, och bara du kan visa det.

  • Grupptillbehör för den här sidan: Tillbehöret används bara för den här sidan och kan visas av alla Teaming-användare som har åtkomstbehörighet till sidan.

Ändra ordning på tillbehör

Du kan ändra ordning på tillbehören på en viss sida om det finns mer än ett tillbehör.

  1. Klicka på upp- eller nedpilen längst upp till höger i tillbehörets namnlist för att flytta upp eller ned tillbehöret.

Konfigurera om ett tillbehör

I Teaming kan du konfigurera om tillbehör efter att de har skapats.

  1. Klicka på ikonen Ändra längst upp till höger i tillbehörets namnlist.

Ta bort ett tillbehör

  1. Klicka på ikonen Ta bort längst upp till höger i tillbehörets namnlist.

Dölja tillbehör

I Teaming kan du dölja tillbehör så att bara tillbehörets namnlist visas.

  1. Klicka på ikonen Dölj längst upp till höger i tillbehörets namnlist.

Du kan också dölja alla tillbehör på en gång på tillbehörspanelen.

  1. Klicka på ikonen Dölj bredvid Tillbehörspanel.

    Alla tillbehören på den aktuella sidan döljs.

    Klicka på ikonen Dölj igen när du vill visa tillbehören.
