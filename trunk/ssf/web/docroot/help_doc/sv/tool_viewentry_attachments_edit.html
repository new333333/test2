Redigera bifogade filer

Du kan redigera filer som bifogats till poster i Kablink Teaming.

Om du inte kan redigera en fil kan det bero på att Teaming inte konfigurerats att använda det redigeringsprogram som är installerat på datorn. Om Teaming exempelvis har konfigurerats att använda Microsoft Word som standardredigeringsprogram för alla textfiler, men du har OpenOffice i stället för Microsoft Word på datorn, visas ett felmeddelande i Teaming och du kan inte redigera dokumentet.

Du kan åtgärda det här problemet genom att kringgå standardinställningen för redigeringsprogram i Teaming.

Ändra standardinställningen för redigeringsprogram för en enskild filtyp

  1. Klicka på Definiera åsidosättningar för editor under Personliga inställningar i huvudsidofältet.

  2. Klicka på Lägg till.

  3. Välj det filnamnstillägg i listrutan som du vill ändra standardredigeringsprogram för.

  4. Ange namnet på det program i fältet Program som du vill använda när du redigerar den typ av filer du valde i Steg 3.

  5. Markera kryssrutan och klicka på OK.

Ändra standardinställningen för redigeringsprogram för alla filtyper

Du kan konfigurera Teaming så att samma redigeringsprogram används för alla filtyper.

  1. Klicka på Definiera åsidosättningar för editor under Personliga inställningar i huvudsidofältet.

  2. Klicka på Använd och välj det program som du vill använda när du redigerar filer i Teaming, oavsett filens typ.

  3. Markera kryssrutan till vänster om Tillägg och klicka på OK.
