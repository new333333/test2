Dokumentation

Länkarna i det här avsnittet öppnar nya webbläsarfönster med handböcker som hjälper dig att komma igång. Du kan läsa handböckerna online i HTML- eller PDF-format. PDF-formatet är dessutom utskriftsvänligt.

Vi välkomnar kommentarer och synpunkter om våra handböcker och annan dokumentation som medföljer den här produkten. Använd gärna kommentarsfunktionen längst ner på sidorna i onlinedokumentationen eller lämna dina kommentarer på www.novell.com/documentation/feedback.html.

Alla Teaming-användare

Följande handböcker riktar sig till alla användare av Novell Teaming:

  • Novell Teaming 2.1 User Guide

    Den här handboken innehåller detaljerad information som beskriver hur du utför olika åtgärder i Teaming. Den innehåller översiktliga beskrivningar, navigeringsinstruktioner, anpassningsanvisningar och bilder.

  • Novell Teaming 2.1 Advanced User Guide

    Den här handboken riktar sig till arbetsyte- och mappägare som ansvarar för att göra arbetsytorna användbara och effektiva för användarna. Exempel på användare som kan ha nytta av den här informationen är team- och projektledare.

Teaming-administratörer

Följande handböcker riktar sig till Novell Teaming-administratörer:

  • Kablink Teaming 2.1 Administration Guide

    I den här handboken beskrivs olika administratörsspecifika åtgärder för installation, konfiguration och administration av en Teaming-webbplats.

  • Novell Teaming 2.1 Advanced User Guide

    Den här handboken riktar sig till arbetsyte- och mappägare som ansvarar för att göra arbetsytorna användbara och effektiva för användarna. Exempel på användare som kan ha nytta av den här informationen är team- och projektledare.

  • Novell Teaming 2.1 Installation Guide

    Den här handboken innehåller information om hur du installerar Teaming så att alla standardfunktioner fungerar som de ska. Handboken innehåller också information om hur du skapar en Teaming-användare.

  • Utvecklarehandbok för Novell Teaming 2.1

    Den här handboken riktar sig främst till utvecklare som vill utöka funktionaliteten hos Teaming eller programmera andra program till att interagera med systemet.
