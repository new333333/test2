Definición de una URL para una carpeta o un área de trabajo

Utilice esta sección para definir una URL sencilla para la carpeta o el área de trabajo en la que se encuentre actualmente. Definir esta URL le permite desplazarse directamente a la carpeta o área de trabajo desde el navegador.

Si define una URL para una carpeta, podrá añadir entradas a la carpeta mediante el correo electrónico o con mensajes de texto.

  1. En la lista desplegable Definir URL, seleccione una de las palabras clave globales predefinidas.

    A menos que sea administrador del sitio, la URL debe comenzar por una de estas palabras clave globales predefinidas.

  2. En el campo correspondiente, especifique la URL que desee para la carpeta o área de trabajo.

  3. Haga clic en Añadir.

  4. (Condicional) Si se dispone a definir la dirección URL de una carpeta, debe habilitar la URL que acaba de definir seleccionándola. La URL que debe seleccionar se encuentra bajo el botón Añadir.

  5. (Condicional) Si se dispone a definir la dirección URL de una carpeta, haga clic en Aplicar.

  6. Haga clic en Cerrar.

Para suprimir una URL definida, selecciónela en la sección URL definidas actualmente y haga clic en Suprimir las URL seleccionadas.
