Configuración de Control de acceso

Utilice esta página para modificar los ajustes de control de acceso para esta área de trabajo o carpeta.

De manera predeterminada, las áreas de trabajo y las carpetas heredan la configuración del control de acceso de sus primarios. Utilice esta página sólo si desea que haya diferentes configuraciones de acceso.

Antes de poder efectuar cambios en la tabla de control de acceso, debe asegurarse de que la carpeta o el área de trabajo no heredan los usuarios y funciones del directorio padre. Para obtener información sobre cómo hacerlo, consulte la Heredar la pertenencia a funciones.

Para obtener información más detallada sobre el control de acceso que la contenida en esta sección y sus apartados, consulte Controlling User Access (Control del acceso de los usuarios) en la Novell Teaming 2.1 Advanced User Guide (Guía del usuario avanzado de Novell Teaming 2.1).

Las siguientes secciones describen cómo modificar la tabla de control de acceso a la medida de sus necesidades.

Adición de usuarios y grupos

  1. Haga clic en el enlace Añadir un grupo o en Añadir un usuario de la columna izquierda de la tabla.

  2. En el campo correspondiente, escriba las primeras letras del nombre de usuario o del grupo y, a continuación, haga clic en enlace del nombre en la lista desplegable.

    Novell Teaming añade una fila a la tabla para el usuario o el grupo.

Visualización de tareas de función

En la tabla de control de acceso, los nombres de función sirven como encabezados de columna.

  1. Haga clic en el encabezado de una columna (Participante, Miembro del equipo, etc. ) para ver una lista de los derechos asociados con dicha función.

Cambio de las funciones de los usuarios o los grupos

Una función es un conjunto de privilegios. Cuando se asigna alguien a una función, ese usuario adquiere automáticamente todos los derechos asociados a dicha función.

  1. Marque o deseleccione las casillas de verificación para determinar las funciones que tendrá cada usuario.

  2. Haga clic en Aceptar.

Ajuste de los controles de acceso en otro lugar

  1. Diríjase al árbol del área de trabajo de la parte superior de esta página, acceda al lugar donde desee ajustar los controles de acceso y, a continuación, haga clic en el enlace de dicho lugar en el árbol.

    Aparecerá la página Control de acceso de ese lugar.
