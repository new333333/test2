Programación de la sincronización

Esta página le permite activar y configurar una programación de sincronización para las carpetas duplicadas. La página incluye las opciones siguientes:

Habilitar sincronización programada: debe marcar esta casilla de verificación para que tenga lugar la sincronización programada.

Cada día: seleccione esta opción si desea que la sincronización se produzca todos los días.

Semanalmente: seleccione esta opción si desea especificar determinados días de la semana en los que se deba realizar la sincronización.

A la hora: especifique una hora específica a la que deba producirse la sincronización.

Repetir cada n horas: especifique el intervalo de tiempo que se deba esperar entre las sincronizaciones.

Para obtener más información, consulte Información legal.
