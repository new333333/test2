Aplicación de la configuración de definición de forma repetida

Puede utilizar esta función para aplicar la configuración de definición a las carpetas o las entradas que ya se hayan creado.

Aplicación de la configuración de definición a todas las subcarpetas

Puede aplicar toda la configuración de definición de una carpeta padre a todas las subcarpetas.

  1. Seleccione Cambiar carpetas incluidas para que hereden las definiciones de esta carpeta.

  2. Haga clic en Aplicar.

Toda la configuración de definición de la página Configurar valores por defecto se aplicará a todas las subcarpetas.

Aplicación de la configuración de definición a entradas existentes

Si ha añadido entradas previamente a la carpeta y el tipo de definición de las entradas no coincide con el de las demás entradas de la carpeta (por ejemplo, puede contar con una carpeta de blogs que incluya sobre todo tipos de definiciones de blog, pero también algunas con tipos de definiciones de discusión), puede utilizar esta sección para cambiar la configuración de definición de modo que todas las entradas tengan la misma.

  1. Seleccione Cambiar carpetas incluidas para que hereden las definiciones de esta carpeta y después Cambiar entradas incluidas para que sean del siguiente tipo.

  2. En la lista desplegable, seleccione el tipo de entrada que desee aplicar a todas las entradas de la carpeta.

    Para añadir tipos de entrada a la lista desplegable, selecciónelos en la sección Tipos de entrada por defecto y después haga clic en Aplicar.

  3. Haga clic en Aplicar.
