Permitir correo electrónico

En las secciones siguientes se describen las opciones de esta página.

Habilitación del correo electrónico saliente

Puede modificar las siguientes opciones de configuración para el correo electrónico saliente:

  • Habilitar correo electrónico saliente: marque esta casilla de verificación para habilitar el correo electrónico saliente de Teaming. Si no desea permitir que los usuarios puedan enviar mensajes de correo electrónico desde Teaming, deseleccione esta casilla de verificación.

    Esta casilla de verificación debe estar marcada si desea permitir que los usuarios puedan enviar notificaciones de resumen.

  • Enviar notificaciones por correo electrónico (saliente): utilice esta sección para definir la configuración por defecto de las notificaciones de resumen. Si los usuarios de Teaming desean enviar o recibir notificaciones de resumen por correo electrónico, deberán usar los ajustes de notificación de resumen que seleccione aquí. Puede configurar que se envíen notificaciones de resumen cada día, en determinados días de la semana, en un momento determinado de la jornada o a intervalos de una hora.

Habilitación del correo electrónico entrante mediante el servidor de correo electrónico incrustado y URL sencillas

Marque la casilla de verificación Habilitar correo electrónico entrante usando el servidor de correo electrónico incrustado y URL sencillas si desea permitir que las entradas se envíen por correo electrónico a las carpetas que se hayan preconfigurado con una URL sencilla.

Para obtener más información, consulte Información legal.
