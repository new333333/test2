Herramientas de gestión del área de trabajo

Cuando se hace clic en Gestionar en la barra de herramientas Área de trabajo, Kablink Teaming sólo muestra las herramientas de gestión y configuración que tiene permiso para utilizar. Los administradores del sitio o el propietario del área de trabajo pueden otorgar acceso a estas herramientas.

Si tiene acceso no restringido, puede ver las siguientes opciones:

  • Control de acceso: permite determinar quién puede ver y participar en esta área de trabajo.

  • Informes de actividad: permite ver las actividades que se han producido en el área de trabajo.

  • Configurar: permite configurar el área de trabajo, incluida la vista por defecto y la URL.

  • Copiar esta área de trabajo: permite copiar el área de trabajo actual en cualquier destino del árbol del área de trabajo que desee, si dispone de los derechos adecuados. Consulte Navegación por el árbol del área de trabajo.

  • Suprimir esta área de trabajo: suprime el área de trabajo actual.

  • Diseñadores de formulario y vista: permite crear formularios de entrada y flujos de trabajo personalizados que más adelante podrá aplicar al área de trabajo y a sus subcarpetas.

  • Modificar esta área de trabajo: permite modificar el área de trabajo, incluido el nombre, la descripción, la página de inicio, la marca y el icono.

  • Mover esta área de trabajo: permite mover el área de trabajo actual a cualquier destino del árbol del área de trabajo que desee, si dispone de los derechos adecuados. Consulte Navegación por el árbol del área de trabajo.

  • Añadir nueva área de trabajo: permite añadir una nueva área de trabajo dentro del área en que se encuentre.

  • Añadir nueva carpeta: permite añadir una carpeta nueva al área de trabajo.

Para obtener más información, consulte Información legal.
