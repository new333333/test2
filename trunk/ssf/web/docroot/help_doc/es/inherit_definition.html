Herencia de la configuración de definición

Esta sección permite especificar si desea que se herede la configuración de definición de una carpeta padre.

  1. Haga clic en Sí > Aplicar si desea que se herede la configuración de definición.

    O bien

    Haga clic en No > Aplicar si no desea que se herede la configuración de definición.

Para obtener más información, consulte Información legal.
