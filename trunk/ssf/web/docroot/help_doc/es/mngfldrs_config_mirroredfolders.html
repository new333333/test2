Carpetas duplicadas

Una carpeta duplicada es una carpeta de biblioteca cuyos archivos están ubicados en un sistema externo (ya sea un directorio en un sistema de archivos o una carpeta en un servidor SharePoint). Cuando se duplican carpetas desde Teaming, se pueden aplicar las herramientas de Kablink Teaming (por ejemplo, las búsquedas y los flujos de trabajo) a archivos externos.

Tiene que trabajar con un archivo situado en el servidor para habilitar esta opción.

Si desea obtener más información sobre cómo crear carpetas duplicadas, consulte Working with Mirrored Folders (Uso de carpetas duplicadas) en la Kablink Teaming 2.0 Advanced User Guide (Guía del usuario avanzado de Kablink Teaming 2.0).

Para obtener más información, consulte Información legal.
