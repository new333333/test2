Carpetas duplicadas

Una carpeta duplicada es una carpeta de biblioteca cuyos archivos están ubicados en un sistema externo (ya sea un directorio en un sistema de archivos o una carpeta en un servidor SharePoint). Cuando se duplican carpetas desde Teaming, se pueden aplicar las herramientas de Novell Teaming (por ejemplo, las búsquedas y los flujos de trabajo) a archivos externos.

Tiene que trabajar con un archivo situado en el servidor para habilitar esta opción.

Si desea obtener más información sobre cómo crear carpetas duplicadas, consulte Uso de carpetas duplicadas en la Guía del usuario de Novell Teaming 2.1.

Para obtener más información, consulte Información legal.
