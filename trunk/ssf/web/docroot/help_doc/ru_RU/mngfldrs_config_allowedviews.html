Разрешенные представления

Поскольку только владельцы папок имеют права задавать для папки представление по умолчанию, раздел Разрешенные представления позволяет владельцу папки выбрать представления, которые могут использовать внешние пользователи, если им не подходит представление по умолчанию или более удобным кажется другое представление. Например, если настраивается представление календаря, и в этом разделе выбирается Задача, то пользователи получают возможность просматривать папку календаря в представлении задач.

Включение представлений, используемых для папки

  1. В разделе Разрешенные представления выберите представления, которые должны быть разрешены для настраиваемой папки.

  2. Нажмите кнопки Применить > Закрыть.

  3. На главной боковой панели в разделе Параметры папок щелкните Представление папки.

    Отображаются все разрешенные представления папки. (Это представления, выбранные Вами в разделе Разрешенные представления в Шаг 1.) Пользователи папки могут выбирать из этого списка представления, соответствующие их потребностям просмотра папки.

Включение представлений для использования по умолчанию

При выборе представления в разделе Разрешенные представления оно добавляется в раздел Представление по умолчанию. После этого владелец папки может выбрать это представление в качестве представления по умолчанию.

  1. В разделе Разрешенные представления выберите представление, которое следует добавить в раздел Представление по умолчанию.

  2. Нажмите кнопку Применить.

    Выбранное представление будет добавлено в раздел Представление по умолчанию.

О способах задания представления по умолчанию для папки см. Изменение представления папки по умолчанию.

Дополнительную информацию см. в Юридическая информация.
