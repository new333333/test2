Теги

Теги являются ключевыми словами, которые характеризуют рабочую область, папку или запись. Функция тегов в Kablink Teaming позволяет создавать виртуальные контейнеры для мест и записей. Аналогично папкам в электронной почте, пользователи могут распределять информацию по категориям и возвращаться к ним позже.

Присоединение тегов к рабочим областям или папкам

  1. Перейдите к рабочей области или папке, для которой требуется определить тег.

  2. В разделе Тег для рабочей области (или Тег для папки) на главной боковой панели введите имя тега в поле в разделе Персональные теги или Теги сообщества.

    Чтобы решить, следует ли создать персональный тег или тег сообщества, см. Выбор персонального тега или тега сообщества.

    Это поле использует функцию поиска вводящегося текста, поэтому, если уже существует тег с необходимым именем, он появится в раскрывающемся списке. Если имя не появляется в раскрывающемся списке, Вы создаете новый тег.

  3. Завершите ввод имени тега или выберите его в раскрывающемся списке и нажмите кнопку Добавить.

Для рабочей области или папки будет определен указанный тег.

Теги для записей

  1. В записи, для которой следует определить тег, нажмите кнопку Добавить тег.

  2. Введите имя тега в поле в разделе Персональные теги или Теги сообщества.

    Чтобы решить, следует ли создать персональный тег или тег сообщества, см. Выбор персонального тега или тега сообщества.

    Это поле использует функцию поиска вводящегося текста, поэтому, если уже существует тег с необходимым именем, он появится в раскрывающемся списке. Если имя не появляется в раскрывающемся списке, Вы создаете новый тег.

  3. Завершите ввод имени тега или выберите его в раскрывающемся списке и нажмите кнопку Добавить.

Для записи будет определен указанный Вами тег.

Выбор персонального тега или тега сообщества

  • Персональные теги Персональные теги предназначены для использования только Вами. Созданный Вами персональный тег будет виден и доступен для использования только для Вас.

  • Теги сообщества Теги сообщества могут использоваться и просматриваться любыми пользователями, имеющими права на просмотр рабочих областей, папок или записей, для которых определены теги сообщества. Например, если Вы создаете тег сообщества в папке файлов в своей рабочей области команды, любой пользователь, имеющий права на просмотр папки файлов, может использовать созданный Вами тег, чтобы помечать другие места этим же тегом. Они могут также использовать функции Teaming "Поиск" и "Найти" для обнаружения записей с этим тегом.

    Дополнительно об использовании функции "Поиск" для обнаружения записей с тегами см. Использование расширенного поиска.

    Дополнительно об использовании функции "Найти" для обнаружения записей с тегами см. Поиск конкретных людей, мест или элементов с тегами.

    Помечать места и записи тегом сообщества может только пользователь, имеющий достаточные права.

Дополнительную информацию см. в Юридическая информация.
