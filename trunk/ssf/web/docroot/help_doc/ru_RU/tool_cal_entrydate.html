Тип даты записи

Этот инструмент особенно полезен, когда представление календаря используется для просмотра информации, предназначенной для другого типа папок. Например, можно использовать вики или обсуждение с представлением по умолчанию. Тогда, если владелец или администратор папки настраивает ее на поддержку этого действия, можно использовать представление календаря, чтобы видеть, для каких записей имели место последние действия (например, какие записи изменялись недавно). Можно также просматривать записи согласно датам их создания.

Дополнительную информацию см. в Юридическая информация.
