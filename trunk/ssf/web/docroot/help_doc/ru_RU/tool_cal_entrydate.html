Просмотр событий в календаре

В выпадающем списке Показать можно выбрать типы событий, которые будут отображаться в календаре, а также способ их представления. Если пользователь Teaming не находится в персональной рабочей области, он может видеть только те события календаря, которые расположены в текущей папке.

  • Назначенные события Доступно только в рабочих областях пользователей. Позволяет просмотреть все события календаря, назначенные владельцу определенной рабочей области, вне зависимости от расположения событий на портале Teaming.

  • Из папки — События Позволяет просмотреть все события календаря, хранящиеся в текущей папке, с сортировкой по датам, на которые они запланированы.

  • Из папки — Все в порядке создания Позволяет просмотреть все события календаря по датам их создания, а не по датам, на которые они запланированы.

  • Из папки — Все с разбивкой по действиям Позволяет просмотреть все события календаря по датам последнего изменения каждого события, а не по датам, на которые события запланированы.

Дополнительную информацию см. в Юридическая информация.
