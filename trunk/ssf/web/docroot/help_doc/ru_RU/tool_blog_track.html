Отслеживание блога

Панель инструментов отслеживания блога содержит инструменты для быстрого выполнения следующих действий.

  • Добавить комментарий Позволяет добавить комментарий к записи блога Перед добавлением комментария полезно просмотреть комментарии, добавленные ранее. При добавлении комментария в блог, в котором уже имеются комментарии, новый комментарий автоматически добавляется как последний в списке вложенных комментариев.

  • Просмотреть комментарии Позволяет просматривать все комментарии к конкретной записи без открытия записи.

  • Отправить запись другу Позволяет отправить комментарии к записи по электронной почте любому пользователю, имеющему достаточные права на просмотр записи.

  • Подписаться Запускает форму, позволяющую подписаться на запись. При подписке на запись Teaming настраивается на отправку по электронной почте оповещений об обновлениях этой записи.

    Дополнительные сведения см. в разделе Подписка на запись в Руководстве пользователя Novell Teaming 2.1.

Дополнительную информацию см. в Юридическая информация.
