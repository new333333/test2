Документация

Ссылки в этом разделе открывают руководства по началу работы в новом окне навигатора. Эти руководства можно читать в электронном виде в формате HTML или PDF. Кроме того, формат PDF предназначен также для печати.

Мы будем рады получить Ваши комментарии и предложения по данным руководствам и по другой документации, включенной в этот продукт. Для этого воспользуйтесь ссылкой "Комментарий пользователя" в нижней части каждой страницы электронной документации или перейдите на веб-страницу www.novell.com/documentation/feedback.html и оставьте свои комментарии.

Все пользователи Teaming

Следующие руководства будут интересны всем пользователям Novell Teaming:

  • Руководство пользователя Novell Teaming 2.1

    В этом руководстве содержится подробная информация о выполнении различных задач в Teaming. В нем можно найти общее описание портала, инструкции по навигации, рекомендации по настройке портала, а также иллюстративный материал.

  • Расширенное руководство пользователя Novell Teaming 2.1

    Настоящее руководство предназначено для владельцев рабочих областей и папок, которые являются ответственными за то, чтобы рабочие области и папки оказались доступными и полезными для тех, кто их использует. Примерами пользователей, которым может быть интересна эта информация, могут служить руководители команд и менеджеры проектов.

Администраторы Teaming

Следующие руководства представляют интерес для администраторов Novell Teaming:

  • Руководство по администрированию Kablink Teaming 2.1

    Руководство содержит описание задач, необходимых для установки, настройки и поддержания сайта Teaming.

  • Расширенное руководство пользователя Novell Teaming 2.1

    Настоящее руководство предназначено для владельцев рабочих областей и папок, которые являются ответственными за то, чтобы рабочие области и папки оказались доступными и полезными для тех, кто их использует. Примерами пользователей, которым может быть интересна эта информация, могут служить руководители команд и менеджеры проектов.

  • Руководство по установке Novell Teaming 2.1

    Руководство содержит описание задач, необходимых для установки Teaming с заданными функциями по умолчанию. В нем также описываются способы настройки пользователей Teaming.

  • Руководство разработчика Novell Teaming 2.1

    Руководство предназначено в первую очередь для разработчиков, стремящихся расширить функциональные возможности портала Teaming или запрограммировать другие приложения для взаимодействия с ним.

Дополнительную информацию см. в Юридическая информация.
