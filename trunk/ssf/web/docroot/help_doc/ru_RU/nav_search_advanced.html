Использование расширенного поиска

Расширенный поиск позволяет ограничить область поиска с помощью одного или всех следующих условий.

  • Текст Введите текст для поиска.

  • Места Этот раздел позволяет ограничить поиск определенными местами. Разверните дерево, щелкнув значок "плюс" (+) рядом с домашней рабочей областью, а затем перейдите и выберите рабочую область или папку, в которой следует проводить поиск.

    Если требуется, чтобы Novell Teaming выполнял поиск во всех местах дерева рабочих областей, вложенных в выбранное, выберите Поиск во всех вложенных местах.

  • С учетом регистра Установите этот флажок, чтобы выполнить поиск текста, введенного в поле Текст, с учетом регистра. Например, образцу "дерево" не будут соответствовать "Дерево", "ДЕРЕВО" и т. д.

  • Искать удаленные элементы Поиск элементов, которые были удалены. В результатах этого поиска отображаются только удаленные элементы.

  • Параметры представления Укажите, сколько элементов будет показано на каждой странице результатов поиска и сколько слов будет отображаться для каждого результата.

  • Авторы Поиск элементов по их авторам.

  • Теги Поиск элементов, имеющих теги, как теги сообщества, так и персональные. Для получения дополнительной информации о тегах см. Теги.

  • Рабочий поток Поиск элементов, к которым применен конкретный рабочий поток.

  • Атрибуты записи Выберите тип записи для поиска, например запись блога, запись календаря или комментарий.

  • Последнее действие за дни Выберите поиск элементов, которые были активными за последние 1, 3, 7, 30 или 90 дней.

  • Даты создания Поиск элементов на основании дат их создания. В предоставленных полях укажите начальную и конечную даты необходимого интервала.

  • Даты изменения Поиск элементов на основании дат их последнего изменения. В предоставленных полях укажите начальную и конечную даты необходимого интервала.

  • Тип элемента Выберите тип элемента для поиска.

Следующие разделы содержат дополнительную информацию об использовании расширенного поиска.

Добавление условий

Если для поиска требуется несколько условий, щелкните Добавить элемент и Novell Teaming предоставит другое поле для использования.

Чтобы удалить условие из поиска, щелкните значок X рядом с условием.

Доступ к расширенному поиску

Доступ к расширенному поиску можно получить с панели навигации или со страницы результатов поиска.

Панель навигации

  1. Щелкните Расширенный. (Находится над полем Поиск.)

Страница результатов поиска.

  1. Щелкните Расширенный.

  2. Если требуется добавить условие к ранее заданным, щелкните ссылку Измените свой расширенный запрос в разделе Сводка условий поиска.

Дополнительную информацию см. в Юридическая информация.
