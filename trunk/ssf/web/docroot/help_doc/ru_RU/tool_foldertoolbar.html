Панель инструментов "Запись папки"

Novell Teaming помещает на панель инструментов "Запись папки" параметры, характерные для просматриваемой папки. Однако можно настроить панель инструментов "Запись папки" так, чтобы она содержала различные типы записей. Например, если имеется папка вики, и требуется, чтобы папка также содержала записи фото, можно настроить папку так, чтобы разрешать записи фото.

Параметры панели инструментов

Каждая папка имеет параметры по умолчанию, которые автоматически отображаются на панели инструментов "Запись папки". Ниже перечислены все типы папок и параметры, которые автоматически отображаются на панели инструментов "Запись папки".

Блог

  • Добавить запись блога Добавляет новую запись в папку блога. В запись можно добавить заголовок, описание и вложение. Можно также подписаться на запись и отправить пользователям по электронной почте оповещение о подписке на запись.

  • Новая страница блога Добавляет новую папку блога к существующей папке блога. Новая папка блога будет вложена в главную папку блога. Новые страницы блога можно использовать для распределения содержимого блога по определенным темам.

  • Сортировать по Сортировка блогов по номерам, датам действий, авторам, состояниям или заголовкам.

  • Добавить файлы в папку Предоставляет значок, позволяющий перетаскивать файлы или копировать и вставлять их в папку блога. Это действие добавляет файл в качестве новой записи блога.

Календарь

  • Добавить запись календаря Добавляет новую запись в папку календаря. В запись можно добавить заголовок, описание и вложение. Можно также подписаться на запись и отправить пользователям по электронной почте оповещение о подписке на запись и т. п.

  • Добавить файлы в папку Предоставляет значок, позволяющий перетаскивать файлы или копировать и вставлять их в папку календаря. Это действие добавляет файл как новую запись календаря.

  • Показать В этом выпадающем списке можно выбрать события, которые будут отображаться в календаре. Имеются следующие категории событий.

    • Назначенные события Доступно только в рабочих областях пользователей. Позволяет просмотреть все события календаря, назначенные владельцу определенной рабочей области, вне зависимости от расположения событий на портале Teaming.

    • Из папки — События Позволяет просмотреть все события календаря, хранящиеся в текущей папке, с сортировкой по датам, на которые они запланированы.

    • Из папки — Все в порядке создания Позволяет просмотреть все события календаря по датам создания.

    • Из папки — Все с разбивкой по действиям Позволяет просмотреть все события календаря по датам последнего изменения.

Гостевая книга

  • Добавить файлы в папку Предоставляет значок, позволяющий перетаскивать файлы или копировать и вставлять их в папку гостевой книги. Это действие добавляет файл в качестве новой записи файла.

Этап

Микроблог

  • Добавить микроблог Открывает диалоговое окно микроблога, позволяющее создать и отправить новый миниатюрный блог.

Фотоальбом

  • Добавить запись фотоальбома Добавляет новую запись в папку фотоальбома. Пользователь может именовать, описывать и загружать свои фотографии и другую графику. Можно также отправить конкретным пользователям по электронной почте оповещение о представлении новой записи.

  • Новый фотоальбом Добавляет новую папку в текущую папку фотоальбома. Новая папка становится вложенной в основную папку фотоальбома.

  • Сортировать по Сортировать изображения можно по параметрам Дата действия или Заголовок:

    Сортировка по параметру Дата действия размещает последние загруженные фотографии первыми в списке, а старые фотографии в фотоальбоме последними.

    Сортировка по параметру Заголовок размещает фотографии в алфавитном порядке.

  • Добавить файлы в папку Предоставляет значок, позволяющий перетаскивать файлы или копировать и вставлять их в папку фотоальбома. Это действие добавляет файл в качестве новой записи файла.

Опрос

  • Добавить файлы в папку Предоставляет значок, позволяющий перетаскивать файлы или копировать и вставлять их в папку опросов. Это действие добавляет файл в качестве нового неопределенного опроса.

  • Добавить файлы в папку Предоставляет значок, позволяющий перетаскивать файлы или копировать и вставлять их в папку задач. Это действие добавляет файл в качестве новой неопределенной задачи.

Вики

  • Добавить запись вики Добавляет новую запись в папку вики. Вы можете добавить заголовок, описание и вложение. Можно также отправить конкретным пользователям по электронной почте оповещение о том, что Вы представили запись на страницу вики.

  • Новая тема вики Позволяет добавить новую папку в текущую папку вики и присвоить ей имя. Новая папка становится вложенной в основную папку вики.

  • Сортировать по Сортировать записи вики можно по параметрам Дата действия или Заголовок:

    Сортировка по параметру Дата действия размещает последние записи первыми в списке, а старые записи в папке вики последними.

    Сортировка по параметру Заголовок размещает записи вики в алфавитном порядке.

  • Добавить файлы в папку Предоставляет значок, позволяющий перетаскивать файлы или копировать и вставлять их в папку вики. Это действие добавляет файл как новую запись вики. Она становится видимой в разделе Страницы вики на Боковая панель "Вики".

Разрешение на добавление в папку записей другого типа

  1. Перейдите к папке, в которую требуется добавить другой тип записей.

  2. Щелкните Управление > Настроить.

  3. В разделе Тип записи по умолчанию выберите папки, представляющие типы записей, которые требуется разрешить в папке.

  4. Нажмите кнопки Применить > Закрыть.

  5. Нажмите кнопку Добавить на панели инструментов "Запись папки" и в раскрывающемся списке выберите тип добавляемых записей.
