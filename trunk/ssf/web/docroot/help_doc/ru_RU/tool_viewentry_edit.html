Изменить

Ссылка Изменить позволяет изменять только раздел Описание записи.

Если требуется изменить другие части записи или комментарий, в том числе изменить заголовок, отправить прикрепленный файл или удалить прикрепленные файлы, нажмите кнопку Изменить на панели инструментов "Запись".

Дополнительную информацию см. в Юридическая информация.
