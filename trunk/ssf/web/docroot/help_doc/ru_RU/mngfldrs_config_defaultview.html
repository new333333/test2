Изменение представления папки по умолчанию

  1. Выберите представление, наиболее подходящее для списка папки или рабочей области.

  2. Нажмите кнопки Применить > Закрыть.

    Представление папки будет изменено.

О способах добавления представлений в этот раздел для выбора в качестве представления по умолчанию см. Включение представлений для использования по умолчанию.

При выборе представления папки следует учитывать типы содержащихся в ней записей. Например, для списка папки записей календаря лучше всего подходит представление календаря.

Дополнительную информацию см. в Юридическая информация.
