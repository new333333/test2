Рекурсивное применение параметров определений

Эта функция позволяет применять параметры определений к ранее созданным папкам.

Применение параметров определений ко всем вложенным папкам

Параметры определения, заданные для родительской папки, можно применить ко всем подпапкам.

  1. Выберите Изменить содержащиеся папки, чтобы наследовать определения этой папки.

  2. Нажмите кнопку Применить.

Все параметры определений, доступные на странице "Настроить параметры по умолчанию", будут применены ко всем вложенным папкам.

Применение параметров определений к существующим записям

Если в папку были добавлены записи, тип определения которых не совпадал с типами определений других записей из этой же папки (например, папка "Блог" содержит в основном определения типа "Блог", но некоторые записи отображаются как записи типа "Дискуссия"), в этом разделе можно изменить параметры определений таким образом, чтобы все записи стали одинаковыми.

  1. Щелкните Изменить содержащиеся папки, чтобы наследовать определения этой папки, а затем — Изменить тип содержащихся записей на следующий тип.

  2. Выберите в раскрывающемся списке тип записи, который хотите применить ко всем записям, существующим в данной папке.

    Чтобы добавить тип записи в этот раскрывающийся список, выберите тип записи в разделе Тип записи по умолчанию и нажмите Применить.

  3. Нажмите кнопку Применить.

Дополнительную информацию см. в Юридическая информация.
