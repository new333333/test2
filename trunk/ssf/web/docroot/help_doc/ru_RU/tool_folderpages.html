Перечисление записей и страниц

Эти инструменты позволяют проводить поиск отдельных записей или выполнять переходы по страницам перечня. В папке отображаются записи или информация о записях. Некоторые из этих инструментов полезны, только когда имеется несколько перечней страниц или записей. Слева направо на этой панели доступны следующие инструменты.

  • Перейти к записи Укажите номер записи (отображаемый в столбце Число), которую требуется просмотреть, и нажмите кнопку Перейти.

  • Назад/Далее Позволяют прокручивать страницы папки.

  • На страницу Если ввести в это поле номер страницы и нажать кнопку Перейти, Kablink Teaming отобразит записи на этой странице перечня.

Дополнительную информацию см. в Юридическая информация.
