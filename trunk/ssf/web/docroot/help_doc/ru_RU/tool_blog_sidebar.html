Боковая панель блога

Боковая панель блога в Kablink Teaming предоставляет следующие инструменты для обнаружения и просмотра записей блога.

Страницы блога

Если в папке блога имеется несколько страниц, страницы перечисляются сверху на боковой панели блога. Чтобы перейти к другой странице в папке, щелкните ссылку с именем страницы.

Архивы

Боковая панель блога содержит раздел Архивы, в котором перечисляются все месяцы, когда в папку блога делались записи. Если щелкнуть месяц, все записи, сделанные за этот месяц, отображаются в обратном хронологическом порядке, т. е. сверху в списке отображаются самые новые записи.

Дополнительную информацию см. в Юридическая информация.
