Определение URL-адреса для папки или рабочей области

Этот раздел используют для определения удобного для пользователей URL-адреса для текущей папки или рабочей области. Определение URL-адреса позволяет переходить к папке или рабочей области непосредственно из навигатора.

Определение URL-адреса папки позволяет добавлять в эту папку записи по электронной почте или как текстовые сообщения.

  1. В раскрывающемся списке Определить URL-адрес выберите одно из предопределенных ключевых слов.

    Если Вы не являетесь администратором сайта, Ваш URL-адрес должен начинаться с одного из этих ключевых слов.

  2. В поле укажите требуемый URL-адрес папки или рабочей области.

  3. Нажмите кнопку Добавить.

  4. (условное) После того как для папки был указан URL-адрес, необходимо активизировать его щелчком мыши. URL-адрес, который необходимо выбрать, находится под кнопкой Добавить.

  5. (условное) Чтобы задать URL-адрес для папки, щелкните Применить.

  6. Нажмите кнопку Закрыть.

Чтобы удалить определенный URL-адрес, выберите URL-адрес в разделе Текущие определенные URL-адреса и щелкните Удалить выбранные URL-адреса.

Дополнительную информацию см. в Юридическая информация.
