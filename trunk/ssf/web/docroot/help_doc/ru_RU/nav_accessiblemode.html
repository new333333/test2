Подсказки

Эта ссылка включает и отключает функцию подсказок в Novell Teaming. Для просмотра подсказок щелкните Включить подсказки. Подсказки отображаются при наведении указателя мыши на определенные элементы интерфейса Teaming. Если подсказки больше не нужны, щелкните Выключить подсказки.

Дополнительную информацию см. в Юридическая информация.
