Поиск информации

Функции поиска в Kablink Teaming позволяют находить записи, места или людей. Функции поиска полезны, если требуется найти эксперта по конкретному вопросу, а также для поиска информации по определенной теме.

Если требуется быстро перейти к персональной рабочей области, командной рабочей области или папке, которая существует на сайте Teaming, следует использовать функцию Teaming Найти.

Функции поиска описаны в следующих разделах.

Использование обычного поиска

Чтобы произвести обычный поиск, выполните следующие действия.

  1. Введите слова или фразы в кавычках в поле Поиск, а затем нажмите клавишу ВВОД на клавиатуре или щелкните значок Поиск .

Использование расширенного поиска

Для получения сведений о расширенном поиске см. Использование расширенного поиска.

Сохранение и повторное использование поисков

Teaming позволяет сохранять поиски для их использования в дальнейшем.

Сохранение поиска

Чтобы сохранить поиск для дальнейшего использования, выполните следующие действия.

  1. После выполнения поиска, который требуется сохранить, введите имя в разделе Сохраненные поиски главной боковой панели на левой стороне страницы.

  2. Нажмите кнопку Сохранить.

Доступ к сохраненным поискам

  1. Щелкните стрелку раскрытия списка рядом со значком Поиск.

    Будут отображены ссылки на сохраненные поиски.

Использование страницы результатов поиска

После выполнения поиска открывается страница результатов поиска. В главном разделе этой страницы Kablink Teaming перечисляет все результаты поиска. Кроме того, Teaming предоставляет инструменты на главной боковой панели и отображает условия поиска сверху в правом столбце, как описано в следующих разделах.

Результаты поиска

В главном разделе на странице поиска Kablink Teaming отображает результаты поиска. Чтобы просмотреть результат, щелкните имя результата. Результатами могут быть рабочие области, папки и записи.

Инструменты

На главной боковой панели доступны следующие инструменты, помогающие сделать поиск более эффективным.

  • Последние места При выполнении поисков Teaming помещает ссылки на каждый поиск в раздел "Последние места" главной боковой панели, что позволяет легко повторить один из недавних поисков. Однако Teaming сохраняет только последние посещенные места и поиски. Чтобы сделать поиски постоянно доступными, следует использовать Сохраненные поиски.

  • Сохраненные поиски Введите имя текущего поиска и нажмите кнопку Сохранить. В дальнейшем на панели навигации можно будет щелкнуть стрелку раскрытия списка у значка Поиск и выбрать имя ссылки сохраненного поиска, который требуется выполнить еще раз.

  • Лучшие места Перечень мест, в которых наиболее часто обсуждается тема поиска. Элементы в верхней части списка являются наиболее значимыми.

  • Лучшие люди Перечень пользователей, чаще всего вносивших вклад в обсуждения по теме поиска. Пользователи в верхней части списка являются наиболее значимыми.

  • Теги Отображаются теги, которые были применены к соответствующим элементам.

Быстрый поиск

Если при первоначальном поиске не получены желаемые результаты, раздел Быстрый поиск позволяет выполнить другой поиск. Укажите тему поиска в поле Текст и щелкните Поиск.

Если требуется выполнить более детальный поиск, щелкните Расширенный.

Правила для строк поиска

Для строк поиска существуют следующие правила.

  • Учет регистра Все основные поиски в Teaming выполняются без учета регистра. Например, образцу "дерево" будут соответствовать "дерево", "Дерево", "ДЕреВо" и т. д. Если требуется поиск с учетом регистра, необходимо выполнить расширенный поиск. Чтобы выполнить расширенный поиск, щелкните Расширенный над полем Поиск.

  • Более одного слова Teaming возвращает элементы, содержащие все слова в строке поиска.

  • Фраза в кавычках Teaming возвращает элементы, содержащие все слова в указанном порядке. Правило учета регистра применяется, как описано выше.

  • Исключения Чтобы исключить слово (не возвращать совпадения для элементов, содержащих это слово), поместите перед ним знак "минус" (-). Например, образец "-дерево" задает исключение всех элементов, содержащих "дерево", "Дерево", "ДЕРЕВО" и т. д.

Дополнительную информацию см. в Юридическая информация.
