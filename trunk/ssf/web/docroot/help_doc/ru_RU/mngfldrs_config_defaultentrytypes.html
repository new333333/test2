Типы записей по умолчанию

Этот раздел позволяет выбрать типы записей, которые пользователям разрешается создавать в папке. Можно настроить папку, чтобы разрешить дополнительные типы записей, или изменить текущий тип записей. Если добавляются дополнительные типы записей, пользователи, отправляющие записи в эту папку, могут выбрать тип добавляемой записи, например запись задачи или запись календаря.

  1. Выберите типы записей, которые следует разрешить в этой папке.

  2. Нажмите кнопки Применить > Закрыть.

Если включены дополнительные типы записей, пользователи, отправляющие записи в эту папку, могут нажать кнопку Добавить на панели инструментов "Запись папки", а затем в раскрывающемся списке выбрать тип добавляемой записи из списка разрешенных Вами записей.

Администраторы сайта и владельцы папок могут использовать функцию "Дизайнеры форм и представлений" для создания новых типов записей (пользовательские формы), которые затем появятся в этом списке. (Когда владельцы папок создают новые типы записей, эти типы перечисляются в разделе Локальные определения данного списка.)

Дополнительные сведения о создании пользовательских типов записей см. в разделе Создание пользовательских форм записей в Расширенном руководстве пользователя Novell Teaming 2.1.

Дополнительную информацию см. в Юридическая информация.
