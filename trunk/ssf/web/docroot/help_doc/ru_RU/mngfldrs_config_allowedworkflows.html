Разрешенные рабочие потоки

В этом разделе содержатся сведения о том, как разрешить использование рабочего потока для всех записей определенной папки. Если для какой-либо папки разрешено использование рабочего потока, при просмотре записей в ней пользователь может запустить любой рабочий поток, выбранный в соответствующем разделе. Для этого нужно щелкнуть Рабочий поток > Запустить рабочий поток: wf_name на панели инструментов "Запись".

Будет запущен рабочий поток для этой записи.

Дополнительную информацию см. в Юридическая информация.
