Разрешение электронной почты

В следующих разделах описаны параметры, доступные на этой странице.

Включение исходящей электронной почты

Можно изменять следующие параметры настройки исходящей электронной почты.

  • Включить исходящую почту Установите этот флажок, чтобы разрешить исходящую электронную почту из Teaming. Если не следует разрешать пользователям отправлять сообщения электронной почты из Teaming, снимите этот флажок.

    Этот флажок должен быть установлен, если требуется разрешить пользователям отправлять оповещения-сводки.

  • Отправить оповещение по почте (исходящее) Используйте этот раздел, чтобы задать параметры настройки по умолчанию для отправки оповещений-сводок. Если пользователям Teaming требуется отправлять или получать по электронной почте оповещения-сводки, они должны использовать выбранные Вами здесь настройки сводок. Можно задать отправку сводок каждый день, по определенным дням недели, в определенное время дня или по часовым интервалам.

Включение входящей электронной почты с помощью внедренного сервера электронной почты и простых URL-адресов

Установите флажок Включить входящую электронную почту с помощью внедренного сервера и простых URL-адресов, если требуется разрешить отправку записей по электронной почте в папки, которые были настроены с простыми URL-адресами.

Дополнительную информацию см. в Юридическая информация.
