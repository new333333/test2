Боковая панель "Вики"

Боковая панель "Вики" содержит следующие параметры.

  • Домашняя страница вики Нажмите эту кнопку. чтобы перейти к ранее определенной основной странице вики. По умолчанию Kablink Teaming не определяет основную страницу вики; пользователь должен сделать это вручную. Если необходимо настроить папку вики на отображение основной страницы вики, найдите и отобразите запись, которую следует определить как основную страницу, и щелкните Задать домашнюю страницу вики на панели инструментов записи.

  • Найти страницу Это поле использует функцию поиска вводящегося текста. Начните вводить имя страницы вики, которую требуется найти, и щелкните имя, когда оно будет отображено.

    Teaming выполнит переход на нужную страницу.

  • Темы вики В этом разделе перечислены все темы, существующие в папке вики. Если выбрать тему в списке, все страницы вики для выбранной темы отображаются в разделе Страницы вики.

  • Страницы вики В этом разделе перечисляются все страницы вики для текущей выбранной темы.

Дополнительную информацию см. в Юридическая информация.
