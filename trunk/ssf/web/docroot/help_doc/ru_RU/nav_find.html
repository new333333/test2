Поиск конкретных людей, мест или элементов с тегами

Функция "Найти" позволяет быстро перейти к определенному месту в Kablink Teaming. Функцию "Найти" используют для поиска людей, мест и тегов, про которые известно, что они существуют на сайте Teaming. Если требуется найти специалиста по определенному вопросу или просмотреть доступную информацию по некоторой теме, следует использовать функцию Teaming Поиск.

Функция предоставляет следующие поля.

  • Люди Позволяет быстро перейти к персональной рабочей области пользователя.

  • Места Позволяет быстро перейти к любой папке или рабочей области в Teaming. Например, если требуется найти папку блога сотрудника, просто введите Блог и укажите имя сотрудника. Соответствующее место будет перечислено в раскрывающемся списке.

  • Теги Позволяет находить любые рабочие области, папки или записи, для которых определены теги. Укажите имя тега, по которому проводится поиск, и щелкните имя в раскрывающемся списке. Teaming отображает все рабочие области, папки и записи, имеющие указанный тег.

    Teaming отображает все персональные теги и все теги сообщества, для которых у Вас имеются достаточные права.

При использовании этих полей Teaming применяет функцию поиска вводящегося текста. По мере ввода Teaming перечисляет элементы, соответствующие введенному тексту (например, все элементы, соответствующие "д", затем все элементы, соответствующие "де").

Если предлагается несколько панелей элементов для выбора, то для их просмотра можно использовать значки стрелок вправо и влево.

Дополнительную информацию см. в Юридическая информация.
