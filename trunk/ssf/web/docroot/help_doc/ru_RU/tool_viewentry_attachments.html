Присоединение файлов к записи

В разделе Вложения записи Novell Teaming предоставляет различные возможности прикрепления файлов к записи.

Перетаскивание файлов

Чтобы прикрепить файл к записи, можно перетащить его из другого расположения в эту запись.

  1. Щелкните значок Перетащить файлы.

    Будет отображено окно со значком папки.

  2. Перетащите файл на значок папки.

    Teaming отображает открывающуюся и закрывающуюся папку, чтобы показать, что после перетаскивания файлы были прикреплены как вложения.

Теперь файл прикреплен к записи.

Доступ к вложениям с помощью WebDAV (только для Internet Explorer)

Teaming позволяет использовать WebDAV для доступа к файлам, сохраненным на удаленных веб-серверах, и прикреплять их к записям. Однако Teaming поддерживает WebDAV только с навигатором Internet Explorer. Если используется навигатор, отличный от Internet Explorer, эта возможность не поддерживается.

  1. Щелкните Доступ к вложениям через WebDAV.

  2. Перейдите в нужное расположение и выберите файл, который следует прикрепить.

Дополнительные сведения об использовании WebDAV с Teaming см. в разделе Accessing Attachments Through WebDAV (Supported Environments Only) документа Руководство пользователя Novell Teaming 2.1.

Отправка одного файла

  1. В записи, в которую следует отправить файл, щелкните значок Отправить один файл в разделе Вложения.

  2. Нажмите кнопку Обзор, найдите и выберите файл, который следует прикрепить.

  3. Нажмите кнопку ОК.

Дополнительную информацию см. в Юридическая информация.
