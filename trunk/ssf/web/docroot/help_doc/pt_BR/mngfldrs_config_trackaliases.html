Monitorando álias

Quando proprietários de pastas ou administradores definem as configurações de e-mail para especificar um endereço de álias para publicação na pasta, as informações de configuração aparecem nesta coluna. Use essas informações para monitorar álias existentes na conta de e-mail de entrada para a instalação.

Para obter mais informações, consulte Informações legais.
