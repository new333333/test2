Tipos de Entrada Padrão

Esta seção permite selecionar os tipos de entradas que deseja permitir aos usuários criarem nesta pasta. É possível configurar a sua pasta para permitir tipos de entrada adicionais, ou você pode mudar o tipo de entrada atual. Se você incluir tipos de entrada adicionais, os usuários que enviam entradas para esta pasta poderão escolher o tipo de entrada que desejam adicionar, como entrada de tarefa ou de calendário.

  1. Selecione os tipos de entrada que deseja habilitar na pasta.

  2. Clique em Aplicar > Fechar.

Se você habilitar tipos de entrada adicionais, os usuários que enviam entradas para esta pasta poderão clicar em Adicionar na barra de ferramentas Entrada da Pasta e, depois, usar a lista suspensa para escolher o tipo de entrada que desejam adicionar, dependendo dos tipos de entradas que você habilitou na pasta.

Os administradores de site e proprietários de pastas podem usar a funcionalidade Designers de Formulários e Telas para criarem novos tipos de entrada (formulários personalizados), que serão depois exibidos nessa lista. (Quando os proprietários de pastas criam novos tipos de entrada, esses tipos de entrada são listados na seção Definições de Local dessa lista.)

Para obter mais informações sobre como criar tipos de entrada personalizados, consulte Criando formulários de entrada de pasta personalizados no Novell Teaming 2.1 Advanced User Guide (Guia do Usuário Avançado do Novell Teaming 2.1).

Para obter mais informações, consulte Informações legais.
