Todos os usuários do Teaming

Os seguintes guias são de interesse de todos os usuários do Novell Teaming:

  • Guia do Usuário do Novell Teaming 2.1

    Esse guia inclui informações detalhadas que descrevem como realizar várias tarefas no Teaming. Ele contém descrições gerais, instruções de navegação, instruções de personalização e gráficos.

  • Novell Teaming 2.1 Advanced User Guide (Guia do Usuário Avançado do Novell Teaming 2.1)

    Esse guia destina-se aos proprietários de área de trabalho e pasta responsáveis por tornar suas áreas de trabalho úteis e eficazes para seus consumidores. Exemplos de pessoas que podem se interessar por essas informações são líderes de equipe e gerentes de projeto.

Administradores do Teaming

Os seguintes guias são de interesse de administradores do Novell Teaming:

  • Kablink Teaming 2.1 Administration Guide (Guia de Administração do Kablink Teaming 2.1)

    Esse guia descreve as tarefas necessárias para um administrador instalar, configurar e operar um site do Teaming.

  • Novell Teaming 2.1 Advanced User Guide (Guia do Usuário Avançado do Novell Teaming 2.1)

    Esse guia destina-se aos proprietários de área de trabalho e pasta responsáveis por tornar suas áreas de trabalho úteis e eficazes para seus consumidores. Exemplos de pessoas que podem se interessar por essas informações são líderes de equipe e gerentes de projeto.

  • Novell Teaming 2.1 Installation Guide (Guia de Instalação do Novell Teaming 1.0)

    Esse guia descreve as tarefas necessárias para instalar o Teaming para que todos os recursos padrão funcionem conforme planejado. Ele também descreve como configurar um usuário do Teaming.

  • Novell Teaming 2.1 Developer Guide (Guia do Desenvolvedor do Novell Teaming 2.1)

    Este guia destina-se principalmente a desenvolvedores que desejam estender a funcionalidade do Teaming ou programar outros aplicativos para interagir com ele.

Para obter mais informações, consulte Informações legais.
