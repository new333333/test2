Documentação de introdução

Os links nesta seção exibem manuais de introdução em uma nova janela do browser. É possível ler esses manuais online em formato HTML e PDF. O formato PDF também está otimizado para impressão.

Todos os usuários do Teaming

Os seguintes guias são de interesse de todos os usuários do Kablink Teaming:

  • Guia do Usuário do Kablink Teaming 2.0

    Esse guia inclui informações detalhadas que descrevem como realizar várias tarefas no Teaming. Ele contém descrições gerais, instrução de navegação, instruções de personalização e gráficos.

  • Guia do Usuário Avançado do Kablink Teaming 2.0

    Esse guia destina-se aos proprietários de área de trabalho e pasta responsáveis por tornar suas áreas de trabalho úteis e eficazes para seus consumidores. Exemplos de pessoas que podem se interessar por essas informações são líderes de equipe e gerentes de projeto.

Administradores do Teaming

Os seguintes guias são de interesse de administradores do Kablink Teaming:

  • Guia de Administração do Kablink Teaming 2.0

    Esse guia descreve as tarefas necessárias para um administrador instalar, configurar e operar um site do Teaming.

  • Guia do Usuário Avançado do Kablink Teaming 2.0

    Esse guia destina-se aos proprietários de área de trabalho e pasta responsáveis por tornar suas áreas de trabalho úteis e eficazes para seus consumidores. Exemplos de pessoas que podem se interessar por essas informações são líderes de equipe e gerentes de projeto.

  • Kablink Teaming 2.0 Installation Guide (Guia de Instalação do Kablink Teaming 1.0)

    Esse guia descreve as tarefas necessárias para instalar o Teaming para que todos os recursos padrão funcionem conforme planejado. Ele também descreve como configurar um usuário do Teaming.

  • Guia do Desenvolvedor do Kablink Teaming 2.0

    Este guia destina-se principalmente a desenvolvedores que desejam estender a funcionalidade do Kablink Teaming ou programar outros aplicativos para interagir com ele.
