Substituições do Editor

Quando você edita anexos no Kablink Teaming, os aplicativos padrão do sistema Teaming são usados para editar os anexos. Entretanto, esta tabela permite substituir os aplicativos padrão que são usados para editar anexos por extensões de arquivo específicas no Teaming.

Você pode substituir o aplicativo de edição padrão para um único tipo de extensão de arquivo ou pode substituir os aplicativos padrão para vários tipos de extensão de arquivo.

Substituindo o aplicativo de edição padrão por extensões de arquivo individuais

Para substituir o aplicativo padrão para um tipo de extensão de arquivo individual:

  1. Clique em Adicionar.

  2. Na lista suspensa, selecione o tipo de extensão de arquivo para o qual deseja substituir o aplicativo de edição padrão.

  3. No campo Aplicativo, especifique o nome do aplicativo que deseja usar para editar anexos com a extensão de arquivo que você especificou na Etapa 2.

  4. Clique em OK.

Substituindo o aplicativo de edição padrão para todas as extensões de arquivo

Para substituir todos os tipos de extensão de arquivo:

  1. Clique em Usar.

  2. Na lista suspensa, selecione o aplicativo que deseja usar para editar anexos com todos os tipos de extensão de arquivo no Teaming.

  3. Clique em OK.

Para obter mais informações, consulte Informações legais.
