Fluxos de Trabalho Permitidos

Use esta seção para permitir que um fluxo de trabalho seja usado em todas as entradas desta pasta específica. Quando o usuário vê uma entrada em uma pasta na qual permite um fluxo de trabalho, ele pode iniciar qualquer fluxo de trabalho selecionado nesta seção clicando em Fluxo de Trabalho > Iniciar fluxo de trabalho: nome_do_ft na barra de ferramentas de Entrada.

Isso inicia o processo de fluxo de trabalho nessa entrada.

Para obter mais informações, consulte Informações legais.
