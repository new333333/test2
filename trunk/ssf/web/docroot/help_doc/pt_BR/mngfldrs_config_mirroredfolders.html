Pastas espelhadas

Uma pasta espelhada é uma pasta de biblioteca cujos arquivos estão localizados em um sistema externo (um diretório em um sistema de arquivos ou uma pasta em um servidor do SharePoint). Quando você espelha pastas externas do Teaming, pode aplicar as ferramentas do Kablink Teaming (por exemplo, pesquisa e fluxo de trabalho) aos arquivos externos.

Você precisa trabalhar com um arquivo localizado no servidor para habilitar este recurso.

Para obter mais informações sobre como criar pastas espelhadas, consulte Trabalhando com pastas espelhadas no Guia do Usuário Avançado do Kablink Teaming 2.0.

Para obter mais informações, consulte Informações legais.
