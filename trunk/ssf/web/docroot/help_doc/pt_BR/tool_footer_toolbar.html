Barra de Ferramentas de Rodapé

O conteúdo da barra de ferramentas de Rodapé muda de acordo com o lugar que você está: área de trabalho, pasta ou entrada:

Barra de Ferramentas de Rodapé da Área de Trabalho

  • Permalinks de Área de Trabalho: Permite fazer referência à área de trabalho. Estes links permanecem constantes com o tempo e podem ser usados como favoritos ou enviados por e-mail a outros usuários. Copie e cole-os conforme necessário.

Barra de Ferramentas de Rodapé da Pasta

  • Permalinks de Pasta: Permite fazer referência à pasta. Estes links permanecem constantes com o tempo e podem ser usados como favoritos ou enviados por e-mail a outros usuários. Copie e cole-os conforme necessário.

  • Endereço de E-mail: Se uma pasta já tiver sido configurada para ter um URL amigável, a barra de ferramentas de Rodapé exibirá esse URL quando você clicar no link do rodapé.

    Para obter mais informações sobre como configurar uma pasta para receber entradas por e-mail, consulte Configurando uma pasta para receber entradas por e-mail no Guia do Usuário do Kablink Teaming 2.0.

  • Link RSS: Permite acessar a pasta como feed RSS. Basta copiar o URL do RSS e colá-lo no leitor RSS.

  • URL do WebDAV: Se estiver usando o Internet Explorer 7 ou posterior como browser da Web, poderá usar o URL do WebDAV para acessar os arquivos do Kablink Teaming diretamente do Microsoft Office, e aplicativos similares.

  • URL do iCal: Permite sincronizar as pastas de tarefa e calendário do Teaming com seu aplicativo de calendário de área de trabalho.

    Quando você vê uma pasta ou entrada de calendário, pode usar esse link para exportar os dados do calendário do Teaming para inclusão em outro calendário com suporte ao formato iCal (por exemplo, Google e Outlook). Clique no link, grave o arquivo com uma extensão .ics e importe-o para o outro calendário (consulte a Ajuda do outro calendário para obter mais informações).

  • Ver esta pasta como uma pasta WebDAV: Permite ver a pasta como WebDAV.

    Para obter mais informações, consulte Acessando arquivos do WebDAV com o Teaming[.jsp ref: workspaces_folders?misc_tools/webdav.jsp]![I need to research this too!]![I copied this over from the Help and deleted it from the Help] no Guia do Usuário do Kablink Teaming 2.0.

Barra de Ferramentas de Rodapé da Entrada

  • Permalink da Entrada: Permite fazer referência à entrada. Estes links permanecem constantes com o tempo e podem ser usados como favoritos ou enviados por e-mail a outros usuários. Copie e cole-os conforme necessário.

  • Enviar E-mail: Inicia um formulário de e-mail que lhe permite enviar um e-mail a quem você quiser. O e-mail contém o local da entrada no Teaming e parte do conteúdo da entrada.

  • Assinar esta Entrada: Inicia um formulário que lhe permite assinar a entrada. Quando você assina uma entrada, configura o Teaming para enviar notificações por e-mail informando-o sobre atualizações dessa entrada.

    Para obter mais informações sobre como assinar uma entrada, consulte a Usando notificações por e-mail.

Para obter mais informações, consulte Informações legais.
