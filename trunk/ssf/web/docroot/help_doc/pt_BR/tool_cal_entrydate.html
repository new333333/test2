Entry Date Type (Tipo de Data de Entrada)

Esta ferramenta é útil especialmente quando você está usando a tela de calendário para revisar as informações que foram projetadas para um outro tipo de pasta. Por exemplo, você pode usar um wiki ou uma discussão com sua tela padrão. Em seguida, se o proprietário ou o administrador da pasta configurá-la para suportar essa ação, você poderá usar a tela de calendário para ver quais entradas têm a atividade mais recente (por exemplo, quais entradas foram editadas por último). Também é possível ver as entradas de acordo com a data de criação.

Para obter mais informações, consulte Informações legais.
