Rastreamento do blog

A barra de ferramentas de Rastreamento do Blog oferece maneiras rápidas de:

  • Adicionar um Comentário: Permite fazer um comentário sobre uma entrada de blog. Antes de fazer um comentário, é bom ver os comentários que já foram feitos. Quando você adiciona um comentário a um blog que já tenha sido comentado, o seu comentário será automaticamente adicionado como o último comentário na lista de comentários aninhados.

  • Ver Comentários: Permite ver todos os comentários de determinada entrada, sem abrir a entrada.

  • Enviar esta Entrada a Amigos: Permite enviar por e-mail o conteúdo da entrada a qualquer pessoa com direitos de acesso apropriados para ver a entrada.

  • Assinar: Inicia um formulário que permite a você assinar uma entrada. Quando você assina uma entrada, configura o Teaming para enviar notificações por e-mail informando-o sobre atualizações dessa entrada.

    Para obter mais informações sobre como assinar uma entrada, consulte Assinando uma entrada no Guia do Usuário do Kablink Teaming 2.0.
