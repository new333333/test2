Herdando configurações de definição

Esta seção permite especificar se deseja herdar as configurações de definição de uma pasta-mãe.

  1. Clique em Sim > Aplicar, se quiser herdar as configurações de definição.

    ou

    Clique em Não > Aplicar, se não quiser herdar as configurações de definição.

Para obter mais informações, consulte Informações legais.
